# Glossary

| 中文术语 | English term | Definition | Aliases | Do not confuse with |
| --- | --- | --- | --- | --- |
| 点火框架 | Ignition Framework | 研究新系统如何被应约者主动认领的系统重写框架。 | 点火、Dianhuo | 一般改革理论、纯制度设计 |
| 事件框架 | Event Framework | 将系统理解为重复并凝固的“提议者出方案、应约者接或不接”的事件走向框架。 | 事件版点火框架 | 纯叙事事件、新闻事件 |
| 事件 | Event | 提议者提出方案，应约者接或不接，并由此产生走向的最小结构。 | 提议—应约事件 | 外部冲击 |
| 事件尺度矩阵 | Event Scale Matrix | 用参与者规模 × 持续时间判断事件尺度的二维矩阵。 | 尺度矩阵 | 事件条件矩阵 |
| 参与者规模 | Participant Scale | 真正被卷入“提议—应约—走向”的应约者范围。 | 人数轴 | 现场人数 |
| 持续时间 | Duration Scale | 事件走向跑完、凝固、退化或被替代所需的时间。 | 时间轴 | 单纯存在时间 |
| 事件层级 | Event Layer | 人际、公司、政策、国家、文明等尺度区域的常见标签。 | 尺度标签 | 事件类型 |
| 系统重写 | System Rewriting | 旧系统失灵后，新提议、新协作结构和新认同结构被重新组织的过程。 | 重写过程 | 普通调整、局部优化 |
| 关键人物 | Key Person | 在关键时期通过关键动作改变燃烧条件的人或群体。 | 关键行动者 | 伟人史观中的单一英雄 |
| 关键时期 | Key Period | 旧系统减弱、新系统尚未稳定、应约者必须重新选择的窗口。 | 点火窗口 | 任意历史时期 |
| 关键动作 | Key Action | 改变应约者预期、退出权结构或认同条件的行动。 | 点火动作 | 普通政策动作、宣传动作 |
| 燃烧条件 | Combustion Conditions | 让新系统被接住所需的结构条件。 | 点火条件 | 表面热度、动员声量 |
| 协作系统 | Collaboration System | 制度、流程、组织、资源配置、技术结构。 | 搭房 | 认同系统 |
| 认同系统 | Recognition System | 身份、意义、参与感、归属感、共同方向。 | 点火层 | 协作系统 |
| 提议者 | Proposer | 提出新系统、新秩序、新认同方案的人或群体。 | 新秩序提出方 | 应约者 |
| 应约者 | Responder | 必须决定是否把自己放进新系统的人或群体。 | 接受方、被重组者 | 被动群众、提议者 |
| 提议者意识 | Proposer Awareness | 提议者是否知道自己正在提出方案。 | 提议意识 | 应约者态度 |
| 提议者退出权 | Proposer Exit Right | 提议者能否从自己的方案中撤出。 | 提议者退路 | 应约者退出权 |
| 事件条件矩阵 | Event Condition Matrix | 提议者意识 × 提议者退出权 × 应约者退出权构成的 8 格条件矩阵。 | 8 格矩阵 | 事件尺度矩阵 |
| 提议者姿态 | Proposer Posture | 推方案或备方案，是事件条件矩阵内的次级变量。 | 姿态变量 | 独立第四维度 |
| 格间滑动 | Cell Drift | 同一事件内部因意识、退出权或姿态变化而从一个条件格滑向另一个条件格。 | 格内退化、格间漂移 | 单次静态分类 |
| 退出权 | Exit Right | 应约者承诺被测试的压力条件。 | 退出空间 | 提议者权力限制 |
| 真实退出权 | Real Exit Right | 制度上允许退出，事实上也能退出。 | 可承受退出 | 象征退出权 |
| 事实退出权 | Factual Exit Right | 法理可能允许或不允许，但现实退出成本极高。 | 现实退出约束 | 真实退出权 |
| 心理退出权 | Psychological Exit Right | 人还在系统内，认同已经撤回。 | 人还在，心已走 | 实际离开 |
| 象征退出权 | Symbolic Exit Right | 退出形式存在，但实质被抽空。 | 仪式化退出 | 温和强制 |
| 推方案 | Pushed Proposal | 提议者替应约者决定需求，并将方案推给对方。 | 培养型方案 | 真应约、备方案 |
| 备方案 | Prepared Proposal | 提议者准备方案，让应约者在需要时自主取用。 | 培育型方案、管家型方案 | 消极不作为 |
| 点对了 | Correct Ignition | 认同建在应约者身上，事件健康运转。 | 正向点火 | 短期热度 |
| 点不足 | Insufficient Ignition | 只覆盖部分应约者，事件半活半死。 | 局部点火 | 完整失败 |
| 点歪了 | Misaligned Ignition | 认同建在外部参照或提议者尺子上，退化信号被遮蔽。 | 错位点火 | 延迟点火 |
| 点反了 | Reverse Ignition | 旧认同被拆除，新认同未建立，事件比之前更糟。 | 反向点火 | 负时间差 |
| 负时间差 | Negative Time Gap | 旧系统先被拆，新系统尚未跑通，中间真空足以杀死事件。 | 时间真空 | 延迟点火 |
| 多柱事件 | Multi-Pillar Event | 多个权力、文化或资源柱互相支撑并遮蔽退化信号的事件。 | 多柱互撑 | 单一强制系统 |
| 收费站型结构 | Tollbooth Structure | 只收租、不提供应约价值，因此不应被当成点火事件的结构。 | 收租结构 | 失败点火 |
| 延迟点火 | Delayed Ignition | 点火未立即发生，但燃烧条件仍可能积累。 | 滞后点火 | 失败点火 |
| 失败点火 | Failed Ignition | 协作系统可能建立，但认同系统未建立。 | 点火失败 | 伪点火 |
| 伪点火 | False Ignition | 看似热烈，实则应约者未真正认领。 | 假点火 | 短期成功 |
