# Project Origin

When Systems Catch Fire began from human curiosity and gradually evolved from a book project into a function-and-case knowledge base. The current goal is not to preserve a fixed manuscript, but to make ignition functions and case relations queryable, traceable, verifiable, and extensible by humans and AI agents.

The legacy book structure is preserved under `archive/book-legacy/` for historical tracing. The current primary entries are `data/functions/`, `data/cases/`, and `data/discoveries/`.
