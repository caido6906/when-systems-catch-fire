# Discoveries

The discovery layer is a human-facing synthesis layer. It is not required for machine reasoning over functions and cases, but it can introduce major findings derived from the function network, case network, and source notes.

The current structure is initialized first. Candidate discoveries are not marked as confirmed findings until they are explicitly written and reviewed.

- `data/discoveries/index.md`
- `data/discoveries/index.jsonl`
- `data/discoveries/items/`
