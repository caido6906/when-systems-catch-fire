# D268 容斥加速临界函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M7的容斥加速临界点取决于pᵢ增长模式：pᵢ线性增长→Σpᵢpⱼ∝t²（可控），pᵢ指数增长→Σpᵢpⱼ超指数（失控）。皮凯蒂r>g→财富指数增长→p_income指数增长→容斥加速失控→阶层固化不可逆。
