# D398 趋势-波动主导切换函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M1的R_tw = (a_excl·τ)/(Σpᵢpⱼ/ΣVar)。R_tw>>1趋势主导，<<1波动主导。早期波动主导（随机波动被放大），晚期趋势主导（确定性加速）。衰退从"随机"切换到"确定"——D316转折点的动态版本。
