# D247 自我模型相变函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M_self从0变正的相变条件： M_self = Σᵢ G_i(gᵢ) × ε_self_i > 0 需要至少一个i满足： 1. G_i(gᵢ) > 0：gᵢ在g*_i附近（门控面在正常工作范围） 2. ε_self_i > 0：∂gᵢ/∂(self_action_i) > θ_self_i（自我修改感知超过阈值） 条件1几乎总是满足的（正常运行的系统门控面都在工作范围内）
