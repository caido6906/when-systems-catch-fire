# D347 维度回复非线性函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M6的d_eff偏离4较大时回复力非线性：F_restore∝-k·δ·(1+δ²/δ_c²)。δ>δ_c时回复力减弱——偏离太远弹不回来。δ_c∝1/√(Σαᵢ²/(1-pᵢ)²)——门控面越均匀线性区越宽，越不均匀越窄。不均匀系统的d=4稳定性是脆弱的——小扰动能弹回来，大扰动弹不回来。
