# D360 弱容斥-不可逆边界函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M5的弱容斥主导区是否在不可逆线之前取决于n。n小时弱容斥=不可逆（无窗口），n大时有"容斥主导但还可逆"窗口，宽度∝(√n-1)/n∝D310。大系统有"容斥主导但还有救"的窗口，小系统没有。
