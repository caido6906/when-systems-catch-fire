# D265 极小点漂移方向函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M2的极小点漂移方向取决于门槛结构的"重心"：高门槛项权重>低门槛项权重→δμ>0（极小点上移），反之→δμ<0。学习新技能（新增低p门控面）→极小点下移→系统在更低能量即可稳定→更鲁棒。
