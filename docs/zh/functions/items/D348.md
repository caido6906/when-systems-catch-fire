# D348 容斥加速-时间权重联合函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M7的容斥加速使D316转折点提前：无加速时Σpᵢ≈0.5容斥主导，有加速时转折提前到Σpᵢ≈0.5/(1+a_excl·τ)。加速越强转折越早——"突然变坏"来得更早。a_excl∝n²使高n系统转折大幅提前——复杂系统不仅"变坏更快"而且"变坏更早"。双重加速：更早+更快。
