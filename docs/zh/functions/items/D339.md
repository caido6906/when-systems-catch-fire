# D339 僵尸态救援函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M12的僵尸态外部救援最小力度F_rescue∝n·(g_critical-g_eff)²。n越大需要救援力度越大。但F_rescue存在上限F_max∝n·ḡ——超过此力度干预本身造成新伤害（D312噪声放大）。F_rescue>F_max时僵尸态不可救——组织太大且g_eff太低时，任何干预要么不够要么造成附带损伤。
