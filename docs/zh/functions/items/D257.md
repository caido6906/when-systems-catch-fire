## 函数内容 / Function Content

中文：
门控面间的耦合强度不仅取决于否决概率pᵢpⱼ，还取决于门槛距离g(Λᵢ,Λⱼ)=min(Λᵢ/Λⱼ,Λⱼ/Λᵢ)。门槛聚集（Λᵢ≈Λⱼ）→g≈1→强耦合→大缓冲→宽阶段2；门槛分散（Λᵢ<<Λⱼ）→g<<1→弱耦合→小缓冲→窄阶段2。D255是g=1（全等门槛）的特例。
