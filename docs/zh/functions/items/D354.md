# D354 正反馈延迟函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M13的正反馈有延迟τ_delay时，频率ω=π/τ_delay处正反馈变负反馈。延迟足够大时系统自激振荡。自激振荡条件：K·τ_delay>1。政策反馈延迟过大时本意正反馈的改革变成振荡——每次纠偏都矫枉过正→来回摆。
