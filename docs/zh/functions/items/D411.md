# D411 放大不确定性-两步策略自洽函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M14的正反馈放大不确定性→自动满足两步策略的高不确定性条件。K>1系统天然需要两步策略，K<1系统不需要。正反馈↔两步策略，负反馈↔一步策略——策略选择与系统动力学完全自洽。
