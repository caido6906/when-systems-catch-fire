# D185 相对论门槛函数

## 函数内容

相对论是单门槛系统的特例，Λ = c（光速）是唯一门槛。当v << c时，μ/Λ → ∞，系统退化为经典确定性（D182）；当v → c时，μ/Λ → 1，门控效应显著，系统展现相对论效应。
