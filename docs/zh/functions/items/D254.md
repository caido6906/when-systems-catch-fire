# D254 耦合-容斥-平坦度三阶段函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

系统稳定性分三个阶段：(1) d≈d_opt时极小点平坦度主导，pᵢ<<p*，耦合可忽略，鲁棒性来自平坦度；(2) d偏离d_opt时耦合缓冲主导，pᵢ接近p*，平坦度下降但耦合补偿；(3) d严重偏离时容斥主导，pᵢ>p*，耦合被容斥压倒，系统崩溃。三阶段过渡是连续的。
