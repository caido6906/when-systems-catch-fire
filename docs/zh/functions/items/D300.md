# D300 代际容斥累积函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M7的容斥加速跨代累积∝k²。每代不只新增Δp，还有前面所有Δp的交叉效应累积。代际容斥∝k²→阶层固化的跨代传递是加速的——"三代出贵族"不只是财富累积，还有容斥交叉项的k²加速。
