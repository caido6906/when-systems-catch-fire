# D297 基本常数-容斥约束函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M6的d=4稳定性约束基本物理常数：α增大~100倍→pᵢ>p*→极小点消失。精细结构常数α≈1/137不能太大→否则电磁否决概率超p*→d=4不稳定。常数不是任意的，必须让d=4稳定。
