## 函数内容 / Function Content

中文：
M13的正反馈回路放大噪声：δp_max经过k轮反馈后放大为δp_max·K^k，K为反馈增益(D274)。K>1时噪声指数放大→系统对初始条件极度敏感→蝴蝶效应。K<1时噪声衰减→系统稳定。临界K=1对应D295不可逆点。K>1的系统不可预测——不是因为模型不够好，而是正反馈使噪声放大到宏观尺度。经济危机、社会动荡的不可预测性有数学根源。
