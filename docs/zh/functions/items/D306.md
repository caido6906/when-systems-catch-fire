# D306 去容斥条件函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M7的容斥加速逆过程"去容斥"需两条件同时满足：①p_max下降速率|ḃ|>容斥加速度d²(Σpᵢpⱼ)/dt²，②p分布必须集中化（σ<σ_opt）。只满足①不满足②→容斥项基数仍大→去容斥不可持续。只满足②不满足①→p_max继续上升→容斥加速继续。D290"休克疗法"的精确版：休克疗法同时满足①②，温和改革通常只满足①。
