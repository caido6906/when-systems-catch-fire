# D368 延迟-噪声频谱函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M13的延迟使噪声放大成为低通滤波：低频放大K倍，高频放大降至K/(1+ω²τ_delay²)。延迟过滤了高频噪声但代价是D354自激振荡风险。政策延迟使系统对长期趋势敏感但对短期波动不敏感——延迟不全是坏事。
