# D333 维度回复阻尼函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M6的d_eff在4附近振荡时阻尼γ∝Σαᵢ/(1-pᵢ)²。γ>0衰减→d=4稳定吸引子；γ<0发散→不稳定；γ=0持续振荡。物理系统γ>>0（强阻尼），社会系统γ≈0（弱阻尼，长期振荡）。宇宙d=4不是恰好卡在4，而是衰减振荡后停在4。
