# D53 信号最优流速函数（凯利公式同构）

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

凯利公式 f* = (bp-q)/b 映射： - f* ↔ ε信号最优流速 v* - b（赔率）↔ 锁的硬化速度 s_lock（硬化越快，每次弱ε相变收益越大） - p（胜率）↔ P(L₁=1 | ε相变发生)（第一锁在位的概率） - q=1-p ↔ P(L₁=0 | ε相变发生)
