## 函数内容 / Function Content

中文：
M10的缓冲耗尽后可重建条件：g_eff>g_critical≈√(2ΔΦ/n)。g_eff低于此阈值时，即使p_max大幅下降，耦合强度也不足以重建缓冲——"缓冲不可逆"状态。与D295正反馈不可逆点形成双重不可逆：D295是p_max不可逆，D309是缓冲不可逆。两个不可逆点可以不同时到达——缓冲可能先于p_max进入不可逆。
