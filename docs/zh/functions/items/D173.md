## 函数内容 / Function Content

中文：
μ = R_retreat / C_exit。显态是外部驱动力主导的认知状态,粘性由撤退成本R_retreat和退出成本C_exit的比值决定。R_retreat越大或C_exit越小,显态粘性越高,系统越难从显态退回隐态。

English:
μ = R_retreat / C_exit. 显态是外部驱动力主导的认知状态,粘性由撤退成本R_retreat和exit costC_exit的比值决定. R_retreat越大或C_exit越小,显态粘性越高,系统越难从显态退回隐态.
