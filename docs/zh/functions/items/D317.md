# D317 p*敏感度函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M4的p*对系统参数的敏感度：∂p*/∂n=p*/(2n)（弱敏感），∂p*/∂σ∝p*·(σ_opt-σ)/σ_opt²（强敏感）。p*对分布分散度σ的敏感度远高于对n的敏感度。改变分布形态（集中化）比增加门控面数n更能有效移动p*——D306"去容斥需同时集中分布"的敏感度论证。
