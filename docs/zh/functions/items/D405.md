# D405 共振频率方向竞争函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M8的感知延迟使频率下移，复杂度使频率上移。交叉n_cross∝(1+τ_ratio)²。感知延迟越严重需要越大的系统才从"怕低频"切换到"怕高频"。
