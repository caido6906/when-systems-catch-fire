# D172 自举激活的乘法条件

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

B_active = σ(M_cog+θ_M) × σ(F_form-θ_F) × σ(P_track-1)。自举激活需要三因子同时满足 认知函数M_cog显著为负(好奇心极强)、形式化程度F_form超过阈值、轨道数P_track>1(多轨并行)。三因子乘法,任一归零则自举不激活。
