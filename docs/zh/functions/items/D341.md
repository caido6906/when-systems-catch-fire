# D341 共存震荡阻尼函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M14的共存震荡阻尼∝|K_benign-α_exclusion|——偏离临界越远阻尼越大。临界处阻尼为零→持续震荡不衰减，每次震荡有随机偏移，最终随机漂移到良性或恶性方向。改革力量和保守力量势均力敌时最不稳定——不是静止而是持续震荡。
