# D396 延迟临界-不可逆点统一函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M13的D382与D295在参数空间形成两条不可逆线。交点以下正反馈不可逆先到，交点以上振荡不可逆先到。小系统先触及正反馈不可逆，大系统先触及振荡不可逆。
