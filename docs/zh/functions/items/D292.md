# D292 维度最优吸引域函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M8的d_opt吸引域深度∝Σαᵢ²/(1-pᵢ)²。高p门控面α²越大→吸引越强但偏离后崩溃越猛。d_opt的"引力"和偏离后的"暴力"成正比——强吸引域=稳定但一旦失稳更致命。
