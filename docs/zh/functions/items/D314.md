# D314 ΔΦ-P传导非线性阈值函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M1的ΔΦ通过P_survival=e^{-Φ}传导。ΔΦ<0.1时P变化≈ΔΦ（线性区），ΔΦ>1时P变化≈e^{-ΔΦ}（指数区），0.1<ΔΦ<1为过渡区。非线性阈值ΔΦ_c≈0.3——低于此值D158线性近似可用，高于此值必须用指数形式。与D296三级精度体系衔接：ΔΦ_c=0.3对应pᵢ≈0.5，与D260敏感度阈值一致。
