## 函数内容 / Function Content

中文：
Λ(t) = Λ₀ × e^(σ×t)，μ(t) = μ₀ × e^(-γ×t)，碾压时间t_crush = ln(μ₀/Λ₀)/(σ+γ)。当门槛Λ以指数增长而可用资源μ以指数衰减时，系统必然被门槛碾压。门槛增速越大、资源衰减越快，碾压时间越短。

English:
Λ(t) = Λ₀ x e^(σ x t), μ(t) = μ₀ x e^(-γ x t), 碾压时间t_crush = ln(μ₀/Λ₀)/(σ+γ). 当门槛Λ以指数增长而可用资源μ以指数衰减时, 系统必然被门槛碾压. 门槛增速越大, 资源衰减越快, 碾压时间越短.
