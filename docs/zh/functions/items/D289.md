# D289 良性循环逃逸速度函数

## 函数内容

M14的逃逸速度v_escape∝λ·p_max²·g_eff。p_max≈0.5是最佳逃逸窗口（黄金逃逸点）。p_max<0.5逃逸容易但效果弱，p_max>0.8几乎不可能逃逸。D279干预时机悖论的精确化。
