# D130 调度-执行接口效率函数

## 函数内容

η_interface = P_encode × P_transfer × P_decode 三因子乘法（D127归零律适用）： P_encode = σ(ε_aware^S × B_semantic^S - θ_encode) - 调度方必须有ε_aware>0才能形成意图，必须有B_semantic>0才能把意图编码为可传递信号 - ε_aware=0 → P_encode=0 → 无法形成可传递意图 - B_semantic=0 → P_encode=0 → 有意图但无法编码
