# D286 p*-分布形态函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M4的p*对门槛分布形态有强依赖：均匀分布p*=√((n-1)/2)，极端分散p*=(p_min/p_max)√(n-1)。降低p_max不仅直接降Φ，还通过集中p分布提高p*→双重收益。
