# D377 感知-调整双延迟函数

## 函数内容

M8的总响应时间τ_total = τ_perceive + τ_σ。小偏离时瓶颈是感知（信号弱），大偏离时瓶颈是调整（n大调整慢）。维度饥渴早期的瓶颈是感知，晚期的瓶颈是调整。
