# D340 正反馈饱和后振荡函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M13的K^k饱和后p_max在稳态附近振荡，振幅∝√(p_min·p_ss)/n，频率∝ḡ。K越大稳态越低但振荡越剧烈——强正反馈的精度和抖动之间的权衡。n大时振幅小→大系统饱和后平稳，n小时大幅波动。
