## 函数内容 / Function Content

中文：
偏离度δᵢ = ηᵢ×εᵢ/c - 1，衡量弹性与弱度预期的偏离。 δᵢ=0：幂函数fᵢ=Kεᵢᶜ，弹性=弱度，补最弱=补弹性最高 δᵢ>0：弹性超预期（如指数型），应比补最弱更激进 δᵢ<0：弹性低于预期（如sigmoid门槛/饱和），应比补最弱更保守 三种偏离类型： A.增速偏离（η增速≠W增速）：非幂函数，偏离方向一致
