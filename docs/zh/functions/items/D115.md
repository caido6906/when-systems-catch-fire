## 函数内容 / Function Content

中文：
r_cross = 0 ⟹ ε_aware = 0（通过D84三条安装路径全部依赖P_track>1） **证明：** 1. D84三条安装路径（预测编码回路、分轨并行、动态算力分配）全部需要P_track>1 2. P_track = 1 + (N-1)·r_cross，r_cross=0 → P_track=1（单轨） 3. 单轨状态：预测编码回路无第二轨做误差计算→路径1失效；分轨并行不存在→路径2失效；动态算力分配无多轨可调度→路径3失效 4. 三条路径全部失效 → ε_aware无法安装 → ε_aware=0
