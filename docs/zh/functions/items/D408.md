# D408 Jensen-慢性消耗统一极限函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M11的连续极限下Jensen项∝(dΦ/dt)²·τ_min/2。τ_min→0时Jensen→0，τ_min有限时Jensen有限。慢性消耗=有限τ_min下的Jensen效应。D394"极限等价"需修正：不是严格等价而是τ_min有限时的近似等价。
