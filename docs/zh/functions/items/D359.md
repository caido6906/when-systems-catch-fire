# D359 p*放大涨落-有限n联合函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M4的p*放大涨落对有限n系统：K^k>√n时放大后涨落超过p*→竞争结果完全随机。临界K_c=√n。n越大临界K越大（大系统更耐受正反馈放大），但一旦超过临界崩溃更彻底。小系统先乱但乱得有限，大系统后乱但乱得致命。
