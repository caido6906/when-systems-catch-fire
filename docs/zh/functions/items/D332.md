# D332 容斥-耦合不可逆函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M5的容斥主导区一旦进入，退回需要p*上升，但系统崩溃方向使p*下降→退回条件与动态方向相反。容斥主导是自锁的。与D295(p_max不可逆)、D309(缓冲不可逆)形成三级不可逆：p_max→缓冲→容斥主导，逐层加深的不可逆结构。
