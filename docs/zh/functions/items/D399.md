# D399 漂移速率-复活代价联合函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M2的快速漂移使复活代价快速变化。极小点向深处漂移（学习）→拐点Φ增大→复活代价增大→复活窗口关闭。学习使极小点更深但使旧极小点复活更难——进步的代价是旧路不可逆。
