# D266 容斥偏差加速函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M3的容斥高阶项ΔΦ=Σ[e^{pᵢ}-1-pᵢ]的二阶导数=e^{pᵢ}>0→ΔΦ始终凸→加速增长。容斥偏差一旦启动就加速，不是线性偏离而是凸性加速。Jensen凸性+容斥凸性双重加速→恶性循环是默认路径。
