# D253 信息维度-容斥权衡函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

生物体的最优信息维度d_opt使否决概率总和最小且所有pᵢ<p*。d<d_opt时冗余不足→pᵢ大→容斥主导→脆弱；d>d_opt时维护成本超过容斥收益→系统臃肿→Φ因维护负担而增长。衰老过程是d从d_opt向d<d_opt漂移（信息丢失→pᵢ增大→容斥接管→加速衰亡），对应Gompertz定律的加速段。
