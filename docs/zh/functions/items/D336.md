# D336 多频叠加阶段响应函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M9的多频扰动叠加：低频+高频时高频被缓冲过滤，低频驱动过渡。但两个接近共振的频率产生拍频，拍频周期≈τ_buffer时产生二次共振——比单频共振更强的消耗。经济系统短周期+长周期的拍频可能产生超常衰退。
