# D350 阶段过渡不可逆标记函数

## 函数内容

M9的阶段2→3后能否退回取决于C_consumed/C_max > η_irreversible ≈ 1-1/n。n大时η接近1（大系统有回旋余地），n小时η小（小系统缓冲薄）。但D323重建时间∝n²使大系统理论可逆但实际重建太慢——理论可逆与实际可逆的差距随n²增长。
