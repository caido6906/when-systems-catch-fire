# D369 震荡阻尼-分支步长联合函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M14的阻尼影响步长：高阻尼→步长小→决定慢但方向确定；低阻尼→步长大→决定快但方向随机。T_decision∝|K_benign-α_exclusion|·T_osc。改革力量明显占优时方向确定但推进慢，势均力敌时推进快但方向不确定——速度和确定性的反比关系。
