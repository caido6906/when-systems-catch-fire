# D288 g_eff有限时间崩溃函数

## 函数内容

M12的g_eff演化方程g_eff(t)=1-Δα·t/(p₀+α_max·t)。Δα>α_max时g_eff在有限时间t_c=p₀/(Δα-α_max)内归零→耦合消失→系统崩溃。"突然崩溃"的精确条件：增速极化程度Δα/α_max>1。
