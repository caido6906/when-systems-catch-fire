# D366 共振频率-消耗效率函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M11的高频共振消耗效率低于低频（表面消耗vs均匀消耗），但高频更难被缓冲过滤→穿透力强。快速冲击每次伤害不大但无法被缓冲挡住——慢性消耗比急性冲击更难防御。
