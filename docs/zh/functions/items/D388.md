# D388 不可逆线相交函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M5的D332与D295在(p_max,n)空间相交于p_max=p*且n=n_c。n<n_c时p_max不可逆先到，n>n_c时容斥主导不可逆先到。大系统先进入容斥主导再触及p_max不可逆，小系统反过来。
