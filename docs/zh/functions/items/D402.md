# D402 不可逆相交-临界标度联合函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M5的D388交点处K=1（确定性不可逆），D359临界K=√n（统计不可逆）。确定性不可逆先于统计不可逆——K=1时系统确定性地进入不可逆，K=√n时涨落使不可逆变得随机。
