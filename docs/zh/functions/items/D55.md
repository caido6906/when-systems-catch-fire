# D55 ε_eff昼夜分时函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

ε_base × [1 + A_diel×sin(2πt/T_diel) + A_season×sin(2πt/T_season)]。长沙A面B面的共存有了数学解释：同一城市的ε_eff在昼夜和季节尺度上波动，最优商业模式跟着波动。振幅由城市特征决定，可从F5+H推导，是F7的周期性派生。
