## 函数内容 / Function Content

中文：
M5的容斥主导区有子结构：弱容斥主导（容斥占比50-70%，还有部分缓冲）vs强容斥主导（占比>90%，缓冲完全无效）。弱→强渐变，但D288有限时间崩溃在强容斥区加速。干预在弱容斥主导区还有可能，强容斥主导区无望。
