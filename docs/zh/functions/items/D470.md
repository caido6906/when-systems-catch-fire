# D470 幽灵跳变阻尼函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

D469×D464深入碰撞——振荡优化的阻尼系数γ在优化周期T = t_ghost_diss处不连续跳变： $$\gamma = \begin{cases} \gamma_{dirty} & T < t_{ghost\_diss} \\ \gamma_{clean} & T > t_{ghost\_diss} \end{cases}$$ γ_dirty < γ_clean。跳变来自D464超指数衰减——幽灵在t_ghost_diss前几乎不衰减，之后突然消失。 关键推论： - 存在最优优化周期T* = t_ghost_diss——刚好等幽灵消失就启动下一轮 - T < t_ghost_diss：旧幽灵干扰→γ小→改善慢→浪费能量对抗幽灵
