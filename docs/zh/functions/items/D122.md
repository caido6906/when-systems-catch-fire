# D122 退化加速函数

## 函数内容

a(t) = d²A_Fisher/dt² = -∑ᵢ ∂/∂t[σ'(εᵢ-θC(i))×|dεᵢ/dt|×d_F(εᵢ)/λ] / (1+d_F(εᵢ)/λ)² 关键项展开：对第k个接近门槛的维度， aₖ ≈ -σ''(εₖ-θC)×(dεₖ/dt)²×d_F(εₖ)/λ / (1+d_F(εₖ)/λ)² 当εₖ<θC时σ''>0，且dεₖ/dt<0（退化中），所以aₖ<0——退化在加速。 与D112的统一：D112中β=γ×max(σ')，dβ/dt=γ×σ''×dε/dt。退化加速函数中的σ''×(dε/dt)²与D112的σ''×dε/dt共享同一个σ''变号结构。区别：D112是β对ε的一阶导（策略权重如何随改善变化），D122是A_Fisher对t的二阶导（退化如何随时间加速）。一个是策略空间的动力学，一个是状态空间的动力学，同一个σ''驱动。 加速的临界条件：a(t)从接近0变为显著负值=系统进入退化加速区。临界
