# D367 僵尸态传染免疫函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M12的免疫裕度Δg_i = g_eff(i) - g_critical。Δg_i > δg_spread时免疫。但D351溢出使免疫子系统也被消耗→免疫不是永久的。组织中"健康部门"不能独善其身——溢出最终消耗免疫裕度。
