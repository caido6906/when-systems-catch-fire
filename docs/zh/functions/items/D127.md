# D127 认知路径积分函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

A_ignition = ∫ e^{i·S_ignition/ℏ_eff} D[策略路径] ℏ_eff = σ_ε（认知噪声水平） ℏ_eff→0：经典D118，策略唯一确定 ℏ_eff>0：所有策略都有贡献，最优策略概率幅最大 退化加速↔ℏ_eff增大的正反馈：退化→ℏ_eff↑→策略偏离→退化更快。三阶段协议中阶段2（刀刃期）ℏ_eff等效最大——最需要精确策略的时候策略最不确定。 ---
