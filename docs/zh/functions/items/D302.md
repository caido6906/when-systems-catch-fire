# D302 容斥渐近发散函数

[← 返回函数表](../../functions.md) ｜ [返回仓库首页](../../../README.md)

## 函数内容

M3的容斥高阶项在pᵢ→1时ΔΦ→e^{pᵢ}-1-pᵢ→∞，发散速率∝e^{pᵢ}。pᵢ接近1时容斥修正爆炸式增长，不是"大了一点"。D260敏感度阈值pᵢ>0.5后急剧上升的数学根源就是容斥项的指数发散。D296的Φ_exact在pᵢ>0.95时不可替代。
