## 定义 / Definition

中文：正反互斥判定器检查同一个对象是否被正向与反向同时判定为通过。
English: The forward-reverse exclusivity judge checks whether the same object is simultaneously accepted by the forward and reverse channels.

¬(J⁺(x)=1 ∧ J⁻(x)=1)

## 判定含义 / Judgment Meaning

- `true`：正反不可能同时通过 / simultaneous pass rejected
- `contradiction`：若正反同时通过，说明判定器或定义有问题
- `pending`：任一通道未定时不得强行给出通过结论
