## 预测内容 / Prediction Content

预测判断 / Prediction Statement

中文：
当模型尝试同时覆盖物理、系统、认知和工程问题时，单一门控函数的泛化精度会下降；把问题拆成两类或更多门控族后，预测会更稳定。

English:
When a model tries to cover physical, systems, cognitive, and engineering problems at once, the generalization accuracy of a single gate function will drop; splitting the problem into two or more gate families will produce more stable predictions.

## 推出依据 / Basis

中文：
物理类函数已经多次显示，单一统一模型会在门槛处失去解释力，而工程与系统案例则要求更细的门控分层。

English:
Physical-domain functions repeatedly show that a single universal model loses explanatory power at thresholds, while engineering and systems cases require finer gate stratification.

## 可验证条件 / Test Condition

中文：
用未来的新案例对单门控与多门控模型做交叉验证，比较精度和可解释性。

English:
Cross-validate a single-gate model and a multi-gate model on future cases and compare accuracy and interpretability.

## 可证伪条件 / Falsification Condition

中文：
如果一个单门控模型在跨域新案例上始终优于多门控模型，则预测失效。

English:
If a single-gate model consistently outperforms the multi-gate model on new cross-domain cases, the prediction is falsified.
