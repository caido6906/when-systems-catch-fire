## 全局变更（Agent 行为）

| 变更 | 说明 |
|------|------|
| 角色切换机制 | Agent 在处理点火任务时根据任务切换反方/审稿人/编码员/整理者四种角色 |
| 输出语气约束 | 禁止谄媚式确认，禁止空喊原创，允许强洞见存在但必须降级为可验证命题 |
| 文明级判断降级 | 所有文明/身份尺度的判断一律标记为「假说」，不得写成定论 |
| 案例自检强制 | 每个新案例必须通过 6 项自检问题 |
| 引用分类强制 | 引用时必须区分「框架主张」「案例假说」「已验证结论」「开放问题」 |
