# 点火框架·全案例校准总表（8格版·2026.06.05）

框架核心：系统 = 协作系统（搭房）+ 认同系统（点火）
角色：提议者 / 应约者
条件层：8格（提议者意识 × 提议者退出权 × 应约者退出权，应约者状态移至方向层）
退出权四分法：真实 / 事实 / 心理 / 象征
五步流程：①旧系统失灵 → ②提议者出现 → ③应约者应约 → ④点火跑通 → ⑤稳定期足够

8格编号规则：
格1：有意识·提议者有退出·应约者有退出
格2：有意识·提议者有退出·应约者无退出
格3：有意识·提议者无退出·应约者有退出
格4：有意识·提议者无退出·应约者无退出
格5：无意识·提议者有退出·应约者有退出
格6：无意识·提议者有退出·应约者无退出
格7：无意识·提议者无退出·应约者有退出
格8：无意识·提议者无退出·应约者无退出

原24格→新8格映射：格1/2/3→新格1，格4/5/6→新格2，格7/8/9→新格3，格10/11/12→新格4，格13/14/15→新格5，格16/17/18→新格6，格19/20/21→新格7，格22/23/24→新格8

## 七、核心发现

1. **格1和格3是点火的黄金条件**：28个案例（40%），退出权真实，认同可验证。格3的"提议者被绑=信誉"是最关键的点火机制之一——周公、邓公、罗斯福、纳德拉都在这个格子里用信誉点燃了认同。

2. **格4是最大的温室**：17个案例（24%），事实退出权集中（11个）。认同看起来是真的，但从未被退出权验证过。退出权出现时冲击最大——新中国、清朝、凯末尔改革、海尔·塞拉西。

3. **格2是象征退出权的重灾区**：7个案例中5个象征退出权。系统保留了退出的形式但把代价设计到不可承受——秦统一、元朝、计划生育、安然。"你可以走但走了等于死"。

4. **应约状态是结果不是输入**：同一格内横跨积极/消极/无人三种状态，说明应约者状态由方案质量和外部条件决定，不是系统初始配置。这验证了8格重构的核心判断。

5. **格子漂移只发生在输入变量变化时**：格4→格3（应约者获得退出权）、格8→格7（提议者意识觉醒）、格1内退化（提议者退化导致应约者从积极滑向消极）。应约状态的变化是漂移的结果不是原因。
