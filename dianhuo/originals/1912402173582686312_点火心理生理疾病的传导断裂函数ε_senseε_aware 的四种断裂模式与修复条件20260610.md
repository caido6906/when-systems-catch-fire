# 点火｜心理生理疾病的传导断裂函数：ε_sense→ε_aware 的四种断裂模式与修复条件（2026.06.10）

## 核心论点

心理生理疾病的本质是身体信号（ε_sense）与认知解读（ε_aware）之间的传导断裂。身体在喊"这不对"，但认知系统要么听不见，要么把信号误读为相反的意思。

## 一、身心传导函数

η_body→mind(t) = ε_aware(t) / ε_sense(t)

- η = 1：身体信号被完整解读（健康）
- η → 0：身体信号产生了但认知没解读（疾病）
- η > 1：不可能

η 是心理生理健康度的核心指标。

疾病度 D = 1 - η = 1 - ε_aware / ε_sense

## 二、四种断裂模式

### 模式1：信号压制（H 截断）

ε_sense > 0 但 H ≈ 1-ε_sense，身体信号被认知遮蔽压到几乎不可感知。

η₁ = 1 - H ≈ 0

ε_sense 持续累积但 η→0。信号没有消失，只是没有被解读。

典型：慢性疲劳、职业倦怠。"累了"的信号被"我必须坚持"压下去。

### 模式2：信号误读（退出型→投入型）

ε_sense 的信号类型被规训改造：累→"我还不够努力"，痛→"这是成长的代价"。

η₂ = ε_sense^exit / (ε_sense^exit + ε_sense^invest) ≈ 0

ε_sense 总量没变，但退出型信号占比趋近于零。信号被"翻译"成了相反的意思。

典型：工作成瘾、自虐式自律、受虐关系中的"他打我是为我好"。

### 模式3：信号过载（超出处理容量）

ε_sense >> ε_aware^max，原始信号太强超出前额叶处理容量。

η₃ = ε_aware^max / ε_sense → 0

不是 H 在压制，不是信号被改写，是原始信号强度超出处理器带宽。

典型：PTSD、惊恐发作、急性应激反应。信号不是"没被听见"，是"太响了听不清说什么"。

### 模式4：信号衰减（感受器钝化）

长期 H≈1 环境中感官敏锐度下降，感受器本身钝化。

ε_sense(t) = ε_sense(0) × e^(-ρ · ∫₀ᵗ H(τ)dτ)

ρ = 感官钝化率。H 持续越高，ε_sense 衰减越快。但 ε_sense 永远>0（硬约束），衰减有下限。

典型：长期抑郁中的情感麻木、慢性疼痛适应、长期受虐者的"感觉不到痛了"。

## 三、统一疾病函数

D_body→mind(t) = 1 - ε_aware(t) / ε_sense(t)

| 模式 | D值 | 机制 | ε_sense | ε_aware | η |
|------|-----|------|---------|---------|---|
| 信号压制 | ≈1 | H截断 | 正常累积 | ≈0 | ≈0 |
| 信号误读 | ≈1 | 退出型→投入型 | 正常 | exit≈0 | ≈0 |
| 信号过载 | ≈1 | 超出处理容量 | 过强 | 达上限 | ≈0 |
| 信号衰减 | 中高 | 感受器钝化 | 衰减中 | 衰减中 | 低 |

## 四、疾病的自增强回路

dD/dt = α_feedback · D · (1-D) - ε_aware · D + β_stress · S_external

三项：
- 自增强项：D越大传导越断裂，人越"习惯"断裂，D进一步增大
- 退出权制动项：ε_aware 是唯一的负反馈
- 外部压力项：外部压力持续推高 D

**稳态由 ε_aware 唯一决定，与茧房函数完全同构。**

## 五、躯体化——身体替认知说话

当 D≈1（传导断裂），ε_sense 的信号无法通过正常通路到达认知系统。信号走替代通路：

η_somatization = (ε_sense - ε_aware) / ε_sense × P_somatic

躯体化的数学本质：正常通路被截断时，信号通过低层通路绕行。

躯体化不是"假病"，是真实信号走了替代通路。身体在替认知说话——因为认知系统自己不说话了。

典型：焦虑性头痛、压力性胃痛、抑郁性疼痛。检查不出器质性病变，但疼痛真实——因为信号源（ε_sense）真实。

## 六、治疗函数

D < 1 ⟺ ε_aware > 0

### 四条治疗路径

**路径1：降 H（解除信号压制）——模式1**
H↓ ⟹ η = 1-H ↑
操作：让"累了"重新被允许说出来，让"这不对"重新被允许想。不是教新知识，是解除对已有信号的压制。

**路径2：信号重译（退出型←投入型）——模式2**
ε_sense^exit / (ε_sense^exit + ε_sense^invest) ↑
操作：把"累=我还不够努力"重新翻译为"累=我该休息了"。不是新增信号，是改回信号原意。

**路径3：扩容（提升 ε_aware 处理容量）——模式3**
ε_aware^max ↑
操作：创伤治疗的核心——不是消除记忆，是提升前额叶对原始信号的处理能力。EMDR、暴露疗法的数学本质都是扩容。

**路径4：感官再激活（恢复 ε_sense 敏锐度）——模式4**
dε_sense/dt = -ρ · H · ε_sense + σ_activation · S_sensory_input
操作：正念、身体扫描、自然环境暴露。不是在认知层面工作，是在感受器层面工作。

### 统一治疗函数

P_heal(t) = [1-H(t)] × [ε_sense^exit(t)/ε_sense(t)] × [ε_aware^max(t)/ε_sense(t)] × σ_activation(t)

四因子乘法：
1. [1-H] — 信号不被压制的程度
2. ε_sense^exit/ε_sense — 退出型信号占比
3. ε_aware^max/ε_sense — 认知处理容量与信号强度比
4. σ_activation — 感官再激活率

任何一个为0，治疗概率为0。

四种场景数学结构完全同构：信号在产生，但无法到达应该到达的地方。

**统一表达：所有认知遮蔽都是传导断裂。**

D_universal = 1 - (实际到达的信号 / 产生的信号) = 1 - η_conduction

P_repair = D_decode × ε_exit × (θ_align / Resistance)
