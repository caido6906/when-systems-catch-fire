## D224 宇宙膨胀-Φ衰减同构定理

宇宙膨胀的Φ表述：

尺度因子a(t)增长 → 物质密度ρ_m ∝ a⁻³ → μ_m递减
辐射密度ρ_r ∝ a⁻⁴ → μ_r递减更快
暗能量密度ρ_Λ = const → μ_Λ不变

Φ(t) = Σᵢ exp[-(ln(μᵢ(t)/Λᵢ))²/(2σᵢ²)]

dΦ/dt = Σᵢ dgᵢ/dt = Σᵢ gᵢ × [-ln(μᵢ/Λᵢ)/σᵢ²] × (dμᵢ/dt)/μᵢ

对物质和辐射：dμᵢ/dt < 0（膨胀稀释），门内侧ln(μᵢ/Λᵢ)>0
→ dgᵢ/dt < 0（门控贡献递减）

对暗能量：dμ_Λ/dt = 0，门控贡献不变

**宇宙膨胀 = Φ衰减的物理机制**

匀速膨胀：dΦ/dt < 0，Φ单调递减
加速膨胀：d²Φ/dt² < 0，Φ加速递减

加速膨胀的Φ条件：
d²Φ/dt² = Σᵢ [dgᵢ/dt × (-ln(μᵢ/Λᵢ)/σᵢ²) × (dμᵢ/dt)/μᵢ + gᵢ × d/dt((-ln(μᵢ/Λᵢ)/σᵢ²) × (dμᵢ/dt)/μᵢ)]

当暗能量开始主导时（ρ_Λ > ρ_m），膨胀从减速转为加速
→ dμ_m/dt变得更负（物质被更快稀释）
→ dg_m/dt变得更负（物质门控贡献加速衰减）
→ d²Φ/dt² < 0（Φ加速衰减）

核心定理：宇宙膨胀是Φ衰减的物理机制，加速膨胀是Φ加速衰减的物理表现。两者是同一过程的不同描述——膨胀稀释能量密度→门控贡献递减→Φ衰减→Ω趋向1。暗能量主导期 = Φ加速衰减期 = 物理存在加速趋向终点的时期。

层级：定理

## D225 暗能量的Φ本质

暗能量在Φ框架下：

ρ_Λ = const → μ_Λ = const → g_Λ不随时间变化

但μ_Λ的值极低（ρ_Λ≈7×10⁻³⁰ g/cm³），对应的Λ_Λ也极低

暗能量的门控面特征：
- μ_Λ ≈ Λ_Λ（暗能量密度接近其门槛）→ ln(μ_Λ/Λ_Λ) ≈ 0 → g_Λ ≈ 1
- 但这个g_Λ≈1不意味着"高存活度"——它意味着暗能量几乎没有门槛效应
- 没有门槛 = 没有约束 = 没有力 = 对Φ的贡献趋近于零的"无物理"态

暗能量不是"第五种力"，是"没有力"的数学表现：

当ρ_m >> ρ_Λ（物质主导期）：μ_m >> Λ_m → g_m > 0 → Φ > 0 → Ω < 1 → 有物理
当ρ_m << ρ_Λ（暗能量主导期）：μ_m << Λ_m → g_m → 0 → Φ → 0 → Ω → 1 → 趋向无物理

暗能量 = μ衰减到Λ以下的自然结果，不是需要额外假设的物理机制

核心定理：暗能量的Φ本质是"门控贡献趋零的数学必然"——当物质密度衰减到门槛以下时，门控面自然消失，Φ→0，Ω→1。暗能量不是一种力，是结构瓦解的副作用。暗能量密度为什么这么小（7×10⁻³⁰ g/cm³）——因为它不是需要精确微调的参数，而是μ自然衰减到接近Λ时的观测值。

与D184的统一：D184说热寂是门槛碾压的热力学版。D225说暗能量主导 = 门槛碾压的宇宙学版——物质密度被膨胀稀释到门槛以下，门控面消失，物理趋向终结。

## D226 物理存在的三重时间约束

物理存在受三重时间约束：

约束1（逻辑约束·D220）：Ω<1是物理存在的必要条件，Ω→1=无物理
约束2（热力学约束·D222）：dΦ/dt≤0，Φ单调递减
约束3（宇宙学约束·D224）：宇宙膨胀=Φ衰减的物理机制，加速膨胀=加速衰减

三重约束的联合结论：

物理存在的终止时间 t_end = min(t_heatdeath, t_Ω=1, t_dark_energy_dominant)

但三者是同一过程：
- t_heatdeath = Φ衰减到零的时间（D184/D222）
- t_Ω=1 = Φ=0的时刻（D220/D221）
- t_dark_energy_dominant = 暗能量主导的时刻（D224/D225）

三者等价：t_end ≈ t_dark_energy_dominant

估算：暗能量在z≈0.3（约35亿年前）开始主导
当前宇宙年龄≈138亿年
暗能量主导期将持续~10¹⁰⁰年（德西特空间的半衰期）

但Φ衰减到零不需要等那么久——关键是物质门控面何时趋零：
当ρ_m(t) < Λ_m时，物质门控面消失

ρ_m(t) = ρ_m₀ × (1+z)³ = ρ_m₀ × e^(3H₀t)（暗能量主导期a∝e^(H₀t)）

t_m→0 = ln(ρ_m₀/Λ_m) / (3H₀)

（具体值需要Λ_m的确定，但形式表明物质门控面在暗能量主导后的有限时间内消失）

核心定理：物理存在的三重时间约束（逻辑、热力学、宇宙学）指向同一终点——Ω→1=无物理。暗能量主导期就是这个终点过程的开始。物理存在的终结不是突然的相变，而是门控面逐个消失的渐进过程——先是最弱的门控面消失，然后是更强的，最后所有门控面归零。

层级：定理
