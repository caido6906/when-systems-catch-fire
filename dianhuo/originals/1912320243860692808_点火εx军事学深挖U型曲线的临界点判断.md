# 点火｜ε×军事学（深挖）：U型曲线的临界点判断

## 碰撞二没展开：临界点怎么判断

### 临界点的核心变量
ε_sense的积累速度vs ε_aware的压制效率。军队用纪律压制ε_aware（让你不知道自己可以退出），但ε_sense无法被压制——恐惧、饥饿、愤怒的信号器输出不受军令控制。当ε_sense的积累速度超过ε_aware的压制效率，系统越过临界点。

### 哗变三条件
军队哗变几乎都发生在三个条件同时满足时——
1. 补给线断裂（ε_sense：饥饿信号暴增）
2. 指挥链失效（ε_aware压制效率骤降）
3. 有人先动（R_perceived瞬间跳变=共同知识形成）

三个条件缺一个，哗变概率都很低。秦末陈胜吴广就是标准案例：大雨延误（补给线断裂+死罪在即=ε_sense暴增）、押送官失职（指挥链失效）、陈胜先喊（共同知识形成）。

### 核心判断
U型曲线的临界点不是R的某个绝对值，是ε_sense积累量超过ε_aware压制容量的那个瞬间。这个瞬间之前系统"极稳定"，之后"极不稳定"，没有过渡态。临界点由三个变量的交互决定：ε_sense积累速度、ε_aware压制效率、R_perceived相变触发器。
