# 菲律宾安理会非常任理事国竞选失利事件分析

2026-06-05 00:09:25

### **📅 事件背景与结果**

**联合国安理会非常任理事国选举**
- **选举时间**：联合国大会3日举行。
- **当选国家**：奥地利、吉尔吉斯斯坦、葡萄牙、特立尼达和多巴哥、津巴布韦。
- **菲律宾结果**：竞争对手吉尔吉斯斯坦经四轮投票选出，菲律宾落选。

### **🇵🇭 菲律宾官方回应**

菲律宾总统府4日称，**国内政治紧张局势可能是导致此次落选的原因之一**。

### **🎓 专家分析与评价**

广西民族大学东盟学院副院长葛红亮观点：
1. **执政能力质疑**：把责任推给国内政治局势，暴露了菲律宾现任政府执政能力有问题。
2. **外交政策批判**：
   - 菲律宾打造所谓"多边主义人设"，但实际做法暴露了马科斯政府的虚伪。
   - 对比：杜特尔特政府采取**务实、全面的外交政策**，而马科斯政府大幅改变，在军事和外交方面采取**冒进做法**。
   - 具体表现：在**美国"印太战略"**中发挥了一定作用。
3. **未来潜在风险**：菲律宾2026年担任**东盟轮值主席国**，可能借机推进自身利益；若当选安理会非常任理事国，可能在安理会搞操弄以推进自身利益。

### **🔍 联合国安理会相关机制**

| 项目 | 具体内容 |
| :--- | :--- |
| **成员构成** | 15个成员，包括5个常任理事国与10个非常任理事国 |
| **选举方式** | 联合国大会通过无记名投票方式选举产生 |
| **当选条件** | 候选国需获得总投票数至少**三分之二赞成票**方可当选 |
