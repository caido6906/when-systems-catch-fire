# 认知信道与信息茧房

开跑。

---

**信息茧房 = 信源选择（静态筛选）× 谣言传播（动态灌入）的闭环运行。** 两个函数合并的关键在于：信源选择函数决定了信道的开闭状态，谣言传播函数决定了信号在已开信道上的通量——合在一起就是一个自增强的认知信道系统。

---

**第一步：定义认知信道带宽**

一个人在时刻 t 的认知信道带宽，衡量他实际能接收的信息频谱宽度：

$$W(L, t) = \sum_{i} P_{accept}(S_i, L) \times B(S_i)$$

对所有信源 S_i 求和。W 越大，信息频谱越宽；W 越小，信道越窄。

**代入层级：**

- L₁-L₂：P_accept(local) ≈ 1, P_accept(expert) ≈ 0 → W ≈ B_local（只有身边人信源进入，带宽极窄）
- L₄-L₅：P_accept(local) ≈ 0.3, P_accept(expert) ≈ 0.9 → W ≈ 0.3·B_local + 0.9·B_expert（宽频接收）

**信息茧房的数学定义：**

$$\text{茧房} \iff W(L, t) < W_{critical}$$

即认知信道带宽收窄到临界值以下，只剩单一类型信源。

---

**第二步：茧房的自增强回路——关键创新**

信源选择不是静态的。你接受什么信息，会反过来改变你的认知框架，进而改变你对下一轮信息的接受概率。这是茧房最致命的机制：**它自己把自己锁死。**

定义认知框架更新函数：

$$\theta_{align}(S, L, t+1) = \theta_{align}(S, L, t) + \eta \cdot P_{accept}(S, L, t) \cdot [\theta_{align}^{target}(S) - \theta_{align}(S, L, t)]$$

η 是框架更新率（你被接受的信息改变你认知框架的速度）。

**这个更新规则的意思是：** 你越接受某类信源，你的认知框架就越对齐这类信源，下次对齐度更高，接受概率更大——正反馈。

**对 L₁-L₂ 层跑这个回路：**

1. 初始：θ_align(local) ≈ 1, θ_align(expert) ≈ 0
2. 只接受 local 信息 → 认知框架被 local 信息塑造
3. θ_align(local) 进一步升高（已经接近1，变化不大）
4. θ_align(expert) 进一步降低（专业框架更陌生了）
5. P_accept(expert) 进一步降低 → 更不接受专业信息
6. 回到步骤2，循环

**对 L₄-L₅ 层：**

1. 初始：θ_align(expert) 已经较高
2. 接受 expert 信息 → 认知框架被专业信息塑造
3. θ_align(expert) 进一步升高
4. 但 ε 充足，能主动检测 θ_align 是否过度偏向 → 退出权提供负反馈
5. 系统收敛到宽频稳态而非窄频锁死

**关键差异：ε 是茧房自增强回路的唯一制动器。**

---

**第三步：合并谣言传播——茧房的加速器**

谣言传播函数给茧房的自增强回路加了第二台发动机。

定义茧房深度（茧房化程度）：

$$\Phi_{cocoon}(L, t) = 1 - \frac{W(L, t)}{W_{max}}$$

Φ ∈ [0, 1]，0 = 完全开放，1 = 完全茧房化。

谣言对茧房深度的贡献是双重的：

**贡献1：直接增加 local 信道通量**

谣言走 local 信道，E_rumor > 1，使 C_rumor 远超普通 local 信息。这等价于 B_local 被人为放大：

$$B_{local}^{effective}(t) = B_{local} + \sum_{j} I_{rumor,j}(t) \times E_{rumor,j}$$

谣言越多，local 信道的有效信号强度越大，W 的计算中 local 信源权重增大但频谱不增——**带宽不变，信噪比恶化。**

**贡献2：通过免疫衰减维持 local 信道持续高负载**

δ 大 → Γ 快速衰减 → 同类谣言反复进入 → local 信道始终满载 → 没有剩余带宽给 expert 信号。

**合并后的茧房深度动力学：**

$$\frac{d\Phi_{cocoon}}{dt} = \underbrace{\eta \cdot P_{accept}(local, L) \cdot \Phi_{cocoon} \cdot (1 - \Phi_{cocoon})}_{\text{自增强项（信源选择正反馈）}} + \underbrace{\sum_j \beta_j(L) \cdot I_j \cdot (1 - \Phi_{cocoon})}_{\text{谣言加速项}} - \underbrace{\epsilon(L) \cdot \Phi_{cocoon}}_{\text{退出权制动项}}$$

三项：
- **自增强项**：logistic 增长，Φ 越大增长越快（直到接近1时饱和）
- **谣言加速项**：谣言传播直接推动茧房化
- **退出权制动项**：ε 是唯一的负反馈

**稳态分析：**

令 dΦ/dt = 0：

- **ε ≈ 0（L₁-L₂）**：制动项消失，Φ → 1（完全茧房化），唯一稳态
- **ε 中等（L₃）**：Φ 稳定在中间值，但谣言冲击可以暂时推高
- **ε 充足（L₄-L₅）**：Φ → 0（开放稳态），谣言冲击被快速吸收

---

**第四步：茧房的刚性——为什么一旦形成极难打破**

定义茧房刚性（打破茧房所需的外部信号强度）：

$$K_{cocoon}(L) = \frac{R(S_{expert}, L)}{C(S_{expert}, L)} = \frac{F_{form}(L) + \epsilon(L)}{B_{expert} \times D_{expert}(L) \times \theta_{align}(expert, L)}$$

K 越大，越难打破。

**L₁-L₂ 层的 K 值分析：**

- D_expert ≈ 0 → 分母趋近于 0
- F_form 高 → 分子大
- ε ≈ 0 → 分子无法被减小
- θ_align(expert) 因自增强回路持续降低 → 分母持续缩小

**K → ∞。茧房刚性趋近无穷大——不是"很难打破"，是"在现有信道结构下不可能打破"。**

这解释了为什么单纯加大辟谣力度（增大 B_expert）无效：B_expert 乘以 D ≈ 0 还是 0。信号进不去。

---

**第五步：破壁条件**

茧房破壁的统一条件：

$$K_{cocoon}(L) < K_{critical} \iff \frac{F_{form}(L) + \epsilon(L)}{B_{expert} \times D_{expert}(L) \times \theta_{align}(expert, L)} < K_{critical}$$

**四条破壁路径，按因果优先级排序：**

**路径1：装 D（提升专业信号解码能力）**

这是最根本的路径。D 从 0 变为正，分母从 0 变为正，K 从 ∞ 降为有限值——从"不可能"变成"理论上可以"。

操作：专业信息翻译进 D=1 的信道（日常语言、故事、视觉化），同时逐步引入专业框架元素，让 D 从 0 缓慢上升。

**路径2：装 ε（安装退出权）**

ε 从 0 变为正，制动项出现，自增强回路被抑制，Φ 从 1 开始下降。

操作：让人意识到"我可以不信身边人说的"——这不是信息问题，是认知主权问题。

**路径3：降 F_form（降低形式化服从惯性）**

降低对专业权威的防御反应。

操作：专业信源放弃权威姿态，用平等对话而非居高临下的"我来告诉你"。

**路径4：提升 θ_align（提升认知对齐度）**

让专业信息的表述框架与接收者既有框架有交集。

操作：从接收者已有的概念出发，逐步搭建通往专业概念的桥梁，而不是直接投放专业术语。

**四条路径的依赖关系：**

路径1（装D）是前提——没有 D，其他三条路径的信号都进不去。路径2（装ε）是制动——没有 ε，即使信号进去了也会被自增强回路推回。路径3和4是加速器——降低阻力、提升对齐度，让已经能进入的信号进入得更顺畅。

**统一破壁函数：**

$$P_{break}(L) = D_{expert}(L) \times \epsilon(L) \times \frac{\theta_{align}(expert, L)}{F_{form}(L)}$$

四因子乘法关系——跟你统一跳变函数 P(L₃→L₅) = σ × A × D × |M_cog<0| 结构完全同构。任何一个为 0，破壁概率为 0。

---

**第六步：统一认知信道函数**

把信源选择、谣言传播、茧房形成合并为一个统一系统：

$$\Psi(L, t) = W(L, t) \times [1 - \Phi_{cocoon}(L, t)] \times \Gamma(L, t)$$

- W(L, t) — 认知信道带宽（信源选择函数决定）
- [1 - Φ_cocoon(L, t)] — 信道开放度（茧房深度的补数）
- Γ(L, t) — 免疫强度（谣言传播函数决定）

**Ψ 是认知信道的健康度指标。** Ψ → 0 = 完全茧房化 + 免疫丧失；Ψ → max = 宽频开放 + 免疫充足。

**Ψ 的动力学：**

$$\frac{d\Psi}{dt} = \frac{\partial \Psi}{\partial W} \cdot \frac{dW}{dt} + \frac{\partial \Psi}{\partial \Phi} \cdot \frac{d\Phi}{dt} + \frac{\partial \Psi}{\partial \Gamma} \cdot \frac{d\Gamma}{dt}$$

三个偏导数分别对应三个子系统（信源选择、茧房形成、免疫衰减），通过 W、Φ、Γ 耦合。

**核心结论：Ψ 的稳态由 ε 唯一决定。** ε = 0 → Ψ → 0（唯一稳态是茧房）；ε > 0 → Ψ > 0（存在开放稳态）。ε 是整个认知信道系统的相变参数。

---存好了，已归入点火知识库。

三个函数现在统一了：信源选择解决"为什么信这个不信那个"，谣言传播解决"为什么反复爆发"，认知信道解决"茧房怎么形成怎么破壁"。底层共享同一组参数（D, ε, θ_align, F_form），由 ε 做相变控制。

最狠的结论是茧房刚性 K → ∞——不是"很难打破"，是"在现有信道结构下不可能打破"。加大辟谣力度完全无效，因为 B_expert × D ≈ 0 还是 0。破壁的前提是装 D，让信号先能进去；然后装 ε，让自增强回路有制动。四因子乘法关系，跟你统一跳变函数完全同构。
