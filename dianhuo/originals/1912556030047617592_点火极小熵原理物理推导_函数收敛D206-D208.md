## D206 门控-路径积分同构定理

经典力学中的路径积分：
Z = ∫ D[q(t)] exp[-S[q(t)]/ℏ]

在经典路径q_cl附近展开：S = S_cl + ½(δS)² × G⁻¹ + O(δS³)
其中G为传播子矩阵

一阶展开给出高斯积分：Z ~ exp[-S_cl/ℏ] × (det G)^(-1/2)

门控函数在M_Planck处的展开：
g(μ) = exp[-f(ln(μ/M_Planck))]

在μ=M_Planck附近展开：f(x) = ½f''(0)x² + O(x⁴)
其中x = ln(μ/M_Planck)

一阶展开给出高斯门控：g ~ exp[-f''(0)x²/2] = exp[-(ln(μ/M_Planck))²/(2σ²)]
其中σ² = 1/f''(0)

同构关系：
| 路径积分 | 门控函数 |
|---------|---------|
| 作用量S | f(ln(μ/M_Planck)) |
| 经典路径q_cl | μ=M_Planck |
| δS=0 | dg/dμ=0 |
| 传播子G | 1/f''(0)=σ² |
| ℏ压制高阶项 | σ²压制高阶项 |
| Z的高斯近似 | g的高斯近似 |

核心定理：门控函数在M_Planck处的数学结构与路径积分在经典路径处的数学结构同构。两者共享：极值条件（δS=0 ↔ dg/dμ=0）、一阶展开为高斯、高阶修正被参数压制（ℏ ↔ σ²）。这不是类比，是同一数学结构在不同物理语境下的表现。

## D207 极小熵原理的物理推导

从D206同构定理推导极小熵原理：

步骤1：路径积分的数学结构保证——在作用量极值点附近，积分的主导贡献来自高斯项。高阶项的贡献比例为O(ℏ/S_cl)，在经典极限ℏ→0时被压制。

步骤2：门控函数的同构结构保证——在M_Planck极值点附近，门控函数的主导行为来自高斯项。高阶项的贡献比例为O(σ²/f''(0)×x²)，在|x|<<1时被压制。

步骤3：物理门控函数描述的是"系统在M_Planck附近的行为"——这个行为由路径积分的数学结构决定，不是由额外的假设决定。

步骤4：因此，高斯门控 = 路径积分在M_Planck极值点附近的一阶展开。这不是"选择高斯"，而是"物理系统在稳定点附近的行为必然是高斯的"——正如经典路径附近的量子涨落必然是高斯分布。

步骤5：极小熵原理的物理含义——极小熵 = 高斯 = 路径积分的一阶展开 = 物理系统在稳定点附近的最经济描述。这不是审美偏好，是路径积分的数学结构对物理预言的约束。

核心定理：极小熵原理有严格的物理推导——它等价于路径积分在稳定极值点附近的一阶展开。高斯门控不是"等价类中的一个选择"，而是"物理系统在M_Planck处稳定极值点附近的必然行为"。高阶修正对应路径积分的高阶项，被ℏ（或等效参数σ²）压制，物理上可忽略。

D204从"审美偏好"升级为"物理定理"。
