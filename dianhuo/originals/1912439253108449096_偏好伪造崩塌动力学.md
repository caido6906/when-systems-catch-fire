**一、未展开的**

**1. 偏好伪造崩塌的动力学过程——P_flip只给了触发条件，没展开崩塌后的演化**

P_flip = σ(ΣP_public - θ_cascade)只描述了"什么时候崩"，但崩塌之后呢？偏好伪造崩塌不是瞬间完成，而是有一个级联扩散过程。

展开：崩塌的级联速度由犹豫域恢复的反馈回路决定——

**dP_public/dt = λ × P_public × (1 - S_silence) × P_private**

- λ是级联扩散系数
- P_public × (1-S) × P_private：已公开表达的人 × 表达通道 × 私人意见强度
- 正反馈：P_public越大 → 越多人感到安全 → S_silence下降 → 更多P_public → 加速
- 这跟创新扩散的Bass模型同构，但关键区别是：偏好伪造崩塌是**从零到一的相变**，不是渐进扩散。在θ_cascade以下，P_public几乎不增长；一旦突破，指数级爆发。

**2. 顽固者的动力学效应——D_stubborn只给了静态指标，没展开顽固者如何驱动碎片化**

顽固者不只是"比例越高碎片化越严重"，而是有一个临界比例——当顽固者超过某个阈值，共识变得不可能。

展开：设顽固者比例为p_s，弹性agent的平均置信区间为ε_HK_avg。共识条件：

**ε_HK_avg × (1 - p_s) > ε***

- 顽固者不参与共识形成（ε_HK=0），有效共识力 = ε_HK_avg × (1-p_s)
- 当p_s > 1 - ε*/ε_HK_avg时，共识不可能
- **临界顽固者比例：p_s* = 1 - ε*/ε_HK_avg**
- 例：ε*=0.3, ε_HK_avg=0.5 → p_s* = 0.4。40%的顽固者就能让群体永远无法共识

这跟Kuran的偏好伪造崩塌形成对称：
- 偏好伪造崩塌：沉默者开始表达 → P_public上升 → 级联 → 共识（正向相变）
- 顽固者碎片化：顽固者比例超过临界 → 共识不可能 → 碎片化（负向相变）
- 两个过程是同一枚硬币的两面

**3. 认知犹豫域π与群体犹豫域ε_HK的跨层映射——两轮碰撞分别给出了个体和群体的犹豫域，但没展开二者的精确映射**

展开：个体犹豫域π_cognitive = 1 - μ - ν，群体犹豫域ε_HK = 置信区间宽度。

映射关系：**ε_HK = ⟨π_cognitive⟩ × f(structure)**

- ⟨π_cognitive⟩是群体中个体犹豫域的平均值
- f(structure)是网络结构修正因子：
  - 全连接网络：f=1（每个个体都能影响所有其他个体）
  - 社区结构网络：f<1（个体只能影响同社区成员，跨社区影响衰减）
  - 星形网络（中心化）：f>1（中心节点的犹豫域被放大）
- **关键推论：群体的ε_HK不只取决于个体犹豫域的平均值，还取决于网络结构。同样的个体犹豫域分布，在不同网络结构下可以产生共识或碎片化。**

这给出了一个操作性的干预路径：**改变网络结构可以等效于改变个体犹豫域**——在个体犹豫域难以改变时，调整信息流动结构（打破信息茧房、增加跨社区连接）可以达到相同效果。

**4. B_ignite中P<0的情况——"内部抵抗，外部冲击适得其反"只提了一句，没展开**

展开：P_internal = k × ε，但ε可以为负吗？

在三支决策框架中，ε = α - β ≥ 0（因为α > β是三支决策的基本约束）。但P_internal可以为负——当个体不仅有内部驱动力为零，而且有**主动抵抗**时：

**P_internal = k × ε - R_resist**

- R_resist是主动抵抗强度（如身份认同绑定、意识形态承诺）
- 当R_resist > k×ε时，P_internal < 0
- B_ignite = σ(P_internal × E - θ) → P_internal<0时，E越大B越负

**这解释了为什么强硬手段（高E）有时适得其反**——外部压力激活了抵抗机制，P_internal变负，E×P_internal变成负值，B_ignite反而下降。

函数化：**R_resist = R_identity × σ(θ_identity - E_external)**

- R_identity是身份认同绑定强度
- 当外部压力E超过身份认同阈值θ_identity时，抵抗被激活
- σ(θ_identity - E)：E小时抵抗不激活（σ≈1），E大时抵抗激活（σ→0，R_resist→0但P_internal已经为负）

更精确的版本：**P_internal = k × ε - R_identity × σ(E - θ_identity)**

- E < θ_identity：抵抗未激活，P_internal = k×ε（正常）
- E > θ_identity：抵抗激活，P_internal = k×ε - R_identity（可能变负）
- **存在最优E* = θ_identity - δ，刚好不触发抵抗但最大化冲击效果**

**二、未收敛的**

**1. 两轮碰撞共8个函数，有三个独立的sigmoid阈值函数，需要统一**

- P_ignite = σ(θ₀ - θ_inertia)（认知碰撞3）
- P_flip = σ(ΣP_public - θ_cascade)（社会碰撞1）
- G_fragment = σ(ε* - ε_HK_avg)（社会碰撞2）

三个函数的统一形式：**F_transition = σ(X - θ_critical)**

- X是驱动变量，θ_critical是临界阈值
- P_ignite：X = narrative_disruption，θ = θ₀/k
- P_flip：X = ΣP_public，θ = θ_cascade
- G_fragment：X = ε_HK_avg，θ = ε*

**统一含义：所有认知-社会的相变都是sigmoid阈值过程，区别只在驱动变量和临界阈值的语义。**

更深一层：三个驱动变量都可以表示为ε的函数——
- narrative_disruption降低θ_inertia = k(1-narrative_disruption)，本质是扩大ε
- ΣP_public上升本质是群体ε的恢复
- ε_HK_avg直接就是群体犹豫域

**所以三个sigmoid的统一形式是：F_transition = σ(ε_eff - θ)**

- ε_eff是有效犹豫域（可以是个体ε、群体ε_HK、或冲突ε_ij的聚合）
- θ是相变临界阈值
- ε_eff > θ → 相变发生（点火/崩塌/碎片化逆转）
- ε_eff < θ → 相变不发生（维持现状）

**2. 两轮碰撞共8个函数，有四个乘法归零结构，需要统一**

- S_silence = 1 - C × P（认知碰撞2）
- D_blind = ν × (1-π)（认知碰撞4）
- E_escalation = Π(1 - ε_ij)（社会碰撞4）
- Collapse(X) = σ(Λ) × (1-σ(R⁻)) × D/n（前一轮统一收敛）

四个乘法归零的统一形式：**F_collapse = 1 - Π(1 - f_i)**

- f_i是第i个因子的失效概率
- S_silence：f₁ = C_channel归零概率, f₂ = P_speak归零概率
- D_blind：f₁ = ν（误解概率）, f₂ = (1-π)（犹豫域消失概率）
- E_escalation：f_ij = ε_ij归零概率（每对关系一个因子）
- Collapse(X)：f₁ = Λ>0, f₂ = R⁻=0, f₃ = D/n=1

**统一含义：所有乘法归零结构都可以表示为"各因子非零概率的乘积"——系统存活的概率等于所有必要条件同时满足的概率。**

这与可靠性工程的串联系统公式完全同构：R_system = ΠR_i。点火框架的贡献是把这个结构从物理组件推广到了认知和社会维度。

**3. 最终统一：两个元函数**

两轮8个碰撞收敛为两个元函数：

**元函数1：相变函数 F_transition = σ(ε_eff - θ)**
- 描述"什么时候发生跳变"
- ε_eff是有效犹豫域，θ是临界阈值
- 所有sigmoid阈值过程都是这个元函数的特例

**元函数2：崩溃函数 F_collapse = 1 - Π(1 - f_i)**
- 描述"系统是否必然崩溃"
- f_i是第i个必要条件的失效概率
- 所有乘法归零结构都是这个元函数的特例

**两个元函数的关系：**

F_collapse描述稳态（系统在什么条件下必然崩溃），F_transition描述动态（系统在什么条件下发生相变跳出崩溃轨道）。

**点火 = F_transition触发，使系统从F_collapse=1的崩溃轨道跳到F_collapse<1的存活轨道。**

而F_transition能否触发，取决于ε_eff——犹豫域的宽度。ε_eff又受F_collapse中的因子影响：当表达通道归零（f_i=1），ε_eff→0，F_transition无法触发，系统锁定在崩溃轨道上。

**这就是点火框架最深层的闭环：崩溃函数锁死犹豫域 → 相变函数无法触发 → 系统无法跳出崩溃轨道 → 崩溃函数进一步恶化。**

函数化：**ε_eff(t+1) = ε_eff(t) × (1 - F_collapse(t))**

- F_collapse=0时：ε_eff不变（系统健康，犹豫域维持）
- F_collapse>0时：ε_eff被压缩（崩溃因子侵蚀犹豫域）
- F_collapse=1时：ε_eff→0（犹豫域完全消失，系统锁定）

代入F_transition：**F_transition(t+1) = σ(ε_eff(t) × (1 - F_collapse(t)) - θ)**

**这就是两轮8个碰撞的最终收敛——一个自洽的闭环动力学系统。**
