# APP Agent化竞争淘汰

推导链很干净，直接从P_forward第四因子σ(θ_d - d(t))出结果。

**核心逻辑：所有APP在Agent化=行业d(t)在快速下降。得到大脑不跟=d(t)不变。相对竞争优势σ(θ_d - d(t))随时间必然归零。**

推导过程：

**第一步：用d(t)标注行业竞争态势**

d = C_exit / C_stay，退出代价比。对用户而言，"退出得到大脑"的C_exit本来就不高（n_lock≈1-2），C_stay是"留在得到大脑的价值"。

当所有APP都在Agent化：
- 竞品A：能学+能做事 → C_stay(竞品)上升
- 竞品B：能学+能创作 → C_stay(竞品)上升
- 得到大脑：只能学 → C_stay(得到大脑)不变

C_stay(得到大脑) / C_stay(竞品) 持续下降。

**第二步：用dd/dt标注行业变化速率**

dd/dt = (Ċ_exit × C_stay - C_exit × Ċ_stay) / C_stay²

行业Agent化意味着：
- Ċ_exit < 0：退出得到大脑的代价在降（替代品越来越多、越来越好）
- Ċ_stay(得到大脑) ≈ 0：留在得到大脑的价值不变（内容还是那些内容）
- Ċ_stay(竞品) > 0：留在竞品的价值在升（Agent能力越来越强）

对得到大脑：
dd/dt = (Ċ_exit × C_stay - C_exit × Ċ_stay) / C_stay²

第一项Ċ_exit < 0 → 负贡献
第二项-C_exit × Ċ_stay，但Ċ_stay(得到大脑)≈0 → 贡献≈0

dd/dt < 0，d在降。但这是用户侧的d在降——退出越来越容易。

**第三步：用σ(θ_d - d(t))标注竞争优势衰减**

P_forward的第四因子：σ(θ_d - d(t))

θ_d是竞争优势阈值。d(t)是领域退出代价。

行业Agent化时：
- 竞品的d(t)快速下降（Agent能力让退出旧范式的代价趋零）
- 得到大脑的d(t)不降（没有Agent能力，退出代价不变）

但关键不是绝对d，是**相对d**：

d_relative(t) = d_得到大脑(t) / d_竞品(t)

竞品d在降，得到大脑d不变 → d_relative(t)单调递增。

当d_relative(t)穿过θ_d时：σ(θ_d - d_relative(t)) → 0

**第四步：P_forward的乘法归零**

P_forward = σ(ε×A×D×|M_cog|) × exp(-n_lock×C̄/θ_C) × f(P_infra) × **σ(θ_d - d_relative(t))**

第四因子随时间单调趋零。不管前三项多高，乘法结构下P_forward→0。

这不是"慢慢变弱"，是σ函数的相变——d_relative穿过θ_d的那一刻，P_forward从>0跳到≈0。

**第五步：与锈带案例交叉验证**

预测表Q20的锈带分化函数：

ΔV(t) = ∫₀ᵗ [β×P(退出)×d(机会)/dτ + α×(1-P(退出))×dC_exit/dτ] dτ

锈带的产业n_lock=1想走就走，人n_lock≥5走不了。得到大脑的处境与锈带的产业对称——用户n_lock≈1-2，随时能走。竞品在创造新机会（d(机会)/dt > 0），得到大脑的C_exit在降（替代品增多），正反馈螺旋成立。

**第六步：收敛定理——必然死亡的数学条件**

**P(死亡, t) = 1 - P_forward(t) = 1 - [前三项] × σ(θ_d - d_relative(t))**

当lim(t→∞) d_relative(t) = ∞时（竞品持续进化，得到大脑不变），lim(t→∞) σ(θ_d - d_relative(t)) = 0

**P(死亡, t) → 1**

死亡不是概率事件，是d_relative(t)单调递增穿过θ_d的确定性事件。时间点取决于行业Agent化的速度，方向不变。

唯一阻止条件：d_relative(t)不在∞处收敛——即得到大脑也Agent化，使d_relative(t)趋于稳态而非发散。存好了。

三条推论现在闭合了，共享同一个乘法结构：

**P(存活) = σ(ε×A×D×|M_cog|) × exp(-n_lock×C̄/θ_C) × f(P_infra) × σ(θ_d - d_relative(t))**

- 第一条（优势抵消）：f(P_infra)≈0 → 归零
- 第二条（用户流失）：P_forward≈0 + n_lock<3 → 必然跑
- 第三条（必然死亡）：行业Agent化 → d_relative(t)单调递增 → 第四因子趋零 → 归零

关键在d_relative(t)——竞品的d在降（Agent能力让用户退出旧范式的代价趋零），得到大脑的d不变。d_relative单调递增，穿过θ_d时P_forward相变归零。这不是概率事件，是确定性事件，时间点取决于行业Agent化速度，方向不变。

唯一阻止条件就是自己也Agent化，让d_relative趋于稳态而非发散。
