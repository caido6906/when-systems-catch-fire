# 门控与参数数学

"D102最核心的发现是门控和参数是两套数学"——展开这个命题本身。

---

## 展开：门控数学 vs 参数数学

**参数数学的特征：**
- 因子取值连续 ∈ (0, +∞) 或 (0, 1)
- 改善任何因子都有效，效果与当前值和边际贡献相关
- 乘法系统中改善最弱因子效果最大（D87）
- 不存在"否决"——没有哪个因子能让系统瞬间归零
- 代表函数：T2乘法归零律（理论上有归零，但连续系统中因子=0是边界条件不是常态）

**门控数学的特征：**
- 因子取值离散 ∈ {0, 1}，过渡区极窄
- 改善已开(=1)的因子无效，改善已关(=0)的因子从0→1是质变
- 乘法系统中关着的门否决一切（D102）
- 存在"否决"——一个因子=0让系统瞬间归零
- 代表函数：D101多维门关闭、D102最弱门否决

**两套数学的统一：sigmoid是桥梁**

σ(x) = 1/(1+e^(-x))把两套数学连起来了——

当x >> 0时，σ(x) ≈ 1（门开，参数域）
当x << 0时，σ(x) ≈ 0（门关，门控域）
当x ≈ 0时，σ(x)在(0,1)之间过渡（混合域）

**关键：sigmoid的过渡区宽度由什么决定？**

σ'(x)的最大值在x=0处，σ'(0) = 0.25。σ从0.1到0.9的过渡区间约为x∈(-2.2, 2.2)，宽度4.4。

在门控场景中，x = εᵢ - θC(i)。过渡区宽度4.4意味着εᵢ在θC(i)±2.2范围内时系统处于混合域——既不是纯门控也不是纯参数。

**过渡区宽度函数：**

w_transition = 4.4 / α，其中α为sigmoid的陡度参数

α→∞时w→0，sigmoid退化为step函数，纯门控数学
α→0时w→∞，sigmoid退化为线性，纯参数数学

**现实系统的α值：**

- 心理阈值（如εaware）：α大，过渡窄，接近门控——"要么想通要么没想通"
- 经济阈值（如εeconomic）：α中等，过渡中等——收入连续变化但某些临界点有质变
- 物理阈值（如εgeographic）：α小，过渡宽，接近参数——距离连续变化

**两套数学的精确边界：**

定义门控度：φ(i) = α(i) / (α(i) + α_ref)

α_ref为参考陡度。φ→1时纯门控（D102适用），φ→0时纯参数（D87适用），0<φ<1时混合。

**混合域的行为：**

在混合域中，改善最弱因子的效果介于D87和D102之间：

ΔG/G ≈ φ × (否决式：0→1质变) + (1-φ) × (渐变式：边际改善)

当φ=0.7（如心理维度），70%是否决式、30%是渐变式——改善最弱因子有质变效果但不是绝对否决。

**与已有函数的全面对接：**

| 已有函数 | 适用域 | φ值 |
|---|---|---|
| D87 乘法临界漂移 | 纯参数 | φ≈0 |
| D102 最弱门否决 | 纯门控 | φ≈1 |
| D96 统一门函数 | 混合 | 0<φ<1 |
| D99 门槛降低放大 | 过渡区 | φ取决于α |
| D100 双阈值生存域 | 门控+参数 | φ_low≈1, φ_high≈0.5 |
| D101 多维门关闭级联 | 门控主导 | φ≈0.8-1 |

**深层发现：门控是参数的极限**

当参数系统中某个因子逼近0时，改善它的效果趋近无穷大（D87的边际贡献→∞）。这个极限就是门控——因子=0时改善从0→ε的效果=∞/0=质变。

**数学表达：**

lim(εᵢ→0) [∂G/∂εᵢ] / G = lim(εᵢ→0) 1/εᵢ = ∞

参数改善的边际效应在因子→0时发散，发散点就是门控的否决点。门控不是另一套数学，是参数数学在因子=0处的奇点。

**但这不是数学技巧，有物理含义：**

奇点对应相变。水在100°C附近比热发散，是液态→气态的相变点。εᵢ→0时边际效应发散，是"有退出权"→"无退出权"的相变点。门控数学描述的就是这些相变点上的行为。

**收敛为函数：**

**D103 门控-参数统一函数**

φ(i) = α(i)/(α(i) + α_ref)

G_unified(i) = φ(i) × gate(i) × exp(-λ×Σ(1-gate(j))) + (1-φ(i)) × σ(εᵢ - θC(i))

门控度φ由sigmoid陡度α决定。φ→1时退化为D102纯门控否决，φ→0时退化为D87纯参数渐变，0<φ<1时混合。

核心结论：
1. **门控是参数在因子→0处的奇点**——不是两套独立的数学，是同一套数学在不同区间的行为
2. **sigmoid是桥梁**——α→∞退化为step（纯门控），α→0退化为线性（纯参数）
3. **门控度φ可量化**——心理维度φ高（"想通"是质变），物理维度φ低（距离是渐变）
4. **混合域行为是加权混合**——φ×否决式 + (1-φ)×渐变式
5. **门控=相变**——因子=0处的奇点对应退出权的相变，不是数学技巧是物理现实

函数163条，案例397个跑完了。最核心的发现：**门控不是另一套数学，是参数数学在因子→0处的奇点。**

lim(ε→0) 边际效应 = ∞，这个发散点就是门控的否决点。就像水在100°C比热发散是液态→气态的相变，ε→0时边际效应发散是"有退出权"→"无退出权"的相变。门控数学描述的就是相变点上的行为。

sigmoid是桥梁——α→∞时退化为step函数（纯门控），α→0时退化为线性（纯参数），中间是混合域。门控度φ把D87和D102统一了：φ=0是D87的参数渐变，φ=1是D102的门控否决，0<φ<1是加权混合。

这也解释了为什么心理维度"想通"是质变（φ≈0.8），而距离维度是渐变（φ≈0.1）——不是维度本质不同，是sigmoid陡度不同，同一套数学在不同α值下的行为。
