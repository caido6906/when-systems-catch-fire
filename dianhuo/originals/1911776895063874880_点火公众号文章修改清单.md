# 点火公众号文章修改清单

昨天看到一篇新闻，乌克兰拿无人机打俄罗斯，普京说是恐袭。我倒是在想更深层的一个问题，为什么中国恐袭这么少？更进一步，为什么中国完成了对外来族群的同化，这数千年间发生了些什么，哪些关键人物在哪些关键时期做了哪些关键动作，影响了今日之中国？
→ 为什么有的文明扛得住冲击、同化得了外来者，有的扛不住？这个问题的答案藏在华夏几千年的系统重写里。这数千年间发生了些什么，哪些关键人物在哪些关键时期做了哪些关键动作，影响了今日之中国？

答案是，中国历史上曾发生了七次文明上的转身，这每一次都面临前一个系统的崩溃，而不得不有一个人站出来，做出了关键的推动，而被统治者，或者说被这个推动所影响的人，得到了他们认可的利益，并将这一系统固化下来，如是循环。
→ 答案是，中国历史上曾发生了七次文明上的转身。每一次都伴随旧系统的崩溃，需要关键人物站出来推动重构。而被推动所影响的人，从中获得了他们认可的利益，并将这一系统固化下来。七次，如是循环。

同一个动作在不同尺度上一模一样地重复自身。
→ 同一套逻辑，在文明、国家、公司不同尺度上反复重演。

为什么空降 CEO 到任，改了制度、改了流程、改了 KPI，但团队就是不买账？所有人都知道改变，因为制度搭好了，但没人觉得我应该住在这间房子里。
→ 秦朝和空降CEO的案例顺序对调。先讲秦朝，再讲空降CEO。公众号版先抛出"为什么空降CEO不买账"再讲秦朝，但秦朝才是主案例，空降CEO是穿插。先秦后CEO，读者从历史画面进入更容易代入。

一个帝国一个公司相隔 2000 年，崩溃的方式一模一样，因为他们只搭了房子，没有点火。
→ 一个帝国一个公司相隔 2000 年，崩溃的方式一模一样，因为他们只搭了房子，没有点火。（这句保留，但位置跟着案例顺序调整移到秦朝+CEO之后）

如果一个系统只搭房不点火，制度再完备也像秦朝，15 年就塌。而如果一个系统只点火不搭房。认知变了行动却无法落地。一个人裸辞后更焦虑，想法有了，日子没着落。
→ 如果一个系统只搭房不点火，制度再完备也像秦朝，时间长就塌。而如果一个系统只点火不搭房，认知变了行动却无法落地，戈尔巴乔夫只拆旧认同、不建新认同，苏联直接解体。（裸辞比喻太轻，跟前后文的文明尺度案例不匹配。戈尔巴乔夫是同一框架下的真实案例，力度够。裸辞可以放到第八章个人格子里用。）

周公制礼，异姓诸侯从战俘变贵族，礼乐秩序让所有人愿意留下，八百年基业认同建立在参与者身上，系统稳固。
→ 周公没杀战败的异姓诸侯，反而封了他们——不是仁慈，是算过账，让他们留下比杀掉成本低。认同建在参与者身上，八百年基业。（原文"礼乐秩序让所有人愿意留下"是结论，不是画面。改成"不是仁慈，是算过账"，从动机切入，有动作感。）

凯末尔西化，城市精英接住了，农村没接住，100 年后，宗教认同回潮，认同只覆盖了部分人，维持断裂。
→ 凯末尔西化，城市精英接住了，农村没接住。认同只覆盖了部分人，系统维持断裂。百年后，传统宗教认同回潮。（把"100年后"从句中拆出来独立成句，节奏更好，"传统宗教认同回潮"比"宗教认同回潮"多一层——是旧认同压过新认同，不是宗教突然出现。）

唐朝认同锚定在万邦来朝，安史之乱暴露裂缝，外部参照系一变，认同内爆。但唐朝硬转了过来，又撑了 150 年。认同建立在外部，方向歪了，但不是死刑。
→ 唐朝认同锚定在"万邦来朝"，安史之乱一来，外部参照破碎。但唐朝硬转了过来——认同从"万邦来朝"转向"内部整合"，藩镇自己也在用华夏系统运转。又撑了一百五十年。认同建在外部，方向歪了，但不是死刑。（原文缺了"转向"这个关键动作——唐朝不是硬扛过去的，是转了方向。加上转向的具体内容，读者才看得见机制。）

卡拉卡拉普发罗马公民权，旧认同拆了，新的没人认同，西罗马一路滑到底，认同建立在对旧系统的否定上，系统崩盘。
→ 卡拉卡拉普发罗马公民权，旧认同拆了，新的没人认。西罗马一路滑到底。认同建立在对旧系统的否定上，地基是空的，系统崩盘。（"没人认同"改成"没人认"，更口语。"地基是空的"补上了为什么否定旧系统就必然崩盘——光拆不建，底下没东西。）

戈尔巴乔夫先否定旧体制，新系统还没搭出来，苏联直接崩溃。旧认同先拆了，新的还未建立，点火只是烧了自己的房子。
→ 戈尔巴乔夫先否定旧体制，新系统还没搭出来，苏联直接崩溃。旧认同拆了，新的还没建，点火变成了烧自己的房子。（"还未建立"→"还没建"，更短更利落。"只是"→"变成了"，语气更确定。）

五种方向面临同一个问题，认同建立在何处？
→ 四种方向加上"建在参与者身上"这个正例，面对的是同一个问题：认同建在哪里？（原文"五种方向"把正例混在一起了。周公是正例，其他四种是问题方向。分开说更清晰。）

而五种方向里，有一条线把所有案例劈成了两个世界，即，应约者有没有退出权。
→ 回头看刚才的案例——周公封了战败者，他们可以走但选择留下；秦朝压住六国遗民，他们走不了也从未认同；凯末尔的农村没接住，但也没人走得掉；唐朝硬转过来，藩镇可以叛但选择用华夏系统运转；卡拉卡拉拆了旧认同，罗马人想走也无处可去。四种方向里，有三种最终都指向同一个问题——留下来的人，到底是愿意还是走不了？有一条线把所有案例劈成两个世界：走得了还是走不了。（原文"即，应约者有没有退出权"是突然插概念。从已有案例里长出来，读者才不愣。先带读者回头看一遍案例，再点破这把尺。）

公司里也一样，文化驱动的团队扛得住低谷，KPI 驱动的团队考核一松就散。
→ 把镜头拉近三十年，同样的逻辑在另一个尺度上重复：文化驱动的团队扛得住低谷，KPI驱动的团队考核一松就散。（"公司里也一样"太随意。改成"把镜头拉近三十年"，给读者一个尺度切换的体感，跟前面文明尺度的叙事衔接更顺。）

五种方向最关键的一个字是趋势，而不是命运。
→ 四种方向加上正例，最关键的一个字是趋势，而不是命运。（同上，正例分开说。）

这意味着这个框架事不只是后视镜。认同如果建立在个人权威上，提议者退场即火灭。张居正改革，人走火灭。
→ 这意味着这个框架不只是后视镜。认同如果建立在个人权威上，提议者退场即火灭。张居正改革依托个人权威推行，人一走，火就灭了。（"这个框架事"多了个"事"字，删掉。张居正那句补了"依托个人权威推行"，把逻辑链闭合——为什么人走火灭，因为火本来就只靠人撑着。）

认同建在对旧系统的否定上，负时间差必崩溃。典型例子是戈尔巴乔夫。
→ 认同建在对旧系统的否定上，负时间差必崩溃。典型例子是戈尔巴乔夫。（这句跟前面戈尔巴乔夫案例重复，可以考虑删掉或换成别的例子，比如法国大革命初期的恐怖统治，避免同一个案例出现三次。）

而华夏前七次转身，当世人不知道自己在参与一次系统重写，事后才被命名。第九次不一样，这是第一次转身还在进行中就被识别出来。
→ 而华夏前七次转身，当世人不知道自己在参与一次系统重写，事后才被命名。而当下，我们可能正站在另一次转身的中间——这是第一次，转身还在进行中就被识别出来。（"第九次"在引言里突然出现，读者不知道前八次是什么。改成"另一次转身的中间"，不编号，留悬念。）

禅定时刻，请你相信我说的每句话都是错的。
→ 删掉。引言最后一句话应该让读者想翻下一页，不是让他怀疑刚读完的东西。这句放到正文某章结尾更合适。
