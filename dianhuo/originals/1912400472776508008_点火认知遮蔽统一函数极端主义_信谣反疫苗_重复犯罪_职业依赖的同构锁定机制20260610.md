# 点火｜认知遮蔽统一函数：极端主义·信谣反疫苗·重复犯罪·职业依赖的同构锁定机制（2026.06.10）

## 核心论点

极端主义、信谣反疫苗、重复犯罪、职业依赖是认知遮蔽的四种形态。表面不同，底层同构：认知操作系统被寄生体劫持后，焊死入口（D_alt→0）、焊死出口（ε→0）、改写标记（θ_align反转）。宿主不是"选择相信"，是"只能相信"。

## 一、极端主义的信道结构

C_extreme(L) = B_extreme × D_extreme(L) × θ_align(L) × E_extreme

**B_extreme = B_promise × f(despair)**
承诺不是信号，是期货。绝望程度越大，承诺的感知强度越大。"奶与蜜"的解码不是"这有没有证据"，是"我还有没有别的选项"。

**D_extreme = 1**（二元框架：我们vs他们、圣洁vs污秽，解码门槛为零）

**E_extreme = α_fear × α_anger + α_meaning**
α_meaning 是极端主义独有的——不只放大情绪，还提供"你的痛苦有意义"的叙事。普通谣言没有这个增幅器。

**退出成本陷阱：**
C_exit(S_extreme, L) = C_social + C_identity + C_existential
- C_social — 失去唯一社交网络
- C_identity — 承认过去相信的一切都是错的，身份归零
- C_existential — "奶与蜜是假的"→ 死亡是终结→ 存在性恐惧

C_exit → ∞ ⟹ ε → 0 ⟹ 永久锁定。不是"不愿意退出"，是"退出成本在结构上不可能承受"。

## 二、信谣反疫苗——双重寄生与敌意标记

"全世界政府联合控制人民"不只占用信道，还把专业信源标记为敌意信源：
θ_align(expert, L) → -1

不是"听不懂专家"（D≈0），而是"专家是敌人"（θ_align变负）。专业信道不仅关闭，还被标记为威胁，F_form暴增：
F_form^post-rumor(L) = F_form^baseline(L) + ΔF_form^hostility

茧房刚性从"进不去"升级为"主动拒绝"。

## 三、重复犯罪与职业依赖——身份锁定

身份锁定函数：
L_identity(L, t) = P_accept(S_current_identity, L) / P_accept(S_alternative_identity, L)

L_identity → ∞ = 完全锁定；L_identity ≈ 1 = 身份流动性

**重复犯罪者的信道：**
- P_accept(crime_identity) ≈ 1：D=1（"我只能这样"解码门槛为零），θ_align=1（命中"我没别的出路"），ε≈0（"我学不会别的"=退出成本∞）
- P_accept(alternative) ≈ 0：D_alt≈0（"我学不会"），θ_align_alt≈0（替代叙事不对齐），F_alt高（社会偏见=阻力）

**职业依赖完全同构：** "我就是干这个的" = D_alt≈0, θ_align_alt≈0, ε≈0。

## 四、统一遮蔽函数

H_lock(L, t) = 1 - [D_alt(L) × ε(L) × θ_align(alt, L)] / [C_exit(L) × F_form(L)]

分子 = 打破遮蔽的能力（解码×退出权×对齐度）
分母 = 维持遮蔽的力量（退出成本×形式化阻力）

H_lock ∈ [0, 1]。分子任意一项为0或分母趋向∞，H_lock → 1。

**四种形态的H_lock值：**
- 极端主义：D_alt≈0, ε≈0, C_exit→∞ → H_lock≈1
- 信谣反疫苗：θ_align(expert)→-1, F_form暴增 → H_lock≈1
- 重复犯罪：D_alt≈0, ε≈0, C_exit高 → H_lock≈1
- 职业依赖：D_alt≈0, ε低, C_exit中等 → H_lock≈0.6-0.8

## 五、寄生体劫持的三步操作

1. 焊死入口（D_alt → 0，替代路径不可解码）
2. 焊死出口（ε → 0，退出成本无穷大）
3. 改写标记（θ_align反转，专业信源标记为敌意）

三步完成后，宿主不再是"选择相信"——是"只能相信"。不是态度问题，是信道结构问题。

## 六、打破遮蔽——插入新认知的条件

H_lock < 1 ⟺ D_alt(L) × ε(L) × θ_align(alt, L) > 0

必要条件：分子三项全部>0。任何一项为0，遮蔽不可打破。

**四条路径：**

### 1. 装D_alt（让替代路径可解码）——前提
- 极端主义：用他自己的语言讲"退出的人后来怎样了"——故事形式、D=1
- 重复犯罪：用他能理解的具体操作——"有人跟你一样出来后做了XX，第一步是这样"
- 职业依赖：用已有技能的可迁移映射——"你做XX时的那个能力，在YY领域值钱"

### 2. 装ε（安装退出权）——制动器
退出权不能从外部赋予，必须从内部生长：
- 极端主义：ε_sense的种子是身体信号——让他重新听见"这不对"
- 重复犯罪：ε的种子是微小成功体验——"我做成了这件事"→"我可能不只是那样的人"
- 信谣反疫苗：ε的种子是身边信任的人的亲身经历——"我打了疫苗没事"走D=1信道+退出许可

### 3. 降C_exit（降低退出成本）——加速器
- 极端主义：提供退出后的社交网络替代（否则退出=社会性死亡）
- 重复犯罪：消除前科标签的制度性障碍（否则退出=经济性死亡）
- 职业依赖：提供转型期经济缓冲（否则退出=生存性死亡）

### 4. 降F_form（降低对替代信源的防御）——加速器
- 信谣反疫苗：专业信源放弃权威姿态——"我也曾经不确定，后来看到这些数据"走D=1+低F_form

**依赖关系：装D_alt（前提）→ 装ε（制动）→ 降C_exit + 降F_form（加速器）**

## 七、统一打破遮蔽函数

P_unlock(L) = D_alt(L) × ε(L) × [θ_align(alt, L) / (C_exit(L) × F_form(L))]

四因子乘法关系，与所有统一函数完全同构：

| 函数 | 表达 | 场景 |
|------|------|------|
| P(L₃→L₅) | σ × A × D × \|M_cog\| | 层级跳变 |
| P_break | D × ε × (θ_align / F_form) | 茧房破壁 |
| P_unlock | D_alt × ε × (θ_align / C_exit × F_form) | 认知遮蔽解锁 |

结构完全一致：D（解码）× ε（退出权）× (对齐度 / 阻力)。任何一个为0，整体为0。
