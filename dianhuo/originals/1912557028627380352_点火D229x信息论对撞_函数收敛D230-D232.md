## D230 双通道信息衰减定理

每个门控面提供的信息量（D197）：

Hᵢ = ½ln(2πeσᵢ²)（高斯门控的微分熵）

总信息量：
I_total = Σᵢ Hᵢ = Σᵢ ½ln(2πeσᵢ²)

值衰减通道：门控面消失 → 某些Hᵢ→0 → I_total中对应项归零
精度衰减通道：σᵢ递减 → 每个Hᵢ递减 → I_total中每项的值减小

dI/dt = Σᵢ dHᵢ/dt = Σᵢ (1/σᵢ) × dσᵢ/dt

值衰减贡献：门控面i消失时Hᵢ→0，贡献一个负脉冲
精度衰减贡献：dσᵢ/dt < 0 → dHᵢ/dt < 0，连续负贡献

**dI/dt ≤ 0**（宇宙总信息量单调递减）

与双通道衰减的对应：
| 衰减通道 | Φ框架 | 信息论 |
|---------|-------|--------|
| 值衰减 | g→0（门控贡献归零） | Hᵢ→0（信息源消失） |
| 精度衰减 | σ→0（门控退化） | Hᵢ递减（信息精度降低） |
| 双通道终态 | Φ→0, σ→0 | I→0（完全无信息） |

核心定理：双通道衰减在信息论下统一为信息衰减律dI/dt≤0。值衰减=信息源消失（可区分的状态减少），精度衰减=信息精度降低（每个信息源的分辨率降低）。宇宙的总信息量单调递减——不是信息被"擦除"（Landauer），是信息源在消失和退化。

与Landauer原理的关系：Landauer说擦除1比特产生kTln2热量——这是信息→热的转换。D230说的是更基本的过程：信息本身在衰减，不需要"擦除"操作。门控面消失=信息源消失=不需要擦除，信息自然归零。

层级：定理

## D231 信息-热力学-门控三统一定理

三条衰减律的等价性：

1. 热力学第二定律：dS/dt ≥ 0（熵增）
2. Φ衰减律（D222）：dΦ/dt ≤ 0（门控贡献递减）
3. 信息衰减律（D230）：dI/dt ≤ 0（信息量递减）

三者的关系：

S = -Σᵢ pᵢ ln(pᵢ)（Shannon熵，pᵢ为系统处于态i的概率）
I = Σᵢ Hᵢ = Σᵢ ½ln(2πeσᵢ²)（门控信息量）
Φ = Σᵢ gᵢ = Σᵢ exp[-(ln(μ/Λᵢ))²/(2σᵢ²)]（门控贡献）

在门控面消失时：gᵢ→0 → Hᵢ→0 → 对应态的pᵢ→均匀 → S增大

更精确：dS/dt = -Σᵢ dpᵢ/dt × ln(pᵢ)（熵增来自概率分布的展平）
门控面消失 → 区分能力降低 → 概率分布展平 → 熵增

**dΦ/dt ≤ 0 ⟺ dI/dt ≤ 0 ⟺ dS/dt ≥ 0**

三条定律是同一过程的三个面：
- 热力学面：约束减少→概率展平→熵增
- 门控面：门控贡献归零→Φ递减→Ω趋向1
- 信息面：信息源消失/退化→I递减→可区分态减少

核心定理：热力学第二定律、Φ衰减律、信息衰减律三者等价——它们是同一物理过程（约束消失→结构瓦解→物理趋向终结）的三个数学表述。热力学看到了熵增，门控框架看到了Φ递减，信息论看到了I递减——它们看到的是同一件事。

层级：定理

## D232 信息守恒-衰减悖论与黑洞

量子力学要求信息守恒（么正演化）：封闭系统的I不变
D230说宇宙的I单调递减：dI/dt≤0

矛盾？不矛盾。

封闭系统：dI/dt = 0（量子力学，么正演化）
膨胀宇宙：dI/dt ≤ 0（D230，开放系统）

区别：封闭系统的相空间不随时间变化，膨胀宇宙的相空间在增长（新自由度出现）但门控面不增长（没有新门槛产生）→ 自由度增加但区分能力不增加 → 信息密度降低

黑洞的信息悖论在Φ框架下：

黑洞：μ>>M_Planck → g_B = exp[-(ln(μ/M_Planck))²/(2σ²)] → 0
→ 引力门控贡献归零 → 值衰减终态（D229）
→ H_grav → 0 → 引力信息源消失

但黑洞有Bekenstein熵：S_BH = A/(4l_Planck²) = 4πM²（自然单位）
这个熵对应什么信息？

在Φ框架下：黑洞的Bekenstein熵 = 引力门控面消失前"记住"的信息
= 门控贡献归零前的"残余信息"
= g_B从非零衰减到零的过程中释放的信息量

Hawking辐射 = 门控贡献归零的渐进过程：
g_B(t) → 0，释放的信息 ∝ |dg_B/dt| ∝ g_B × |ln(μ/M_Planck)|/(σ²) × |dμ/dt|

信息释放率 = |dH_grav/dt| = |dg_B/dt|/g_B × H_grav

核心定理：量子力学的信息守恒和D230的信息衰减律不矛盾——前者适用于封闭系统，后者适用于膨胀宇宙。黑洞的信息悖论在Φ框架下重新表述：黑洞=引力门控面的值衰减终态，Hawking辐射=门控贡献归零过程中释放的残余信息。Bekenstein熵=门控面消失前的"信息记忆"。

层级：定理
