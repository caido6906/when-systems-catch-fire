# 点火｜信源选择的层级函数（2026.06.10）

# 点火｜信源选择的层级函数：为什么普通人信身边人，高认知者信专业人士（2026.06.10）

## 核心问题

为什么多数人信身边人（P_local）不信专业人士（P_expert），而高认知者反过来？

## 建模

### 接受概率函数

P_accept(S, L) = C(S, L) / [C(S, L) + R(S, L)]

S = 信源（S_local 或 S_expert），L = 接收者认知层级（L₁-L₅）

### 感知可信度 C(S, L) = B(S) × D(S, L) × θ_align(S, L)

- **B(S)** — 信源基础信号强度。专业人士 B 值高（有数据、有方法），身边人 B 值低。B 是客观属性，能否被解码取决于 D。
- **D(S, L)** — 解码能力，∝ σ(L)（感知锐化函数）。L₁-L₂ 对专业信号 D → 0（听不懂、无法验证），对身边人信号 D → 1（语言同构、经验可触）。
- **θ_align(S, L)** — 认知对齐度。信源表述框架与接收者既有认知框架的重合度。身边人用日常语言，θ_align ≈ 1；专业人士用学科框架，与 L₁-L₂ 的 θ_align ≈ 0。

### 感知阻力 R(S, L) = F_form(S, L) + ε(S, L)

- **F_form** — 形式化服从惯性。L₁-L₂ 对学科规范有强宜人性服从，专业权威姿态反而触发防御，F_form 增大；身边人无权威姿态，F_form ≈ 0。
- **ε** — 退出权信号。L₁-L₂ 几乎没有 ε，无法退出身边人共识的信息茧房；L₄-L₅ ε 充足，能主动切换信源。

## 代入结果

### 普通人（L₁-L₂）接受身边人 ≈ 1

信号虽弱但全部可解码（D=1）、零阻力（F=0）、无退出能力（ε≈0）——几乎全盘接受。

### 普通人（L₁-L₂）接受专业人士 ≈ 0

信号虽强但无法解码（D≈0）、高阻力（F高）、无退出能力（ε≈0）——几乎全盘拒绝。

### 高认知者（L₄-L₅）反转

D 升高（能解码专业信号）、F_form 降低（不再宜人性服从）、ε 充足（有退出身边人共识的能力）→ P_expert > P_local。

## 统一函数

ΔP = P_expert(L) - P_local(L)

ΔP 的符号由 L 决定：
- L₁-L₂：ΔP < 0（信身边人）
- L₃：ΔP ≈ 0（摇摆区）
- L₄-L₅：ΔP > 0（信专业人士）

## 两个核心发现

### 1. 转折点在 L₃

L₃ 的摇摆本质是 D 和 ε 的联合临界——D 刚够解码专业信号但不够稳定，ε 刚够退出身边人共识但不够坚决。这与 θ₂ 层间跳变阈值的定义完全对齐。

### 2. "别人说"是信道劫持

"别人说"作为经典开头，本质是把 B_local 从低值人为抬高——用"别人"这个虚拟群体背书制造伪信号增强，对 D(L₁-L₂) = 1 的信道做信号注入。这不是信息传递，是信道劫持。
