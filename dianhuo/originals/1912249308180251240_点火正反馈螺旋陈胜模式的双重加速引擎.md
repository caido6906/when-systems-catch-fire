# 点火｜正反馈螺旋：陈胜模式的双重加速引擎

来源：系统动力学视角下反馈回路极性分析中深挖。

## 双重正反馈引擎

回路1（R_perceived→I）和回路5（应约者状态→R_perceived）可以咬合成双重正反馈。

螺旋启动：一个人退出并被看见（信号源出现）→ R_perceived跳变 → 周围人意识到"原来我也可以走" → I从0跳到1 → 更多有意识的人选择退出 → 退出行为本身变成更强的信号源 → R_perceived继续升高 → 更多人意识到……

两个加速器叠在一起：
- 认知加速器：R_perceived升高→I觉醒→退出行为更有目的性（不是盲目逃跑，是有意识地选择离开）→退出行为更有组织→信号更强
- 社会加速器：应约者状态从消极变积极→参与感强化→R_perceived继续升高→更多人从消极变积极→社会网络密度增大→横向激活速率加快

两个加速器叠加 = 指数传播，不是线性传播。陈胜一喊几个月传遍全国就是这个机制。

## 螺旋的自然极限

R_perceived ≤ R。当R_perceived追上R时，差值归零，激活动力消失，螺旋停止。正反馈螺旋不会无限加速，天花板就是R本身。

## 螺旋的人工极限

惩罚机制重新生效。螺旋启动后如果惩罚迅速介入（杀鸡儆猴成功），R_perceived被压回去，螺旋中断。惩罚机制不是防止螺旋启动的（防止不了，信号源出现是随机的），而是在螺旋启动后抢时间——在正反馈加速到不可控之前把R_perceived压回去。

这解释了为什么有些系统能在陈胜式事件后存活（惩罚机制抢到了时间，如唐朝安史之乱后虽然大乱但系统没崩），有些系统崩了（惩罚机制没抢到时间，如秦末和苏联末期）。

## 与退化传播方向的关系

正反馈螺旋的方向是从底层向上引爆（横向激活绕过层级），退化传播方向是从顶层向下传染（I重置→Posture翻转→P退化→R降级）。两个方向相反——退化是从上往下的慢性病，正反馈螺旋是从下往上的急性发作。一个系统可以同时存在两种过程：顶层在退化，底层在积累势能。两种过程在中间层相遇时，就是最危险的时刻。
