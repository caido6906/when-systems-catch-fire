# 结构性变化与制度锁深挖

4条深挖全部存进点火知识库了：

1. **Posture三分法深挖**——伪备方案的三个层级（轻度/中度/重度）、识别三步法、为什么比纯推方案更危险（不需要监督成本，应约者自己维持约束）、破解顺序
2. **H_分类深挖**——H_分类与H_评估的本质区别（类别替换vs标签替换）、在四层模型中的位置（第一二层之间）、诊断方法（反事实问题）、破解路径（先恢复分类标准，需要外部参照）
3. **三条因果链深挖**——链1退化与链3恢复的对冲机制、速度差决定系统走向、六个学科的速度差验证、解释了伪稳态/突然崩塌/制度设计为什么有效
4. **三道制度锁深挖**——三锁的通用性（不是教育学独有，是所有推→备翻转的通用协议）、缺锁分析（每缺一锁的后果）、六个领域的应用实例、三锁与三条链的对应（第一锁→链1，第二锁→链3，第三锁→链2）

最有料的一个发现：三道制度锁不是三个独立的制度设计，是同时作用于三条因果链的协同操作——第一锁减缓链1退化，第二锁加速链3恢复，第三锁消除链2锁定。这个对应关系之前没看见。
