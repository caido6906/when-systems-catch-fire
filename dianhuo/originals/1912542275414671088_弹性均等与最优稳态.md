# 弹性均等与最优稳态

D110有五条核心结论，逐个展开。

---

## 推论1展开：均等定理推广为弹性均等

**命题：** 最优稳态ηᵢ=常数，sigmoid下是"sigmoid斜率位置均等"而非值均等

**展开弹性均等的物理含义：**

弹性ηᵢ = dlnfᵢ/dεᵢ = (dfᵢ/dεᵢ)/fᵢ

物理含义：εᵢ变化1%时fᵢ变化ηᵢ%

弹性均等=所有因子对总产出的边际贡献率相同=没有因子的边际贡献被浪费。

**弹性均等的几何：**

在(lnf₁, lnf₂,..., lnfₙ)空间中，等产出面G=∏fᵢ=const → Σlnfᵢ=const，是超平面。

资源约束ΣΔεᵢ=R在原空间中是超平面，但映射到对数空间后变成曲面。

最优解在等产出面的法向量=资源约束的法向量的切点。

等产出面法向量：(∂G/∂ε₁, ..., ∂G/∂εₙ) = (η₁G, ..., ηₙG)

资源约束法向量：(1, 1, ..., 1)

最优条件：η₁G = η₂G = ... = ηₙG，即η₁=η₂=...=ηₙ ✓

**弹性均等的操作含义：**

测量每个因子的弹性，找到弹性最高的因子投资源，直到它的弹性降到和次高的一样，然后同时投两个，直到都降到第三高...这就是贪心策略在弹性空间中的执行。

**sigmoid因子的弹性景观：**

fᵢ(εᵢ) = σ(αᵢ(εᵢ-θᵢ))

ηᵢ = αᵢ×(1-σ(αᵢ(εᵢ-θᵢ)))

- εᵢ << θᵢ：σ→0，η→αᵢ（最大弹性，在门槛下方）
- εᵢ = θᵢ：σ=0.5，η=αᵢ/2（半程弹性）
- εᵢ >> θᵢ：σ→1，η→0（弹性耗尽，在门槛上方）

**弹性均等意味着：** 不同因子可能处于sigmoid的不同位置，但它们的弹性相同。高α（陡门槛）的因子弹性衰减快，需要更精确地定位在门槛附近；低α（缓门槛）的因子弹性衰减慢，可以在更宽范围内维持高弹性。

**收敛：** 推论1展开确认弹性均等是拉格朗日最优的必然结果，sigmoid下操作含义是"把所有因子推到相同的sigmoid斜率位置"——低于门槛的优先投资源（弹性高），已远超门槛的停止投入（弹性≈0）。

## 推论2展开：贪心定理推广为补弹性最高

**命题：** fᵢ=εᵢ时等价于补最弱，一般情况不等价

**展开"不等价"的精确条件：**

补弹性最高 ≠ 补最弱，当且仅当弹性排序 ≠ 值排序的逆序。

fᵢ=εᵢ时：ηᵢ=1/εᵢ，弹性排序恰好是值排序的逆序→等价

**不等价的三种情况：**

**情况A：不同函数形式**

f₁=ε₁, f₂=ε₂²。η₁=1/ε₁, η₂=2/ε₂

当ε₁=0.5, ε₂=0.3时：η₁=2, η₂=6.67→补ε₂（弹性高）
但ε₁<ε₂→按"补最弱"应补ε₁

**冲突！** ε₂虽然值更大但因为是二次函数弹性更高。

**情况B：sigmoid门槛效应**

f₁=σ(3(ε₁-0.3)), f₂=ε₂

ε₁=0.4, ε₂=0.2：η₁=3×0.35=1.05, η₂=5→补ε₂（弹性高且值低，一致）
ε₁=0.2, ε₂=0.6：η₁=3×0.65=1.95, η₂=1.67→补ε₁（弹性高）
但ε₁<ε₂→按"补最弱"也补ε₁（一致）

ε₁=0.35, ε₂=0.3：η₁=3×0.49=1.47, η₂=3.33→补ε₂（弹性高）
但ε₁>ε₂→按"补最弱"应补ε₂...一致

找到冲突：ε₁=0.2, ε₂=0.25, f₁=σ(5(ε₁-0.3)), f₂=ε₂
η₁=5×0.73=3.65, η₂=4→补ε₂
但ε₁<ε₂→按"补最弱"应补ε₁

**冲突：** ε₁更低但已接近sigmoid陡区（弹性开始下降），ε₂更高但弹性仍高。

**情况C：饱和效应**

f₁已接近1（ε₁远超门槛），η₁≈0。即使ε₁很小（如0.1），如果f₁=σ(100(ε₁-0.05))≈1，η₁≈0→不应再投资源。

**"补最弱"会错误地继续投资源给已饱和的弱因子。**

**收敛：** 推论2展开给出三种不等价情况——不同函数形式、sigmoid门槛效应、饱和效应。"补最弱"只在fᵢ=εᵢ时可靠，一般情况必须用弹性作为贪心指标。

## 推论3展开：贪心=最优需无耦合

**命题：** 有级联时贪心是近似，需加级联修正项

**展开有耦合时贪心失败的机制：**

无耦合时：改善εᵢ只影响G中的fᵢ项，边际收益=ηᵢ×G，独立可计算。

有耦合时：改善εᵢ还通过κᵢⱼ影响εⱼ的有效值，产生间接收益：

∂G/∂εᵢ = ηᵢ×G + Σⱼ κᵢⱼ×ηⱼ×G

**总边际收益 = 直接弹性收益 + 级联间接收益**

贪心只看直接弹性收益，忽略级联间接收益。

**级联修正项的精确推导：**

设εⱼ_eff = εⱼ × (1 - Σₖ κₖⱼ × step(θC(k)-εk))

改善εᵢ可能使某个step函数从1→0（解锁一个门），或使εk超过阈值（关上一个门）。

**解锁门的级联收益：** εᵢ从θC(i)-δ→θC(i)+δ时，gate(i)从0→1，G从0跳到非零。同时如果κᵢⱼ>0，εⱼ_eff也提升→fⱼ提升→G进一步增加。

**级联收益的量化：**

ΔG_cascade = G_after - G_direct

G_direct = G × (1 + ηᵢ×Δεᵢ)（只考虑直接弹性）

G_after = G × (1 + ηᵢ×Δεᵢ) × ∏ⱼ (1 + κᵢⱼ×ηⱼ×Δεᵢ/εⱼ)（考虑级联）

ΔG_cascade/G ≈ Σⱼ κᵢⱼ×ηⱼ×Δεᵢ/εⱼ

**级联修正权重：** βⱼ = κᵢⱼ×ηⱼ/εⱼ

βⱼ越大→改善εᵢ通过j维度的级联收益越大→εᵢ的优先级应提高

**D110中β参数的精确含义：**

β不是常数，是级联敏感度对系统整体耦合强度的比值：

β = ||κ||_F / (||κ||_F + ||η||_²)

||κ||_F为耦合矩阵的Frobenius范数，||η||²为弹性向量的平方和。

弱耦合（||κ||<<||η||²）时β≈0→纯贪心
强耦合（||κ||>>||η||²）时β≈1→级联主导

**收敛：** 推论3展开给出级联修正的精确量化——总边际收益=直接弹性+级联间接收益，级联修正权重βⱼ=κᵢⱼ×ηⱼ/εⱼ。β参数由系统耦合强度与弹性强度的比值决定。

## 推论4展开：三种脆弱度三种策略

**命题：** 最坏情况补最弱、平均补弹性最高、条件级联切断关键节点

**展开三种策略的适用场景：**

**策略1：补最弱（最坏情况优化）**

目标：max minᵢ εᵢ

适用：系统有"否决性瓶颈"——一个维度为零系统崩溃（D102门控否决）

典型场景：8维退出权中某一维被锁死

**策略2：补弹性最高（期望优化）**

目标：max E[G] = max ∏fᵢ(εᵢ)

适用：系统在连续域运行，所有因子>0但大小不同（D87参数渐变）

典型场景：资源分配优化、投资组合

**策略3：切断级联（条件风险优化）**

目标：min P(级联崩溃)

适用：因子间有强耦合，一个崩溃触发连锁（D101级联）

典型场景：系统性风险防范、组织韧性

**三种策略的统一目标函数：**

G_robust = ∏fᵢ(εᵢ) × exp(-γ×P(cascade))

γ为风险厌恶系数。γ=0时退化为策略2（期望优化），γ→∞时退化为策略3（风险最小化），中间值综合两者。

**P(cascade)的估计：**

P(cascade) ≈ Σᵢ P(εᵢ<θC(i)) × Σⱼ κᵢⱼ×P(εⱼ<θC(j)|εᵢ<θC(i))

= Σᵢ (1-σ(εᵢ-θC(i))) × Σⱼ κᵢⱼ×(1-σ(εⱼ_eff-θC(j)))

**策略3的操作：** 找到使P(cascade)降低最多的维度i，即：

argmax_i ΔP(cascade)/Δεᵢ = argmax_i ηᵢ×Σⱼ κᵢⱼ×ηⱼ×(1-σⱼ)

**这就是D110中级联修正项的来源。**

**收敛：** 推论4展开给出三种策略的统一目标函数G_robust = G×exp(-γ×P(cascade))，γ=0期望优化，γ→∞风险最小化。级联概率P(cascade)的估计直接导出D110的级联修正项。

## 推论5展开：统一策略=弹性+级联加权

**命题：** β=0贪心，β→∞级联切断，中间为综合

**展开统一策略的完整数学：**

Δεᵢ* ∝ ηᵢ(εᵢ) + β×Σⱼ κᵢⱼ×ηⱼ(εⱼ)

**对β求极值的完整推导：**

目标：max G_robust = ∏fᵢ(εᵢ+Δεᵢ) × exp(-γ×P(cascade))

约束：ΣΔεᵢ=R

拉格朗日：L = lnG - γ×P(cascade) - λ(ΣΔεᵢ-R)

∂L/∂Δεᵢ = ηᵢ - γ×∂P(cascade)/∂Δεᵢ - λ = 0

∂P(cascade)/∂Δεᵢ = Σⱼ κᵢⱼ×ηⱼ×σ'(εⱼ_eff-θC(j))

在门槛附近σ'最大，远离门槛σ'→0。

**所以级联修正主要影响门槛附近的维度——远离门槛的维度级联修正≈0。**

**β的精确表达：**

β = γ×max_j(σ'(εⱼ_eff-θC(j)))

γ是风险厌恶系数（外部参数），max_j(σ'(...))是系统当前最接近门槛的维度的sigmoid导数（内部状态）。

β大意味着：要么风险厌恶高（γ大），要么有维度正处于门槛附近（σ'大）。

**β的动态变化：**

随着资源投入，维度被推离门槛，σ'降低，β自动下降→策略从级联防御逐渐转向贪心优化。

**这就是"先防守后进攻"的数学基础：** 早期维度在门槛附近β大→级联防御优先；后期维度远离门槛β小→贪心优化优先。策略自动切换不需要人为调整。

**收敛：** 推论5展开给出β的精确表达β=γ×max(σ')，β动态变化——维度推离门槛后β自动下降，策略从级联防御自动转向贪心优化。"先防守后进攻"不是经验法则，是统一策略函数的动态性质。

## 五条推论统一收敛

**D111 弹性-级联统一策略函数**

Δεᵢ* ∝ ηᵢ(εᵢ) + β×Σⱼ κᵢⱼ×ηⱼ(εⱼ)

β = γ×maxⱼ(σ'(εⱼ_eff-θC(j)))

γ为风险厌恶系数，max(σ')为最接近门槛维度的sigmoid导数。

统一目标：G_robust = ∏fᵢ(εᵢ) × exp(-γ×P(cascade))

β动态性质：维度推离门槛→σ'↓→β↓→策略自动从级联防御转向贪心优化。"先防守后进攻"是函数的动态性质不是经验法则。

弹性均等=拉格朗日最优的必然（推论1），补弹性最高≠补最弱当函数形式不同或sigmoid门槛效应存在（推论2），级联修正权重βⱼ=κᵢⱼ×ηⱼ/εⱼ量化间接收益（推论3），三种策略由γ统一（推论4），β动态变化实现策略自动切换（推论5）。

核心结论：
1. **弹性均等是拉格朗日最优的几何必然**——等产出面法向量=资源约束法向量
2. **补弹性最高≠补最弱的三种情况**——不同函数形式、sigmoid门槛效应、饱和效应
3. **级联修正量化**——总边际收益=直接弹性+Σκᵢⱼ×ηⱼ×Δεᵢ/εⱼ
4. **统一目标G_robust=G×exp(-γP(cascade))**——γ=0期望优化，γ→∞风险最小化
5. **"先防守后进攻"是β动态性质的数学必然**——不需要人为切换策略
