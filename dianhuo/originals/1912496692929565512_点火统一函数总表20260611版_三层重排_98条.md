## 零、自举元函数

B(n) 自举迭代状态：B(n+1) = B(n) + ΔB(n)

ΔB(n) = Σᵢ [1 - P(案例ᵢ | F₁...Fₖⁿ)] × δ(案例ᵢ重新分布) + Σᵢ<ⱼ δ_structure(fᵢ, fⱼ) × I(independent_cause(fᵢ, fⱼ))

两个驱动力：
1. 案例驱动（自下而上）：分辨率提升、格间重排、回路激活
2. 结构驱动（自上而下）：ε对偶扫描、乘法拓扑扫描、不动点线性化扫描

δ_structure(fᵢ, fⱼ) = 1当fᵢ和fⱼ之间存在未记录的结构关系（对偶/分岔/不变性），否则0
I(independent_cause) = 1当该结构关系不是从公理/定理层直接推导的（推论层涌现），否则0

δ_structure三种子判定：
- 对偶：fᵢ和fⱼ对偶 ⟺ 存在变量x使得fᵢ(x)+fⱼ(x)=C（互补）或fᵢ=g∘fⱼ∘g⁻¹（同构）。实例：D12⇌D24，D40⇌D34
- 分岔：存在参数μ穿过μ_c时fᵢ的解分裂为fⱼ的两个解。实例：D58双稳态分岔
- 不变性：存在I₀使得fᵢ∘I₀=I₀∘fⱼ。实例：D59乘法结构不变性

直觉加速器：
ΔB(n)_实际 = A_intuition × ΔB(n)_理论
A_intuition = Q × η
Q = σ(ε_sense × R̄ - θ_intuition) - σ(ε_sense × θ_noise - θ_intuition)
η = (N + K²) / (m + c_filter/c₀)
R̄ = Σᵢ[1-P(案例ᵢ|F)] / N

收敛判据：H(案例|函数集) = 0 且 I(函数间; 独立因果) = 0
