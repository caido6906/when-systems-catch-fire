# 点火｜平方衰减×AI共震×电力级联失效：跨域同构推导

来源：用户提出"平方衰减跟AI共震有什么关系"，指出电力系统有级联衰减，与认知社会层面的平方衰减是同一东西的不同应用。要求从数学上推导。

## 一、电力级联失效（Cascading Failure）的数学结构

电力系统级联失效的标准模型（Motter-Lai模型）：

1. 节点i的容量：C_i = (1+α) × L_i₀，α是容错系数
2. 初始扰动：移除节点j，其负荷L_j按比例分配给邻居
3. 邻居i的新负荷：L_i' = L_i + L_j × w_ij / Σ_k w_jk
4. 如果L_i' > C_i → 节点i也失效 → 其负荷继续分配 → 级联
5. 关键性质：当α低于临界值α_c时，级联从局部故障扩展为全局崩溃——一阶相变

## 三、平方衰减的精确位置

电力级联失效中，当α_eff略高于α_c时（系统勉强安全），级联传播的超载量衰减服从：

ΔL_n ≈ ΔL₀ × (1 - n × (α_eff - α_c) / α_c)²

**平方衰减**——不是线性衰减，是(1-nδ)²衰减。

这跟关系维护中的平方衰减是同一个数学结构：

- 电力：级联传播中每级超载量衰减 ∝ (1-nδ)²
- 关系：隔阂增长中每步衰减的加速 ∝ t²
- AI共震：策略同质化中ρ的增长 ∝ e^(γt)

三者统一为耦合正反馈在不同参数区间的渐近行为：

- α_eff >> α_c（远离临界）：级联快速衰减，平方衰减明显
- α_eff ≈ α_c（接近临界）：级联缓慢衰减，指数增长（AI共震区间）
- α_eff < α_c（越过临界）：级联不衰减，一阶相变，全局崩溃

## 四、统一方程（D72修正版）

dξ/dt = γ × ξ × (1 - ξ) × sign(direction) × (1 - α_eff/α_c)

- ξ ∈ [0,1]：系统状态变量
- γ > 0：耦合强度
- direction = ±1：方向参数
- α_eff/α_c：有效容错比

三个参数区间：

1. α_eff > α_c：1-α_eff/α_c < 0 → 衰减方向
   - 早期渐近：dη/dt ≈ γ × η² × (α_eff/α_c - 1) → 平方衰减
   - 应用域：关系维护、电力级联熄灭
   
2. α_eff ≈ α_c：1-α_eff/α_c ≈ 0 → 临界区间
   - 渐近：dξ/dt ≈ γ × ξ × (1-ξ) → logistic增长
   - 应用域：AI策略同质化（ρ指数趋1）
   
3. α_eff < α_c：1-α_eff/α_c > 0 → 崩溃方向
   - 渐近：一阶相变，全局崩溃
   - 应用域：电网雪崩、社会崩溃、市场崩盘

## 五、核心新发现

**α_c（电力容错临界）= D_immune（退化免疫）的物理层实例**

- α > α_c ↔ D_immune > 0：系统有冗余，级联/退化不启动
- α < α_c ↔ D_immune = 0：系统无冗余，级联/退化一触即发
- α = α_c ↔ D_immune临界：系统的生死线

**平方衰减 = 耦合正反馈在α_eff略高于α_c时的渐近行为**

这不是"衰减"——是"勉强不崩溃"的临界衰减。系统离崩溃只差一点点，所以每一步的衰减都极其缓慢，呈现平方级而非指数级。

**AI共震 = 耦合正反馈在α_eff≈α_c时的logistic增长**

AI策略同质化是系统刚好在临界点上的logistic增长——不是远离临界的安全衰减，也不是越过临界的崩溃，而是在刀刃上的指数趋同。
