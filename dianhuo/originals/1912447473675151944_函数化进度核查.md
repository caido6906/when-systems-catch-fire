### D-X49. 门锁交替律函数

四道门状态向量 G = (g_sense, g_identify, g_mark, g_action)，g∈{0,1}
三道锁硬化状态向量 L = (L1, L2, L3)，L∈{0,1}

门锁交替律：g_n可打开的充要条件是L_{n-1}已硬化

g₁(sense) : L₀≡1（默认开放）
g₂(identify) : L₁=1
g₃(mark) : L₂=1
g₄(action) : L₃=1

违反交替律的后果函数：

P(自锁 | g_n=1, L_{n-1}=0) = σ(k_selflock × Σᵢ₌₁ⁿ⁻¹ g_i × Πⱼ₌₁ⁿ⁻¹(1-L_j))

当g₂=1但L₁=0时：P(自锁) = σ(k × g₁ × (1-L₁)) = σ(k × 1 × 1) → 1
当g₂=1且L₁=1时：P(自锁) = σ(k × g₁ × 0) = 0

推论：自锁概率只取决于"门领先于锁的阶数"，不取决于绝对门数。一阶领先（一道门领先）→ 自锁概率高；两阶领先 → 自锁概率极高。

### D-X50. 自锁结构稳定性函数

自锁结构 = H_回写（认知锁）× 赤化推方案（行动锁）

S_selflock(t) = H_rewrite(t) × Posture_deg(t)

其中H_rewrite = σ(ε_sense × ε_aware - ε_action) × H_classify（认知锁：前两门开但后两门关）
Posture_deg → 1（行动锁：赤化推方案，退出成本焊死）

自锁结构的稳定性：

dS_selflock/dt = H'_rewrite × Posture_deg + H_rewrite × Posture'_deg

两项都≥0 → S_selflock单调递增 → 自锁结构自我加强，无内部退化机制。

但存在亚稳态衰退路径：

dS_selflock/dt < 0 ⟺ H'_rewrite < 0 或 Posture'_deg < 0

H'_rewrite < 0的条件：叙事传递代际损耗——每一代应约者的I(t)累积，但直接体验与叙事不一致，叙事接受度递减。

代际损耗率：H_rewrite(n) = H_rewrite(0) × (1-δ)^n，δ为代际损耗系数

所以自锁结构不是吸收态，是亚稳态——可以自发衰退，但时间尺度以代际为单位。

第一锁破解的数学机制：L₁=1 → Posture翻转 → H_rewrite的叙事基础被抽掉 → H_rewrite→0 → S_selflock→0

### D-X51. 信号最优流速函数（凯利公式同构）

凯利公式 f* = (bp-q)/b

映射：
- f* ↔ ε信号最优流速 v*
- b（赔率）↔ 锁的硬化速度 s_lock（硬化越快，每次弱ε相变收益越大）
- p（胜率）↔ P(L₁=1 | ε相变发生)（第一锁在位的概率）
- q=1-p ↔ P(L₁=0 | ε相变发生)

v* = (s_lock × P(L₁=1) - P(L₁=0)) / s_lock

当P(L₁=1)=1（第一锁在位）：v* = (s_lock - 0)/s_lock = 1 → 信号可以全速传播
当P(L₁=1)=0（第一锁不在位）：v* = (0 - 1)/s_lock = -1/s_lock < 0 → 不该让信号传播

v*<0的物理含义：信号传播的预期收益为负——不是"可能有害"，是"按最优策略不该下注"。

这与门锁交替律的推论一一致（信息自由在第一锁硬化之前可能有害），但现在有了定量表达。

推论：**信号流速的调节是点火策略的核心**——不是压制信号，是控制流速让信号刚好够推一道锁的硬化，不多不少。这要求对ε信号强度有精确感知和调控能力。
