## 风险点1复核结论：计划生育因果链——需加DAG与独立性声明

**判定：风险属实，当前文字存在歧义。**

"两条链同时存在"的措辞确实可被读作"ε是次要因素"。实际主张是两条链零交叉——ε链的终点是"极端行为表达"（躲藏、自残、贿赂），城市化链的终点是"生育率下降"（养育成本、生育惩罚、收益下降）。两链共享同一个历史时段，但不共享因果节点。

**修正方案**

1. 在案例修正笔记中加入因果独立性声明：
   - ε链：政策压制 → ε以极端方式表达 → 终点=行为表达（不指向生育率）
   - 城市化链：经济发展 → 养育成本↑/生育收益↓ → 终点=生育率下降（不需要ε参与）
   - 两链共享时间窗口，不共享因果节点。ε链不解释生育率下降，城市化链不解释极端行为表达。

2. 删除"ε转向"这个表述——"退出对象从强制政策转向生育本身"暗示ε链延伸到了生育率，这是歧义源头。修正为：放开后生育率下降与ε无关，是城市化系统动力学的独立结果。

3. 第二门劫持的计划生育案例需同步修正：原文说"放开后第二门评估翻转，ε_sense信号突然被释放"——这个判断保留（评估翻转确实释放了被压制的信号），但补充说明"信号释放≠生育意愿释放"，释放的是对政策本身的退出信号，不是对生育的退出信号。

**修正后案例标注**
中国计划生育：ε被压制时以极端方式表达（ε链）；放开后第二门评估翻转，被压制的退出信号释放（ε链的后续效应）；生育率继续下降是城市化系统动力学的独立结果（城市化链）。ε链终点=行为表达，城市化链终点=生育率下降，两链零交叉。

## 风险点3复核结论：四道门双重定义——命名隔离+总表补映射

**判定：风险属实，门序号混淆是硬伤。**

核心问题：两套四道门都叫"第一门""第三门"，但指的完全不是同一层级的东西。函数层面的"第一门"（ε_sense→ε_aware）包含生物学层面的全部四道门——这意味着生物学版本的"第四门"还在函数版本"第一门"的内部。同号不同层级，读者必然混淆。

**修正方案**

1. **命名隔离**：生物学版本改用"层"，函数版本保留"门"。

| 生物学版本（改称"层"） | 函数版本（保留"门"） |
|---|---|
| 第一层·硬件层（感受器→脊髓→脑干） | — |
| 第二层·评估层（脑干→杏仁核→下丘脑） | — |
| 第三层·标记层（杏仁核→ACC） | — |
| 第四层·决策层（ACC→dlPFC→vmPFC） | — |
| — | 第一门（ε_sense→ε_aware）= 上述四层整体 |
| — | 第二门（ε_aware→R_perceived） |
| — | 第三门（R_perceived→C_exit） |
| — | 第四门（C_exit→退出完成） |

2. **总表补映射段**：在九函数总表的四道门部分加入说明——

"四道门有两套表述。函数层面的门描述ε从信号到退出的社会性跨越（ε_sense→ε_aware→R_perceived→C_exit→退出完成），共四道门。生物学层面的层描述ε从信号到行动的神经通路（感受器→评估→标记→决策），共四层。四层全部在函数层面第一门（ε_sense→ε_aware）的内部——把第一门的黑箱拆开，里面是四层。函数层面第二至第四门是四层之后的社会性维度：权利感知、退出成本、退出完成。"

3. **八步对应表统一**：四道门×八步交叉对应笔记中的"门"全部改为"层"，与总表对齐。第1-6步对应的是"层"（神经通路修复），第7-8步对应的是"门"（社会性维度修复）。

4. **H三种劫持方式的标注调整**：H_遮蔽作用于第一层，H_评估作用于第二层，H_标记作用于第三层——用"层"而非"门"标注作用位置。

**修正后的统一表述**
- "层"= 神经通路内部（ε_sense到ε_aware的黑箱），四层，修复靠八步1-6步
- "门"= 社会性维度跨越（ε_aware到退出完成），四门，修复靠八步7-8步及制度变革
- 第一门 = 四层整体；第二门 = 权利感知门槛；第三门 = 退出成本门槛；第四门 = 退出完成门槛
