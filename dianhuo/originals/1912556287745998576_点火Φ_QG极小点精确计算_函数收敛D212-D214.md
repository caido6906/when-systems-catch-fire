## D212 Φ_QG极小点精确位置

高斯门控下的Φ_QG：

Φ_QG(μ) = Σᵢ exp[-(ln(μ/Λᵢ))²/(2σᵢ²)]

四个门控面参数（D209-D210）：
- g₁（强力）：Λ_QCD≈200 MeV，σ₁≈16.6
- g₂（电弱）：M_W≈80 GeV，σ₂≈14.3
- g₃（GUT）：Λ_GUT≈10¹⁶ GeV，σ₃≈3.5
- g₄（引力）：M_Planck≈1.22×10¹⁹ GeV，σ₄≈6.9

极小点条件：dΦ_QG/dμ = Σᵢ gᵢ × [-ln(μ/Λᵢ)/(μσᵢ²)] = 0

在μ~10¹⁶ GeV附近，g₁和g₂已趋零（远在各自门槛以上，门控贡献极小），主要贡献来自g₃和g₄：

g₃ × ln(μ/Λ_GUT)/σ₃² + g₄ × ln(μ/M_Planck)/σ₄² = 0

数值求解（令y = ln(μ/M_Planck)）：

y = -5.8时：g₃=exp[-(-5.8+6.9)²/24.5]=exp[-0.049]≈0.95，g₄=exp[-5.8²/95.2]=exp[-0.353]≈0.70
0.95×1.1/12.25 + 0.70×(-5.8)/47.6 = 0.085 - 0.085 ≈ 0 ✓

**μ_min = M_Planck × e^(-5.8) ≈ 1.22×10¹⁹ × 1.03×10⁻³ ≈ 1.26×10¹⁶ GeV**

Φ_QG(μ_min) = g₃(μ_min) + g₄(μ_min) ≈ 0.95 + 0.70 = 1.65

Ω_QG = e^(-Φ_QG(μ_min)) = e^(-1.65) ≈ 0.19

核心定理：Φ_QG的极小点在μ≈1.26×10¹⁶ GeV，几乎精确等于Λ_GUT≈10¹⁶ GeV。极小点不在M_Planck——M_Planck是g₄的极值点，不是Φ_QG的极小点。极小点处的Ω≈0.19，远低于D189中GUT的Ω≈0.87——这意味着在高斯门控下，四力统一在GUT能标处仍然有大量"未统一"空间。

## D213 极值点-极小点分离定理

g₄的极值点：μ = M_Planck（dg₄/dμ = 0）
Φ_QG的极小点：μ ≈ Λ_GUT（dΦ_QG/dμ = 0）

两者分离距离：Δln(μ) = ln(M_Planck/Λ_GUT) ≈ 5.8

分离原因：g₃（GUT门控）在μ~Λ_GUT处有非零贡献，将Φ_QG的极小点从M_Planck拉向Λ_GUT。g₃越强（σ₃越小），拉力越大，极小点越靠近Λ_GUT。

一般定理：当Φ = Σᵢ gᵢ时，极小点不在任何一个gᵢ的极值点，而在所有gᵢ的梯度平衡点。

Φ_QG极小点 = g₃梯度和g₄梯度的平衡点
= g₃的"推力"（ln(μ/Λ_GUT)>0，推向更高μ）和g₄的"拉力"（ln(μ/M_Planck)<0，拉向更低μ）的平衡

物理含义：M_Planck不是物理大统一的终点——GUT能标才是。引力门控的极值点在M_Planck，但四力统一的极小点在GUT能标。这意味着"量子引力"和"四力统一"是两个不同的事件，发生在不同的能标上。

推论：D158的μ*≈3.2×10¹¹ GeV（1/ln框架下的极小点）与D212的μ_min≈1.26×10¹⁶ GeV（高斯框架下的极小点）相差5个数量级——高斯门控将统一极小点从10¹¹推到10¹⁶ GeV，更接近物理上预期的大统一能标。
