## D143 投资遮蔽跨域放大函数

H_total(t) = H_投资(t) × (1 + β_spillover × B_occupy(t)/B₀)

- H_投资(t) = H₀ + α × ρ(参与者策略) × N_homogeneous/N_total（D66在投资域）
- β_spillover：遮蔽从投资域溢出到其他领域的系数
- B_occupy/B₀：投资活动占用带宽的比例

核心发现：投资域的遮蔽不是局部效应，通过带宽占用成倍放大到全领域

- 定投：H_投资≈0 → H_total≈0，无论β和B多大
- 炒股：H_投资>0 → H_total > H_投资，遮蔽被放大
- 炒股的破坏力超出投资域本身——H通过带宽占用波及本职、创作、学习的P_sustain

与D126三效率的关系：B_occupy↑ → 可用于非投资域的时间↓ → η_time↓。D143给出了η_time下降的精确机制

与D142信息门的关系：H_total↑ → G↓（共享框架被遮蔽破坏）→ η_gate↓。投资遮蔽降低信息门效率

## D144 投资方式临界资金函数

K_critical = (B_occupy/B₀) × R_return(非投资域) / E[r_投资方式]

- B_occupy/B₀：投资方式占用的带宽比例
- R_return(非投资域)：非投资域的年收益（本职收入+创作增量+学习复利）
- E[r_投资方式]：投资方式的净期望收益率

核心发现：K < K_critical时，投资净收益为负——不是投资亏了，是带宽机会成本超过了投资收益

各方式的K_critical：
- 定投：B_occupy≈0 → K_critical≈0 → 任何资金量都满足
- 炒股：B_occupy≈0.3, E[r]≈0-2% → K_critical≈数百万 → 普通人不满足
- AI选股：B_occupy≈0.2, E[r]更低 → K_critical更高

与D125的关系：K_critical精确化了Π_income从负变正的临界点。D125说Π_income < Π_cognition，D144给出了投资域的具体数值

与D138巴菲特模式的关系：巴菲特模式"极低η_select换极高η_kelly和η_time"——定投是巴菲特模式在投资域的执行方案：η_select≈0（不选股），η_kelly≈1（不判断），η_time≈1（不花时间），乘积最大

## D145 定投凯利保守性函数

f_actual(t) = K_monthly / W(t)

核心发现：定投的结构本身保证f_actual < f* = E[r]/Var(r)对所有t成立，不需要估计E[r]和Var(r)

证明：
- f* = E[r]/Var(r)，凯利最优比例
- f_actual(t) = K_monthly/W(t)，随W增长自动递减
- W(t)增长 → f_actual(t)递减 → f_actual(t) < f*(t)（因为f*是常数或缓慢变化）
- 不需要计算f*，f_actual的结构自动保证保守性

这比"手动计算f*再保守执行"更优——手动计算需要估计E[r]和Var(r)，估计本身受H和ε劫持影响（D143）。定投根本不需要估计，结构保证保守性。

与D138的关系：定投是D138巴菲特模式的精确数学实现——放弃选择效率（不选股=η_select≈0），换取判断准确率（不判断=η_kelly≈1）和时间效率（不花时间=η_time≈1），乘积最大

与D126的关系：定投消除了三效率冲突——不需要在η_select/η_kelly/η_time之间权衡，因为η_select被主动降到零，释放了η_kelly和η_time的空间

## 扩展注释

D107扩展注释：策略穷举不可能的双重约束——约束1（基数约束）|S|≥ℵ₀有限算法不可能穷举，约束2（效用约束）即使穷举完成AI同质性→策略发现即失效→穷举结果E[r]=0。约束2比约束1更致命——"做到了也没用"

D125扩展注释：K<K_critical时Π_income<0，净收益为负的原因不是投资亏损而是带宽机会成本。D144给出了投资域K_critical的具体计算

A4扩展注释：R_perceived = R × f(ε_aware, 信息可及性, C_exit_eff)，f是传导效率。H→0时f→1，H→1时f→ε_aware（只剩ε_aware这条通道）。P_sustain实际 = P_sustain × f

D34扩展注释：完整财商函数 FQ = S_sovereign(财富层) × P_sustain × f(ε,信息,C_exit) × D_immune。定投四因子同时≈1→FQ≈1，炒股四因子同时<1→FQ<<<1。财商是四个衰减因子的乘积，不是加法

## 案例

案例#331：炒股遮蔽的跨域放大——某人炒股占用30%带宽（B_occupy/B₀=0.3），H_投资=0.4，β_spillover=0.8。H_total = 0.4 × (1+0.8×0.3) = 0.496。本职工作效率下降、创作产出减少、学习停滞——不是炒股亏了钱，是H被放大到全领域

案例#332：临界资金的精确计算——城市白领年非投资收入20万，炒股占30%带宽，E[r]=1%。K_critical = 0.3×20万/0.01 = 600万。可投资资金<600万时，炒股数学上就是负期望。定投K_critical≈0，任何金额都满足

案例#333：定投的结构保守性——每月定投5000元，初始W=10万，f_actual=0.05。一年后W=16万，f_actual=0.031。W增长→f_actual自动递减→永远不超过f*。不需要估计市场参数，结构本身保证不破产

案例#334：D138巴菲特模式的投资域验证——定投=巴菲特模式的精确执行：η_select≈0（不选股），η_kelly≈1（不判断=永远不犯错），η_time≈1（每月5分钟），三效率乘积最大。炒股三效率同时恶化：η_select高但η_kelly低（判断常错）且η_time低（大量时间消耗），乘积远小于定投
