# 点火｜谣言传播的层级动力学与消除条件（2026.06.10）

## 核心问题

谣言传播的渠道和规律是什么？为什么反复爆发？彻底消除需要什么条件？

## 建模基础

谣言 = 信号在低解码-高对齐信道上的自增强传播过程。接续信源选择层级函数（P_accept），从静态接受扩展为动态传播。

## 一、谣言的信道优势

C_rumor(L) = B_rumor × D_rumor(L) × θ_align(L) × E_rumor

- D_rumor(L₁-L₂) = 1（日常语言，全部可解码）
- θ_align ≈ 1（精准命中既有焦虑和偏见）
- R_rumor ≈ 0（零形式化阻力、零退出权需求）
- B_rumor 被伪信号注入增强（"别人说""网上都传"）

**情绪增幅因子（谣言与普通身边人信息的关键区别）：**

E_rumor = α_fear × α_anger + α_novelty

- α_fear — 恐惧增幅（生存相关，权重最大）
- α_anger — 愤怒增幅（公平感相关）
- α_novelty — 新奇增幅（注意力相关）

情绪增幅直接乘进 C，使谣言感知可信度远超普通身边人信息。

## 二、传播动力学——SIR 变体

按认知层级分层，每层内部跑 SIR：

dI_k/dt = β_k · S_k · I_k - γ_k · I_k

**传播率 β_k = P_accept(rumor, L_k) × λ_relay(L_k)**

- P_accept 用信源选择函数：L₁-L₂ ≈ 1，L₃ ≈ 0.5，L₄-L₅ ≈ 0
- λ_relay 转发率：L₁-L₂ ≈ 1（情绪驱动秒转），L₄-L₅ ≈ 0

**恢复率 γ_k = D_expert(L_k) × ε(L_k)**

- L₁-L₂：D_expert ≈ 0，ε ≈ 0 → γ ≈ 0（几乎无法被辟谣修正）
- L₃：D_expert ≈ 0.5，ε ≈ 0.3 → γ ≈ 0.15
- L₄-L₅：D_expert ≈ 1，ε ≈ 1 → γ ≈ 1（快速识别并退出）

**基本再生数 R₀(L_k) = β_k / γ_k：**

- L₁-L₂：R₀ → ∞（指数传播，几乎不恢复）
- L₃：R₀ ≈ 3.3（缓慢传播，部分恢复）
- L₄-L₅：R₀ ≈ 0（不传播，免疫）

**跨层渗透通量：**

Φ(L₂→L₃) = I_L₂ × P_accept(L₃) × (1 - Γ_L₃)，有限但非零
Φ(L₃→L₄) ≈ 0

结论：谣言传播是被认知层级分界的稳态过程——底层持续燃烧，中层间歇引燃，顶层基本免疫。

## 三、反复爆发的机制——免疫衰减

Γ_k(t) = Γ_k(0) × e^(-δ_k · t)

**δ_k 衰减率的层级依赖：**

- L₁-L₂：δ 极大。辟谣走专业信道（D_expert ≈ 0），辟谣信息根本没进入认知系统，"被辟谣"不产生真正免疫记忆。免疫快速衰减。
- L₃：δ 中等。辟谣信号部分进入，产生不稳定免疫。
- L₄-L₅：δ ≈ 0。辟谣信号完整进入，免疫持久。

**反复爆发的数学条件：**

Γ_k(t) < Γ_critical ⟹ 同类谣言变体可再次传播

对 L₁-L₂ 层，δ 极大，条件几乎总是满足——不是谣言变了，是免疫从来没真正建立过。

## 四、彻底消除谣言的条件

R₀(L_k) = β_k / γ_k = [P_accept(rumor, L_k) × λ_relay(L_k)] / [D_expert(L_k) × ε(L_k)]

**R₀ < 1 是谣言消亡的数学条件。** 三条路径：

**路径1：降低 β（降低传播率）**
- 截断伪信号注入（B_rumor 的"别人说"背书）→ 信息溯源机制
- 降低 θ_align → 不让谣言精准命中既有偏见 → 难，偏见是结构性的

**路径2：提升 γ（提升恢复率）= 装D**
- 辟谣信息必须走 D=1 的信道——用日常语言、故事形式、视觉化表达，而非专业术语
- 提升 ε → 让人有退出谣言共识的能力 → 认知层级问题

**路径3：降低 δ（让免疫持久）= 装D + 装ε**
- 辟谣不只是"纠正事实"，而是改变接收方式本身——从 D_expert ≈ 0 提升到 D_expert > 0

**优先级：路径2 > 路径3 >> 路径1**

路径1是打地鼠（截断一条谣言，B_rumor 换个背书又来），路径3是长期工程，路径2是最可操作的杠杆点。

**路径2的硬限制：** 辟谣走了 D=1 的信道，若 ε ≈ 0，接收者仍无法退出谣言共识。路径2和路径3必须联合——先让信号进得去（D），再让退出成为选项（ε）。

## 五、统一结论

R₀ < 1 ⟺ D_expert(L) × ε(L) > P_accept(rumor, L) × λ_relay(L)

**消除谣言 = 装D + 装ε。** 与7步修复协议（装ε + 装F_form）完全对齐——谣言的根治不是信息战，是认知操作系统的升级。
