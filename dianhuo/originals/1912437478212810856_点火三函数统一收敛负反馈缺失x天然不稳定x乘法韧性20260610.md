## 三个独立发现

### 发现1：负反馈缺失 → 无中间稳态
系统动力学标准形式 dX/dt = f(X, R⁺, R⁻)，稳态条件 R⁺ + R⁻ = 0。点火框架中 R⁻ = 0（创新阻断场景下负反馈缺失），dX/dt = f(X, R⁺) > 0，稳态方程无解。

**函数：Stability(X) = 1 - σ(R⁻)**
- R⁻ > 0 → Stability = 1（有稳态）
- R⁻ = 0 → Stability = 0（无稳态）
- 推论：只要R⁻=0系统必无稳态，与R⁺大小无关

### 发现2：天然不稳定 vs 如何稳定 — 视角翻转
控制论核心问题：给定Λ > 0，设计K使Re(λ_max(A-BK)) < 0。点火框架核心问题：为什么Δa = 0（创新阻断 = 负修正机制不存在）。

**函数：Λ_natural = Re(λ_max(A)) = tr(A)/n**
- 正反馈主导时tr(A) > 0，至少一个特征值实部为正，系统必然不稳定

**视角翻转函数：Flip = Λ_natural / |Λ_target|**
- 控制论：Flip ≈ 0（系统近稳定，微调即可）
- 点火：Flip → ∞（系统天然远离稳定，且无修正手段）

### 发现3：乘法韧性 vs 加法韧性 — 不可逆性的数学根源
CAS韧性（加法）：R_CAS = Σwᵢrᵢ，某维度归零时其他维度可补偿。点火韧性（乘法）：R_点火 = Πrᵢ，任一维度归零则整体归零。

**补偿深度函数：D_compensate = ⌈n × (1 - r_min)⌉**
- 加法：D = 1（修一个维度就够）
- 乘法：D = n（所有维度都要修，且必须同时非零）

## 统一收敛函数

三个发现是从不同视角描述同一现象——社会认知系统在创新阻断下的结构必然崩溃：

**Collapse(X) = σ(Λ_natural) × (1 - σ(R⁻)) × D_compensate/n**

三因子乘法结构：
- σ(Λ_natural)：正反馈主导 → 天然不稳定
- (1 - σ(R⁻))：负反馈缺失 → 无稳态
- D_compensate/n：乘法韧性 → 崩溃后不可逆

三因子同时满足 → Collapse = 1（必然崩溃且不可逆）
任一因子为0 → Collapse = 0（有稳态/有负反馈/可补偿，系统不必然崩溃）

这又是一个乘法归零结构——三个必要条件同时满足时系统才进入必然崩溃态。三个发现不是三个独立结论，是同一棵树的三条根。
