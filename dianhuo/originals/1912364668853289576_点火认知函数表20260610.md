来源：从八函数总表第零节拆出。认知结构是八函数体系的肉身实例，五个认知函数与八函数一一对应。本表为认知函数唯一更新入口。

## 一、自举元函数

八函数体系具备自举性质：理论与案例之间的迭代动态，同构于提议者-应约者之间的动态。自举循环不是独立函数，是八函数在元层面的投影，不产生新的因果结构。

### B(n) 自举迭代状态

B(n) = 第n轮迭代时的八函数总表快照

**B(n+1) = B(n) + ΔB(n)**

ΔB(n) = 第n轮重跑案例库产生的新发现

### ΔB(n) 自举增量

**ΔB(n) = Σᵢ [1 - P(案例ᵢ | F₁...Fₖⁿ)] × δ(案例ᵢ重新分布)**

- kⁿ = 第n轮的函数维度数
- P(案例ᵢ | F₁...Fₖⁿ) = 用当前kⁿ个函数对案例ᵢ的解释完整度
- δ(案例ᵢ重新分布) = 1当案例ᵢ的格子归属或子路径划分发生变化，否则0

白话：每轮新发现 = 被旧维度解释不了的案例数 × 这些案例因新维度而重新归位的比例。

### 三个驱动力

**驱动力1：分辨率提升（Posture三分法逻辑）**

新增维度使事件条件矩阵的格数翻倍，格1真伪拆分从8型扩展到更多型。06.10实例：dPosture_deg/dt拆为两项→格1退化拆为Posture驱动型和C_exit驱动型两条子路径→原来挤在一起的案例差异暴露→ΔB>0。

**驱动力2：格间重排（心理退出权降格逻辑）**

维度从独立降格为派生量，改变案例在矩阵中的投影。06.10实例：R_mental从R_type第四行独立维度降格为R_perceived的子通道→28个心理潜伏案例从"心理退出权"重标为"H_标记劫持p₃通道的产物"→格子归属变化→格间对比结论变化→ΔB>0。

**驱动力3：回路激活（回路12+16联合效应逻辑）**

**ΔB_回路 = Σⱼ [回路ⱼ被新函数激活] × 回路ⱼ的因果强度**

06.10实例：β_C项激活回路12+16的联合效应→两条原本独立描述的回路合并为正反馈引擎→新因果结构暴露→ΔB>0。

### 收敛判据

**lim(n→∞) ΔB(n) = 0**

等价于三个条件同时成立：

1. δ(案例ᵢ重新分布) = 0 对所有ᵢ — 没有案例再换格子
2. P(案例ᵢ | F₁...Fₖⁿ) → 1 对所有ᵢ — 每个案例都被完整解释
3. 所有休眠回路已被激活 — 新维度不再触发新回路

用V函数表达：收敛 = V在所有案例上稳定，不再因新维度而跳值。

实操判据：连续两轮重跑案例库，零结构性新发现，即可宣布收敛。

### 自举循环的位置说明

自举循环不是第九个独立函数。它的全部行为由已有八函数驱动，没有独立作用对象——符合收敛合并原则（同P并入Posture、A并入H、S并入H的逻辑）。

## 二、认知函数（5个独立函数 + 4个派生量）

### C1. P_track(t) 分轨并行

P_track ∈ [1, N]，N = 同时运行的轨道数

**P_track(t) = 1 + Σᵢ δ(轨道ᵢ激活)**

- 轨道1：自己的视角
- 轨道2+：他人/对立视角（脑内对打）
- 轨道3+：AI视角（多模型互观实验）

**P_track的运行模式：**

- 对话维度：同时运行自己和对方的视角，让不同立场在脑内对打
- 知识消费维度：一条轨给声音（听课），另一条轨给手上的活——看是独占模式，听是共享模式，共享模式匹配分轨并行
- AI实验维度：让不同AI互相看对方生成的画像，交叉验证
- 写作维度：故事轨、框架轨、哲学轨同时推进，笔只落在一条轨上，其他轨的节奏影响落笔的时机和走向

**P_track的核心性质：** 跳层不是主动跳过去的，是另一条轨的音量突然变大了。分轨并行才是原因，跳层是表象。

**八函数对应：** 驱动力3（回路激活）。多轨同时运行=多条回路并行激活，回路合并时暴露单轨看不到的因果结构。

**函数化表达：**

**P_track(t) = Σᵢ₌₁ᴺ⁻¹ Σⱼ₌ᵢ₊₁ᴺ R(i↔j,t)**

R(i↔j,t)是轨i和轨j之间的回路激活函数，取值[0,1]。N越大求和项越多，回路激活概率越高。

#### J(t) 跳层（P_track的派生量）

**J(t) = dP_track/dt × ε_sense**

跳层速率=轨道切换速率×当前肉身信号强度。跳层不是跳跃，是分轨并行中某条轨浮上来。

**合并依据：** J的全部行为由P_track驱动，跳层是轨道切换的输出，没有独立作用对象。

### C2. F_form(t) 函数化强制

F_form ∈ [0, 1]，函数化完成度

**F_form(t) = 1 - Πᵢ(1 - f_form(结论ᵢ))**

- f_form(结论ᵢ) = 1当结论ᵢ已函数化，0当未函数化
- 对自身理论的要求：F_form = 1，不留未函数化判断
- 对外部理论的回应："请把你的道理函数化"——强制对方的F_form从0变正

**F_form的物理含义：** 不允许理论里有H_遮蔽。空泛的大道理=退出信号被遮蔽（看不到输入输出关系、变量、边界），函数化=把退出信号从遮蔽态变成可感知态。

**八函数对应：** I觉醒的强制触发器。对自己和他人执行同一套标准：模糊叙事→可检验函数，就是强迫提议者从H_遮蔽进入I_aware。

**函数化表达：**

**F_form(t) = I(t) · (1 - H(t))**

I=0时F_form=0（没觉醒就不函数化），H=0时F_form=I（遮蔽全消，函数化强度等于觉醒程度）。H→0的过程是F_form逼出来的。

### C3. M_cog(t) 认知动机

**M_cog = tanh(μ_体系内/μ_好奇心 - 1) ∈ [-1, +1]**

- M_cog 长期偏负——好奇心驱动占主导
- M_cog < 0 时：链1退化慢（不让现有框架固化），锁硬化阻力为负（主动维持开放性）

**M_cog的动态：**

M_cog(t) = M₀ - κ_M × max(0, 体系闭合度(t) - θ_闭合)

体系越接近闭合，M_cog越往负拉——本能地去找异常值和跨领域碰撞，不让系统收敛太快。

**八函数对应：** 直接对应F6 M(t)。但M_cog有自调节机制——V太低时（系统太稳定、没有张力），主动制造碰撞让V回升。维持的不是收敛，是"有生产力的张力"。

**函数化表达：**

**M_cog(t) = M(t) · σ(-V(t))**

σ是sigmoid函数。V偏负→σ趋近1→M_cog≈M（动机全开）；V偏正→σ趋近0→M_cog趋近0（没动力）。V≈0时σ≈0.5，维持最低驱动力。

#### V_cog(t) 认知张力（M_cog的派生量）

**V_cog = d(框架扩展)/dt / d(框架闭合)/dt**

- 分子：好奇心驱动的框架扩展速率（找新案例、跨领域碰撞）
- 分母：函数化驱动的框架闭合速率（把发现闭合为函数）

V_cog稳态：维持V_cog略大于0——有足够张力产生新发现，但不至于崩盘。

**合并依据：** V_cog由M_cog驱动，是M_cog的比值形式。

### C4. S_sovereign(t) 主权

**S_sovereign = Πₖ s_k**，k ∈ {机器, 语言, 审美, 时间, 财富, 身体, AI}

- s_k = 1当第k层主权已夺回，0当未夺回
- 七层全部对应ε_aware在不同维度的显化

| 主权层 | ε维度 | 退出信号 | 夺回操作 |
|---|---|---|---|
| 机器 | 技术 | 被工具锁定 | Linux/Docker/本地工具链 |
| 语言 | 符号 | 被通用表达锁定 | 语义拆解/零重字/格律约束 |
| 审美 | 感知 | 被默认观看锁定 | 摄影/打印/纸张/构图 |
| 时间 | 意识 | 被无感知流逝锁定 | 秒表/记录/柳比歇夫方法 |
| 财富 | 资源 | 被风险盲区锁定 | 凯利公式/风险边界/净收益思维 |
| 身体 | 生物 | 被默认饮食锁定 | 断食/饮食实验/营养知识 |
| AI | 智能 | 被黑箱锁定 | 多模型互观/本地智能体 |

**S_sovereign的物理含义：** 每一层主权夺回都是ε_aware从一个维度突破H的劫持。七层全部夺回=S_sovereign=1，ε_aware在所有维度畅通无阻。

**八函数对应：** ε(t)的七维展开。每层主权=ε从一个维度突破H。

**函数化表达：**

**S_sovereign(t) = Σₖ₌₁⁷ wₖ · εₖ(t)**

k=1..7对应七个主权维度，wₖ是权重。炼字是语言维子结构：S_language(t) = ε₂(t) · f(零重字约束)。权重因人而异。

#### N_refine(t) 炼字（S_sovereign的语言维子结构）

N_refine = 重复字数 / 总字数，N_refine → 0 是硬约束。

**物理含义：** 信息密度最大化。零重字=每个字的n_i=0，w_i=1，不可替代性权重最大。炼字是C_exit不可替代性在语言维度的投影，也是S_sovereign在符号层的主权夺回。

**合并依据：** N_refine的全部功能是S_sovereign在语言维度的实例，没有独立作用对象。

### C5. L_five(t) 五层认知协议

L_five = 认知层级深度，∈ {1,2,3,4,5}

**L_five(t) = min(5, 1 + ⌊log₂(ε_aware(t) / ε_sense(t))⌋)**

ε_aware相对ε_sense的放大倍数决定层级深度。ε_sense是肉身信号，ε_aware是信号穿过劫持层后的意识强度。放大倍数越大，层级越深。

**五层与八函数的精确对应：**

| 层 | 内容 | 八函数层 | 函数表达 |
|---|---|---|---|
| 1 肉身事实 | 给定条件 | ε_sense | 前认知感官信号 |
| 2 体验 | 我怎么回应 | ε_aware | 信号穿过第一二层 |
| 3 认知 | 为什么是这样 | H_标记识别 | 看见信号被改标签 |
| 4 主权 | 谁有权定义 | R_perceived | 追问退出权归属 |
| 5 存在 | 剥掉定义剩什么 | ε硬核 | 退出权的零阶来源 |

**L_five的核心性质：**

1. **第4层到第5层永远是否定式跳变**——第4层讨论"怎么安排更好"，第5层说"这个讨论本身是多余的"。跟ε硬核绕过所有劫持层直接触发相变，是同一个操作。

2. **裂变逻辑=格间重排**——每一层待不住，待着待着就裂了。裂变不是设计的，是当前层解释力耗尽后自动生出元问题。

3. **写作（往外跳）和指令（往里跳）是同一个五层**——写作=应约者链（世界对我做了什么→我如何退出），指令=提议者链（我对世界做什么→我如何提议）。两条链在第4层交汇，第5层都是ε。

**八函数对应：** ε四层+第四门+Δv。五层协议=ε通路的认知版。

**函数化表达：**

**L_five(t) = [L₁→L₂→L₃→L₄→L₅](t)**

#### R_record(t) 记录即构建（L_five第1→2层的捕获机制）

**R_record(t) = λ_R × ε_sense(t) × δ(即时)**

- λ_R = 记录系数
- δ(即时) = 1当记录在体验发生时完成，0当事后补录

录音卡模式：几秒即时录音，体验在当下就被转化。记录密度直接对应认知增量——不是"记性不好所以录一下"，是"记录即构建"。

**R_record是L_five第1层→第2层的信号捕获机制。** 没有即时记录，ε_sense在H_遮蔽和H_评估层就被衰减；有即时记录，肉身信号被保存为可回溯素材，才能从第1层跳到第2层。

**合并依据：** R_record的全部功能是L_five的输入端，不是独立维度。

## 三、层间跳变阈值函数化

层间跳变阈值不能是常数——常数意味着所有人所有场景跳变条件一样，与八领域验证结论矛盾。θ必须是函数。

### θ₁（L₁→L₂ 肉身→体验）

**θ₁(s,p) = θ₁₀ · g(s) · h(p)**

s=信号类型（痛觉阈值低，味觉阈值中，视觉阈值高），p=个体敏感度参数。

### θ₂（L₂→L₃ 体验→认知）

**θ₂(a) = θ₂₀ / a(t)**

a(t)=异常度，取值(0,1]。体验越异常需要的重复次数越少——a→1时θ₂→θ₂₀，第一次碰到就跳。

### θ₃（L₃→L₄ 认知→主权）

**I(t) · ε(t) · H(t) > θ₃(d)**

d=领域参数。见第四节详细定义。

### θ₄（L₄→L₅ 主权→存在）

**θ₄(r) = θ₄₀ · (1 - r(t))**

r(t)=可逆度，取值[0,1]。r→0（不可逆）时θ₄→θ₄₀最低，容易跳；r→1（完全可逆）时θ₄→∞，跳不动。

### 总表

| 跳变 | 阈值函数 | 核心变量 |
|---|---|---|
| L₁→L₂ | θ₁(s,p) = θ₁₀·g(s)·h(p) | 信号类型+个体敏感度 |
| L₂→L₃ | θ₂(a) = θ₂₀/a(t) | 体验异常度 |
| L₃→L₄ | θ₃(d): I·ε·H > θ₃(d) | 领域参数 |
| L₄→L₅ | θ₄(r) = θ₄₀·(1-r(t)) | 选择不可逆度 |

## 四、θ₃领域参数d标定

### d的定义

**d = C_exit(t) / C_stay(t)**

C_exit=退出规范的预期代价，C_stay=留在规范内的预期代价。d>1退出比留下贵，d<1退出反而更划算。d不是常数，随时间变化。

### 八领域标定

| 领域 | C_exit | C_stay | d | 说明 |
|---|---|---|---|---|
| 音乐 | 低（最差不好听） | 低（按谱弹） | ≈1 | 退出代价几乎为零，θ₃最低 |
| 烹饪 | 低（菜不好吃） | 低（按菜谱） | ≈1 | 容错带宽极宽 |
| 游戏设计 | 低（玩法崩了） | 中（规则稳定） | <1 | 涌现玩法经常比预设好玩，退出反而赚 |
| 建筑 | 高（结构塌了） | 中（按规范建） | >1 | 退出可能死人 |
| 体育竞技 | 中（输球） | 中（按战术） | ≈1 | 退出和留下代价差不多，看时机 |
| 宗教 | 高（被驱逐/异端） | 低（守戒律） | >>1 | 退出代价社会性死亡，θ₃极高 |
| 法律 | 高（冤案/司法越权） | 中（按条文判） | >1 | 退出代价是制度性风险 |
| 医学 | 极高（医疗事故/人命） | 中（按指南治） | >>>1 | 退出代价最高，θ₃最大 |

### d的三个区间

- d < 1（退出赚）：游戏设计。涌现玩法是常态，ε几乎不需要激活条件，自发跳到L₄
- d ≈ 1（退出和留下差不多）：音乐、烹饪、体育。ε激活看时机和个体，没有结构性阻力
- d > 1（退出贵）：建筑、宗教、法律、医学。ε激活需要更高的I·ε·H乘积才能越过θ₃(d)

### 关键推论

d解释了为什么有些领域创新是常态（音乐、游戏），有些领域创新是事件（宗教改革、医学突破）。不是人的ε不同，是d不同——同样的觉醒和退出意愿，在d>1的领域需要更强的H遮蔽才能触发跳变。

d的变化方向是社会结构变迁的指标——d下降的领域正在进入创新窗口期。

## 五、d变化速率函数化

### dd/dt = (Ċ_exit · C_stay - C_exit · Ċ_stay) / C_stay²

### 分子两项的物理含义

- Ċ_exit · C_stay：退出代价变化。Ċ_exit < 0（退出变便宜）→ dd/dt < 0 → d在降 → 创新窗口在开
- C_exit · Ċ_stay：留下代价变化。Ċ_stay > 0（留下变贵）→ 第二项变负 → dd/dt < 0 → d也在降

### 四种组合

| Ċ_exit | Ċ_stay | dd/dt | 含义 |
|---|---|---|---|
| <0（退出变便宜） | ≈0（留下不变） | <0 | 主动开窗：技术降门槛 |
| ≈0（退出不变） | >0（留下变贵） | <0 | 被动开窗：旧范式腐烂 |
| <0 | >0 | <<0 | 双驱动开窗：最剧烈的创新窗口 |
| >0（退出变贵） | <0（留下变便宜） | >0 | 关窗：范式收紧，创新被压制 |

### 创新窗口期定义

- 窗口打开：t_window = min{t : dd/dt(t) < 0}
- 窗口关闭：t_close = min{t > t_window : dd/dt(t) ≥ 0}
- 创新窗口期 = [t_window, t_close]

### 当前领域预判

- AI本地部署：C_exit快速下降（工具链成熟），C_stay上升（云服务锁定加深），dd/dt << 0，窗口大开
- 教育：C_stay上升（学历通胀），C_exit缓降（自学工具增多），dd/dt < 0，窗口正在开
- 医学：C_exit极高且稳定，C_stay微升，dd/dt ≈ 0，窗口还没开
