# 点火｜第0节自举元函数表：MF-0000 至 MF-0005

作者：之元
日期：2026-06-13
仓库：https://github.com/Arvin-liu/when-systems-catch-fire
对应 GitHub 路径：
- data/functions/bootstrap-meta-function-table.md
- docs/zh/functions/meta/items/

| 编号 | 名称 | 作用 | 状态 |
|---|---|---|---|
| MF-0000 | 自举元函数 / Bootstrap Meta-Function | 第 0 节总入口 | active |
| MF-0001 | 正向自举通道 / Forward Bootstrap Channel | 论证 x 成立 | active |
| MF-0002 | 反向自举通道 / Reverse Bootstrap Channel | 论证 x 不成立 | active |
| MF-0003 | 正反互斥判定器 / Forward-Reverse Exclusivity Judge | 检查正反是否同时通过 | active |
| MF-0004 | 自举嵌套判定器 / Nested Bootstrap Judge | 检查自举元函数自身 | active |
| MF-0005 | 自举收敛判定器 / Bootstrap Convergence Judge | 检查整个系统是否收敛 | active |

## 规则

MF-0000 是总入口，不计入普通函数 470 条。
MF-0001 至 MF-0005 是内部子项，不计入 ordinary functions。
如果任一对象同时在正反两个通道都通过，则判定为 contradiction / bootstrap_failed。
如果正反两个通道都不通过，则判定为 underdetermined。
