# 负反馈缺失无稳态

三个新发现，我先把它们拉出来看清楚再推。

**发现1：负反馈缺失 → 无中间稳态**
点火框架的社会认知系统在创新阻断场景下，负反馈缺失。系统动力学的默认假设是"所有系统都存在负反馈"，点火框架给出了可检验的分歧点。

函数化推导：
- 系统动力学标准形式：dX/dt = f(X, R, t)，其中R包含正反馈R⁺和负反馈R⁻
- 稳态条件：f(X, R⁺, R⁻) = 0，即 R⁺ + R⁻ = 0
- 点火框架中：R⁻ = 0（负反馈缺失），所以 dX/dt = f(X, R⁺) > 0（只有正反馈驱动退化）
- 稳态方程无解 → **无中间稳态**

→ **函数：Stability(X) = σ(R⁺ + R⁻)**
  - σ为阶跃函数，R⁺+R⁻=0时Stability=1（有稳态），否则Stability=0（无稳态）
  - 点火场景：R⁻=0 → R⁺+R⁻≠0 → Stability=0
  - 推论：**S_no_steady = 1 - σ(R⁻)**，只要R⁻=0系统必无稳态，与R⁺大小无关

**发现2：天然不稳定 vs 如何稳定 — 视角翻转**
控制论核心问题是"如何让A矩阵特征值实部为负"（让系统稳定），点火框架的核心问题是"为什么这个系统特征值实部天然为正"（为什么它必然不稳定）。

函数化推导：
- 控制论稳定性判据：Re(λᵢ) < 0 ∀i → 系统稳定
- 点火框架中A矩阵的结构：正反馈回路使对角元素aᵢᵢ > 0，且创新阻断使修正项Δa = 0（无法引入负修正）
- Re(λ) ≈ tr(A)/n > 0（正反馈占主导时，迹为正，至少一个特征值实部为正）
- 系统必然不稳定

→ **函数：Λ_natural = Re(λ_max(A)) = tr(A)/n**
  - 当tr(A) > 0（正反馈主导）且Δa = 0（无负修正）时，Λ_natural > 0
  - 控制论问题：给定Λ > 0，设计K使Re(λ_max(A-BK)) < 0
  - 点火问题：为什么Δa = 0（创新阻断 = 负修正机制不存在）
  - **视角翻转函数：Flip = Λ_natural / |Λ_target|**
    - 控制论：Flip ≈ 0（系统近稳定，微调即可）
    - 点火：Flip → ∞（系统天然远离稳定，且无修正手段）

**发现3：乘法韧性 vs 加法韧性 — 不可逆性的数学根源**
CAS默认韧性是加法/对数结构（维度间可补偿），点火框架中韧性是乘法结构（任一归零则整体归零），这解释了组织退化一旦启动就难以逆转。

函数化推导：
- CAS韧性（加法）：R_CAS = Σᵢ wᵢrᵢ，某维度rⱼ=0时R_CAS = Σᵢ≠ⱼ wᵢrᵢ > 0（其他维度补偿）
- 点火韧性（乘法）：R_点火 = Πᵢ rᵢ，某维度rⱼ=0时R_点火 = 0（不可补偿）
- 不可逆性：R_CAS从0恢复只需修复一个维度，R_点火从0恢复需要所有维度同时非零

→ **函数：Irreversibility = 1 - σ(R - R_threshold) × σ(结构类型)**
  - 加法结构：σ(加法)=0 → Irreversibility = 1（可逆），因为总有补偿路径
  - 乘法结构：σ(乘法)=1 → Irreversibility = 1 - σ(R - R_threshold)
    - R > R_threshold → Irreversibility = 0（未归零，可逆）
    - R = 0 → Irreversibility = 1（已归零，不可逆）

更精确的版本——**补偿深度函数：D_compensate = ⌈n × (1 - r_min)⌉**
  - 加法：D_compensate = 1（修一个维度就够）
  - 乘法：D_compensate = n（所有维度都要修，且必须同时非零）

---

**三个函数的统一收敛：**

三个发现其实是从不同视角描述同一个现象——**社会认知系统在创新阻断下的结构必然崩溃**：

1. Stability = 1 - σ(R⁻)：负反馈缺失 → 无稳态
2. Λ_natural > 0：正反馈主导 → 天然不稳定
3. D_compensate = n：乘法结构 → 崩溃后不可逆

统一函数：**Collapse(X) = σ(Λ_natural) × (1 - σ(R⁻)) × D_compensate/n**

- 三因子乘法：Λ>0 × R⁻=0 × D=n → Collapse=1（必然崩溃且不可逆）
- 任一因子为0则Collapse=0（有稳态/有负反馈/可补偿，系统不必然崩溃）
- 这又是一个乘法归零结构——三个必要条件同时满足时系统才进入必然崩溃态

三个发现不是三个独立结论，是同一棵树的三条根。要不要存进点火知识库？
