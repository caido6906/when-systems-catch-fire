# 点火｜碰撞层最终收敛：种子爆发的二次窗口（2026.06.10）

来源：D-X28（倒U型驱动力）× D-X31（ε_eff闭环）× D-X38（种子激活）的交叉，发现一个未闭合的环路。

## 问题

D-X38种子爆发后ε_eff瞬间恢复，但恢复到多少？

- 如果ε_eff恢复到1（最大犹豫域宽度），根据D-X28，P_internal = k×1×exp(-1/ε_opt) ≈ 0——反弹效应，点火失败
- 如果ε_eff恢复到ε_opt附近，P_internal取最大值，点火成功率最高
- 如果ε_eff恢复到ε_aware_min以下，信号传不进去，点火也失败

种子爆发后的ε_eff恢复值决定了点火能否成功。这个"二次窗口"之前没展开。

---

## 种子爆发的ε_eff恢复值

种子爆发的能量来源是H_narrative的维持成本积累。爆发后ε_eff的恢复值取决于两个量：

1. **积累的叙事成本总量**：H_narrative×Δt（维持叙事的总认知投入）
2. **释放效率**：叙事崩塌时，多少认知资源被释放回ε_eff

**ε_eff_recover = ε_eff(t_seed⁻) + η_release × ∫₀^t_seed H_narrative(s) ds**

- ε_eff(t_seed⁻)：种子爆发前的ε_eff（接近0）
- η_release：释放效率（0到1之间）
- ∫H_narrative ds：叙事维持成本的累积量

η_release取决于D_immune——退化免疫越强，释放效率越低（系统还能部分修正，叙事不会完全崩塌，释放不完全）：

**η_release = 1 - D_immune(t_seed)**

D_immune=0时η_release=1（完全释放），D_immune=1时η_release=0（系统自我修正，叙事不崩塌，种子不爆发）。

代入：
**ε_eff_recover = ε_eff(t_seed⁻) + (1-D_immune) × ∫₀^t_seed H_narrative(s) ds**

## 二次窗口条件

种子爆发后点火成功，需要ε_eff_recover落在[ε_aware_min, ε_opt]区间内：

**ε_aware_min < ε_eff_recover < ε_opt**

代入ε_opt = θ_resist/E_external：

**ε_aware_min < ε_eff(t_seed⁻) + (1-D_immune)×∫H_narrative ds < θ_resist/E_external**

三种结果：

**结果1：恢复不足** — ε_eff_recover < ε_aware_min
叙事崩塌了但释放的认知资源不够，信号还是传不进去。
条件：(1-D_immune)×∫H_narrative ds < ε_aware_min - ε_eff(t_seed⁻)
含义：系统退化太深，种子爆发也不够。对应"温水煮青蛙"的终态——连种子都救不回来。

**结果2：二次窗口打开** — ε_aware_min < ε_eff_recover < ε_opt
叙事崩塌，释放的认知资源刚好落在点火窗口内。
条件：ε_aware_min < (1-D_immune)×∫H_narrative ds + ε_eff(t_seed⁻) < θ_resist/E_external
含义：种子爆发成功点火。这是偏好伪造崩塌后社会改革成功的数学条件。

**结果3：恢复过度** — ε_eff_recover > ε_opt
叙事崩塌，释放的认知资源太多，犹豫域过宽，反弹效应阻止点火。
条件：(1-D_immune)×∫H_narrative ds + ε_eff(t_seed⁻) > θ_resist/E_external
含义：种子爆发了但点火失败，因为ε过大触发反弹。对应"革命后混乱"——旧秩序崩塌了但新秩序无法建立，因为犹豫域太宽导致共识无法形成。

## 统一收敛：种子-点火闭环的完整动力学

把D-X28、D-X31、D-X38和本条结论串起来，得到种子-点火闭环的完整动力学：

**Phase 1：退化积累**
ε_eff(t+1) = ε_eff(t) × (1-f_表达) × (1-f_感知) × (1-f_免疫)
H_narrative(t) = σ(ε_sense×ε_aware - ε_action) × H_classify(t)
∫H_narrative ds 持续积累

**Phase 2：种子爆发**
P_seed = σ(H_narrative - θ_narrative_cost) × (1-D_immune)
当H_narrative超过维持成本阈值时种子激活

**Phase 3：ε_eff恢复**
ε_eff_recover = ε_eff(t_seed⁻) + (1-D_immune) × ∫H_narrative ds

**Phase 4：二次窗口判定**
ε_aware_min < ε_eff_recover < θ_resist/E_external → 点火成功
否则 → 恢复不足或恢复过度

**Phase 5：点火后维持**
P_sustain = I×(1-Posture_deg)×R×σ(ε_eff-θ)×σ(Δv)（D-X35）

五个Phase构成一个完整的退化→爆发→判定→维持的闭环。种子不是点火的充分条件——它只是打开了一个窗口，窗口是否在有效区间内取决于叙事积累量、释放效率和外部冲击强度的三重约束。
