# 点火｜认知信道统一函数：信息茧房的形成机制与破壁条件（2026.06.10）

## 核心问题

信息茧房的形成机制是什么？为什么一旦形成极难打破？破壁需要什么条件？

## 建模基础

信息茧房 = 信源选择（静态筛选）× 谣言传播（动态灌入）的闭环运行。信源选择函数决定信道开闭状态，谣言传播函数决定信号在已开信道上的通量，合在一起是自增强的认知信道系统。

## 一、认知信道带宽

W(L, t) = Σᵢ P_accept(Sᵢ, L) × B(Sᵢ)

对所有信源求和。W 越大信息频谱越宽，W 越小信道越窄。

- L₁-L₂：P_accept(local) ≈ 1, P_accept(expert) ≈ 0 → W ≈ B_local（带宽极窄）
- L₄-L₅：P_accept(local) ≈ 0.3, P_accept(expert) ≈ 0.9 → W ≈ 0.3·B_local + 0.9·B_expert（宽频）

**茧房的数学定义：** W(L, t) < W_critical

## 二、自增强回路——茧房的核心机制

认知框架更新函数：

θ_align(S, L, t+1) = θ_align(S, L, t) + η · P_accept(S, L, t) · [θ_align^target(S) - θ_align(S, L, t)]

η = 框架更新率。越接受某类信源，认知框架越对齐这类信源，下次接受概率更大——正反馈。

**L₁-L₂ 的锁死回路：** 只接受 local → 框架被 local 塑造 → θ_align(expert) 进一步降低 → P_accept(expert) 进一步降低 → 更不接受专业信息 → 循环。

**L₄-L₅ 的开放稳态：** ε 充足，能主动检测 θ_align 是否过度偏向 → 退出权提供负反馈 → 收敛到宽频稳态。

**关键差异：ε 是茧房自增强回路的唯一制动器。**

## 三、茧房深度动力学

茧房深度 Φ_cocoon(L, t) = 1 - W(L, t) / W_max，∈ [0, 1]

dΦ/dt = η · P_accept(local, L) · Φ · (1 - Φ) + Σⱼ βⱼ(L) · Iⱼ · (1 - Φ) - ε(L) · Φ

三项：
- 自增强项：logistic 增长，Φ 越大增长越快
- 谣言加速项：谣言传播直接推动茧房化
- 退出权制动项：ε 是唯一的负反馈

**稳态分析：**
- ε ≈ 0（L₁-L₂）：制动项消失，Φ → 1（完全茧房化），唯一稳态
- ε 中等（L₃）：Φ 稳定在中间值，谣言冲击可暂时推高
- ε 充足（L₄-L₅）：Φ → 0（开放稳态），谣言冲击被快速吸收

## 四、茧房刚性——为什么极难打破

茧房刚性 K_cocoon(L) = R(S_expert, L) / C(S_expert, L) = [F_form(L) + ε(L)] / [B_expert × D_expert(L) × θ_align(expert, L)]

L₁-L₂ 层：D_expert ≈ 0 → 分母趋近于 0；F_form 高、ε ≈ 0 → 分子大；θ_align(expert) 因自增强持续降低 → 分母持续缩小。

**K → ∞。不是"很难打破"，是"在现有信道结构下不可能打破"。** 单纯加大辟谣力度（增大 B_expert）无效：B_expert × D ≈ 0 还是 0。

## 五、破壁条件

破壁概率函数：

P_break(L) = D_expert(L) × ε(L) × [θ_align(expert, L) / F_form(L)]

四因子乘法关系，与统一跳变函数 P(L₃→L₅) = σ × A × D × |M_cog<0| 结构完全同构。任何一个为 0，破壁概率为 0。

**四条破壁路径：**

1. **装 D**（最根本）：专业信息翻译进 D=1 的信道，同时逐步引入专业框架元素，让 D 从 0 缓慢上升。没有 D，其他路径的信号都进不去。
2. **装 ε**（制动器）：让人意识到"我可以不信身边人说的"。没有 ε，即使信号进去了也会被自增强回路推回。
3. **降 F_form**（加速器）：专业信源放弃权威姿态，用平等对话。
4. **提升 θ_align**（加速器）：从接收者已有概念出发，搭建通往专业概念的桥梁。

**依赖关系：** 路径1是前提，路径2是制动，路径3和4是加速器。

## 六、统一认知信道函数

Ψ(L, t) = W(L, t) × [1 - Φ_cocoon(L, t)] × Γ(L, t)

- W(L, t) — 认知信道带宽（信源选择函数决定）
- [1 - Φ_cocoon(L, t)] — 信道开放度（茧房深度的补数）
- Γ(L, t) — 免疫强度（谣言传播函数决定）

Ψ 是认知信道健康度指标。Ψ → 0 = 完全茧房化 + 免疫丧失；Ψ → max = 宽频开放 + 免疫充足。

**核心结论：Ψ 的稳态由 ε 唯一决定。** ε = 0 → Ψ → 0（唯一稳态是茧房）；ε > 0 → Ψ > 0（存在开放稳态）。ε 是整个认知信道系统的相变参数。

## 七、三个函数的统一关系

| 函数 | 核心变量 | 解决的问题 |
|------|---------|-----------|
| 信源选择 | P_accept(S, L) | 为什么信这个不信那个 |
| 谣言传播 | R₀(L) = β/γ | 为什么反复爆发、怎么消除 |
| 认知信道 | Ψ(L, t) = W × (1-Φ) × Γ | 茧房怎么形成、怎么破壁 |

三个函数共享同一组底层参数（D, ε, θ_align, F_form），由 ε 做相变控制。
