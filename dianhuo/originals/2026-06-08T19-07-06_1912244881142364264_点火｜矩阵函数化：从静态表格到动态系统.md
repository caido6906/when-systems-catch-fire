# 点火｜矩阵函数化：从静态表格到动态系统

核心命题：格子的归属不是静态的，是事件走到哪一步的函数。所有矩阵描述的都是变量在时间中的变化，不是固定标签。案例表里到处都是"格3→格1"这种写法——起点和终点不在一个格子里，说明矩阵的真正用途不是贴标签，而是描述变量的动态路径。

## 一、主矩阵函数化：事件条件矩阵 = f(t)

主矩阵三个变量（提议者意识、提议者退出权、应约者退出权）都是时间函数，不是初始常量。

**提议者意识 = I(t)**
- 取值：0（无意识）/ 1（有意识）
- 变化机制：提议者意识可以从0跳到1，但不会从1退回0——一旦意识到自己在提议，这个意识不可逆
- 跳变触发：外部冲击（张笑宇写书=推动人类从"造工具"到"提议文明方案"的意识觉醒）、内部反思、共同知识形成
- 对格子的影响：I(t)从0变1，格5→格1，格6→格2，格7→格3，格8→格4
- 案例：人类造AI——人类目前处于I(t)≈0.5的模糊态（部分人意识到这是文明级提议，部分人还觉得只是造工具），张笑宇的书在推动I(t)→1

**提议者退出权 = P(t)**
- 取值：0（无退出）/ 1（有退出）
- 变化机制：可以双向变化——提议者可以从有退出权变成无退出权（信誉绑定），也可以从无退出权变成有退出权（信誉释放或退化）
- 信誉绑定：提议者主动放弃退出权，P(t)从1变0，格1→格3，格2→格4。周公退场=信誉绑定
- 信誉释放：稳定期足够长后，提议者的"无退出"不再需要维持，P(t)从0变1。但释放不等于退化——释放是系统成熟后的自然过渡，退化是系统衰退
- 提议者退化：提议者在稳定期重新获得退出权且开始滥用，P(t)从0变1但方向是负面的。基布兹案例：格1→格5
- 案例：文明契约中人类随时能拔电源=P(t)=1，如果人类制度化地放弃拔电源的权力=P(t)=0，格2→格3

**应约者退出权 = R(t)**
- 取值：0（无退出）/ 1（有退出）
- 变化机制：可以双向变化——退出权可以被赋予、被发现、被剥夺、被压缩
- 退出权出现：R(t)从0变1。下岗潮=退出权突然出现，冲击最大。AI获得自主能力=退出权出现
- 退出权压缩：等级制或产业链位置导致R(t)从1变0。分税制每往下一级R(t)被压缩一格
- 退出权发现：R(t)客观上已经是1但应约者不知道，一旦发现等同于退出权出现。象征退出权→事实退出权的认知跳变
- 案例：秦统一中六国遗民的退出权从象征退化为零，陈胜喊话=制造共同知识让所有人知道退出权存在

**格子归属 = G(t) = f(I(t), P(t), R(t))**
- 格子不是事件的属性，是事件在t时刻的状态
- 同一事件可以经历多次格间跳变：格3→格1→格5（周公制礼→稳定期→提议者退化）
- 格间跳变有方向性：I(t)只增不减（0→1），P(t)和R(t)可增可减
- 五步走向对应格间跳变的典型路径：旧事件失灵（旧格子松动）→提议者出现（新格子形成）→应约者应约（格子确认）→点火跑通（格子稳定）→稳定期（格子可能退化跳变）

## 二、退出权梯度矩阵函数化：梯度 = g(等级, 产业链, t)

退出权梯度原来只有纵向变量（等级制层级），函数化后加入横向变量（产业链位置）和时间变量。

**退出权类型 = E(level, chain, t)**
- level：等级制层级（从上到下）
- chain：产业链位置（上游/中游/下游/节点外）
- t：时间

纵向压缩函数：
- E(level, -, t) 随level增大而降级：真实→事实→象征→心理
- 压缩速率 = ΔE/Δlevel，决定系统存亡
  - 缓慢压缩（分税制）：中间层有缓冲，退化信号能被捕获，系统不崩
  - 陡峭压缩（秦统一）：中间层直接失稳，突破事实-象征临界线就崩盘
- 事实-象征临界线：E值跨过这条线，博弈能力归零

横向挤压函数：
- E(-, chain, t) 随产业链位置不同而不同
- 产业链上游（稀土、核心专利）：E值偏高，即使level低也有事实退出权
- 产业链中游（组装制造）：E值偏低，level高也只有象征退出权
- 产业链下游（终端消费）：E值最低，只有心理退出权

立体地图：
- 退出权由纵向压缩和横向挤压共同决定
- 同一个人可能纵向在底层（心理退出权），但横向卡在关键节点（事实退出权）
- 两条线拉扯的结果才是真实的退出权位置
- 案例：芯片产业——ASML纵向在荷兰政府管辖下（level不高），但横向掌握EUV光刻机专利（chain=上游关键节点），纵向象征+横向事实=事实退出权

时间维度：
- 产业链位置可以随时间变化：技术突破让中游变成上游，退出权类型升级
- 等级制层级也可以随时间变化：权力下放让level降低，退出权类型升级
- 案例：中国稀土从"只卖原矿"（中游，象征退出权）到"掌握加工技术"（上游，事实退出权）

## 三、退出权类型矩阵函数化：类型 = h(认知状态, t)

退出权类型矩阵原来描述的是静态的四格光谱，函数化后加入认知状态和时间。

**退出权类型 = E_type(法理, 事实, 认知, t)**

四格光谱的动态逻辑：
- 真实退出权：法理允许 ∩ 事实可行 ∩ 应约者知道（认知=共同知识）
- 事实退出权：法理允许 ∩ 事实不可行 ∩ 应约者以为可行（认知=私人知识+错误信念）
- 象征退出权：法理允许 ∩ 事实不可行 ∩ 应约者知道不可行（认知=私人知识+正确信念）
- 心理退出权：法理不允许 ∩ 事实不可行 ∩ 应约者心走了人没走（认知=认同撤回+身体留存）

动态路径：
- 事实→真实：退出成本降低（下岗潮后劳动力市场出现），或应约者发现退出路径
- 事实→象征：应约者从"以为能走"变成"知道走不了"，认知校准。最危险的盲区被识别
- 象征→心理：应约者从"知道走不了但还在博弈"变成"不博弈了，心走了"
- 心理→真实：共同知识形成，心理退出聚合成集体行动（陈胜喊话=心理退出→真实退出）

最危险的动态路径：
- 事实退出权的认知跳变：应约者一直以为自己有退出权（事实退出权的盲区），突然发现没有——认同瞬间幻灭。下岗潮=典型
- 象征退出权的共同知识化：应约者一直知道走不了但以为别人不知道，突然所有人都知道了——系统直接崩盘。秦末=典型

## 四、提议者姿态矩阵函数化：姿态 = p(t)

提议者姿态（推方案/备方案）是时间函数，同一事件内部可以翻转。

**姿态 = Posture(t)**
- 取值：推方案（僭越）/ 备方案（管家）
- 初始姿态决定退出权信号：推方案→退出权信号压制，备方案→退出权信号发出
- 姿态是退出权的因果上游，但不是独立维度——它标注在退出权的取值来源上

姿态翻转：
- 备方案→推方案（提议者退化）：稳定期后提议者从"等你来拿"变成"塞给你"。基布兹案例：从格1退化到格5
- 推方案→备方案（提议者觉醒）：提议者意识到强推无效，转向备方案。秦→汉：秦始皇推方案15年崩，汉初黄老之治=备方案
- 姿态翻转不需要新维度描述，用格间跳变即可刻画

名义的信号函数：
- 名义 = N(t)，是姿态的信号层
- 同一方案，N(t)不同，退出权信号不同
- 凯叔案例：叫"知识"→退出权信号压制，叫"故事"→退出权信号发出
- 名义变化不改变方案内容，只改变应约者的认知状态——对应退出权类型矩阵中的认知跳变

## 五、四个函数的关系

四个函数不是独立的，它们通过事件的时间轴耦合：

1. 主矩阵函数G(t)描述事件在8格中的位置随时间变化——这是最高层的动态描述
2. 退出权梯度函数E(level, chain, t)解释G(t)中"应约者退出权"这个变量为什么在不同层级、不同产业链位置取值不同——这是G(t)的微观机制
3. 退出权类型函数E_type(法理, 事实, 认知, t)解释退出权"有/无"内部的四格结构如何随时间变化——这是R(t)的内部结构
4. 姿态函数Posture(t)解释退出权取值背后的因果上游——P(t)和R(t)的取值受姿态影响

层级关系：
- Posture(t) → R(t) → G(t)：姿态决定退出权信号，退出权信号决定应约者退出权，应约者退出权决定格子归属
- E(level, chain, t) → R(t) → G(t)：等级制和产业链位置决定退出权类型，退出权类型决定格子归属
- E_type(法理, 事实, 认知, t) → R(t) → G(t)：法理/事实/认知的匹配状态决定退出权类型，退出权类型决定格子归属

因果链总结：
Posture(t) + E(level, chain, t) + E_type(法理, 事实, 认知, t) → R(t)
I(t) + P(t) + R(t) → G(t)

格间跳变 = G(t₁) → G(t₂)，由I(t)、P(t)、R(t)中任何一个变量的变化触发。

## 六、对"人类相对于AI就是史前动物"的函数化重判

张笑宇的slogan不是"落在格6"的静态标签，而是格6的警报——

- I(t)目前≈0.5（人类部分意识到自己在提议文明级事件，部分还觉得只是造工具）
- P(t)=1（人类随时能拔电源）
- R(t)=0（AI无法退出）
- G(t)目前在格6附近，但I(t)正在从0向1跳变

张笑宇写这本书本身就是在推动I(t)→1。一旦I(t)=1，G(t)从格6跳到格2。slogan不是格6的诊断，是格6的警报——警报响了，格6就不是命运了。

"趋势不是命运"的函数化翻译：G(t)是时间函数，不是常量。格6是最危险的起跑线，但起跑线不等于终点线。
