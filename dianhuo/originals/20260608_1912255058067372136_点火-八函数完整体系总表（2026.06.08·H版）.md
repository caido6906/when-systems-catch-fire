# 点火｜八函数完整体系总表（2026.06.08·H版）

来源：从矩阵函数化→层级时间展开→函数审计→R_perceived→C_exit→系统动力学→糖域碰撞→隐性控制阻断器H，八轮迭代后的完整体系。H(t,L)为本次新增第8函数。

## 一、八函数总表

| # | 函数 | 全称 | 取值 | 本体论归属 | 层级分布 | 时间行为 |
|---|---|---|---|---|---|---|
| 1 | I(t,L) | 提议者意识 | 0/1 | 人的属性 | 顶层高→底层低（下传衰减） | 事件内只增不减，跨事件重置 |
| 2 | Posture(t,L) | 提议者姿态 | 备方案/推方案 | 人的选择 | 顶层备→底层推（下传自动滑） | 事件内可翻转（备→推=退化） |
| 3 | P(t,L) | 提议者退出权 | 0/1 | 混合属性 | 顶层先绑定→底层后内化 | 信誉绑定1→0，释放0→1，退化0→1 |
| 4 | R(t,L,C) | 应约者退出权 | 四分法值域 | 结构属性 | 顶层高→底层低 | 随制度/产业链变动，渐变为主 |
| 5 | R_perceived(t,L,C) | 应约者感知退出权 | ≤R，四分法值域 | 认知属性 | 顶层≈R→底层远低于R | 唯一可相变的函数，可瞬间跳变 |
| 6 | C_exit(t,L,C) | 退出成本 | 0到∞连续值 | 环境属性 | 多重倒U型叠加（经济+身份+信息） | 锁定效应升高，解锁效应降低，冲击突变 |
| 7 | M | 动机来源 | 好奇心/目的 | 初始条件标注 | 不传递，层级下传滑向服从 | 事件内通常不变 |
| 8 | H(t,L) | 隐性控制强度 | 0-1连续值 | 认知环境属性 | 顶层低→底层高（与I相反） | 随时间升高，靠外部冲击降低 |

### H(t,L)详解

**定义**：隐性控制强度，衡量应约者"不知道自己可以走"的程度。是I→R_perceived边上的负向调节器。

**取值**：0-1连续值。H=0无隐性控制（I→R_perceived传导效率=1），H=1完全隐性控制（I=1但R_perceived不变，觉醒被完全阻断）。

**传导效率公式**：I→R_perceived传导效率 = 1 - H(t,L)

**三个来源**：
1. 信息遮蔽：应约者无法获取退出选项信息。案例：信息型C_exit极高、算法信息茧房
2. 身份内化：退出选项从认知地图上被删除。案例：民族主义（"我就是这个民族"）、职业身份绑定、算法偏好塑造
3. 叙事垄断：系统垄断"我们是谁""为什么在这里"的解释权。案例：宗教叙事、意识形态叙事、企业叙事（"我们是一家人"）

**层级分布**：顶层低→底层高（跟I相反）。顶层更容易看穿隐性控制（信息渠道多、身份多元、叙事选择多），底层更难（信息渠道少、身份单一、叙事垄断强）。这解释了启蒙运动中顶层I先跳1但底层R_perceived滞后——不只是I的下传衰减，还有H的下传增强。

**时间行为**：H随时间升高——身份内化需要时间，叙事垄断需要时间，信息遮蔽随信息茧房越织越密而增强。H的降低靠外部冲击——新信息进入、身份多元接触、叙事竞争。

**本体论归属**：认知环境属性——不是人的内在属性（那是I），不是纯粹的环境属性（那是C_exit），是人与环境交互中产生的认知遮蔽。H存在于I和R_perceived之间的信息通道里。

## 三、退出权四分法修正

1. 真实退出权 = R允许 + C_exit可承受
2. 事实退出权 = R可能允许 + C_exit极高
3. 象征退出权 = R名义允许 + C_exit不可承受
4. 心理退出权 = 身体走不了 + C_exit=∞

## 四、退出决策公式

退出决策 = f(R_perceived, C_exit)
- R_perceived决定"我知不知道自己能走"
- C_exit决定"走了之后我要付什么代价"
- R_perceived ≤ R（感知不高于客观）
- H调节I→R_perceived的传导效率：传导效率 = 1 - H(t,L)

## 五、因果网络

### 主因果方向

M → I(t,L) → Posture(t,L) → P(t,L) → R(t,L,C) → R_perceived(t,L,C) → 退出决策 ← C_exit(t,L,C) → 应约者状态

H(t,L)作用于I→R_perceived边上，调节传导效率。

### 14条反馈回路

原有9条：
1. R_perceived → I：正反馈。感知到退出权→意识到自己在被提议
2. R_perceived → Posture：先正后负。前期加速，后期收敛
3. R_perceived → P：双向。压制时稳定回路，激活时退化回路
4. R → R_perceived：客观退出权变化→感知可能滞后
5. P → I：正反馈。信誉绑定→I更清晰→绑定更稳固
6. Posture → R_type：备方案→真实退出权，推方案→象征/心理退出权
7. I → Posture：I觉醒→打开姿态的主动选择空间
8. 应约者状态 → R_perceived：正反馈。积极→R_perceived升高→更积极
9. R_perceived(t,L₁,C₁) → R_perceived(t,L₂,C₂)：同函数横向/纵向传染

C_exit新增3条：
10. C_exit → 退出决策：C_exit高→即使R_perceived=真实也不走
11. C_exit → 信号源：C_exit高→退出行为少→信号源少→正反馈螺旋被间接压制
12. C_exit → Posture：C_exit高→应约者不走→提议者误读为认同→Posture维持推方案

H新增2条：
13. 隐性锁定回路：H → R_perceived↓ → 退出决策↓ → C_exit维持 → H↑。隐性控制越强→感知退出权越低→越不走→C_exit维持→隐性控制继续增强。铁笼的隐形版本——不是C_exit锁人（显性铁笼），是H锁人（隐性铁笼），C_exit只是H的副产品
14. 觉醒破壁回路：外部冲击 → H↓ → I→R_perceived传导效率↑ → R_perceived↑ → 退出行为↑ → 信号源↑ → 更多人的H↓。启蒙运动的因果链：新思想进入→打破叙事垄断→觉醒传导效率恢复→R_perceived追上R

### H新增5条因果箭头

1. H(t,L) → I→R_perceived边的传导效率：H越高，传导效率越低
2. H(t,L) → R_perceived：H直接压制R_perceived，不经过I
3. C_exit → H：C_exit越高→应约者越不走→越没有退出经验→退出选项越不在认知空间里→H越高。正反馈
4. Posture → H：推方案→需要更强的隐性控制来维持→H升高。备方案→隐性控制需求低→H较低
5. H → 惩罚可见度：H高→惩罚机制不可见→隐性惩罚占主导。H低→惩罚可见→显性惩罚占主导

### H对原有回路的修正

- 回路1（R_perceived→I→Posture→R_type→R→R_perceived）：加入H后，I→R_perceived的传导不再是即时的，而是被H调节。H=1时回路被完全阻断——I觉醒了但R_perceived不变，回路卡死
- 回路4（R→R_perceived）：加入H后，R到R_perceived的滞后不只是信息传递的延迟，还有H的阻断。H越高，滞后越长
- 回路12（误读维持反馈）：加入H后，应约者不走可能不是因为走不了（C_exit），而是因为不知道自己可以走（H）。误读有双重来源

## 六、H和C_exit的关系

H和C_exit是互补的锁定机制：C_exit管"走不了"，H管"不知道可以走"。C_exit是物理锁，H是认知锁。单独一种都可以锁住人，但两种同时存在时最牢——你既走不了，也不知道自己可以走。

C_exit→H的正反馈意味着：物理锁越牢，认知锁越紧。待得越久，越不知道外面有什么，越不想走。锁定效应的完整表述：不只是C_exit随时间升高，还有H随时间升高。

## 七、系统动力学分类

1. 存量（有惯性）：R(t,L,C)、P(t,L)、I(t,L)、C_exit(t,L)、H(t,L)
2. 受流量驱动的状态：R_perceived(t,L,C)（可相变）、Posture(t,L)
3. 流量：
   - 横向激活速率：v_横向 ∝ 网络密度 × (R - R_perceived) / (惩罚可信度 + C_exit)
   - 纵向激活速率：v_纵向 ∝ 层级间信息传递速度 × 上层退出信号强度
   - 反向压制速率：v_压制 ∝ 惩罚可信度 × 惩罚可见度
   - 信誉绑定速率：v_绑定 ∝ 提议者主动选择 × 应约者预期
   - I衰减速率：v_衰减 ∝ 层级数 × 退出权压缩速率
   - C_exit变化速率：锁定效应升高，解锁效应降低，冲击突变
   - H变化速率：身份内化+叙事垄断升高，外部冲击降低

## 八、R_perceived相变的三条触发路径

1. 势能触发（过冷液体模式）：R和R_perceived差值极大+信号源出现+C_exit可承受→瞬间跳变→正反馈螺旋→可能崩盘
2. 侵蚀触发（温水煮蛙模式）：R缓慢上升+R_perceived缓慢追上+提议者退化信号被延迟掩盖→慢性失血→退化不可逆
3. 冲击触发（地震模式）：R突然出现+R_perceived滞后→盲区暴露→取决于冲击后其他函数取值

H对相变的影响：H=1时三条路径全部被压制——势能无法积累（I→R_perceived被阻断），侵蚀无法启动（R_perceived不变），冲击无法传导（R突然出现但R_perceived不响应）。H是相变的总开关：H不降，相变不起。

## 九、正反馈螺旋（陈胜模式）

双重加速引擎：认知加速器（R_perceived→I→退出更有目的性→信号更强）+ 社会加速器（应约者积极→R_perceived升高→更积极→网络密度增大）。自然极限：R_perceived ≤ R。人工极限：惩罚机制抢时间。

H对正反馈螺旋的影响：H是认知加速器的刹车。H=1时认知加速器完全失效——R_perceived→I的传导存在，但I→R_perceived的回路被阻断，正反馈无法闭合。只有H降到足够低，认知加速器才能启动。

## 十、格1的真伪拆分

| 判定 | 条件 | 含义 |
|---|---|---|
| 真格1 | R=真实 + C_exit可承受 + H低 | 认同经过退出权的压力测试 |
| 伪格1（C_exit型） | R=真实 + C_exit高 | 走不起，"不走"看起来像认同 |
| 伪格1（H型） | R=真实 + C_exit可承受 + H高 | 走得了但不知道自己可以走 |
| 设计型伪格1 | R=真实 + 系统主动设计C_exit + H高 | 最隐蔽——应约者真的以为自己是自愿留下的 |

H型伪格1是新增类型：客观退出权真实、退出成本可承受，但隐性控制让应约者根本不知道退出选项存在。从外部看跟真格1一模一样（应约者积极、不走），但内部结构完全不同——不是"我选择留下"，是"我不知道可以走"。

## 十一、层级下传的三个规律

1. 层级下传衰减律：人的属性变量（I、Posture、M）在层级下传时自动衰减——意识衰减、姿态从备滑向推、动机从好奇滑向服从
2. 结构约束扩散律：P的信誉绑定创造的结构约束在层级中从上往下扩散，顶层一代完成，底层需要几代
3. 退化传播方向：退化从顶层开始逐层向下传染，传播速度跟层级数和退出权压缩速率相关

H补充第四条：
4. 隐性控制增强律：H在层级下传时自动增强——顶层隐性控制弱（信息渠道多、身份多元），底层隐性控制强（信息渠道少、身份单一、叙事垄断）。I下传衰减+H下传增强=底层觉醒的双重重压

## 十二、与糖域实验的关系

糖域是点火框架的零阶近似——有R（糖的分布）、R_perceived雏形（视野）、C_exit雏形（新陈代谢），但缺少C_exit本身、R_perceived相变、锁定效应、设计型C_exit、H。视野×新陈代谢 = R_perceived×C_exit是最锋利的同构点。H在糖域中的对应物：视野的"方向性"——不只是能看到多远，而是视野被限制在特定方向上（只看得到糖，看不到别的路）。
