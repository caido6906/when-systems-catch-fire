# 柬埔寨金边中国公民遇害案件详情及进展

### **🔍 案件基本情况**

**案发信息**
- **时间**：2026年5月30日（遗体发现时间），5月29日晚（失踪时间）。
- **地点**：柬埔寨金边市朗哥区。
- **受害者**：52-53岁中国籍杨姓男子，当地地产公司负责人。
- **死因**：谋杀（生前遭非法控制、殴打与虐待）。

### **📅 事件时间线**

| 时间节点 | 关键事件 |
| :--- | :--- |
| **5月29日 20:16** | 受害者在公寓停车场被3名不明身份男子强行带走 |
| **5月30日 3:00左右** | 受害者手机向妻子发送信息，要求支付200万美元赎金 |
| **5月30日 10:30左右** | 货车司机发现白色丰田普锐斯轿车及后座死者遗体并报警 |
| **5月31日 8:47** | 嫌疑人发信息称“事情到此为止”，随后失去联系 |
| **5月31日 11:30左右** | 受害者妻子接警方通知，证实丈夫遇害 |
| **6月1日** | 中国驻柬埔寨大使馆发布消息，敦促柬方全力侦办 |

### **💬 各方响应与调查进展**

#### **(一) 中国驻柬使馆行动**
- **介入时间**：第一时间向柬警方核实情况。
- **工作对象**：柬政府、警察总署和金边市警察局。
- **核心要求**：查明真相、缉捕嫌犯、依法严惩、维护中国公民安全。
- **家属联系**：已与遇害者家属建立联系，密切跟踪案件进展。

#### **(二) 柬埔寨方面回应**
- **态度**：柬有关部门表示高度关注此案。
- **行动**：正全力推进案件侦办工作，已立为蓄意谋杀案。

#### **(三) 案件调查方向**
- **赎金情况**：嫌疑人曾索要200万美元（约合1360万元人民币）赎金。
- **潜在动机**：警方调查发现受害者自2014年起因生意失败与人产生债务纠纷，2025年对方曾上门追讨，双方矛盾持续多年，案件或与商业纠纷有关。

### **📝 补充细节**
- **案发现场**：遗体在一辆白色丰田普锐斯轿车后座被发现，现场已用警戒线隔离，有警务人员及相关人员在场。
- **信息来源**：中国驻柬埔寨大使馆消息、南方都市报报道。
