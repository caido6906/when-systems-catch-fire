# 点火｜ε×行为经济学：路径A'与系统1自动判断的改造

## 核心碰撞：路径A'改造的是系统1的自动判断，不是系统2的理性计算

行为经济学的核心：人的决策不是完全理性的，受认知偏差和启发式判断的影响。卡尼曼区分系统1（快速、自动、直觉）和系统2（缓慢、费力、理性）。前景理论说人对损失的敏感度远高于对收益的敏感度——损失厌恶。

996员工用系统2算"我应该换工作"——算得出来，凯利公式的最优解很清楚。但系统1的自动判断已经被路径A'改造了——"累是正常的"已经写进了直觉层面的启发式判断，你不需要思考就自动把疲劳归类为"正常"，不是"该退出的信号"。

## 损失厌恶与路径A'的叠加

退出被视为损失（放弃已有的稳定、收入、身份），留下被视为维持现状。损失厌恶让人天然倾向于不退出——而路径A'改造后的先验模型又让你感知不到退出的收益。两重偏差叠加：高估退出的损失（损失厌恶），低估退出的收益（路径A'感知失真）。

## ε_aware的操作：用系统1对抗系统1

不要试图用系统2对抗系统1——用系统1对抗系统1。跟996员工讲凯利公式的最优解（系统2），他点头同意但不会行动——因为系统1的自动判断没变。更有效的方式是制造一个直觉层面的退出信号——让他直接体验不加班的生活（哪怕一天），让身体重新产生退出信号的预测误差，用新的直觉经验覆盖旧的自动判断。

## 碰撞总结

| | 行为经济学 | 路径A' |
|---|---|---|
| 改造对象 | 系统1（自动判断）vs 系统2（理性计算） | 路径A'改造系统1，不是系统2 |
| 损失厌恶 | 退出被视为损失 | 损失厌恶+感知失真=双重偏差叠加 |
| ε_aware的操作 | — | 用系统1对抗系统1，不是用系统2说服系统1 |

核心判断：路径A'改造系统1的自动判断，损失厌恶叠加感知失真让退出更难。ε_aware要用系统1对抗系统1。
