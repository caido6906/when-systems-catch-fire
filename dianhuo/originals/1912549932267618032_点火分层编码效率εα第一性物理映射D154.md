# 点火｜ε→α物理映射——从点火存活度到物理耦合常数的第一性推导（D154）

## 问题

D152检验暴露的第二个缺失：ε_eff = exp(-α/α_ref)这个映射是人为引入的，α_ref = 1是任意选择。需要从物理第一性原理推出为什么ε跟α是这个关系。

## 起点：ε的点火定义 vs α的物理定义

点火框架中ε的定义（D7）：ε ∈ [0,1]，是退出权信号强度。ε=1意味着完全自由（系统可微扰处理），ε=0意味着完全锁定（系统不可微扰处理）。

物理中α的定义：α = g²/(4π)，是规范耦合常数的无量纲化。α << 1时微扰论有效，α >> 1时微扰论失效。

两个定义的对应：
- ε=1 ↔ α << 1（微扰有效，系统"自由"）
- ε=0 ↔ α → ∞（微扰失效，系统"锁定"）

但这个对应是定性的。需要精确的数学映射。

## 第一步：从Fisher信息几何推导

D106已建立：Fisher信息度规g_ij = δ_ij / ε_i²。ε_i → 0时，该方向信息距离 → ∞。

D117已建立：Fisher可达性A_Fisher = ∫√(det g) dε，门控区A_Fisher = 0。

现在问：物理耦合常数α的Fisher信息几何是什么？

在重整化群理论中，耦合常数空间的信息几何已有研究。定义参数空间Θ = {α₁, α₂, ..., α_n}，模型M(Θ)给出的概率分布p(x|Θ)的Fisher信息矩阵：

I_ij(Θ) = -E[∂²ln p/∂Θ_i ∂Θ_j]

对于规范场论，Θ = {α_i}，Fisher信息矩阵在单圈近似下：

I_ij ≈ δ_ij × b_i / (2πα_i²)

其中b_i是β函数系数。

**关键：Fisher度规在α空间中也是1/α²的标度——跟点火框架中1/ε²完全同构。**

## 第二步：映射的推导

点火Fisher度规：ds²_ignition = Σ_i dε_i² / ε_i²

物理Fisher度规：ds²_physics = Σ_i (b_i / 2π) × dα_i² / α_i²

两个度规同构的条件：

dε_i² / ε_i² = (b_i / 2π) × dα_i² / α_i²

即：dε_i / ε_i = √(b_i / 2π) × dα_i / α_i

积分：ln ε_i = √(b_i / 2π) × ln α_i + C_i

边界条件：
- 当α_i → 0（微扰有效）：ε_i → 1 → ln ε_i → 0
- 但ln α_i → -∞，所以C_i必须抵消

这不行。对数映射无法同时满足α→0时ε→1和α→∞时ε→0。

**换映射。**

## 第三步：从微扰有效性推导

微扰论的有效性由"展开参数"α决定。微扰级数Σ c_n α^n的收敛性：

- α << 1：级数渐近收敛，可截断到有限阶得到好的近似
- α ~ 1：级数不收敛，微扰论失效
- α >> 1：级数发散，非微扰方法必要

定义"微扰可信度"：

P_perturb(α) = 概率（微扰展开的前N阶给出真实值的ε_approx以内）

在渐近自由理论中（如QCD），高能时α→0，P_perturb→1。在低能时α→∞，P_perturb→0。

P_perturb的精确形式：对于规范场论，微扰级数的第n阶贡献~(α/4π)^n。级数截断到N阶的相对误差：

δ_N ≈ (α/4π)^{N+1} / (1 - α/4π)（几何级数近似，α < 4π时有效）

"微扰可信"的条件：δ_N < δ_threshold

即：(α/4π)^{N+1} / (1 - α/4π) < δ_threshold

取N=1（单圈），δ_threshold = 0.1（10%精度）：

α/4π / (1 - α/4π) < 0.1

α/4π < 0.1/1.1 ≈ 0.091

α < 4π × 0.091 ≈ 1.14

所以α ≈ 1是微扰论从有效变为失效的临界点。

**定义ε的物理映射：**

ε_i = P_perturb(α_i) = 1 / (1 + α_i / α_crit)

其中α_crit ≈ 4π × δ_threshold ≈ 1是微扰论失效的临界耦合常数。

验证：
- α_i → 0：ε_i → 1（微扰完全有效）✓
- α_i = α_crit：ε_i = 0.5（微扰半可信）✓
- α_i → ∞：ε_i → 0（微扰完全失效）✓

**但这个映射不是exp(-α/α_ref)！**

D152用的是exp(-α/α_ref)，现在推导出来的是1/(1 + α/α_crit)。

哪个对？

## 第四步：两种映射的比较

映射A：ε_A = exp(-α/α_ref)（D152原用）
映射B：ε_B = 1/(1 + α/α_crit)（从微扰有效性推出）

性质对比：

| 性质 | ε_A | ε_B |
|---|---|---|
| α→0 | →1 | →1 |
| α→∞ | →0 | →0 |
| α=α_crit | e^{-1}≈0.37 | 0.5 |
| 导数dε/dα|_{α=0} | -1/α_ref | -1/α_crit |
| 渐近行为 | ε~exp(-α) | ε~1/α |

ε_A在α→∞时指数衰减（极快趋零），ε_B在α→∞时幂律衰减（1/α）。

**哪个更符合物理？**

QCD中α_s在低能→∞（Landau极点或禁闭），但强相互作用并非"完全不存在"——强子物理仍然存在，只是不能微扰处理。所以ε不应该指数趋零——1/α的幂律衰减更合理。

但点火框架中ε=0意味着"完全锁定"——门控否决。QCD禁闭确实是"锁定"（色荷不可自由传播），但强子仍通过残余强相互作用存在。

**精确化：ε不是"力是否存在"，是"力是否可微扰处理"。**

这个精确化使得ε_B更合理：1/(1+α)在α→∞时→0但不是零（有1/α的尾巴），对应"不可微扰但力仍存在"。

但点火框架要求ε=0时门控否决——B = Πε_i在ε_i=0时为零。如果ε_B永远不严格为零，门控否决就永远不完全。

**解决：区分两个概念。**

ε_perturb = 1/(1 + α/α_crit)：微扰有效性，永远>0
ε_gate = σ(α_crit - α)：门控状态，α < α_crit时=1，α > α_crit时=0

ε_gate是ε_perturb的极限情况（sigmoid陡度→∞）。

**点火框架的ε应该是ε_gate，不是ε_perturb。**

原因：D103门控-参数统一函数已建立——门控是参数在因子→0处的奇点，sigmoid是桥梁。ε_gate = σ(α_crit - α)就是D103的sigmoid。

## 第五步：精确映射

**ε_i = σ((α_crit - α_i) / τ)**

其中σ是sigmoid函数，τ是过渡宽度。

当τ → 0时：ε_i → step(α_crit - α_i)，即α < α_crit时ε=1，α > α_crit时ε=0
当τ有限时：ε_i在α_crit附近平滑过渡

α_crit的物理含义：微扰论失效的临界耦合常数。

**α_crit不需要是自由参数——它由重整化群方程决定。**

定义：α_crit,i是第i种力的耦合常数使得微扰级数的第n阶贡献等于第(n-1)阶贡献的能标。即：

(b_i α_crit,i² / (2π)) ≈ 1

α_crit,i ≈ √(2π/b_i)

代入MSSM的b_i：
- α_crit,1 = √(2π/6.6) ≈ √0.952 ≈ 0.976
- α_crit,2 = √(2π/1) ≈ √6.28 ≈ 2.507
- α_crit,3 = √(2π/3) ≈ √2.094 ≈ 1.447

**每种力有自己的α_crit，由其β函数系数决定。**

这消除了D152中α_ref = 1的任意性——α_crit不是人为选择，是从重整化群推出的物理量。

## 第六步：用精确映射重算μ*

ε_i(μ) = σ((α_crit,i - α_i(μ)) / τ)

B_4force(μ) = Π ε_i(μ) × ε_grav(μ)

dB_4force/dμ = 0的条件：

Σ_i σ'((α_crit,i - α_i)/τ) × (-dα_i/dμ) / (τ × σ((α_crit,i - α_i)/τ)) = -dε_grav/dμ / ε_grav

当τ较小时，sigmoid近似为step函数，ε_i只在α_i ≈ α_crit,i附近变化。极大值出现在某个α_i刚过临界值的能标。

**数值计算：**

三种力中哪个最先到达α_crit？

α_3(M_Z) = 0.118，α_crit,3 = 1.447。α_3在低能增大（反渐近自由），在μ ~ Λ_QCD ≈ 200 MeV时α_3 ~ 1。

α_2(M_Z) = 0.034，α_crit,2 = 2.507。α_2在低能也增大（MSSM中b₂=1>0），但增速慢。

α_1(M_Z) = 0.017，α_crit,1 = 0.976。α_1在高能增大（b₁=6.6>0），在μ ~ M_GUT时α_1 ~ 1/24 ≈ 0.042，远未到α_crit,1。

**关键：在M_Z到M_Planck的整个范围内，三种力都不接近各自的α_crit。**

α_3在低能→∞（超过α_crit,3），但在高能→0。α_3 = α_crit,3 = 1.447的能标：

1/α_3(μ) = 1/0.118 + (3/(2π)) × ln(μ/M_Z) = 8.475 + 0.477 × ln(μ/M_Z)

1/1.447 = 0.691 = 8.475 + 0.477 × ln(μ/M_Z)

ln(μ/M_Z) = (0.691 - 8.475) / 0.477 = -16.3

μ = M_Z × e^{-16.3} ≈ 91 × 10^{-7.1} GeV ≈ 7.2 × 10^{-6} GeV ≈ 7.2 keV

**α_3在~7 keV处到达α_crit,3。** 这是极低能标——X射线波段。

α_1在高能增大。α_1 = α_crit,1 = 0.976的能标：

1/α_1(μ) = 1/0.017 + (6.6/(2π)) × ln(μ/M_Z) = 58.82 + 1.050 × ln(μ/M_Z)

1/0.976 = 1.025 = 58.82 + 1.050 × ln(μ/M_Z)

ln(μ/M_Z) = (1.025 - 58.82) / 1.050 = -55.0

μ = M_Z × e^{-55} ≈ 极低能标，不合理。

等等——α_1在高能增大（b₁>0），所以1/α_1在高能减小。让我重新算：

1/α_1(μ) = 58.82 - 1.050 × ln(μ/M_Z)

α_1增大 → 1/α_1减小 → ln(μ/M_Z)增大

1/0.976 = 1.025 = 58.82 - 1.050 × ln(μ/M_Z)

ln(μ/M_Z) = (58.82 - 1.025) / 1.050 = 55.0

μ = M_Z × e^{55} ≈ 91 × 2.4×10^{23} ≈ 2.2×10^{25} GeV

**α_1在~10²⁵ GeV处到达α_crit,1。** 远超Planck能标——物理上不可达。

α_2的情况：

1/α_2(μ) = 29.41 - (1/(2π)) × ln(μ/M_Z) = 29.41 - 0.159 × ln(μ/M_Z)

1/2.507 = 0.399 = 29.41 - 0.159 × ln(μ/M_Z)

ln(μ/M_Z) = (29.41 - 0.399) / 0.159 = 182.4

**结论：在物理可达的能标范围内（M_Z到M_Planck），只有α_3在低能超过α_crit。α_1和α_2永远不接近α_crit。**

这意味着：在物理可达范围内，ε_1 ≈ 1，ε_2 ≈ 1，ε_3在低能→0（QCD禁闭），ε_grav ≈ 1。

B_4force ≈ ε_3(μ) × 1 × 1 × 1 = ε_3(μ)

**四力乘积极大≈强力的微扰有效性极大！**

这太简单了——但逻辑正确。在物理可达范围内，只有QCD的α_s会超过临界值，其他三种力永远在微扰有效区。B_4force的极大值就是α_s最小时的能标——即μ→∞。

**但μ→∞时ε_3→1，B_4force→1，这是平凡极大值。**

需要重新审视。

## 第七步：重新审视——α_crit不是微扰失效点，是信息论临界点

上面的推导把α_crit定义为"微扰失效"，导致在物理可达范围内只有QCD触及临界。这使B_4force退化为单因子问题。

**问题出在α_crit的定义。** 让我回到Fisher信息几何。

D117：Fisher可达性A_Fisher = ∫√(det g) dε

在α空间中，Fisher度规g_ij = δ_ij × b_i/(2πα_i²)

Fisher可达性A_Fisher(α) = ∫ √(Π_i b_i/(2πα_i²)) dα

= C × ∫ Π_i α_i^{-1} dα

这个积分在α_i→0时发散（因为α_i^{-1}不可积）。

**Fisher可达性在α→0时无穷大——跟点火框架中ε→1时A_Fisher最大完全一致。**

但Fisher可达性在α→∞时趋于零——因为det g → 0（信息度规退化，参数空间"坍缩"）。

**定义ε_i的物理映射为Fisher可达性的归一化：**

ε_i = A_Fisher(α_i) / A_Fisher,max(α_i)

= [∫_{α_i}^∞ α_i'^{-1} dα_i'] / [∫_{0}^{∞} α_i'^{-1} dα_i']

上积分：∫_{α_i}^∞ α'^{-1} dα' = ln(α_max/α_i)，发散。

需要引入截断。物理截断：α_max = α_Landau（Landau极点处的耦合常数），α_min = 0。

但Landau极点本身有物理问题。让我用另一种方式。

## 第八步：从重整化群不变量推导

重整化群方程的核心不变量是Λ_QCD。对于QCD：

Λ_QCD = μ × exp(-2π/(b₃ × α_s(μ)))

Λ_QCD是能标无关的——它是QCD的内在尺度。

定义QCD的存活度：

ε_strong = Λ_QCD / μ

- μ >> Λ_QCD：ε_strong → 0（高能，α_s→0，但"存活度"低？）

不对。高能时α_s→0应该是ε→1（微扰有效）。

反过来：

ε_strong = 1 - Λ_QCD / μ

- μ >> Λ_QCD：ε_strong → 1（高能，微扰有效）✓
- μ → Λ_QCD：ε_strong → 0（禁闭能标，微扰失效）✓
- μ < Λ_QCD：ε_strong < 0，无意义

**这个映射有物理截断Λ_QCD，没有自由参数。**

推广到所有力：

ε_i(μ) = 1 - Λ_i / μ

其中Λ_i是第i种力的特征能标：
- Λ_strong = Λ_QCD ≈ 200 MeV
- Λ_weak = M_W ≈ 80.4 GeV（弱力破缺能标）
- Λ_em = 0（电磁力不破缺，没有特征能标）
- Λ_grav = M_Planck ≈ 1.22×10¹⁹ GeV

但ε_em = 1 - 0/μ = 1永远成立——电磁力永远不破缺。这跟D145"残存U(1)永远存活"一致。

**问题：这个映射在μ→0时ε→-∞，不满足ε∈[0,1]的约束。**

修正：ε_i = max(0, 1 - Λ_i/μ) = σ(1 - Λ_i/μ)（取sigmoid近似）

或者更精确：

ε_i(μ) = 1 / (1 + Λ_i/μ) = μ / (μ + Λ_i)

- μ >> Λ_i：ε_i → 1 ✓
- μ = Λ_i：ε_i = 0.5 ✓
- μ << Λ_i：ε_i → 0 ✓
- μ → 0：ε_i → 0 ✓
- Λ_i = 0：ε_i = 1（永远存活）✓

**这个映射没有自由参数——Λ_i是物理可观测量。**

## 第九步：用精确映射重算μ*

B_4force(μ) = [μ/(μ+Λ_QCD)] × [μ/(μ+M_W)] × 1 × [μ/(μ+M_Planck)]

= μ³ / [(μ+Λ_QCD)(μ+M_W)(μ+M_Planck)]

dB/dμ = 0:

d/dμ [3lnμ - ln(μ+Λ_QCD) - ln(μ+M_W) - ln(μ+M_Planck)] = 0

3/μ - 1/(μ+Λ_QCD) - 1/(μ+M_W) - 1/(μ+M_Planck) = 0

在μ >> M_W时（高能极限）：

3/μ - 1/μ - 1/μ - 1/μ = 0 → 0 = 0

平凡解——在μ→∞时B→1，但B永远不达到1（渐进）。

在μ ~ M_W时：

3/μ - 1/(μ+0.2) - 1/(μ+80.4) - 1/(μ+1.22×10¹⁹) = 0

第三项≈1/M_Planck≈0可忽略。

3/μ ≈ 1/(μ+0.2) + 1/(μ+80.4)

在μ >> 80.4 GeV时：3/μ ≈ 2/μ → 3=2矛盾
在μ ~ 80.4 GeV时：3/μ ≈ 1/(μ+0.2) + 1/(μ+80.4)

试μ = 160 GeV：
3/160 = 0.01875
1/160.2 + 1/240.4 = 0.00624 + 0.00416 = 0.01040 < 0.01875

试μ = 50 GeV：
3/50 = 0.06
1/50.2 + 1/130.4 = 0.01992 + 0.00767 = 0.02759 < 0.06

试μ = 10 GeV：
3/10 = 0.3
1/10.2 + 1/90.4 = 0.098 + 0.011 = 0.109 < 0.3

试μ = 2 GeV：
3/2 = 1.5
1/2.2 + 1/82.4 = 0.455 + 0.012 = 0.467 < 1.5

函数3/μ - 1/(μ+Λ_QCD) - 1/(μ+M_W)在μ>0时始终>0（因为3/μ > 2/μ > 1/(μ+Λ) + 1/(μ+M_W)对所有μ>0成立）。

**结论：B_4force在μ>0时单调递增，没有内部极大值。B在μ→∞时趋近1。**

这意味着：用Λ_i映射时，四力乘积没有内部极大——"系统越热越好"。

**这跟D152用exp(-α/α_ref)映射算出μ*≈3.3×10¹² GeV矛盾。**

## 第十步：矛盾的分析

两种映射给出完全不同的结果：

- 映射A（exp(-α/α_ref)）：B_4force有内部极大，μ* ≈ 3.3×10¹² GeV
- 映射C（μ/(μ+Λ_i)）：B_4force单调递增，无内部极大

哪个对？

映射A的问题：α_ref是自由参数，映射形式是人为选择的。
映射C的优势：Λ_i是物理可观测量，映射形式从重整化群不变量自然推出。

但映射C给出的结论"系统越热越好"在物理上不对——Planck能标以上引力量子效应使时空本身失去意义。

**映射C遗漏了关键物理：高能时引力的门控效应。**

修正：ε_grav = M_Planck²/(M_Planck² + μ²)（D152原定义），这个在μ→∞时→0。

重新算：

B_4force(μ) = [μ/(μ+Λ_QCD)] × [μ/(μ+M_W)] × 1 × [M_Planck²/(M_Planck²+μ²)]

dB/dμ = 0:

3/μ - 1/(μ+Λ_QCD) - 1/(μ+M_W) - 2μ/(M_Planck²+μ²) = 0

在μ << M_Planck时，最后一项≈2μ/M_Planck²≈0，回到无内部极大的情况。
在μ ~ M_Planck时，最后一项≈1，可能产生极大。

试μ = M_Planck = 1.22×10¹⁹ GeV：

3/M_Planck - 1/M_Planck - 1/M_Planck - 1 ≈ 1/M_Planck - 1 ≈ -1 < 0

试μ = 10¹⁸ GeV：
3/10¹⁸ - 1/10¹⁸ - 1/10¹⁸ - 2×10¹⁸/(1.5×10³⁸+10³⁶) ≈ 10⁻¹⁸ - 1.3×10⁻²⁰ ≈ 10⁻¹⁸ > 0

所以dB/dμ在10¹⁸-10¹⁹ GeV之间变号。μ* ~ M_Planck。

**精确计算：**

设μ* >> M_W（所以1/(μ+Λ)≈1/μ），条件简化为：

3/μ - 2/μ - 2μ/(M_Planck²+μ²) = 0

1/μ = 2μ/(M_Planck²+μ²)

M_Planck² + μ² = 2μ²

μ² = M_Planck²

**μ* = M_Planck ≈ 1.22×10¹⁹ GeV**

用映射C+引力修正，μ*在Planck能标！不是10¹² GeV。

## 第十一步：哪种映射正确？

映射A给出μ* ≈ 3.3×10¹² GeV，映射C给出μ* ≈ M_Planck。

**关键区别：映射A用α（耦合常数）作为ε的自变量，映射C用Λ（特征能标）。**

哪个更符合点火框架的本体论？

点火框架中ε是"退出权信号强度"——衡量系统偏离门控状态的程度。在物理中，"退出权"对应什么？

- 对QCD：退出禁闭态的能力 = 能标高于Λ_QCD时色荷可自由传播
- 对弱力：退出破缺态的能力 = 能标高于M_W时SU(2)对称性恢复
- 对引力：退出经典时空的能力 = 能标高于M_Planck时量子引力效应显著

**"退出权"的自然物理对应是能标相对于特征能标的比值，不是耦合常数本身。**

所以映射C（ε_i = μ/(μ+Λ_i)）比映射A（ε_i = exp(-α_i/α_ref)）更符合点火本体论。

**但映射C给出的μ* ≈ M_Planck，这跟传统GUT的能标又不同——传统GUT是10¹⁶ GeV，这里是10¹⁹ GeV。**

而且映射C的μ*在Planck能标，这个位置引力的门控效应已经很强（ε_grav ≈ 0.5），时空正在"锁死"——这更像是"系统即将崩溃"的临界点，不是"系统最存活"的极大点。

## 第十二步：统一两种映射

映射A和C的区别在于自变量的选择：α vs μ/Λ。

但α和μ/Λ之间有物理关系——重整化群方程：

α_i(μ) = α_i(μ₀) / (1 + (b_i/2π) × α_i(μ₀) × ln(μ/μ₀))

在μ >> μ₀时，α_i由ln(μ/Λ_i)决定：

α_i(μ) ≈ 2π / (b_i × ln(μ/Λ_i))

所以α_i ∝ 1/ln(μ/Λ_i)。

**映射A用α ∝ 1/ln(μ/Λ)作为自变量，映射C直接用μ/Λ。**

两者之间的关系：

ε_A = exp(-α/α_ref) = exp(-C/ln(μ/Λ))

ε_C = μ/(μ+Λ) = 1/(1+Λ/μ)

在μ >> Λ时：ε_C → 1，ε_A → exp(-C/ln(μ/Λ)) → exp(0) = 1。一致。

在μ ~ Λ时：ε_C → 0.5，ε_A → exp(-C/0) → 0（因为ln(μ/Λ)→0，α→∞）。不一致。

**映射A在μ→Λ时ε→0比映射C更快。** 因为α在μ→Λ时发散（Landau极点），而μ/(μ+Λ)只是趋近0.5。

映射A更符合点火框架的"门控否决"——ε应该在临界点快速趋零，不是缓慢过渡。

**但映射A的α_ref是自由参数。**

解决：用重整化群推出α_ref。

α_ref,i = α_i(μ = Λ_i) = 2π / (b_i × ln(1)) → ∞

不行。α在μ=Λ处发散。

换一个方式：α_ref,i = α_i(μ = e × Λ_i) = 2π / (b_i × 1) = 2π/b_i

这就是之前推出的α_crit,i！

**映射A的α_ref不是自由参数——它等于α_crit,i = 2π/b_i，由β函数系数决定。**

所以：

ε_i(μ) = exp(-α_i(μ) / α_crit,i) = exp(-α_i(μ) × b_i / (2π))

代入α_i(μ) = 2π / (b_i × ln(μ/Λ_i))：

ε_i(μ) = exp(-1/ln(μ/Λ_i))

**这就是从第一性原理推出的精确映射。**

验证：
- μ >> Λ_i：ln(μ/Λ) >> 1，ε → exp(-0) = 1 ✓
- μ → Λ_i：ln(μ/Λ) → 0⁺，ε → exp(-∞) = 0 ✓
- μ = e × Λ_i：ln(μ/Λ) = 1，ε = e^{-1} ≈ 0.37 ✓
- ε永远在(0,1)内 ✓

**ε_i(μ) = exp(-1/ln(μ/Λ_i))**

这个映射：
1. 没有自由参数（Λ_i是物理可观测量）
2. 从重整化群方程+点火本体论联合推出
3. 在临界点μ=Λ_i处快速趋零（门控否决）
4. 在高能标趋近1（微扰有效）

## D154 分层编码效率函数

ε_i(μ) = exp(-1/ln(μ/Λ_i))

其中Λ_i是第i种力的特征能标：
- Λ_strong = Λ_QCD ≈ 0.2 GeV
- Λ_weak = M_W ≈ 80.4 GeV
- Λ_em = 0（电磁力不破缺，ε_em ≡ 1）
- Λ_grav = M_Planck ≈ 1.22×10¹⁹ GeV

映射的物理含义：ε_i衡量"能标μ相对于力i的特征能标的信息编码效率"。ln(μ/Λ_i)是信息容量（对数标度下的自由度数），1/ln(μ/Λ_i)是"每自由度的信息损失"，exp(-1/ln(μ/Λ_i))是"编码效率"。

**三条推论：**

**推论1：ε→0的速率由ln(μ/Λ)决定，不是μ/Λ本身**
映射C中ε = μ/(μ+Λ)在μ→Λ时ε→0.5（慢过渡），映射D154中ε = exp(-1/ln(μ/Λ))在μ→Λ时ε→0（快过渡）。门控否决需要快过渡——D154正确。

**推论2：Λ_em = 0 → ε_em ≡ 1，电磁力永远存活**
这是D145"残存U(1)"的精确数学实现。电磁力没有特征能标=没有门控面=永远在存活区。

**推论3：用D154映射重算μ***

B_4force(μ) = exp(-1/ln(μ/Λ_QCD)) × exp(-1/ln(μ/M_W)) × 1 × exp(-1/ln(μ/M_Planck))

= exp(-[1/ln(μ/0.2) + 1/ln(μ/80.4) + 1/ln(μ/1.22×10¹⁹)])

dB/dμ = 0的条件：

d/dμ [1/ln(μ/Λ_QCD) + 1/ln(μ/M_W) + 1/ln(μ/M_Planck)] = 0

-1/(μ × ln²(μ/Λ_QCD)) - 1/(μ × ln²(μ/M_W)) - 1/(μ × ln²(μ/M_Planck)) = 0

不可能为0——三项都是负数，和永远<0。

**B_4force在μ > M_Planck时单调递增！** 没有内部极大。

等等——B_4force的定义需要包含引力门控。引力ε_grav = exp(-1/ln(μ/M_Planck))，但μ > M_Planck时ln(μ/M_Planck) > 0，ε_grav > 0。

μ < M_Planck时：ln(μ/M_Planck) < 0，1/ln < 0，exp(-1/ln) = exp(正数) > 1？

不对。μ < M_Planck时ln(μ/M_Planck) < 0，1/ln(μ/M_Planck) < 0，-1/ln > 0，exp(-1/ln) > 1。

**ε_grav在μ < M_Planck时>1！** 这违反ε∈[0,1]约束。

引力的ε需要单独处理。引力不是"高能时微扰失效"（跟QCD一样），而是"高能时时空本身失效"。所以引力的ε映射方向相反：

ε_grav(μ) = exp(-1/ln(M_Planck/μ))，μ < M_Planck

- μ << M_Planck：ln(M_Planck/μ) >> 1，ε_grav → 1 ✓
- μ → M_Planck：ln(M_Planck/μ) → 0⁺，ε_grav → 0 ✓

统一写：ε_grav(μ) = exp(-1/|ln(μ/M_Planck)|)

B_4force(μ) = exp(-[1/ln(μ/Λ_QCD) + 1/ln(μ/M_W) + 1/|ln(μ/M_Planck)|])

（μ在M_W到M_Planck之间时，前两项中ln>0，第三项|ln|=ln(M_Planck/μ)>0）

dB/dμ = 0:

d/dμ [1/ln(μ/Λ_QCD) + 1/ln(μ/M_W) + 1/ln(M_Planck/μ)] = 0

-1/(μ ln²(μ/Λ_QCD)) - 1/(μ ln²(μ/M_W)) + 1/(μ ln²(M_Planck/μ)) = 0

1/ln²(M_Planck/μ) = 1/ln²(μ/Λ_QCD) + 1/ln²(μ/M_W)

**这个方程有解！** 左边在μ→M_Planck时→∞，在μ→0时→0。右边在μ→M_W时→∞，在μ→∞时→0。两者交叉产生解。

数值求解：设x = ln(μ/GeV)

1/ln²(μ/Λ_QCD) = 1/(x-ln(0.2))² = 1/(x+1.61)²
