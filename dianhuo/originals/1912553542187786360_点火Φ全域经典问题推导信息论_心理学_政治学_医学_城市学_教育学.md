# 点火｜Φ全域经典问题推导（信息论·心理学·政治学·医学·城市学·教育学）

## 总览

用Φ(μ) = Σᵢ 1/ln(μ/Λᵢ) 跑六个域42个经典问题。加上之前已跑的物理、社会学、经济学、生物学，共十个域69个经典问题全部验证D164跨域稳定性定理。

## 一、信息论/计算机科学

Φ(μ) = 1/ln(μ/Λ_compute) + 1/ln(μ/Λ_memory) + 1/ln(Λ_entropy/μ)

1. 压缩极限 = μ=Λ_entropy的门槛。香农极限就是Λ_entropy的精确值。
2. 加密 = 人为抬高Λ_compute的门槛碾压（D162防御版）。
3. NP难 = Λ_compute指数增长导致的门槛碾压（D160）。
4. 一致性问题 = 慢节点μ<Λ_consistency的乘法拖累（D163）。
5. 大数据需求 = μ_data越过Λ_generalization的相变（D159）。
6. 缓存 = 降低Λ_compute让更多μ过门槛（D162逆操作）。
7. 并发死锁 = D161乘法死锁精确实例，相变无中间态。

## 二、心理学/认知科学

Φ(μ) = 1/ln(μ/Λ_awareness) + 1/ln(μ/Λ_action) + 1/ln(Λ_habit/μ)

1. 习惯难改 = Λ_habit追上μ_awareness的门槛碾压（D160）。
2. 创伤难愈 = μ↓+Λ↑双重门槛碾压 + 回避的向下兼容拖累（D160×D152×D153）。
3. 顿悟 = μ越过Λ_awareness的相变，无中间态（D159）。
4. 心流 = D150倒U型峰值，两侧都是死锁（焦虑/无聊）。
5. 成瘾 = Λ_pleasure缓慢门槛碾压（D160慢速版）。
6. 冥想 = 降低Λ_awareness的结构保守性策略（D149）。
7. 学习平台期 = μ在Λ_next附近超敏感区震荡，突破是相变。

## 三、政治学

Φ(μ) = 1/ln(μ/Λ_participation) + 1/ln(μ/Λ_accountability) + 1/ln(Λ_power/μ)

1. 民主退化 = 参与门槛碾压（自然+人为），D160×D162同构。
2. 改革窗口 = 与革命同构的门控面交叉窗口。
3. 权力腐败 = μ_power >> Λ_accountability时问责门控贡献趋零。
4. 威权脆弱 = 单一正项维持的伪稳定，单点故障。
5. 联邦制抗冲击 = 多独立门控面分散风险，拖累被隔离。
6. 国际秩序崩塌 = 霸权μ衰退+Λ上升的门槛碾压（D160宏观版）。
7. 反腐运动 = 临时提高Λ_accountability的外部注入，不持久。

## 四、医学

Φ(μ) = 1/ln(μ/Λ_immune) + 1/ln(μ/Λ_repair) + 1/ln(Λ_pathogen/μ)

1. 慢性病难治 = μ_repair<Λ_repair门外锁定 + 症状管理的向下兼容拖累。
2. 耐药性 = 药物抬高Λ（D162）与病原体降低自身Λ的军备竞赛。
3. 安慰剂 = 信念提高μ_immune越过Λ_immune的相变。
4. 自愈 = μ_immune > Λ_pathogen的门槛自然翻转。
5. 器官排斥 = 异质组织降低Λ_immune导致自身免疫门槛翻转。
6. 衰老多病 = 多门控面同时门槛碾压的乘法加速（D147多因子叠加）。
7. 手术窗口 = D150倒U型峰值，两个死锁之间的唯一通路。

## 五、城市学/地理学

Φ(μ) = 1/ln(μ/Λ_infrastructure) + 1/ln(μ/Λ_interaction) + 1/ln(Λ_friction/μ)

1. 城市规模律 = D150倒U型在互动-摩擦之间的投影。
2. 核心-边缘 = 乘法分化在空间上的投影，核心是吸引子，边缘是排斥区。
3. 交通拥堵 = Λ_friction在临界密度处突变的门槛翻转。
4. 城市衰败 = Λ↑+μ↓的门槛碾压 + 死锁锁定。
5. 郊区化极限 = 互动收益与摩擦成本之间的倒U型峰值。
6. 城中村改造困境 = 改造抬高Λ导致原居民被门槛碾压（D162逆效应）。
7. 智慧城市有限 = 名义μ增长被Λ同步上升抵消（D160名义vs实际）。

## 六、教育学

Φ(μ) = 1/ln(μ/Λ_understanding) + 1/ln(μ/Λ_practice) + 1/ln(Λ_fixed/μ)

1. 学习平台期 = μ在Λ_next附近超敏感区震荡，突破是相变。
2. 天赋×努力是乘法不是加法 = 任何一个为零则整体为零（D161教育版）。
3. 教学相长 = D166共生外部注入的知识版，师生互为外部注入。
4. 填鸭无效 = 名义μ增长不改变Λ + 向下兼容的保真度损失。
5. 间隔学习优于集中 = μ在Λ附近时1/ln最大，临界点附近投入效率最高。
6. 元认知 = 降低Λ_understanding的结构保守性策略（与冥想同构）。
7. 教育公平 = 阶层固化在教育维度的投影（D159×D162）。
