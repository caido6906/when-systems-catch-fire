# 点火｜分属矩阵一：退出权梯度矩阵

## 这个矩阵描述的是什么

退出权在等级制中的纵向分布。同一个事件里，不同层级的应约者拥有不同类型的退出权——等级制对退出权做结构性压缩，每往下一级，退出权被压低一个类型。

主矩阵（8格）的每一格只对应一组条件，描述的是"事件的初始条件"。但主矩阵回答不了"同一个事件在不同层级为什么跑出不同走向"——这是退出权梯度矩阵的事。

退出权四分法（真实/事实/心理/象征）是横向光谱，描述同一个应约者的退出权性质。退出权梯度是纵向展开，描述不同层级应约者的退出权分布。两者正交。

## 两个变量

- 等级制层级：从上往下的行政/权力层级（如中央→省→市→县→百姓）
- 退出权类型：真实/事实/象征/心理

## 矩阵的格子

以分税制为例：

| 层级 | 真实 | 事实 | 象征 | 心理 |
|---|---|---|---|---|
| 中央→省 | | ✓ | | |
| 省→市 | | | ✓ | |
| 市→县 | | | ✓ | |
| 县→百姓 | | | | ✓ |

以秦统一为例：

| 层级 | 真实 | 事实 | 象征 | 心理 |
|---|---|---|---|---|
| 皇帝→官僚 | | ✓ | | |
| 官僚→百姓 | | | | ✓ |

分税制的梯度表和秦统一的梯度表放在一起比，能看见"缓慢压缩"和"陡峭压缩"的区别——这是主矩阵的注释做不到的，注释只能各写各的，没法对齐比较。

## 事实-象征临界线

退出权梯度有一条临界线：事实退出权与象征退出权之间。

- 线之上：还有博弈能力（虽然走不了，但能谈判、拖延、灰色操作）
- 线之下：博弈能力归零，只剩心理退出

中间层级的"假选择"不是bug，是等级制运行的必要条件——让中间层以为自己还有博弈空间，不至于彻底心理退出，同时确保走不了。

## 压缩速率

退出权梯度的压缩速率是事件走向的预测变量：

- 缓慢压缩（每级只降一格）→ 系统有时间适应，退化信号能被中间层捕获 → 可能"点不足"但不会崩
- 陡峭压缩（跳级降档）→ 中间层没有缓冲，退化信号被压制 → 一旦突破临界线就崩盘

分税制：缓慢压缩，土地财政填补了县级的事实退出权缺口，系统没崩。
秦统一：陡峭压缩，从真实直接跳到象征，15年崩。

## 共同知识制造发生在梯度底层

心理退出的应约者无法通过正常渠道博弈，唯一的集体行动方式是把心理退出变成共同知识——让所有人都知道所有人都想走。陈胜喊话就是把分散的心理退出聚合成共同知识，让退出权从心理层直接跳到真实层（造反=暴力夺回退出权）。

## 矩阵在什么条件下成立

1. 事件内部存在等级制结构（不是所有事件都有——双人事件、平级组织之间没有梯度）
2. 不同层级的应约者面对的不是同一个提议者（中央面对省、省面对县，每一级的"提议者-应约者"关系不同）
3. 退出权在等级制中从上往下递减（如果递增，比如联盟制，梯度方向相反但矩阵结构同样适用）

## 主要用来描述什么

1. 解释"同一个事件在不同层级跑出不同走向"——主矩阵只能标一个格子，梯度矩阵能画出每个层级的格子
2. 预测退化信号的可见性和爆发方式——压缩速率决定信号是被中间层吸收还是被压制到崩盘
3. 定位共同知识制造的层级——梯度底层是心理退出权集中的地方，也是共同知识爆发的起点
4. 跨案例比较等级制的退出权压缩模式——不同事件的梯度表可以对齐比较

## 与主矩阵的关系

退出权梯度矩阵是主矩阵的分属矩阵，只在主矩阵的特定格子里展开——格2、格4、格6、格8（应约者退出权为"无"的格子），因为只有应约者退出权被压缩时才有梯度可画。格1、格3、格5、格7（应约者有退出权）通常不需要展开梯度矩阵——退出权是真实的，没有压缩空间。
