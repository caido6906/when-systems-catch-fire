# 点火｜三层因果链：动机来源→姿态→退出权信号（2026.06.07）

## 定位

动机来源不是退出权的前置条件，而是最上游的因果种子。前置条件意味着"没有好奇心驱动就没有退出权"，这不成立——周公制礼是有意识的目的驱动，但退出权是真的（诸侯可叛可留）。好奇心驱动是退出权的充分但不必要条件：好奇心驱动一定能产出退出权信号，但退出权信号不一定来自好奇心驱动。

## 三层因果链

动机来源 → 姿态 → 退出权信号

好奇心驱动 → 备方案姿态 → 退出权信号发出 → 应约者开放进入 → 认同建在应约者身上 → 点对了
目的驱动 → 推方案姿态（默认）→ 退出权信号被压 → 应约者防御启动 → 认同建在强制力上 → 点歪了或点反了
目的驱动 → 备方案姿态（主动选择）→ 退出权信号发出 → 同上

## 关键区分：自动 vs 主动选择

好奇心驱动到备方案姿态是自动的——好奇心驱动的提议者天然就是备方案姿态，不需要额外条件。没有预设结论要强推，退出权信号天然就发出去了。

目的驱动到姿态是不定的——可以走推方案（秦始皇，默认路径），也可以走备方案（周公，主动选择）。走哪条取决于提议者是否主动选择保留退出权。周公选择让诸侯可叛可留，这不是自动的，是刻意的设计。

动机来源决定的是姿态的默认值：好奇心驱动默认备方案，目的驱动默认推方案。目的驱动的提议者如果想走备方案，需要额外一步主动选择。

## 与现有框架的对接

这条因果链把之前分散在知识库里的几个结论串起来了：

1. 动机来源（好奇心vs目的）是提议者意识的因果底座——之前已存入
2. 姿态（推vs备）是退出权的次级变量——之前已存入
3. 名义是姿态的信号层——之前已存入
4. 退出权四分法（真实/事实/心理/象征）的取值来源——之前已存入

三层因果链把1→2→3→4串成一条完整的因果路径：动机来源决定姿态默认值，姿态决定名义信号，名义信号决定退出权真假，退出权真假决定应约者反应，应约者反应决定点火趋势。

## 对8格标注的影响

退出权列的取值来源标注现在可以追溯到最上游：

- 真实退出权（备方案型提议者·好奇心驱动）— 自动路径
- 真实退出权（备方案型提议者·目的驱动·主动选择）— 非自动路径
- 象征退出权（推方案型提议者·目的驱动）— 默认路径
- 事实退出权（客观条件限制型）— 与动机来源无关
