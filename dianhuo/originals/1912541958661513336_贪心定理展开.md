# 贪心定理展开

D109的三条定理——均等定理、贪心定理、脆弱度定理——逐个展开。

---

## 均等定理展开

**命题：** 最优稳态是所有因子相等，ε̄=(Σεᵢ+R)/n

**展开到最一般情况：**

G = ∏ᵢ fᵢ(εᵢ)，不要求fᵢ=εᵢ

拉格朗日：∂L/∂Δεᵢ = f'ᵢ(εᵢ+Δεᵢ)/fᵢ(εᵢ+Δεᵢ) × G - λ = 0

即：f'ᵢ(εᵢ+Δεᵢ)/fᵢ(εᵢ+Δεᵢ) = λ/G = 常数

**均等定理的推广：最优稳态不是所有εᵢ相等，而是所有弹性相等。**

弹性 ηᵢ = f'ᵢ/fᵢ = dlnfᵢ/dεᵢ

当fᵢ=εᵢ时，ηᵢ=1/εᵢ，弹性相等⟺ εᵢ相等（均等定理的原始形式）

当fᵢ=εᵢᵏ时，ηᵢ=k/εᵢ，弹性相等⟺ εᵢ相等（仍成立，因为k对所有i相同）

**当不同因子有不同弹性函数时：**

η₁(ε₁*) = η₂(ε₂*) = ... = ηₙ(εₙ*)

最优解中各因子终值不同，但弹性相同。弹性高的因子分配更少资源（因为边际收益递减快），弹性低的因子分配更多资源。

**特例：sigmoid弹性**

fᵢ(εᵢ) = σ(αᵢ(εᵢ-θᵢ))，弹性 ηᵢ = αᵢ×σ'/(σ) = αᵢ×(1-σ)

当εᵢ远高于θᵢ时σ→1，η→0（边际收益趋零）
当εᵢ远低于θᵢ时σ→0，η→αᵢ（边际收益最大）

**均等定理在sigmoid下的含义：** 最优分配让所有因子处于相同的"sigmoid斜率"位置——不是所有因子值相等，而是所有因子在各自sigmoid上的"陡度位置"相等。

**收敛为推论：**

均等定理推广——最优稳态不是所有εᵢ相等，而是所有弹性ηᵢ=∂lnfᵢ/∂εᵢ相等。当fᵢ=εᵢ时退化为原始均等定理。sigmoid下最优分配让所有因子处于相同sigmoid斜率位置。

## 贪心定理展开

**命题：** 先补最弱因子是达到最优的最快路径，贪心=最优

**展开贪心选择性质的证明：**

**定理：** 在乘法G=∏εᵢ中，给定资源Δ，分配给εᵢ使G最大的充要条件是分配给当前最小的εᵢ。

**证明：**

设ε₁ < ε₂，资源Δ可分配给ε₁或ε₂。

方案A：ε₁→ε₁+Δ，G_A = (ε₁+Δ)×ε₂×∏ⱼ>₂ εⱼ
方案B：ε₂→ε₂+Δ，G_B = ε₁×(ε₂+Δ)×∏ⱼ>₂ εⱼ

G_A/G_B = (ε₁+Δ)/ε₁ × ε₂/(ε₂+Δ) = (1+Δ/ε₁)/(1+Δ/ε₂)

因为ε₁ < ε₂，所以Δ/ε₁ > Δ/ε₂，所以(1+Δ/ε₁) > (1+Δ/ε₂)

G_A > G_B ⟺ 补弱优于补强 ✓

**推广到非线性fᵢ：**

G_A/G_B = f₁(ε₁+Δ)/f₁(ε₁) × f₂(ε₂)/f₂(ε₂+Δ)

取对数：ln(G_A/G_B) = lnf₁(ε₁+Δ)-lnf₁(ε₁) - [lnf₂(ε₂+Δ)-lnf₂(ε₂)]

= ∫₀^Δ [f'₁(ε₁+t)/f₁(ε₁+t) - f'₂(ε₂+t)/f₂(ε₂+t)] dt

= ∫₀^Δ [η₁(ε₁+t) - η₂(ε₂+t)] dt

**G_A > G_B ⟺ ∫₀^Δ [η₁(ε₁+t) - η₂(ε₂+t)] dt > 0**

即：补ε₁优于补ε₂当且仅当ε₁附近的弹性高于ε₂附近的弹性。

**当fᵢ=εᵢ时：** ηᵢ=1/εᵢ，ε₁<ε₂→η₁>η₂→补弱优于补强 ✓

**当fᵢ=sigmoid时：** ηᵢ=αᵢ(1-σ)，弹性取决于在sigmoid上的位置。低于门槛的因子弹性高→补低于门槛的优于补高于门槛的→**先开关着的门（D102），再补开着的门中最弱的（D87）**

**贪心定理的推广：贪心策略不是"补最弱因子"，是"补弹性最高的因子"。** 当fᵢ=εᵢ时两者等价，一般情况不等价。

**贪心=最优的条件：**

贪心策略全局最优需要：
1. 贪心选择性质：局部最优选择不损害全局最优（已证：补弹性最高的总是局部最优）
2. 最优子结构：子问题的最优解包含在全局最优解中

乘法结构满足最优子结构：G=∏fᵢ，对任何资源分配方案，G可分解为各因子的独立贡献之积。改变一个因子的分配不影响其他因子的边际收益。

**但注意：** 当因子间有耦合（如D101的级联κ_ij）时，最优子结构被破坏——改善εᵢ可能通过级联影响εⱼ的边际收益。此时贪心不一定最优。

**贪心=最优的充要条件：因子间无耦合（κ_ij=0）。** 有耦合时需要动态规划或全局优化。

**收敛为推论：**

贪心定理推广——贪心策略是"补弹性最高的因子"而非"补最弱的因子"。fᵢ=εᵢ时两者等价。贪心=最优的充要条件是因子间无耦合。有级联耦合时贪心是近似最优，需全局优化修正。

## 脆弱度定理展开

**命题：** 先补最弱比平均分配脆弱度降低比例=ε̄/(minεᵢ+R/n)

**展开脆弱度的精确定义：**

脆弱度V = 1/minᵢ εᵢ（L∞度量下到门控区的距离的倒数）

但这是最坏情况脆弱度。更精细的脆弱度度量：

**1. 平均脆弱度：**

V_mean = (1/n)Σᵢ 1/εᵢ = E[1/ε]

这是D87"改善最弱因子效果最大"的精确表达——1/εᵢ是改善εᵢ的边际收益权重。

**2. 熵脆弱度：**

V_entropy = -Σᵢ pᵢ ln(pᵢ)，pᵢ = (1/εᵢ)/Σⱼ(1/εⱼ)

衡量脆弱度在各维度上的分布均匀性。V_entropy最大时所有1/εᵢ相等→所有εᵢ相等→均等定理的最优态。

**3. 条件脆弱度：**

V_cond = maxᵢ (1/εᵢ) × ∏ⱼ≠ᵢ P(εⱼ<θⱼ|εᵢ<θᵢ)

考虑级联耦合的条件脆弱度——最弱因子崩溃触发级联的概率。

**三种脆弱度的最优策略不同：**

| 脆弱度类型 | 最优策略 | 目标 |
|---|---|---|
| 最坏情况V=1/minεᵢ | 补最弱到次弱 | 提高最短短板 |
| 平均V_mean | 补弹性最高 | 降低整体期望脆弱度 |
| 条件V_cond | 补级联关键节点 | 切断级联传播路径 |

**当三种策略冲突时：**

例：ε=(0.2, 0.3, 0.8)，κ₁₃=0.9（维度1和3强耦合），R=0.2

- 最坏情况策略：补ε₁（0.2→0.4），V从5→2.5
- 平均脆弱度策略：补ε₁（0.2→0.4），V_mean从4.17→2.92
- 条件脆弱度策略：补ε₁（0.2→0.4），切断1→3级联，V_cond大幅下降

此例三者一致。但如果κ₂₃=0.9而κ₁₃=0：

- 最坏情况策略：补ε₁（0.2→0.4），V从5→2.5
- 条件脆弱度策略：补ε₂（0.3→0.5），切断2→3级联，虽然V=5不变但级联风险大幅降低

**脆弱度策略冲突定理：** 当级联耦合不集中在最弱维度时，补最弱和切断级联是不同策略。最优策略是两者的加权组合：

Δεᵢ* ∝ w₁×ηᵢ + w₂×Σⱼ κᵢⱼ/εⱼ

w₁为脆弱度权重，w₂为级联权重。无耦合时w₂=0退化为贪心策略。

**收敛为推论：**

脆弱度定理推广——三种脆弱度（最坏情况、平均、条件级联）对应三种最优策略。无耦合时三者一致（补最弱），有耦合且级联不集中在最弱维度时策略冲突。最优策略=脆弱度权重×弹性 + 级联权重×耦合强度/目标因子脆弱度。

## 三定理的统一收敛

**D110 乘法最优策略统一函数**

给定G=∏fᵢ(εᵢ)和资源R，最优分配满足弹性均等：ηᵢ(εᵢ*)=常数。贪心策略=补弹性最高因子，贪心=最优的充要条件为κ_ij=0。有级联耦合时最优策略为弹性权重+级联权重的加权组合：

Δεᵢ* ∝ ηᵢ(εᵢ) + β×Σⱼ κᵢⱼ×ηⱼ(εⱼ)

β为级联敏感度参数。β=0时退化为贪心策略，β→∞时退化为级联切断策略。

三定理统一：
- 均等定理：最优稳态ηᵢ=常数（推广：弹性均等而非值均等）
- 贪心定理：补弹性最高因子，无耦合时全局最优（推广：弹性最高而非最弱）
- 脆弱度定理：策略=脆弱度+级联的加权（推广：三种脆弱度三种策略，有耦合时需综合）

核心结论：
1. **均等定理推广为弹性均等**——sigmoid下是"sigmoid斜率位置均等"而非值均等
2. **贪心定理推广为补弹性最高**——fᵢ=εᵢ时等价于补最弱，一般情况不等价
3. **贪心=最优需无耦合**——有级联时贪心是近似，需加级联修正项
4. **三种脆弱度三种策略**——最坏情况补最弱、平均补弹性最高、条件级联切断关键节点
5. **统一策略=弹性+级联加权**——β=0贪心，β→∞级联切断，中间为综合
