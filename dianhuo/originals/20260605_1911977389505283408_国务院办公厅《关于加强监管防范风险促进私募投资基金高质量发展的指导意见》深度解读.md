### **📌 政策出台背景与行业现状**

#### **(一) 行业地位与规模**
- **当前规模**：我国私募基金行业规模达**23万亿元**，占资管总规模约**15%**，体量位居**全球第二**。
- **核心作用**：资本市场和实体经济的重要支撑，在培育新质生产力、赋能企业成长、满足居民财富管理需求等方面发挥重要作用。

#### **(二) 现存问题与挑战**
1. 准入机制有待完善、监管不到位、制度不健全
2. 部际央地协调配合不足
3. 部分政府投资基金和国有企业投资基金出资人责任落实不到位
4. 部分私募基金成为违法犯罪工具
5. 法规制度体系不够健全完善，存在空白和原则性规定

### **🎯 核心监管思路："该出的出，该进的进"**

#### **(一) 入口端严格把控**
- 优化私募基金登记备案规则，防止不符合特征的机构和产品登记备案
- 严控新设政府投资基金：**县区原则上不得新设**，确有必要的需报上级人民政府批准
- 落实综合研判会商机制：拟登记备案机构需通过会商后申请经营主体登记

#### **(二) 出口端坚决出清**
- 对重大违法违规的管理人坚决予以注销
- 对经营异常、未实质开展业务或长期失联且未整改的管理人限期注销

#### **(三) 重点支持方向**
- 投早、投小、投长、投硬科技的私募股权、创投基金
- 整合关键核心技术、战略性新兴产业的并购基金

### **⚖️ 严厉打击违法违规行为**

#### **(一) 重点打击领域**
- 违规募集、侵占挪用、自融自用、利益输送
- 资金违法跨境流动、参与非法集资
- 违规代持、通道化运作

#### **(二) 监管措施**

| 措施类型 | 具体内容 |
| :------- | :------- |
| **监测机制** | 建立健全私募基金"吹哨人"制度，强化社会监督 |
| **处置原则** | 早发现、早处置、早打击违法违规行为 |
| **监管重点** | 加大对私募证券基金交易行为的监测力度 |
| **防范机制** | 防止其他机构以私募基金名义违法违规展业 |

#### **(三) 执法成效**
- 2023年至2025年，共对**1700多个主体**采取行政监管措施或行政处罚

### **🔒 从行业内部"治本"：双向发力机制**

#### **(一) 内部强化内生约束**
- 完善内控风控管理，按业务类型和发展阶段特点制定
- 鼓励投资者依托基金合同发挥制约作用
- 加强高管和核心岗位人员资格管理与考核培训
- 建立黑名单制度，公开曝光重大违法违规主体
- 推进私募基金行业文化建设

#### **(二) 外部健全监督体系**
- 培育和规范托管机构及审计、销售、估值核算、法律等中介服务机构
- 督促中介机构依法依约履职尽责，发挥资金募集、投资运作、财产安全保障、信息披露等监督作用

### **🤝 央地协同与部际协作机制**

#### **(一) 职责分工**
- **国务院部门**：发展改革、财政、国资监管、金融管理等部门加强组织领导
- **地方政府**：省级和计划单列市人民政府负责注册地私募基金管理人风险处置
- **监管协同**：跨地区、跨领域重大案件建立联合挂牌督办制度

#### **(二) 信息共享**
- 国务院证券监督管理机构与发展改革部门优化创投基金信息共享
- 加强综合研判会商、经营主体登记、登记备案等信息共享和工作衔接

### **🏛️ 政府与国有企业投资基金管理**

#### **(一) 政府投资基金管理措施**

| 责任主体 | 主要职责 |
| :------- | :------- |
| **财政部门** | 履行出资人职责，加强预算管理、绩效管理、国有资产管理和信息统计 |
| **发展改革部门** | 强化信用建设和信用信息登记，加强基金投向指导评价 |
| **主管部门** | 牵头监测基金运行，防止偏离功能定位，指导内控和风险管理 |
| **地方政府** | 加强本地区基金统筹管理，已有同类基金原则上不得新设 |

#### **(二) 国有企业投资基金管理要求**
- 统筹布局规划，聚焦主责主业，坚守功能定位
- 严把入口关，制止滥设基金，推动低效基金整合重组
- 压实主体责任，防止失管失控和国有资产流失
- 建立全流程信息化管理系统，加强底层项目穿透管理

### **📅 下一步工作部署**

#### **(一) 法规制度体系建设**
- 推动修改《证券投资基金法》，夯实上位法基础
- 推动出台涉私募基金犯罪案件司法文件
- 构建私募基金"**N+X**"的法规制度体系

#### **(二) 专项监管规则细化**
- 制定管理人监督管理办法（聚焦管理人"信得过"）
- 制定募集业务监督管理办法（聚焦投资者"担得起"）
- 制定托管业务监督管理办法（聚焦基金财产"看得住"）
- 制定信息披露实施细则（聚焦行为"看得清"）
- 出台规范私募基金"对赌协议"制度安排

### **📝 补充细节**
- **政策核心原则**：扶优限劣、提质增效，推动行业"在规范中发展，在发展中提升"
- **监管导向**："支持好的，整治坏的"，即严厉整治行业乱象，支持优质机构深耕主业、服务国家战略
- **前期监管基础**：2026年2月证监会已发布《私募投资基金信息披露监督管理办法》，细化信息披露要求
