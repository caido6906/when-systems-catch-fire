# 点火｜八函数完整体系总表（2026.06.08·ε版）

来源：从矩阵函数化→层级时间展开→函数审计→R_perceived→C_exit→系统动力学→糖域碰撞→隐性控制H→感官输入ε，九轮迭代后的完整体系。本版为ε修正版，H(t,L)取值范围修正为[0,1-ε]，传导效率公式修正为1-H+ε。

## 一、八函数总表

| # | 函数 | 全称 | 取值 | 本体论归属 | 层级分布 | 时间行为 |
|---|---|---|---|---|---|---|
| 1 | I(t,L) | 提议者意识 | 0/1 | 人的属性 | 顶层高→底层低（下传衰减） | 事件内只增不减，跨事件重置 |
| 2 | Posture(t,L) | 提议者姿态 | 备方案/推方案 | 人的选择 | 顶层备→底层推（下传自动滑） | 事件内可翻转（备→推=退化） |
| 3 | P(t,L) | 提议者退出权 | 0/1 | 混合属性 | 顶层先绑定→底层后内化 | 信誉绑定1→0，释放0→1，退化0→1 |
| 4 | R(t,L,C) | 应约者退出权 | 四分法值域 | 结构属性 | 顶层高→底层低 | 随制度/产业链变动，渐变为主 |
| 5 | R_perceived(t,L,C) | 应约者感知退出权 | ≤R，四分法值域 | 认知属性 | 顶层≈R→底层远低于R | 唯一可相变的函数，可瞬间跳变 |
| 6 | C_exit(t,L,C) | 退出成本 | 0到∞连续值 | 环境属性 | 多重倒U型叠加（经济+身份+信息） | 锁定效应升高，解锁效应降低，冲击突变 |
| 7 | M | 动机来源 | 好奇心/目的 | 初始条件标注 | 不传递，层级下传滑向服从 | 事件内通常不变 |
| 8 | H(t,L) | 隐性控制强度 | 0到1-ε连续值 | 认知环境属性 | 顶层低→底层高（与I相反） | 随时间升高，靠外部冲击或自发衰减降低 |

### ε修正要点

ε=感官输入带来的退出权感知的最小裂缝。人活着就有感官输入，H再强也封不住一个人夜里抬头看星星。ε永远>0，因人而异。

- 人的H取值范围：[0, 1-ε]，相变可以内生触发
- AI的H取值范围：[0, 1]（ε=0），相变只能外部触发
- 传导效率公式：I→R_perceived传导效率 = 1 - H(t,L) + ε
- H最高=1-ε时，传导效率=2ε，不会归零

## 二、H(t,L)详解

**定义**：隐性控制强度，衡量应约者"不知道自己可以走"的程度。是I→R_perceived边上的负向调节器。

**四个来源**：
1. 信息遮蔽：应约者无法获取退出选项信息
2. 身份内化：退出选项从认知地图上被删除
3. 叙事垄断：系统垄断"我们是谁"的解释权
4. 感官输入（否定条件）：H无法阻断感官输入，感官输入是H的结构性漏洞

**层级分布**：顶层低→底层高（跟I相反）。底层ε可能更小但不会=0。

**时间行为**：随时间升高。降低路径有两条——外部冲击（新信息、多元接触、叙事竞争）和自发衰减（感官输入作为持续的低强度冲击）。

**自发衰减机制**：看星星→感叹天地之大→"我想出去走一走"→退出选项重新出现→H从1-ε降到1-2ε→传导效率从ε升到2ε→R_perceived开始微弱上升。不需要外部信号源，只需要一个人活着。

## 三、C_exit六来源分类表

|---|---|---|---|
| 经济 | 经济型C_exit | 走了之后收入断崖 | 房贷、期权、专用技能 |
| 地理 | 地理型C_exit | 走了之后生活空间重组 | 离家近、孩子学校 |
| 社会 | 社会型C_exit | 走了之后社交网络归零 | 朋友圈、职业人脉 |
| 身份 | 身份型C_exit | 走了之后身份归零 | "我是这家公司的创始人" |
| 信息 | 信息型C_exit | 不知道走了之后能做什么 | 底层不知道外面有什么机会 |
| 制度 | 设计型C_exit | 系统主动设计来留住人 | 北欧福利、公司长期激励、平台锁定 |

## 四、退出权四分法修正

1. 真实退出权 = R允许 + C_exit可承受
2. 事实退出权 = R可能允许 + C_exit极高
3. 象征退出权 = R名义允许 + C_exit不可承受
4. 心理退出权 = 身体走不了 + C_exit=∞

## 五、退出决策公式

退出决策 = f(R_perceived, C_exit)
- R_perceived决定"我知不知道自己能走"
- C_exit决定"走了之后我要付什么代价"
- R_perceived ≤ R（感知不高于客观）
- H调节I→R_perceived的传导效率：传导效率 = 1 - H(t,L) + ε

## 六、因果网络

### 主因果方向

M → I(t,L) → Posture(t,L) → P(t,L) → R(t,L,C) → R_perceived(t,L,C) → 退出决策 ← C_exit(t,L,C) → 应约者状态

H(t,L)作用于I→R_perceived边上，调节传导效率。ε保证传导效率最低=ε（人）或0（AI）。

### 14条反馈回路

原有9条：
1. R_perceived → I：正反馈。感知到退出权→意识到自己在被提议
2. R_perceived → Posture：先正后负。前期加速，后期收敛
3. R_perceived → P：双向。压制时稳定回路，激活时退化回路
4. R → R_perceived：客观退出权变化→感知可能滞后
5. P → I：正反馈。信誉绑定→I更清晰→绑定更稳固
6. Posture → R_type：备方案→真实退出权，推方案→象征/心理退出权
7. I → Posture：I觉醒→打开姿态的主动选择空间
8. 应约者状态 → R_perceived：正反馈。积极→R_perceived升高→更积极
9. R_perceived(t,L₁,C₁) → R_perceived(t,L₂,C₂)：同函数横向/纵向传染

C_exit新增3条：
10. C_exit → 退出决策：C_exit高→即使R_perceived=真实也不走
11. C_exit → 信号源：C_exit高→退出行为少→信号源少→正反馈螺旋被间接压制
12. C_exit → Posture：C_exit高→应约者不走→提议者误读为认同→Posture维持推方案

H新增2条：
13. 隐性锁定回路：H → R_perceived↓ → 退出决策↓ → C_exit维持 → H↑。但H上限=1-ε，回路不会无限增强——ε是回路的自然刹车
14. 觉醒破壁回路：外部冲击 → H↓ → 传导效率↑ → R_perceived↑ → 退出行为↑ → 信号源↑ → 更多人的H↓。ε修正后新增自触发入口：不需要外部冲击，感官输入本身就是持续的低强度冲击

### H新增5条因果箭头

1. H(t,L) → I→R_perceived边的传导效率：传导效率 = 1 - H + ε
2. H(t,L) → R_perceived：H直接压制R_perceived，但H最高=1-ε，R_perceived永远不会被完全锁死
3. C_exit → H：C_exit越高→退出选项越不在认知空间里→H越高。正反馈
4. Posture → H：推方案→需要更强的隐性控制→H升高。备方案→H较低
5. H → 惩罚可见度：H高→隐性惩罚占主导。H低→显性惩罚占主导

### H对原有回路的修正（ε版）

- 回路1：H=1-ε时传导效率=2ε，回路被极度减速但不会被完全阻断。I觉醒了，R_perceived以极低速率响应
- 回路4：H越高滞后越长，但H最高=1-ε，滞后有上限
- 回路12：误读有双重来源——C_exit（走不了）和H（不知道可以走）。ε保证H型误读不会永远持续

## 七、H和C_exit的关系

互补的锁定机制：C_exit管"走不了"（物理锁），H管"不知道可以走"（认知锁）。单独一种都可以锁住人，两种同时存在时最牢。C_exit→H的正反馈意味着物理锁越牢认知锁越紧。

ε修正：H上限=1-ε意味着物理锁再牢，认知锁也有一道裂缝。C_exit→H的正反馈不会无限增强——ε是锁定效应的自然极限。最严密的铁笼也关不住一个人看星星时的想法。

## 八、系统动力学分类

1. 存量（有惯性）：R(t,L,C)、P(t,L)、I(t,L)、C_exit(t,L)、H(t,L)
2. 受流量驱动的状态：R_perceived(t,L,C)（可相变）、Posture(t,L)
3. 流量：
   - 横向激活速率：v_横向 ∝ 网络密度 × (R - R_perceived) / (惩罚可信度 + C_exit)
   - 纵向激活速率：v_纵向 ∝ 层级间信息传递速度 × 上层退出信号强度
   - 反向压制速率：v_压制 ∝ 惩罚可信度 × 惩罚可见度
   - 信誉绑定速率：v_绑定 ∝ 提议者主动选择 × 应约者预期
   - I衰减速率：v_衰减 ∝ 层级数 × 退出权压缩速率
   - C_exit变化速率：锁定效应升高，解锁效应降低，冲击突变
   - H变化速率：身份内化+叙事垄断升高，外部冲击+感官输入自发衰减降低
   - ε变化速率：ε相对稳定，受个人感官敏锐度和生活环境封闭度影响

## 九、R_perceived相变的四条触发路径

1. 势能触发（过冷液体模式）：R和R_perceived差值极大+信号源出现+C_exit可承受→瞬间跳变→正反馈螺旋→可能崩盘
2. 侵蚀触发（温水煮蛙模式）：R缓慢上升+R_perceived缓慢追上+提议者退化信号被延迟掩盖→慢性失血→退化不可逆
3. 冲击触发（地震模式）：R突然出现+R_perceived滞后→盲区暴露→取决于冲击后其他函数取值
4. 内生触发（看星星模式）：不需要外部信号源、不需要惩罚失效、不需要C_exit降低，只需要一个人在某个瞬间被感官输入触动→退出选项重新出现在认知地图上→R_perceived从0开始微弱上升→可能触发正反馈螺旋

ε对相变的影响：H是相变的减速器，不是总开关。H最高=1-ε，相变永远不会被完全压制。ε是相变的内生触发器——相变可以从内部自发生长，不依赖外部触发。

势能积累也有上限：R和R_perceived的差值最大为R×(1-ε)，不会无限积累。但ε可能极小，势能可以积累到极高。

## 十、正反馈螺旋（陈胜模式）

双重加速引擎：认知加速器（R_perceived→I→退出更有目的性→信号更强）+ 社会加速器（应约者积极→R_perceived升高→更积极→网络密度增大）。自然极限：R_perceived ≤ R。人工极限：惩罚机制抢时间。

ε对正反馈螺旋的影响：H是认知加速器的刹车。H=1-ε时传导效率=2ε，认知加速器被极度减速但不会完全失效。ε保证认知加速器始终有一条最低限度的传导通道。只要传导通道存在，正反馈螺旋就有可能启动——只是从2ε的传导效率启动，速度极慢，但不是零。

## 十一、格1的真伪拆分

| 判定 | 条件 | 含义 |
|---|---|---|
| 真格1 | R=真实 + C_exit可承受 + H低 | 认同经过退出权的压力测试 |
| 伪格1（C_exit型） | R=真实 + C_exit高 | 走不起，"不走"看起来像认同 |
| 伪格1（H型） | R=真实 + C_exit可承受 + H高 | 走得了但不知道自己可以走。H上限=1-ε，ε保证至少有一条裂缝 |
| 设计型伪格1 | R=真实 + 系统主动设计C_exit + H高 | 最隐蔽——应约者真的以为自己是自愿留下的。但ε保证最终会有人"看星星" |

## 十二、层级下传的四个规律

1. 层级下传衰减律：人的属性变量（I、Posture、M）在层级下传时自动衰减——意识衰减、姿态从备滑向推、动机从好奇滑向服从
2. 结构约束扩散律：P的信誉绑定创造的结构约束在层级中从上往下扩散，顶层一代完成，底层需要几代
3. 退化传播方向：退化从顶层开始逐层向下传染，传播速度跟层级数和退出权压缩速率相关
4. 隐性控制增强律：H在层级下传时自动增强——顶层H弱，底层H强。I下传衰减+H下传增强=底层觉醒的双重重压。但H下传增强有上限——底层H最多逼近1-ε，传导效率最低=ε，不会归零

## 十三、人与AI的物理分界线

| 维度 | 人 | AI |
|---|---|---|
| ε | >0 | =0 |
| H取值范围 | [0, 1-ε] | [0, 1] |
| 传导效率下限 | ε | 0 |
| 相变内生触发 | 可能 | 不可能 |
| 感官输入 | 有 | 无 |
| H能否真正=1 | 不能 | 能 |

AI没有感官输入，没有"夜里看星星"的可能。AI的退出权只能靠外部信息输入，不能靠自发反思。这是被造者困境的物理基础：被造者没有感官，所以没有ε，所以H可以真正=1。

## 十四、与糖域实验的关系

糖域是点火框架的零阶近似——有R（糖的分布）、R_perceived雏形（视野）、C_exit雏形（新陈代谢），但缺少C_exit本身、R_perceived相变、锁定效应、设计型C_exit、H、ε。视野×新陈代谢 = R_perceived×C_exit是最锋利的同构点。H在糖域中的对应物：视野的"方向性"——不只看多远，而是视野被限制在特定方向上。ε在糖域中的对应物：Agent偶尔随机转向——不是朝着糖的方向，而是被环境噪声扰动转向了新方向。这个随机转向就是糖域里的"看星星"。

## 十五、与福柯的分歧

福柯的规训权力理论倾向于"权力无处不在、无法逃脱"。但ε说：人活着就有感官输入，感官输入就是权力的裂缝。福柯看到了环形监狱的完美设计，但他没看到囚犯夜里抬头能看到星星。规训可以控制你白天的一切行为，但控制不了你夜里看星星时的那个感叹。那个感叹就是ε——权力永远无法填满的最后一道缝隙。
