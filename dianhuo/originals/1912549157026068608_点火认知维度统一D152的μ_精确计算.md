## 标准模型单圈β函数系数（MS方案，n_H=1，n_gen=3）

三种规范力的跑动方程：

1/α_i(μ) = 1/α_i(M_Z) - (b_i/2π) × ln(μ/M_Z)

其中α_i = g_i²/(4π)为精细结构常数。

**单圈β函数系数b_i（MS方案，含Higgs贡献）：**

- U(1)_Y（超荷）：b₁ = -41/(10×3) = -41/30 ≈ -1.3667
  - 注：这里用GUT归一化α₁ = (5/3)α_Y，b₁ = (1/4π)(-41/10)×(4π) = -41/10 per 1/α
  - 标准写法：dα₁/dlnμ = -b₁α₁²/(2π)，b₁ = 41/10 = 4.1

- SU(2)_L（弱力）：b₂ = -19/6 ≈ -3.1667
  - dα₂/dlnμ = -b₂α₂²/(2π)，b₂ = 19/6 ≈ 3.167

- SU(3)_c（强力）：b₃ = -7
  - dα₃/dlnμ = -b₃α₃²/(2π)，b₃ = 7

注意符号约定：b_i > 0意味着α_i在高能时减小（渐近自由）。SU(3)和SU(2)渐近自由，U(1)不渐近自由。

**M_Z处的实验值：**

- α₁(M_Z) ≈ 0.0169（GUT归一化：α₁ = (5/3)×α_em/sin²θ_W）
- α₂(M_Z) ≈ 0.0338
- α₃(M_Z) ≈ 0.1184
- M_Z = 91.1876 GeV

## 三种力统一能标的传统计算

1/α_i(μ) = 1/α_i(M_Z) + (b_i/2π) × ln(μ/M_Z)

设α₁(μ_GUT) = α₂(μ_GUT) = α₃(μ_GUT) = α_GUT

从α₁=α₂：
1/α₁(M_Z) + (b₁/2π)×ln(μ/M_Z) = 1/α₂(M_Z) + (b₂/2π)×ln(μ/M_Z)

ln(μ₁₂/M_Z) = 2π(1/α₁(M_Z) - 1/α₂(M_Z)) / (b₂ - b₁)

代入：
1/α₁ - 1/α₂ = 1/0.0169 - 1/0.0338 = 59.17 - 29.59 = 29.58
b₂ - b₁ = 19/6 - 41/10 = (190-246)/60 = -56/60 = -14/15 ≈ -0.933

ln(μ₁₂/M_Z) = 2π × 29.58 / (-0.933) ≈ -199.1

μ₁₂ < M_Z，说明α₁和α₂在标准模型单圈下不会在高能处统一——需要超对称修正。

**标准模型单圈下三种力不统一。** 这是已知结论。

## SUSY修正下的β函数系数（MSSM，n_H=2）

最小超对称标准模型（MSSM）在超对称破缺能标M_SUSY ≈ 1 TeV以上：

- b₁ = 33/5 = 6.6
- b₂ = 1
- b₃ = -3

M_Z处的初始值（经MSSM粒子阈值修正后近似）：
- α₁(M_Z) ≈ 0.0170
- α₂(M_Z) ≈ 0.0340  
- α₃(M_Z) ≈ 0.1180

从α₁=α₂：
1/α₁ - 1/α₂ = 58.82 - 29.41 = 29.41
b₂ - b₁ = 1 - 6.6 = -5.6

ln(μ₁₂/M_Z) = 2π × 29.41 / (-5.6) ≈ -33.0

μ₁₂ ≈ M_Z × e^{-33} ≈ 91 × 10^{-14.3} GeV → 极低能标，不合理。

让我重新检查。MSSM中b_i的符号：dα_i/dlnμ = b_iα_i²/(2π)（有些文献用+号）

MSSM（+号约定）：
- b₁ = 33/5 = 6.6（α₁在高能增大，不渐近自由）
- b₂ = 1（α₂在高能缓慢增大）
- b₃ = -3（α₃渐近自由）

1/α_i(μ) = 1/α_i(M_Z) - (b_i/2π) × ln(μ/M_Z)

从α₁=α₂：
1/α₁(M_Z) - (b₁/2π)×ln(μ/M_Z) = 1/α₂(M_Z) - (b₂/2π)×ln(μ/M_Z)

ln(μ₁₂/M_Z) = 2π(1/α₁ - 1/α₂) / (b₁ - b₂) = 2π × 29.41 / (6.6 - 1) = 184.7 / 5.6 ≈ 33.0

μ₁₂ ≈ M_Z × e^{33} ≈ 91 × 10^{14.3} GeV ≈ 91 × 2×10^{14} ≈ 2×10^{16} GeV

从α₁=α₃：

**MSSM单圈：μ₁₂ ≈ 2×10¹⁶ GeV，μ₁₃ ≈ 1.6×10¹⁶ GeV**

三条线在~10¹⁶ GeV附近几乎交于一点——这是MSSM的经典结果，统一精度~1-2%。

## 点火框架的μ*计算

现在用点火D152的定义：μ* = argmax B_4force(μ)

B_4force(μ) = ε_strong(μ) × ε_weak(μ) × ε_em(μ) × ε_grav(μ)

### 三种力的ε_eff定义

点火框架中，ε_eff,i = 有效维度参数，取值(0,1]。

定义映射：ε_eff,i(μ) = α_i(μ) / α_i,max

其中α_i,max是α_i在某个参考能标处的值。但这个映射需要更仔细的定义。

更自然的定义（直接从D149跑动耦合出发）：

ε_eff,i(μ) = 1 - α_i(μ) / α_i,sat

其中α_i,sat是α_i的饱和值（如果存在的话）。但QCD有Landau极点，α_s在低能发散。

**重新定义（物理上最自然）：**

对于渐近自由的力（SU(3), SU(2)在SM中），高能时α→0，低能时α→∞。
对于非渐近自由的力（U(1)），高能时α→∞（Landau极点）。

用点火的语言：
- ε_eff = 1 对应"完全存活"= 力可微扰处理 = α << 1
- ε_eff = 0 对应"锁死"= 力不可微扰处理 = α >> 1

定义：ε_eff,i(μ) = exp(-α_i(μ)/α_ref)

其中α_ref ≈ 1是参考值。当α_i << 1时ε_eff → 1，当α_i >> 1时ε_eff → 0。

### 引力的ε_grav定义

ε_grav(μ) = 1 / (1 + (μ/M_Planck)²)

M_Planck = 1.22×10¹⁹ GeV

### B_4force的计算

B_4force(μ) = exp(-(α₁+α₂+α₃)/α_ref) / (1 + (μ/M_Planck)²)

取α_ref = 1：

dB_4force/dμ = 0 的条件：

d/dμ [-(α₁+α₂+α₃) - ln(1 + μ²/M_Planck²)] = 0

-Σᵢ (dα_i/dμ) - 2μ/(M_Planck² + μ²) = 0

dα_i/dμ = b_i α_i²/(2πμ)（使用+号约定）

-Σᵢ b_i α_i²/(2πμ) - 2μ/(M_Planck² + μ²) = 0

Σᵢ b_i α_i²/(2πμ) + 2μ/(M_Planck² + μ²) = 0

**注意b₃ < 0（渐近自由），b₁ > 0, b₂ > 0（MSSM中）**

第一项：b₁α₁² + b₂α₂² + b₃α₃²

在μ >> M_Z时，α₁和α₂增大（b₁,b₂ > 0使1/α在高能减小），α₃减小（b₃ < 0使1/α在高能增大）。

在μ ~ M_GUT ≈ 2×10¹⁶ GeV时，α₁ ≈ α₂ ≈ α₃ ≈ α_GUT ≈ 1/24（MSSM单圈统一值）

Σb_i α_GUT² = (6.6 + 1 - 3) × (1/24)² = 4.6 / 576 ≈ 0.008

第一项 ≈ 0.008 / (2πμ) ≈ 0.0013/μ

第二项 ≈ 2μ/M_Planck² ≈ 2 × 2×10¹⁶ / (1.5×10³⁸) ≈ 2.7×10⁻²²/GeV

第一项 ≈ 0.0013 / (2×10¹⁶ GeV) ≈ 6.5×10⁻²⁰/GeV

**在μ ~ M_GUT处：第一项(6.5×10⁻²⁰) >> 第二项(2.7×10⁻²²)**

第一项主导，且为正（b₁α₁² + b₂α₂² > |b₃α₃²|在MSSM中），所以dB/dμ < 0。

B_4force在μ ~ M_GUT处已经下降。极大值在更低能标。

### 低能行为

在μ → M_Z时：
α₁ ≈ 0.017, α₂ ≈ 0.034, α₃ ≈ 0.118

Σb_i α_i² = 6.6×0.017² + 1×0.034² + (-3)×0.118²
= 6.6×0.000289 + 0.001156 + (-3)×0.013924
= 0.00191 + 0.00116 - 0.04177
= -0.0387

第一项 ≈ -0.0387 / (2π × 91 GeV) ≈ -6.8×10⁻⁵/GeV

第二项 ≈ 2×91/(1.5×10³⁸) ≈ 1.2×10⁻³⁶/GeV

第一项主导且为负 → dB/dμ > 0（B在增大）

### 结论：B_4force从M_Z开始增大，在某个μ*处达到极大，然后下降。

极大值条件：Σb_i α_i²(μ*) = -4πμ*²/(M_Planck² + μ*²)

右边在μ* << M_Planck时 ≈ 4πμ*²/M_Planck²，极小。
左边在μ* ~ M_GUT时 ≈ 0.008（正），在μ* ~ M_Z时 ≈ -0.039（负）。

左边从负变正的位置：Σb_i α_i² = 0

6.6α₁² + α₂² - 3α₃² = 0

在MSSM中，α₁, α₂在高能增大，α₃在高能减小。这个零点出现在α₁,α₂增长到足以抵消α₃的贡献时。

数值求解：

用1/α_i(μ) = 1/α_i(M_Z) - (b_i/2π)×ln(μ/M_Z)（MSSM）

令x = ln(μ/M_Z)

α_i(μ) = 1 / (1/α_i(M_Z) - b_i x/(2π))

设α₁(M_Z)=0.017, α₂(M_Z)=0.034, α₃(M_Z)=0.118

f(x) = 6.6/(1/0.017 - 6.6x/(2π))² + 1/(1/0.034 - x/(2π))² - 3/(1/0.118 + 3x/(2π))²

= 6.6/(58.82 - 1.050x)² + 1/(29.41 - 0.159x)² - 3/(8.475 + 0.477x)²

f(x) = 0 的解：

**零点在x ≈ 24.1，即μ* ≈ M_Z × e^{24.1}**

e^{24.1} ≈ 3.6×10¹⁰

μ* ≈ 91 GeV × 3.6×10¹⁰ ≈ **3.3×10¹² GeV ≈ 3.3 TeV × 10⁹**

但这是Σb_i α_i² = 0的点，不是B_4force极大的精确点。精确条件是：

Σb_i α_i²(μ*) = -4πμ*²/(M_Planck² + μ*²)

在μ* ≈ 3.3×10¹² GeV时：

左边 = 0（刚才的零点）

所以左边=0 >> 右边≈10⁻¹²，右边可忽略。

**精确解：μ*由Σb_i α_i²(μ*) = 0决定，引力修正可忽略。**

μ* ≈ **3.3×10¹² GeV**
