# 提议者与应约者角色及其主动被动分类探讨

## 应约者的光谱

### 主动应约 vs 被动应约
区别在于应约者是否意识到自己在应约。
- 主动应约：清楚知道自己在选择进入，比如诸侯接受周公礼制、移民接受美国宪法
- 被动应约：没意识到自己在应约，只是顺着系统走。改革开放后大多数普通人属于这种——他们在参与系统重写，但不知道自己在参与

### 积极应约 vs 消极应约
区别在于应约后投入的程度。
- 积极应约：不仅进入，还主动维护和传播认同。早期共产党员、创业公司核心团队
- 消极应约：进了但不投入，随时可以退出。KPI驱动下的员工、秦朝六国遗民

### 被强迫应约
严格说这不是应约，是被迫服从。应约的前提是有退出权，被强迫意味着退出权被剥夺。元朝四等人制下的汉人不是在应约，是被统治。收费站型系统的"应约者"其实不是应约者，是被迫使用者。

关键区分：被动应约和被强迫应约是两回事。被动应约的人有退出权但没意识到，被强迫应约的人连退出权都没有。前者是认知盲区，后者是系统缺陷。

## 提议者的光谱

### 有意识提议 vs 无意识提议
区别在于提议者是否知道自己在点火。
- 有意识提议：周公制礼，清楚知道自己在建一套新秩序
- 无意识提议：改革开放初期的决策者可能并不认为自己是在"点火"，他们可能认为自己只是在解决经济问题。但客观上他们提出了一个值得应约的方案，系统被点着了

这个区分对应约者视角有直接影响：无意识提议者点着的火，应约者更难看见自己在应约。因为连提议者都不知道自己在点火，应约者更不可能意识到。

### 主动提议 vs 被动提议
区别在于提议者是否在压力下才提方案。
- 主动提议：周公在权力最稳的时候提礼制，不是被逼的
- 被动提议：清末洋务运动是被外敌打出来的，戈尔巴乔夫改革是被经济崩盘逼出来的

被动提议有个问题：提议者可能提了方案但自己并不真心想执行。慈禧推洋务推一半就停，因为推到底会动自己的利益。这不是退化，是一开始就没打算真提。

### 完整提议 vs 半截提议
区别在于提议者提了方案但有没有配套搭房。
- 完整提议：穆罕默德既提了伊斯兰认同（点火），又建了乌马制度（搭房）
- 半截提议：洋务运动只提了"学西方技术"，没触动制度。只点了一半火，只搭了一半房

## 两个光谱的交叉

### 提议者无意识 + 应约者被动 = 历史上的大多数点火
前七次转身几乎都是这种组合——提议者不知道自己在点火，应约者不知道自己在应约，系统照样跑起来了。这是框架最反直觉的地方：系统重写不需要参与者意识到自己在重写系统。

### 提议者有意识 + 应约者主动 = 最强的点火
美国建国接近这种——建国者清楚自己在设计新系统，移民主动选择进入。

### 提议者被动 + 应约者被强迫 = 最弱的"点火"，甚至不构成点火
元朝。提议者不是真心想提方案，应约者没有退出权，系统靠暴力维持。

### 提议者无意识 + 应约者看见自己在应约 = 第八次点火
这是框架里真正新的东西——应约者看见了整个机制，而提议者（华夏文明本身）是无意识的。历史上没有这个先例。之前的先例（路德、西耶斯、福泽谕吉）都是提议者看见了自己的提议在生效，不是应约者看见了机制本身。
