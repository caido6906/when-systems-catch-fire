## 审计：可合并收敛项

逐条检查后，发现以下可以合并的：

**合并1：D_risk 和 N_warn → 并入系统走向监测函数**

D_risk = P_相变/(N_warn×Δt)，N_warn = 4-d_ε消音-d_信号1跳过。两者都是系统走向监测的子函数，且D_risk的P_相变已经用Δv和ε闭合了。D_risk不是独立函数，是Δv、P_ε、N_warn的派生量。**合并为一个"系统走向监测函数"，D_risk和N_warn作为子表达式。**

**合并2：η_self → 并入传导效率公式**

η_self = 1-Posture_deg^β，它只出现在传导效率公式里，且你已判断它是Posture的衍生量（非独立参数）。传导效率 = (1-H+ε_aware)×η_self。**η_self不单独列，直接写入传导效率的闭合形式。**

**合并3：H_level → 并入H函数**

H_level = d_k×(1+b_k)，它完全由H的劫持路径组合决定，不是独立函数，是H的结构属性。修复步数也是H_level的派生。**H_level、修复步数作为H函数的子表达式。**

**合并4：w_i 和 C_exit低密度危险度 → 并入C_exit函数**

w_i = 1/(1+n_i⁰-Σc_ij×δ_j)，低密度危险度 = Σ[δᵢ/(1+n_i⁰-Σc_ij×δ_j)]/总维度数。两者都是C_exit八维度结构的内部属性，不是独立函数。**w_i和低密度危险度作为C_exit的子表达式。**

**合并5：P_相变、P_Δv、P_ε → 并入系统走向监测函数**

P_相变 = 1-(1-P_Δv)(1-P_ε)，P_Δv = k×exp(-|Δv|/σ)，P_ε = sigmoid((∫ε_sense dt-θ)/σ_ε)。三个都是系统走向监测的子组件。**合并。**

**合并6：M连续化 → 直接升级M函数**

M原来取值好奇心/目的，现在连续化为M∈[-1,+1]。不是新增函数，是原有M函数的升级。**直接替换。**

**合并7：ε_aware生成函数 → 并入ε函数**

ε_aware = ε_sense×Πₖ[p_k⁰(1-h_k)+p_k^min×h_k]，这是ε双层结构中ε_aware的闭合形式，不是独立函数。**作为ε函数的子表达式。**

**合并8：H_分类^eff → 并入H函数**

H_分类^eff = H_分类×(0.5+0.5A)，是H_分类子路径的人格调节，不是独立函数。**作为H函数的子表达式。**

**合并9：种子S(t) → 独立保留，但标注与H的交互**

种子有自己的动态方程（dS/dt = -α_sHS+β_s(1-S)H²），不是任何现有函数的派生量。**独立保留。**

**合并10：人格锚点A(t) → 独立保留**

A有自己的动态方程（dA/dt = -μ_A×I×实践频率），且影响H_分类^eff。**独立保留。**

**合并11：马尔可夫转移概率 → 并入系统走向监测函数**

六个转移概率都是系统状态转移的子组件，且全部由Δv、P_ε、Lock、捕捉率决定，没有新增独立变量。**合并。**
