# 先发Agent优势分析

这个推导要从四个过程同时展开，直接出结果。

**一、马尔可夫链过程：用户状态转移矩阵**

用户在平台间的状态转移可以用三态马尔可夫链建模：

状态S = {A: 活跃用户, D: 休眠用户, E: 已退出}

转移矩阵P：

| | A | D | E |
|---|---|---|---|
| A | P_AA | P_AD | P_AE |
| D | P_DA | P_DD | P_DE |
| E | P_EA | P_ED | P_EE |

**先发Agent化如何改变转移概率？**

得到大脑先完成Agent化的时刻t₀，转移矩阵发生跳变：

- P_AA↑：活跃用户留存率上升（能办事→不走了）
- P_DA↑：休眠用户回流率上升（能办事→回来试试）
- P_AE↓：活跃→退出的概率下降（C_exit上升——Agent能力制造新的锁定维度）
- P_DE↓：休眠→退出的概率下降（还有理由回来）
- P_EA↑：已退出用户回流率上升（听说能办事了→回来）

关键：P_EA在内容型产品中≈0（走了的人几乎没有理由回来），Agent化后P_EA>0。这是内容型产品不可能有的转移路径。

**稳态分布的变化：**

设先发时间差为Δt。在Δt内，竞品还没Agent化，得到大脑独占Agent态。

稳态活跃用户比例π_A满足：

π_A = lim(n→∞) P^n · π_0

先发Δt的稳态增量：

Δπ_A(Δt) = π_A(Agent先发) - π_A(同步Agent化)

这个增量不是线性的。因为P_AA↑和P_DA↑同时作用，形成正反馈——更多活跃用户→更多内容生产→更多新用户→P_AA进一步↑。Δt内的增量会被正反馈放大。

**二、系统动力学过程：正反馈回路**

先发Agent化激活三条正反馈回路：

**回路1：能力-用户回路**
Agent能力↑ → 用户办事成功率↑ → 用户留存↑ → 更多用户数据 → Agent能力↑（学习效应）

dP_infra/dt = α₁ × N_active × η_learning

N_active是活跃用户数，η_learning是学习效率。先发者有更多N_active，P_infra增长更快。

**回路2：锁定-退出成本回路**
Agent能力↑ → 用户在平台上完成更多任务 → C_exit(时间+信息+身份)↑ → P_AE↓ → 留存↑

dC_exit/dt = α₂ × P_infra × N_active

先发者不仅P_infra更高，C_exit也在增长——用户在Agent上投入的时间、建立的流程、积累的个性化数据，都是退出成本。

**回路3：内容-认知跳变回路**
Agent能力↑ → 内容从"只能看"变为"能用" → σ(ε×A×D×|M_cog|)↑ → 认知跳变概率↑ → 用户获得正向转变 → 口碑传播 → 新用户↑

**回路1×回路2×回路3的乘法效应：**

dV_advantage/dt = α₁ × N × η × P_infra × σ(认知跳变)

三条回路同时运行，产出乘法叠加。先发Δt的初始优势被三条正反馈回路放大。

**三、用户行为学过程：习惯锁定与切换成本**

用户从内容型切换到Agent型，存在学习成本C_switch。先发者的优势在于：

**习惯形成函数：**

Habit(t) = 1 - exp(-λ × t_usage)

t_usage是使用Agent功能的时间。Habit→1时，用户形成操作习惯，切换到竞品的C_switch上升。

先发Δt内，用户的Habit已经接近1。竞品Agent化时，用户面临：

C_switch = C_learning(新平台) + C_migration(数据迁移) + C_habit(打破旧习惯)

C_habit = Habit(得到大脑) × w_habit

先发Δt越长，C_habit越大，切换越难。

**切换成本函数：**

P(切换, t) = σ(R_perceived(竞品) - C_switch(t)) × exp(-n_lock × C̄_exit / θ_C)

先发者通过Δt积累了C_exit（时间维度+信息维度+身份维度），n_lock从1-2升到2-3。竞品Agent化时P(切换)已经低于阈值。

**四、传播学过程：R_perceived的横向传染**

R_perceived有横向传染机制：同层级有人用Agent办事成功→其他人感知到→R_perceived跳变。

**传染速率函数：**

v_传染(t) = ρ × N_已转化(t) × (N_总 - N_已转化(t)) / N_总²

这是SIR模型中的感染速率。ρ是传染系数，取决于：
- 信号强度：Agent办事成功的可见度
- 网络密度：用户之间的社交连接
- C_exit：传播阻力

先发者在Δt内独占传染源。已转化用户N_已转化在Δt内按logistic曲线增长：

N_已转化(t) = N_总 / (1 + exp(-ρ × (t - t₀)))

先发Δt内N_已转化已经过拐点，竞品Agent化时传染势能已经释放大半。

**五、收敛：先发优势函数**

综合四个过程，先发优势函数：

**A_first(Δt) = Δπ_A(Δt) × exp(∫₀^Δt [α₁×N×η×P_infra + α₂×P_infra×N - λ_decay] dτ)**

- Δπ_A：马尔可夫稳态增量（回流+留存）
- exp项：系统动力学正反馈的累积放大
- λ_decay：优势的自然衰减率（竞品追赶速度）

**优势能维持多久？**

优势半衰期T_half满足：

A_first(T_half) = A_first(0) / 2

T_half = ln(2) / (λ_decay - α₁×N×η×P_infra)

当正反馈强度（α₁×N×η×P_infra）> 衰减率（λ_decay）时，T_half→∞——优势不衰减，反而扩大。

**这是关键结论：先发优势不是"先跑一段"然后被追平，是正反馈回路是否已经越过自持临界点。**

自持条件：α₁ × N_active × η_learning × P_infra > λ_decay

越过自持临界点后：
- 马尔可夫稳态已经偏向先发者（π_A↑）
- C_exit已经积累到n_lock≥2（切换成本显著）
- Habit已经接近1（习惯锁定）
- 传染已经过拐点（R_perceived已在用户群中扩散）

竞品即使完成Agent化，面对的是一个n_lock≥2、Habit≈1、π_A已偏向先发者的用户群。P(切换)被压制在阈值以下。

**六、优势大小的量化**

设行业Agent化完成时间为T_industry，得到大脑先发时间差Δt = T_industry - T_得到大脑。

| Δt | 优势等级 | 机制 | 可逆性 |
|---|---|---|---|
| 0（同步） | 无 | 无先发优势 | — |
| 3个月 | 弱 | P_infra略高，C_exit微增 | 高——习惯未形成 |
| 6个月 | 中 | Habit>0.5，n_lock=2，传染过拐点 | 中——切换成本显著 |
| 12个月 | 强 | Habit→1，n_lock≥3，正反馈自持 | 低——需竞品有代际优势才能逆转 |
| 18个月+ | 锁定 | 稳态π_A偏向先发者，P(切换)<阈值 | 极低——数学上不可逆 |

**n_lock≥3是关键阈值**——跟预测表的行动分流阈值完全一致。先发12个月以上，用户锁定维度数超过3，P(切换)进入数学上不可能区间。存好了。

四个过程的收敛函数：

**A_first(Δt) = Δπ_A × exp(∫₀^Δt [α₁×N×η×P_infra + α₂×P_infra×N - λ_decay] dτ)**

关键结论不是"先跑多远"，是**正反馈是否越过自持临界点**。

自持条件：α₁ × N × η × P_infra > λ_decay

越过之后优势不衰减反而扩大，T_half→∞。没越过就只是"先跑一段"，迟早被追平。

四个过程分别对应P_forward的四个因子：
- 马尔可夫稳态 → 认知跳变因子的用户分布
- 系统动力学 → f(P_infra)的正反馈增长
- 用户行为学 → exp项的锁定积累（n_lock从1-2升到2-3）
- 传播学 → σ(θ_d-d)的传染扩散

量化结果：6个月是中等优势（Habit>0.5，传染过拐点），12个月是强优势（n_lock≥3，正反馈自持），18个月以上锁定（数学上不可逆）。n_lock≥3这个阈值跟预测表的行动分流阈值完全一致——不是巧合，是同一个乘法结构在不同场景的投影。
