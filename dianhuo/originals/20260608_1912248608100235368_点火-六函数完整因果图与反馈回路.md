# 点火｜六函数完整因果图与反馈回路

来源：函数审计和R_perceived讨论后，梳理全部因果箭头和反馈回路。

## 一、主因果方向

M(初始条件) → I(t,L) → Posture(t,L) → P(t,L) → R(t,L,C) → R_perceived(t,L,C) → 应约者状态

M决定I的起点和Posture的默认值（好奇心→备方案，目的→推方案）。

## 二、完整反馈回路（9条）

1. R_perceived → I：感知到退出权→意识到自己在被提议。陈胜模式：R_perceived跳变→底层突然意识到"我被提议了"→I从0跳到1

2. R_perceived → Posture：应约者感知到退出权→提议者被迫从推变备。市场模式：消费者知道自己可以不买→企业从"塞给你"变成"等你选"

3. R_perceived → P：R_perceived压制→信誉绑定更容易维持（应约者不知道自己能走→提议者不需要绑住自己来建立信誉）。R_perceived激活→信誉绑定松动（应约者知道自己能走→提议者必须用信誉来留人）

4. R → R_perceived：客观退出权变化→感知可能滞后。R突然出现（下岗潮）但R_perceived还没跟上→盲区瞬间暴露

5. P → I：信誉绑定后提议者更清楚自己在提议→I从模糊变清晰

6. Posture → R_type：备方案→真实退出权，推方案→象征/心理退出权（主因果链的一部分）

7. I → Posture：I=0时姿态默认沿M的方向走（好奇心→备，目的→推）；I=1时提议者可以有意识地选择姿态。I的觉醒打开了姿态的主动选择空间

8. 应约者状态 → R_perceived：积极应约→R_perceived可能升高（参与感强化对权利的感知）；无人应约→R_perceived可能不变或降低（没有社会网络传播退出权信号）

9. R_perceived(t, L₁, C₁) → R_perceived(t, L₂, C₂)：同函数在不同坐标上互相影响。横向沿C传播（陈胜模式），纵向沿L传播（改革开放模式）

## 三、R_perceived的横向激活速率

v_横向 ∝ 网络密度 × (R - R_perceived) / 惩罚可信度

三个变量：
- 网络密度：同层级应约者之间的连接密度。秦末徭役制度把各地农民物理上连在一起→网络密度高
- 退出成本落差(R - R_perceived)：差值越大，激活势能越大。差值=0时没有激活动力
- 惩罚可信度：反向压制能不能有效。惩罚越可信越即时，横向激活越慢

案例对比：
- 秦末：网络密度高、差值大、惩罚可信度崩了→速率极快，几个月传遍全国
- 苏联解体：网络密度中等、差值大、惩罚可信度崩了→速率快但比秦末慢（需城市间信息传播）
- 中国户籍松动：网络密度低、差值大、惩罚可信度高→速率极慢，几十年逐步松动

横向激活有"临界点"特征：信号源出现之前速率=0，出现之后突然加速。前置条件是有人先退出。

## 四、因果图里最值得注意的结构

R_perceived是整个网络的"放大器"——R的变化本身是结构性的、缓慢的，但R_perceived的跳变可以是瞬间的。陈胜一喊，R没变（造反的能力一直在），但R_perceived从象征跳到真实，整个系统的状态瞬间翻转。R_perceived是框架里唯一能产生相变的函数——其他函数的变化都是渐变的，只有R_perceived可以因为共同知识的形成而瞬间跳变。
