## D127 乘法归零律

**定义**：对任意函数 F = ∏ᵢ₌₁ⁿ xᵢ，若 ∀i: xᵢ ∈ [0, Mᵢ]，则 F 的零点集为各因子零点集的并集：

Z(F) = ⋃ᵢ₌₁ⁿ Z(xᵢ)

即：任一因子归零 ⟹ F 归零，且仅当所有因子同时非零时 F > 0。

**四条推论**：

1. **无替代性**：∂lnF/∂lnxᵢ = 1，因子间无可补偿性。某个因子足够大不能补偿另一个因子归零。
2. **瓶颈锁定**：F ≤ min(xᵢ) · ∏(Mⱼ)，天花板由最弱因子决定，改善非瓶颈因子不改变上限。
3. **归零吸收**：xᵢ = 0 是吸收态，不可被其他因子拉回。不存在"几乎归零但被救回来"的路径。
4. **对数可加**：lnF = Σlnxᵢ，脆弱性在对数空间退耦为独立贡献，使各因子可独立分析。

**统一收敛**：

ZL(F) := { F = ∏ᵢ₌₁ⁿ xᵢ | xᵢ ∈ [0, Mᵢ], F = 0 ⟺ ∃i: xᵢ = 0 }

## 与D123的深层同构

D123缓存倒U型：因子不是越大越好，存在最优值。
D127乘法归零律：因子不能为零，零即死亡。

两者合起来构成乘法系统的完整约束：
- 下界是零（归零律）
- 上界非单调（倒U型）
- 系统只能在中间活

这是斯多葛底色的数学表达：外部可以混乱但内部不能归零，归零就是失去解释权。

## 在统一函数总表中的位置

D127是元函数，不依赖特定领域变量，描述的是乘法结构本身的数学性质。D86/D121/D124/D126等所有乘法形式函数都是D127的实例化。
