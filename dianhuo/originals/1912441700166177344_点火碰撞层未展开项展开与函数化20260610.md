## 一、α(t)和β(t)的动态驱动方程

来源：认知决策碰撞1，ε = Δ(t) = α(t) - β(t)

三支决策中α是接受阈值、β是拒绝阈值。两者各自受什么驱动？

α(t)（接受阈值）受两个力驱动：
- H(t)遮蔽：H越高，需要更强的证据才敢接受→α升高
- ε_aware(t)：感知退出权越高，接受新选项的阈值越低→α降低

**α(t) = α₀ + k_α × H(t) - m_α × ε_aware(t)**

β(t)（拒绝阈值）受两个力驱动：
- C_exit(t)：退出成本越高，拒绝当前状态的阈值越低（更不敢拒绝）→β升高（拒绝域缩小）
- Posture_deg(t)：提议者退化度越高，应约者拒绝的阈值越低→β升高

**β(t) = β₀ + k_β × C̄_exit(t)/θ_C - m_β × (1 - Posture_deg(t))**

犹豫域宽度：
**ε(t) = α(t) - β(t) = (α₀ - β₀) + k_α·H - m_α·ε_aware - k_β·C̄/θ_C + m_β·(1-Posture_deg)**

当H↑、ε_aware↓、C̄↑、Posture_deg↑时，α↑且β↑，犹豫域被双向压缩。退化路径：α和β从两侧同时向中间逼近，ε→0。

**新增碰撞层派生 D-X16：犹豫域双向压缩函数**
ε(t) = (α₀-β₀) + k_α·H(t) - m_α·ε_aware(t) - k_β·C̄_exit(t)/θ_C + m_β·(1-Posture_deg(t))

## 二、narrative_disruption的函数化

来源：认知决策碰撞3，P_ignite = σ(θ₀ - k(1-narrative_disruption))

narrative_disruption不是外生变量，它由三个基础变量决定：

1. **I(t)提议者意识**：提议者是否意识到自己在叙事——I=0时叙事是惯性延续（无破坏力），I=1时叙事是刻意冲击
2. **ε_aware(t)感知退出权**：应约者能否感知到叙事与现实的断裂——ε_aware=0时再强的叙事也无法制造disruption（听不到）
3. **D_immune(t)退化免疫**：系统是否还有自我修正能力——D_immune=0时叙事被系统过滤为噪音

**narrative_disruption(t) = I(t) × ε_aware(t) × D_immune(t)**

三因子乘法：提议者不意识=0，应约者听不到=0，系统过滤掉=0。任一归零则叙事冲击归零。

代入P_ignite：
**P_ignite = σ(θ₀ - k(1 - I × ε_aware × D_immune))**

当I×ε_aware×D_immune足够大时，1-narrative_disruption足够小，θ_inertia < θ₀，点火成功。

与P(L₃→L₅) = σ(ε×A×D×|M_cog|)对比：σ项内的ε×A×D与I×ε_aware×D_immune结构同构——都是"信号×放大×免疫"的三因子乘法。

**新增碰撞层派生 D-X17：叙事冲击函数**
narrative_disruption(t) = I(t) × ε_aware(t) × D_immune(t)
P_ignite = σ(θ₀ - k(1 - narrative_disruption))

## 三、认知犹豫域π的动态压缩

来源：认知决策碰撞4，D_blind = ν × (1-π)

π_cognitive = 1 - μ_understand - ν_misunderstand 的动态压缩过程：

专业化遮蔽的机制是π随时间单调递减，同时ν可能递增。两条路径同时压缩π：

路径1：经验积累→μ↑→π↓（正常的认知进步，π被理解吸收）
路径2：框架锁定→ν↑但被误认为μ↑→π↓（伪进步，π被误解吸收）

区分两条路径的关键：π↓同时μ↑是真进步，π↓同时ν↑是伪进步。但个体无法区分——因为π↓本身感觉一样（确定性增加了）。

**dπ/dt = -λ₁ × μ(t) - λ₂ × ν(t) × H_framework(t)**

- λ₁：正常学习速率（π被真理解吸收）
- λ₂ × H_framework：框架锁定速率（π被误解吸收，H_framework是框架遮蔽度）
- H_framework = 0时只有路径1，π被真理解吸收，D_blind不增加
- H_framework > 0时路径2激活，π↓但ν↑，D_blind = ν×(1-π)增加

**专业化遮蔽的充要条件：dπ/dt < 0 且 dν/dt > 0**
即犹豫域在缩小的同时误解在增加——确定性增加不等于正确性增加。

**新增碰撞层派生 D-X18：认知犹豫域动态压缩函数**
dπ/dt = -λ₁·μ(t) - λ₂·ν(t)·H_framework(t)
专业化遮蔽充要条件：dπ/dt < 0 且 dν/dt > 0

## 四、偏好伪造级联速度k的函数化

来源：人际社会碰撞1，dP_public/dt = k × P_public × (1 - P_public/P_private)

级联速度k不是常数，它取决于：

1. **网络密度ρ**：节点间连接越密，信号传播越快→k↑
2. **ε_aware平均**：群体感知退出权越高，表达阈值越低→k↑
3. **C_exit_avg**：平均退出成本越低，表达风险越小→k↑

**k = ρ × ε_aware_avg × (1 - C̄_exit/C_max)**

三因子乘法：网络稀疏=0、群体听不到=0、退出成本无穷大=0，任一归零则级联无法启动。

级联动力学的完整形式：
**dP_public/dt = ρ × ε_aware_avg × (1 - C̄_exit/C_max) × P_public × (1 - P_public/P_private)**

与流行病学SIR模型同构：P_public↔感染比例，P_private↔易感人群，k↔传染率。但k的语义绑定不同——不是生物学传染率，而是社会结构参数。

**新增碰撞层派生 D-X19：级联速度函数**
k(t) = ρ × ε_aware_avg(t) × (1 - C̄_exit(t)/C_max)

## 五、顽固者临界比例中β的函数化

来源：人际社会碰撞2，p_critical = 1/(1+β)

β是顽固者对群体碎片化的放大系数。β越大，临界比例越低——越少的顽固者就能导致碎片化。

β由两个变量决定：

1. **ε_HK_avg**：群体平均置信区间——ε_HK_avg越大，顽固者的相对"刚性"越突出→β↑
2. **network_clustering**：网络聚类系数——聚类越高，顽固者的局部影响越集中→β↑

**β = (ε*/ε_HK_avg) × network_clustering**

- ε_HK_avg → ε*时β→1，p_critical → 1/2（群体已经很分裂，一半顽固者就够）
- ε_HK_avg → 0时β→∞，p_critical → 0（群体完全没有犹豫域，任何顽固者都导致碎片化）
- network_clustering = 0（随机网络）时β最低，顽固者影响被稀释
- network_clustering = 1（完全聚类）时β最高，顽固者在局部形成刚性壁

**p_critical = 1/(1 + (ε*/ε_HK_avg) × network_clustering)**

推论：降低碎片化风险有两条路径——提高ε_HK_avg（扩展个体犹豫域）或降低network_clustering（打破信息茧房）。

**新增碰撞层派生 D-X20：顽固者临界比例函数**
p_critical = 1/(1 + (ε*/ε_HK_avg) × network_clustering)

## 六、个体犹豫域到群体犹豫域的跨层映射

来源：人际社会碰撞3，ε_group = f(ε_individuals, network_topology)

具体函数形式：

**ε_group = ε_HK_avg × (1 - p_stubborn) × (1 - network_clustering)**

三因子乘法：
- ε_HK_avg：个体犹豫域的平均值（输入）
- (1-p_stubborn)：顽固者稀释因子——顽固者比例越高，群体犹豫域越低
- (1-network_clustering)：网络开放因子——聚类越高（信息茧房），群体犹豫域越低

退化路径：
- 个体层面ε→0 → ε_HK_avg↓ → ε_group↓
- 顽固者增加 → p_stubborn↑ → ε_group↓
- 信息茧房 → network_clustering↑ → ε_group↓

三条路径独立，任一恶化即可压缩群体犹豫域。与乘法归零律一致。

与D-X20交叉验证：当p_stubborn达到p_critical时，ε_group→0，碎片化不可逆。

**新增碰撞层派生 D-X21：跨层犹豫域映射函数**
ε_group = ε_HK_avg × (1 - p_stubborn) × (1 - network_clustering)

## 七、内部抵抗函数化

来源：人际社会碰撞4，"外部高压产生反作用力，存在最优外部冲击强度"

勒温场动力学中P_internal = k×ε，当外部压力E_external过大时，P_internal可能变为负值——产生抵抗而非顺从。

内部抵抗的机制：外部压力压缩ε→0，个体从"可以商量"变为"绝不退让"。但压力本身也压缩ε，所以存在一个最优压力区间：

**R_resist(t) = σ(E_external - θ_resist) × ε(t)**

- E_external < θ_resist：无抵抗，外部压力不够触发反弹
- E_external > θ_resist且ε > 0：抵抗激活，但ε还有空间，抵抗是弹性的
- E_external > θ_resist且ε → 0：抵抗激活但无犹豫域，抵抗变为刚性壁（与顽固者同构）

最优外部冲击强度：使B_ignite = σ(k×ε×E - θ_ignite)最大化的E值。

dB/dE = 0 → E_optimal = θ_ignite/(k×ε) + 1/(2k×ε)（近似解）

当E > E_optimal时，E继续增大会压缩ε，反而降低B_ignite。这就是"外部高压适得其反"的数学本质。

**新增碰撞层派生 D-X22：内部抵抗函数**
R_resist(t) = σ(E_external - θ_resist) × ε(t)
E_optimal ≈ θ_ignite/(k×ε) + 1/(2k×ε)

## 八、指数衰减的结构化干预函数

来源：全学科同构碰撞4，"衰减率可干预"

exp(-n_lock×C̄/θ_C)的衰减率λ = n_lock×C̄/θ_C，每个分量都可干预：

**λ(t) = n_lock(t) × C̄_exit(t) / θ_C(t)**

干预杠杆：
1. 降低n_lock：解锁维度数。每解锁一个维度，n_lock减1，λ阶跃下降
2. 降低C̄_exit：降低平均退出成本。通过制度设计（如降低竞业协议范围）
3. 提高θ_C：提高认知阈值容忍度。通过叙事冲击（narrative_disruption）或信息透明化

干预函数：
**Δλ = -Δn_lock × C̄/θ_C - n_lock × ΔC̄/θ_C + n_lock × C̄ × Δθ_C/θ_C²**

三项分别对应三种干预路径。最有效的是降低n_lock——因为它是阶跃式的（解锁一个维度=1），而C̄和θ_C的变化是连续的。

**新增碰撞层派生 D-X23：衰减率干预函数**
λ(t) = n_lock(t) × C̄_exit(t) / θ_C(t)
Δλ = -Δn_lock × C̄/θ_C - n_lock × ΔC̄/θ_C + n_lock × C̄ × Δθ_C/θ_C²

## 九、乘法累积/乘法归零的对称变换规则

来源：全学科同构碰撞6

IEEE乘法老化模型：D(n+1) = D(n) × f_shock，f_shock > 1，损伤单调递增
点火框架乘法归零：P(n+1) = P(n) × f_factor，f_factor < 1，生存单调递减

对称变换规则：
- D ↔ 1-P（损伤度↔生存概率的补）
- f_shock ↔ 1/f_factor（损伤放大因子↔生存衰减因子的倒数）
- 累积方向反转：D↑ ↔ P↓

可借用的可靠性工程结论（经对称变换）：

1. **IEEE结论**：多组件系统损伤以乘法累积，系统寿命分布为Weibull
   → **点火推论**：多因子乘法归零下，组织生存时间服从逆Weibull分布，可估计组织剩余寿命

2. **IEEE结论**：组件间依赖通过共享冲击引入，依赖强度β影响系统失效率
   → **点火推论**：因子间依赖通过共享必要条件引入（如P_innovate²），依赖强度影响组织退化速率。低依赖时β因子模型是一阶近似，高依赖时需用精确乘法结构

3. **IEEE结论**：预防性维护可延长系统寿命，最优维护间隔取决于失效率形状参数
   → **点火推论**：预防性"认知维护"（定期引入外部视角、重置承诺、扩展ε）可延长组织健康期，最优维护间隔取决于退化速率的形状参数

4. **IEEE结论**：冗余设计可提高系统可靠度，但冗余组件间可能存在共因失效
   → **点火推论**：认知冗余（多轨道并行P_track）可提高认知可靠度，但多轨道可能共享同一认知框架导致共因失效（H_framework>0时所有轨道同时退化）

**新增碰撞层派生 D-X24：乘法对称变换函数**
D ↔ 1-P，f_shock ↔ 1/f_factor
四条可借用推论：逆Weibull寿命分布、依赖强度与退化速率、认知维护最优间隔、认知冗余的共因失效
