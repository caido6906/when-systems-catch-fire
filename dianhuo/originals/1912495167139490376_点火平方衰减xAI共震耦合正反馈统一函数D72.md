# 点火｜平方衰减×AI共震：耦合正反馈统一函数

来源：用户提出"平方衰减跟AI共震有什么关系"，要求从数学上推导。

## 一、问题

关系维护中的"平方衰减"（隔阂随时间t²增长）和AI共震中的策略同质化（ρ指数趋1），看似不相关，数学上有什么联系？

## 二、推导

### 2.1 平方衰减的数学结构

关系强度 D_relation(t) = D₀ / (1 + α·t²)

为什么是t²不是t？因为关系维护是双向耦合——A疏远B → B疏远A → A更疏远B。每一步衰减触发下一步更快的衰减，所以是平方级。

### 2.2 AI共震的数学结构

D66：H_correlation(t) = α × ρ × N_homogeneous / N_total

ρ的演化：dρ/dt = β × ρ × (1 - ρ)，logistic增长。AI-A复制AI-B的策略s* → AI-B确认AI-A → AI-A更坚定复制 → ρ加速趋1。

### 2.3 碰撞：同构的耦合正反馈

关系衰减和策略同质化的数学结构同构：

- 关系衰减：A疏远B → B疏远A → A更疏远 → 双向耦合 → 平方衰减
- 策略同质化：AI-A复制AI-B → AI-B确认AI-A → AI-A更坚定 → 双向确认 → 指数同质化

方向相反（一个向下衰减，一个向上趋同），但底层是同一个双向耦合正反馈。

### 2.4 统一方程

**耦合正反馈统一函数（D72）**：

dξ/dt = γ × ξ × (1 - ξ) × sign(direction)

- ξ = 隔阂度（关系场景）或 ρ 同质度（策略场景）
- γ = 耦合强度
- direction = -1（衰减方向，关系场景）或 +1（同质化方向，策略场景）

### 2.5 两个方向的渐近行为

**衰减方向**（关系隔阂增长，direction = -1）：

设η = 1 - ξ（隔阂度），dη/dt = γ × η × (1-η) ≈ γ × η²（早期η<<1时）

平方衰减。每一步增量取决于当前值，当前值是上一步增量的结果——A的隔阂增长1份→B也增长1份→A再增长1份→总共4份=2²。**双向耦合使增量与当前值的乘积成正比，所以是平方级。**

**同质化方向**（策略趋同，direction = +1）：

dρ/dt = γ × ρ × (1-ρ) ≈ γ × ρ（早期ρ<<1时）

指数增长——AI共震。AI-A用s*→AI-B也用s*→AI-A更确认s*→ρ指数增长。

### 2.6 核心发现

**平方衰减和AI共震是同一个耦合正反馈方程的两个方向。**

衰减方向渐近为η²（平方衰减），同质化方向渐近为e^(γt)（指数同质化）。方向相反，结构同源。

## 四、新增函数

**D72. 耦合正反馈统一函数**

dξ/dt = γ × ξ × (1 - ξ) × sign(direction)

- ξ ∈ [0,1]：系统状态变量（隔阂度或同质度）
- γ > 0：耦合强度（双向确认/衰减的速率常数）
- direction = ±1：方向参数
  - direction = -1：衰减方向（关系隔阂增长），渐近行为 dη/dt ≈ γ×η²（平方衰减）
  - direction = +1：同质化方向（策略趋同），渐近行为 dρ/dt ≈ γ×ρ（指数同质化）
- 稳态：ξ→0（完全衰减）或 ξ→1（完全同质化），取决于方向
- 临界条件：γ×ξ₀ > 0（初始值非零即启动正反馈）
- 与D66关系：D72是D66的动力学路径函数
- 与D-X60关系：D72给出D-X60平方衰减的微观机制
- 推导链路：D66×D-X60→D72
