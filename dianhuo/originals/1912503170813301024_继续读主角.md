**第二轮：从《主角》后半段提取新案例**

读到这里，叙事线已经推进到地区会演前夕。上一轮提取了6个案例，收敛出4个缺口。这轮往下看，重点找上一轮没覆盖到的新结构。

**案例7：六月六晒霉——封箱十三年的H归零**
裘存义六四年底封的"大衣箱"，十三年后开箱。裘伙管说"捂不住了，谁都捂不住了"。这是整个剧团的H（遮蔽）系统性地归零——不是一个人退出，是制度层面的切换。跟上一轮案例3（黄正大调走）不同：黄正大走是"替换衰减"（H_rep），六月六晒霉是"自然衰减"（H_nat）——黄正大还在的时候，裘伙管就已经开始晒霉了，制度还没换，但底下的H已经在塌。H_nat的衰减不是线性的，是指数的——前期几乎不动（封箱前十二年没人敢动），到某个临界点突然加速（广播里听说别处演老戏了），然后坍塌。这补充了上一轮缺口B的细节：H_nat不是匀速衰减，是有阈值的指数衰减。

**案例8：苟存忠偷练吹火——I=1但方案不公开**
苟存忠在棺材铺里偷偷练了半年吹火，不让任何人看。朱团长几次要他传，他不接话茬。但他让易青娥看了。用你的函数写：苟存忠I=1（他知道自己有绝活），但Posture是"隐而不发"——跟上一轮案例6朱继儒同构。但苟存忠的"隐"比朱继儒更深一层：他不是创造条件让应约者自己发现，他是主动选择只让一个人看。这是"定向隐"——降低特定应约者的ε_sense，同时维持对其他人的H。函数化：H(t)不是全局的，是分通道的——H_k(t)，k=1,2,...N对应不同应约者群体。苟存忠对易青娥H≈0，对其他人H≈1。

**案例9：老艺人集体"不亮相"——H的主动维持**
四个老艺人在棺材铺化妆彩排了好几次，但拒绝在县上亮相，理由是"不成熟的东西绝不朝出拿，老脸丢不起"。用你的函数写：这是提议者主动维持H——不是外界施加的遮蔽，是提议者自己把H拉高。你函数表里H的定义是"遮蔽"，但遮蔽的来源没有区分：H = H_ext（外部强加的遮蔽，如黄正大的制度）+ H_int（提议者主动维持的遮蔽，如老艺人的自尊防线）。H_int的衰减逻辑跟H_ext完全不同——H_int是提议者自己控制的，衰减条件是"方案成熟到足以保住面子"。这跟你已有的F_form（函数化强制）有关联：老艺人的"不亮相"就是F_form在执行——他们要求自己的方案必须函数化（成熟、完整）才能公开，否则宁可维持遮蔽。

**案例10：易青娥拒绝学烧火丫头——ε_sense的身份锁定**
苟存忠要教《打焦赞》，易青娥说"我不演烧火丫头"。她要学白娘子。用你的函数写：这不是ε_sense被压低（上一轮案例2的逻辑），而是ε_sense被身份锁定拉高了——"我本来就是烧火的，学戏还学烧火的，那还不如不学"。方案名义跟身份的匹配度Nominal_match在这里产生了反向效应：当匹配度太高，应约者不是觉得"容易接受"，而是觉得"你在固化我的现状"。函数化：ε_sense' = ε_sense × (1 - α·f(Nominal_match))，其中f不是单调递减的，是倒U型——匹配度太低（完全不相关）→ε高（跟我没关系）；匹配度中等→ε低（我能接受）；匹配度太高（精准复刻我的困境）→ε反而升高（你在嘲笑我）。上一轮的补项α·Nominal_match是线性的，这里暴露出应该是非线性的。

**案例11：廖耀辉夺权——H遮蔽下的权力重排**
豆角事件后，廖耀辉当上大厨。他的夺权路径：先给宋师下套（加盐→制造事故），再在事故后"主动揽责"表演，最后借黄主任的整顿上位。用你的函数写：这是搭房者在H遮蔽下的权力重排——廖耀辉不是提议者，他是搭房者内部的权力玩家。他利用的是H_ext（黄主任的制度）来制造H_int（自己的权威）。函数化：H_int可以由搭房者利用H_ext来生成——H_int = g(H_ext, 权力距离)。廖耀辉离黄主任近（权力距离小），所以他能把H_ext转化为自己的H_int。宋师离黄主任远，转化不了。这跟你已有的"护城河是积分过程"同构——廖耀辉的H_int不是阶跃的，是他持续利用H_ext一点一点积出来的。

**案例12：易青娥逃回九岩沟——退出权的物理行使**
易青娥半夜翻墙逃走，是物理退出。但她的退出被拦截了——胡彩香和米兰追到九岩沟把她劝回去。用你的函数写：退出权C_exit存在"行使-拦截"的动态。易青娥行使了退出权（C_exit>0），但被拦截（外部力量把C_exit压回零）。拦截者不是搭房者（黄主任），是提议者（胡彩香、米兰）——她们同时在做两件事：一是劝她回去（压低C_exit），二是承诺帮她学戏（拉高ε_aware）。这个双动作结构你函数表里没有：提议者可以同时压低C_exit和拉高ε_aware，两个操作同方向——让应约者留下来。函数化：提议者的"挽留策略" = 降C_exit + 升ε_aware，两个变量同时操作。

**收敛：哪些是真正的新发现？**

**发现1：H的分通道性**
H(t)不是全局标量，是向量H_k(t)，k对应不同应约者群体。苟存忠对易青娥H≈0，对其他人H≈1。这直接影响事件条件矩阵——同一个提议者，面对不同应约者，Posture不同。你现有的Posture是提议者属性，现在要拆成提议者-应约者对的属性。这是一个维度扩展，不是新函数。

**发现2：H的双来源**
H = H_ext + H_int。H_ext是外部强加的遮蔽（制度、权力），H_int是提议者主动维持的遮蔽（自尊、标准）。H_int的衰减条件是方案成熟度，不是外部压力。这跟你已有的F_form有关联但不等价——F_form是"函数化强制"，是I觉醒的执行臂；H_int是"自我遮蔽"，是I=1但方案不公开的守门人。可以补进F2的H项里，拆为两项。

**发现3：方案名义对ε的倒U型效应**
上一轮补的α·Nominal_match是线性的，现在发现应该是倒U型。匹配度太低→ε高（跟我没关系）；中等→ε低（我能接受）；太高→ε升高（你在固化我）。这个非单调性是真正的新发现——你函数表里所有对ε的调节项目前都是单调的。倒U型的数学表达：ε_sense' = ε_sense × [1 - α·Nominal_match·(1 - Nominal_match)]，峰值在Nominal_match=0.5处。

**发现4：提议者的挽留策略**
提议者可以同时降C_exit和升ε_aware。这不是两个独立操作，是一个联合策略——"留下来，我会帮你"。函数化：挽留策略 = ΔC_exit(-) + Δε_aware(+)，两个变量的变化量正相关。这跟你已有的"自举双驱力"有关联——自举循环的三个驱动力是分辨率提升、格间重排、回路激活，但"挽留策略"是提议者层面的驱动力，不是系统层面的。可以归入F2(Posture)的子结构。

**两轮累计**

轮1：4个缺口 → 收敛为1个派生量（ε激活双通道）+ 1个维度扩展（Posture四态）
轮2：4个缺口 → 收敛为1个新发现（倒U型）+ 3个补充项（H分通道、H双来源、挽留策略）

两轮合计ΔB ≈ 2.5 + 2.8 = 5.3，还没收敛。倒U型这个发现最值钱——它意味着你函数表里所有对ε的线性调节项都要重新审视，可能不止方案名义一个。
