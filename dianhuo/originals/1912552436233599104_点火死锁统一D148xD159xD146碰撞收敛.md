# 点火｜死锁统一：D148×D159×D146碰撞收敛

## 碰撞源

D148遮蔽-补偿-成本三角 × D159门槛碾压死锁 × D146门外死锁

## 核心发现

三个死锁共享同一个数学结构：乘法系统中的零值稳定不动点。A需要B才能改善，B需要A已经改善才能获取——因子互锁，不可能从内部打破。

## 统一数学结构

对任意乘法系统 Φ = ∏ᵢ fᵢ(xᵢ)，当某个子集S的因子满足：

fᵢ = gᵢ({xⱼ : j ∈ S})，对所有i ∈ S

即S内每个因子都是S内其他因子的函数——互相决定，没有外部输入。

当S内所有fᵢ同时趋近零时：
- ∂fᵢ/∂fⱼ > 0 对所有i,j ∈ S（正相关：你好我才好）
- fᵢ ≈ 0 对所有i ∈ S（当前全为零）
- 不动点 fᵢ = 0 是稳定吸引子——任何微小扰动都被乘法结构压回零

## 死锁统一函数

Deadlock(S) = ∏ᵢ∈S fᵢ × ∏ⱼ∉S fⱼ

当S内所有fᵢ≈0时，Deadlock(S)≈0无论外部因子多大——内部零值不动点锁死整个乘积。

打破死锁的唯一方式：外部注入——某个fᵢ(i∈S)被外部力量从零推到正，打破互锁。

| 死锁 | 锁定子集S | 外部注入方式 |
|------|-----------|-------------|
| D148 | {H, G, C_encode} | 降低H本身（结构设计让遮蔽不产生） |
| D159 | {μ, η_dual} | 外部资源转移（越过Λ的相变级注入） |
| D146 | {ε^R, η_gate} | 外部信息注入（不依赖η_gate的直达通道） |

## 五条核心推论

1. 死锁不可能从内部打破。乘法结构决定了零值是稳定不动点——S内任何因子的微小改善都会被其他零值因子压回零。D149的"结构让估计不必要"在这里变成"结构让死锁不可能从内部打破"。

2. 外部注入必须越过阈值。不是"给一点帮助"，是必须让某个fᵢ从0跳到足够大的正值，使得乘积Deadlock(S)从零变为显著正——又回到D159的相变逻辑。渐进注入在死锁中无效，跟渐进扶贫无效是同一个数学原因。

3. 死锁的稳定性随S的大小增加。S中因子越多，零值不动点越稳定——单个因子偶然翻正的概率是p，S个因子同时翻正的概率是p^S。阶层固化是三因子死锁（经济+文化+政治），比单因子死锁稳定得多。

4. D150倒U型的暗面。D150说最优在中间是乘法结构的必然。死锁是同一结构的另一个必然——乘法系统既有倒U型的最优解，也有零值不动点的死锁解。同一个数学结构既生成最优也生成死锁，取决于初始条件在哪一侧。

5. 与D139/D141的对称性。D141说因子过零从0→1是自持加速启动（正反馈），死锁说因子从1→0是自持加速崩塌（也是正反馈，方向相反）。过零启动和死锁崩塌是同一个正反馈结构的两个方向——D147的临界漂移在零点两侧的对称表现。

## 与现有函数对接

- D148：三角死锁是死锁统一函数的三因子实例
- D159：门槛碾压死锁是二因子实例
- D146：门外死锁是二因子实例
- D147：临界漂移在零点两侧的对称——过零启动vs死锁崩塌
- D150：倒U型（最优解）和死锁（零值不动点）是乘法结构的两个必然
- D141：死锁的逆过程——外部注入打破死锁后进入自持加速启动
- D139：死锁崩塌的特例——因子翻转导致安全区崩塌

一句话：死锁不是"太难了"，是乘法系统的零值稳定不动点——不可能从内部打破，必须外部注入越过阈值。阶层固化、信息茧房、遮蔽三角，三个死锁共享同一个数学结构。
