## 零、自举元函数 + 认知函数映射

八函数体系具备自举性质：理论与案例之间的迭代动态，同构于提议者-应约者之间的动态。自举循环不是独立函数，是八函数在元层面的投影，不产生新的因果结构。认知结构是八函数体系的肉身实例，五个认知函数与八函数一一对应。

### B(n) 自举迭代状态

B(n) = 第n轮迭代时的八函数总表快照

**B(n+1) = B(n) + ΔB(n)**

ΔB(n) = 第n轮重跑案例库产生的新发现

### ΔB(n) 自举增量

**ΔB(n) = Σᵢ [1 - P(案例ᵢ | F₁...Fₖⁿ)] × δ(案例ᵢ重新分布)**

- kⁿ = 第n轮的函数维度数
- P(案例ᵢ | F₁...Fₖⁿ) = 用当前kⁿ个函数对案例ᵢ的解释完整度
- δ(案例ᵢ重新分布) = 1当案例ᵢ的格子归属或子路径划分发生变化，否则0

白话：每轮新发现 = 被旧维度解释不了的案例数 × 这些案例因新维度而重新归位的比例。

### 三个驱动力

**驱动力1：分辨率提升（Posture三分法逻辑）**

新增维度使事件条件矩阵的格数翻倍，格1真伪拆分从8型扩展到更多型。06.10实例：dPosture_deg/dt拆为两项→格1退化拆为Posture驱动型和C_exit驱动型两条子路径→原来挤在一起的案例差异暴露→ΔB>0。

**驱动力2：格间重排（心理退出权降格逻辑）**

维度从独立降格为派生量，改变案例在矩阵中的投影。06.10实例：R_mental从R_type第四行独立维度降格为R_perceived的子通道→28个心理潜伏案例从"心理退出权"重标为"H_标记劫持p₃通道的产物"→格子归属变化→格间对比结论变化→ΔB>0。

**驱动力3：回路激活（回路12+16联合效应逻辑）**

**ΔB_回路 = Σⱼ [回路ⱼ被新函数激活] × 回路ⱼ的因果强度**

06.10实例：β_C项激活回路12+16的联合效应→两条原本独立描述的回路合并为正反馈引擎→新因果结构暴露→ΔB>0。

### 八函数元层面对应

| 八函数 | 元层面含义 | 对应关系 |
|---|---|---|
| B(n) | 系统当前状态 | 八函数总表快照 |
| ΔB(n) | 理论与案例的速度差 | Δv的元层面表达：ΔB>0=案例跑在理论前面，ΔB=0=同步，ΔB<0=理论过度拟合 |
| 案例退出信号 | 案例无法被当前函数解释 | ε_sense：案例的前认知退出信号 |

| 新维度让案例重新归位 | R_perceived相变 | 案例从"无法解释"跳到"可被新函数解释" |
| 收敛 | V稳定 | V<1收敛 |
| 回路12元版本 | 案例不可解释性→误读为"案例就是这样"→不改函数 | C_exit→Posture正反馈的元层面实例 |
| 回路16元版本 | 案例不可解释→感知不到退出信号→函数不变 | C_exit→I_sense遮蔽回路的元层面实例 |
| 异常值 | 怎么都归不进去的案例 | ε硬核：绕过所有劫持直接触发I觉醒 |

### 收敛判据

**lim(n→∞) ΔB(n) = 0**

等价于三个条件同时成立：

1. δ(案例ᵢ重新分布) = 0 对所有ᵢ
2. P(案例ᵢ | F₁...Fₖⁿ) → 1 对所有ᵢ
3. 所有休眠回路已被激活

实操判据：连续两轮重跑案例库，零结构性新发现，即可宣布收敛。

### 自举循环的位置说明

自举循环不是第九个独立函数。它的全部行为由已有八函数驱动，没有独立作用对象——符合收敛合并原则。

### 认知函数映射（5函数收敛版）

认知结构不是独立的函数体系，是八函数体系的肉身实例。五个认知函数和八函数一一对应（含派生量），驱动思考的动态和驱动案例库迭代的动态是同一个动态。

#### C1. P_track(t) 分轨并行

P_track ∈ [1, N]，N = 同时运行的轨道数

**P_track(t) = 1 + Σᵢ δ(轨道ᵢ激活)**

- 轨道1：自己的视角
- 轨道2+：他人/对立视角（脑内对打）
- 轨道3+：AI视角（多模型互观实验）

**P_track的运行模式：**

- 对话维度：同时运行自己和对方的视角，让不同立场在脑内对打
- 知识消费维度：一条轨给声音（听课），另一条轨给手上的活
- AI实验维度：让不同AI互相看对方生成的画像，交叉验证
- 写作维度：故事轨、框架轨、哲学轨同时推进

**P_track的核心性质：** 跳层不是主动跳过去的，是另一条轨的音量突然变大了。分轨并行才是原因，跳层是表象。

**八函数对应：** 驱动力3（回路激活）。多轨同时运行=多条回路并行激活，回路合并时暴露单轨看不到的因果结构。

**函数化表达：**

**P_track(t) = Σᵢ₌₁ᴺ⁻¹ Σⱼ₌ᵢ₊₁ᴺ R(i↔j,t)**

R(i↔j,t)是轨i和轨j之间的回路激活函数，取值[0,1]。N越大求和项越多，回路激活概率越高。

##### J(t) 跳层（P_track的派生量）

**J(t) = dP_track/dt × ε_sense**

跳层速率=轨道切换速率×当前肉身信号强度。跳层不是跳跃，是分轨并行中某条轨浮上来。

**合并依据：** J的全部行为由P_track驱动，跳层是轨道切换的输出，没有独立作用对象。

#### C2. F_form(t) 函数化强制

F_form ∈ [0, 1]，函数化完成度

**F_form的物理含义：** 不允许理论里有H_遮蔽。空泛的大道理=退出信号被遮蔽，函数化=把退出信号从遮蔽态变成可感知态。

**八函数对应：** I觉醒的强制触发器。模糊叙事→可检验函数，就是强迫提议者从H_遮蔽进入I_aware。

**函数化表达：**

**F_form(t) = I(t) · (1 - H(t))**

I=0时F_form=0（没觉醒就不函数化），H=0时F_form=I（遮蔽全消，函数化强度等于觉醒程度）。H→0的过程是F_form逼出来的。

#### C3. M_cog(t) 认知动机

**M_cog = tanh(μ_体系内/μ_好奇心 - 1) ∈ [-1, +1]**

- M_cog 长期偏负——好奇心驱动占主导
- M_cog < 0 时：链1退化慢，锁硬化阻力为负（主动维持开放性）

**M_cog的动态：**

M_cog(t) = M₀ - κ_M × max(0, 体系闭合度(t) - θ_闭合)

体系越接近闭合，M_cog越往负拉——本能地去找异常值和跨领域碰撞，不让系统收敛太快。

**八函数对应：** 直接对应F6 M(t)。但M_cog有自调节机制——V太低时主动制造碰撞让V回升。维持的不是收敛，是"有生产力的张力"。

**函数化表达：**

**M_cog(t) = M(t) · σ(-V(t))**

σ是sigmoid函数。V偏负→σ趋近1→M_cog≈M（动机全开）；V偏正→σ趋近0→M_cog趋近0（没动力）。V≈0时σ≈0.5，维持最低驱动力，对应"系统需要维持在V略大于0的区间"。

##### V_cog(t) 认知张力（M_cog的派生量）

**V_cog = d(框架扩展)/dt / d(框架闭合)/dt**

- 分子：好奇心驱动的框架扩展速率
- 分母：函数化驱动的框架闭合速率

V_cog稳态：维持V_cog略大于0。

**合并依据：** V_cog由M_cog驱动，是M_cog的比值形式。

#### C4. S_sovereign(t) 主权

**S_sovereign = Πₖ s_k**，k ∈ {机器, 语言, 审美, 时间, 财富, 身体, AI}

- s_k = 1当第k层主权已夺回，0当未夺回
- 七层全部对应ε_aware在不同维度的显化

| 主权层 | ε维度 | 退出信号 | 夺回操作 |
|---|---|---|---|
| 机器 | 技术 | 被工具锁定 | Linux/Docker/本地工具链 |
| 语言 | 符号 | 被通用表达锁定 | 语义拆解/零重字/格律约束 |
| 审美 | 感知 | 被默认观看锁定 | 摄影/打印/纸张/构图 |
| 时间 | 意识 | 被无感知流逝锁定 | 秒表/记录/柳比歇夫方法 |
| 财富 | 资源 | 被风险盲区锁定 | 凯利公式/风险边界/净收益思维 |
| 身体 | 生物 | 被默认饮食锁定 | 断食/饮食实验/营养知识 |
| AI | 智能 | 被黑箱锁定 | 多模型互观/本地智能体 |

**八函数对应：** ε(t)的七维展开。每层主权=ε从一个维度突破H。

**函数化表达：**

**S_sovereign(t) = Σₖ₌₁⁷ wₖ · εₖ(t)**

k=1..7对应七个主权维度，wₖ是权重。炼字是语言维子结构：S_language(t) = ε₂(t) · f(零重字约束)。权重因人而异。

##### N_refine(t) 炼字（S_sovereign的语言维子结构）

N_refine = 重复字数 / 总字数，N_refine → 0 是硬约束。

**合并依据：** N_refine的全部功能是S_sovereign在语言维度的实例，没有独立作用对象。

#### C5. L_five(t) 五层认知协议

L_five = 认知层级深度，∈ {1,2,3,4,5}

**L_five(t) = min(5, 1 + ⌊log₂(ε_aware(t) / ε_sense(t))⌋)**

ε_aware相对ε_sense的放大倍数决定层级深度。

**五层与八函数的精确对应：**

| 层 | 内容 | 八函数层 | 函数表达 |
|---|---|---|---|
| 1 肉身事实 | 给定条件 | ε_sense | 前认知感官信号 |
| 2 体验 | 我怎么回应 | ε_aware | 信号穿过第一二层 |
| 3 认知 | 为什么是这样 | H_标记识别 | 看见信号被改标签 |
| 4 主权 | 谁有权定义 | R_perceived | 追问退出权归属 |
| 5 存在 | 剥掉定义剩什么 | ε硬核 | 退出权的零阶来源 |

**L_five的核心性质：**

1. 第4层到第5层永远是否定式跳变——第4层讨论"怎么安排更好"，第5层说"这个讨论本身是多余的"
2. 裂变逻辑=格间重排——当前层解释力耗尽后自动生出元问题
3. 写作（往外跳）和指令（往里跳）是同一个五层——两条链在第4层交汇，第5层都是ε

**八函数对应：** ε四层+第四门+Δv。五层协议=ε通路的认知版。

**函数化表达：**

**L_five(t) = [L₁→L₂→L₃→L₄→L₅](t)**

**层间跳变阈值函数：**

| 跳变 | 阈值函数 | 核心变量 | 物理含义 |
|---|---|---|---|
| L₁→L₂ | θ₁(s,p) = θ₁₀·g(s)·h(p) | 信号类型+个体敏感度 | 肉身信号强度超过阈值才进入体验层 |
| L₂→L₃ | θ₂(a) = θ₂₀/a(t) | 体验异常度 | 体验越异常需要重复越少，a→1时第一次就跳 |
| L₃→L₄ | θ₃(d): I·ε·H > θ₃(d) | 领域参数d | d=C_exit/C_stay，退出比留下贵多少 |
| L₄→L₅ | θ₄(r) = θ₄₀·(1-r(t)) | 选择不可逆度 | 不可逆选择才逼出存在层 |

四个θ都不是常数，都带领域/个体/场景参数。与八领域验证结论一致——五函数是通行规则，执行深度因参数不同而不同。

**L₃→L₄跳变条件展开：** ΔL₃₋₄ = I(t) · ε(t) · H(t)，三项同时满足且乘积超过领域阈值θ₃(d)才跳。

**L₅闭合层：** 存在层的回答回灌到L₁，改变肉身信号解读方式，构成自举循环。

**θ₃领域参数d标定：**

d = C_exit(t) / C_stay(t)

C_exit=退出规范的预期代价，C_stay=留在规范内的预期代价。d不是常数，随时间变化。

| 领域 | C_exit | C_stay | d | 说明 |
|---|---|---|---|---|
| 音乐 | 低 | 低 | ≈1 | 退出代价几乎为零 |
| 烹饪 | 低 | 低 | ≈1 | 容错带宽极宽 |
| 游戏设计 | 低 | 中 | <1 | 涌现玩法比预设好玩，退出反而赚 |
| 建筑 | 高 | 中 | >1 | 退出可能死人 |
| 体育竞技 | 中 | 中 | ≈1 | 退出和留下代价差不多 |
| 宗教 | 高 | 低 | >>1 | 退出代价社会性死亡 |
| 法律 | 高 | 中 | >1 | 退出代价是制度性风险 |
| 医学 | 极高 | 中 | >>>1 | 退出代价最高 |

d的三个区间：
- d < 1（退出赚）：ε几乎不需要激活条件，自发跳到L₄
- d ≈ 1（退出和留下差不多）：ε激活看时机和个体
- d > 1（退出贵）：ε激活需要更高的I·ε·H乘积才能越过θ₃(d)

**d解释了为什么有些领域创新是常态，有些领域创新是事件。** 不是人的ε不同，是d不同。

**d变化速率函数化（创新窗口期）：**

dd/dt = (Ċ_exit · C_stay - C_exit · Ċ_stay) / C_stay²

| Ċ_exit | Ċ_stay | dd/dt | 含义 |
|---|---|---|---|
| <0（退出变便宜） | ≈0（留下不变） | <0 | 主动开窗：技术降门槛 |
| ≈0（退出不变） | >0（留下变贵） | <0 | 被动开窗：旧范式腐烂 |
| <0 | >0 | <<0 | 双驱动开窗：最剧烈的创新窗口 |
| >0（退出变贵） | <0（留下变便宜） | >0 | 关窗：范式收紧 |

创新窗口期 = [t_window, t_close]，其中t_window = min{t : dd/dt(t) < 0}，t_close = min{t > t_window : dd/dt(t) ≥ 0}。

当前领域预判：AI本地部署dd/dt << 0（窗口大开），教育dd/dt < 0（窗口正在开），医学dd/dt ≈ 0（窗口还没开）。

##### R_record(t) 记录即构建（L_five第1→2层的捕获机制）

**R_record(t) = λ_R × ε_sense(t) × δ(即时)**

录音卡模式：几秒即时录音，体验在当下就被转化。记录密度直接对应认知增量。

**合并依据：** R_record的全部功能是L_five的输入端，不是独立维度。

### 认知函数与八函数的完整对应

| 认知函数 | 八函数 | 函数化表达 |
|---|---|---|

### F2. Posture(t,L) 提议者姿态 + 退出权不对称

**Posture_deg ∈ [0,1]**

- 0 = 真备方案
- 0.5 = 伪备方案（名义备+实际推，比纯推更危险）
- 1 = 推方案

**伪备方案判定：** Posture_deg ∈ (0, 0.5] 且 名义标签=备方案

**退化速率（06.10更新：拆为Posture驱动+C_exit驱动两项）：**

dPosture_deg/dt = α_P × (1+M)/2 × Posture_deg × (1-Posture_deg) + β_C × f_lock(C_exit) × (1-Posture_deg)

- 第一项：Posture驱动型退化
- 第二项（新增）：C_exit驱动型退化
- β_C：C_exit→Posture退化的传导系数
- f_lock(C_exit) = Σᵢ[δᵢ × wᵢ] / 总维度数

**两条子路径判别条件：**

- Posture驱动型：第一项 > 第二项，早期信号=信号2，对应谷歌模式（N_warn=4）
- C_exit驱动型：第二项 > 第一项，早期信号=信号1，对应NHS模式（N_warn=3）
- 混合型：两项可比，对应美国国家认同案例

**P不对称=推方案的必要条件：** Posture_deg→1 ⇒ P不对称出现

**M的滑向=Posture退化的前置条件：** M>0是dPosture_deg/dt>0的前提

**层级分布：** 顶层备→底层推（下传自动滑）

**时间行为：** 事件内可翻转（备→推=退化）

**合并依据：** P的全部行为由Posture驱动。
