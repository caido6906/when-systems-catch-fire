# 点火｜P×Q²共源平方加速函数（D73）

## 一、P×Q²的结构

P×Q²描述：当两个因子共享同一个源，联合效应不是P×Q（线性），而是P×Q²（平方级）。

电子工程原始场景：级联放大器共享电源V，第一级增益G₁∝V，第二级增益G₂∝V，总增益G_total∝V²。Q出现两次因为共享源——不是1+1=2，是1×1=1²。

同构结构：

| 领域 | 同构结构 |
|------|---------|
| 电子工程 | 共享电源的级联放大器，G∝V² |
| 贝叶斯网络 | 共同父节点Z的条件概率展开 |
| 可靠性工程 | 共因失效β因子 |
| 点火框架 | 共源平方加速：两个退化因子共享源→P_eff∝(1-Q)² |

## 二、P×Q²在AI共震中的精确对应

AI共震中，两个退化因子共享源ρ（策略相关度）：

- Q₁ = H_correlation = α×ρ×N_h/N_t（同质性遮蔽，D66）
- Q₂ = P(biased) = σ(k×Q_unresolved×H_correlation - ε_aware×R)（解读偏置，D53）

H_correlation→1直接拉高P(biased)——两者共享源ρ。

联合效应：

P_sustain_actual = P_sustain × (1 - H_correlation) × (1 - P(biased))
                ≈ P_sustain × (1 - ρ)²

**P_sustain_actual ≈ P × (1-Q)²**

共享源ρ同时驱动两个退化因子，联合效应是平方级——创新被阻断不是线性地杀死生存概率，是平方级地杀死。

## 四、P×Q²是乘法归零律的平方加速版

乘法归零律：P = Πfᵢ，任一fᵢ→0则P→0。

P×Q²：当两个fᵢ共享同一个源Q时，P的衰减不是∝Q（线性），而是∝Q²（平方级）。共享源让乘法归零加速——不是1×Q=Q，是1×Q²=Q²。

**在所有"两个退化因子共享同一个源"的场景中，乘法归零的杀伤力是平方级的。不是"两个独立问题叠加"，是"一个问题通过两个通道同时杀死你"。**

## 五、新增函数

**D73. 共源平方加速函数**

- P₀：基础生存概率

- 当两个退化因子共享同一个源时，联合效应是(1-Q)²而非(1-Q)——平方级加速
- 电子工程实例：级联放大器共享电源V，G∝V²
