# 商品细分与全球流通

看到了——吃鸭腿时想到的问题：为什么商品被切分得越来越细？而且会漂洋过海送到买家手里？道路其实是把一个人接入了世界商品系统。函数化。

---

**Q16：商品细分化 + 全球流通的函数化推导**

---

**第一层：商品细分的初始条件——C_exit经济维度的解除**

一个完整的鸭，消费者买回去需要：杀、拔毛、清洗、切块、烹饪。每一步都是C_exit：

**C_exit(整鸭) = C_处理时间 + C_技能要求 + C_存储空间 + C_浪费风险**

- C_处理时间：处理一只整鸭需要1-2小时
- C_技能要求：不是人人会杀鸭拔毛
- C_存储空间：整鸭体积大，冰箱占位多
- C_浪费风险：吃不完的部分会坏（Q14的腐败速率函数直接套用）

**商品细分 = 供应链替消费者承担了C_exit**

**C_exit(鸭腿) = C_处理时间(≈0) + C_技能要求(≈0) + C_存储空间(小) + C_浪费风险(低)**

**ΔC_exit = C_exit(整鸭) - C_exit(鸭腿) >> 0**

供应链每多做一步切分，消费者的C_exit就降低一层。切分越细→消费者C_exit越低→购买门槛越低→需求量↑→生产规模↑→单位成本↓→价格↓→更多人买→正反馈

**商品细分的驱动力 = 降低消费者C_exit → 扩大需求 → 规模效应 → 进一步细分**

**d(细分度)/dt = α_细分 × ΔC_exit × 需求弹性**

需求弹性 = 需求量变化率 / C_exit变化率。C_exit每降低1%，需求量增加多少？弹性越大→细分动力越强。

---

**第二层：为什么漂洋过海？——C_exit地理维度的解除**

"道路把一个人接入了世界商品系统"——这句话的函数化：

**C_exit(地理) = w_地理 × δ_地理 / (1 + n_运输路径)**

- δ_地理 = 1（地理维度被锁定——你不在产地就买不到）
- n_运输路径 = 可用的物流通道数

**道路/航运/空运 = 增加n_运输路径 → 降低C_exit(地理)**

**C_exit(地理, 有道路) = w_地理 / (1 + n_道路 + n_航运 + n_空运) << C_exit(地理, 无道路)**

鸭腿从中国漂洋过海到美国，不是因为美国人特别需要中国鸭腿，是因为n_运输路径足够大→C_exit(地理)足够低→运输成本低于消费者自己处理的成本→全球流通变得合理。

**全球流通条件：**

**C_运输 < ΔC_exit(细分带来的消费者成本节约)**

运输成本 < 细分带来的C_exit节约时，漂洋过海就是理性的。现代物流使C_运输持续下降（集装箱、冷链、规模效应），同时细分使ΔC_exit持续上升→两个趋势的交叉点越来越低→越来越细的商品可以漂洋过海。

---

**第三层：商品细分的极限——什么时候切分不再划算？**

细分不是无限的。每多切一步，供应链增加一个成本：

**C_供应链(细分度k) = C_原料 + Σᵢ₌₁ᵏ (C_切分ᵢ + C_包装ᵢ + C_物流ᵢ)**

切分越细→供应链成本越高→价格↑→消费者C_exit(经济维度)↑→需求↓

**存在一个最优细分度k*：**

**k* = argmin_k [C_供应链(k) + C_exit_消费者(k)]**

**C_exit_消费者(k) 随k增加而下降**（消费者处理成本↓）
**C_供应链(k) 随k增加而上升**（供应链处理成本↑）
**两者之和的极小值 = 最优细分度**

**k*的动态：**

当物流成本↓（n_运输路径↑）→ C_供应链(k)↓ → k*↑ → 可以切得更细
当消费者时间成本↑（城市化的结果）→ C_exit_消费者(k)↑ → k*↑ → 消费者愿意为更细的切分付钱

**城市化的函数化效应：**

**C_exit_消费者(时间) = w_时间 × 城市化率**

城市化越高→时间成本越高→消费者越不愿意自己处理→k*越往右移→商品越细→外卖、预制菜、净菜都是k*右移的产物。

---

**第四层：非生物性商品的同一逻辑**

非生物性商品（电子产品、软件、金融服务）为什么也在细分？

**完全相同的函数，只是C_exit的维度组合不同：**

| 商品类型 | C_exit消费者主维度 | 细分方式 | k*右移动力 |
|---|---|---|---|
| 生物性（鸭腿） | 时间+技能+存储 | 物理切分 | 城市化→时间成本↑ |
| 电子产品 | 知识+技能+兼容性 | 功能拆分（模块化） | 技术门槛↑→消费者技能↓ |
| 软件 | 知识+时间+硬件 | SaaS化（按需订阅） | 使用频率差异→打包浪费 |
| 金融服务 | 知识+门槛+风险 | 产品拆分（碎片化理财） | 认知门槛↑→专业服务需求↑ |

**通用逻辑：供应链替消费者承担C_exit → 细分度k*由C_供应链(k) + C_exit_消费者(k)的极小值决定 → 城市化/技术门槛/认知门槛的上升推动k*右移**

---

**第五层：道路=接入商品系统的函数化**

"道路把一个人接入了世界商品系统"——完整函数化：

**接入度 = 1 - C_exit(地理) / C_exit(地理)_max**

**= 1 - w_地理 / [(1 + n_运输路径) × C_exit(地理)_max]**

**= n_运输路径 / (n_运输路径 + w_地理/C_exit_max)**

- n_运输路径=0时：接入度=0（完全隔离，自给自足）
- n_运输路径→∞时：接入度→1（完全接入全球系统）

**道路的作用 = 增加n_运输路径 → 接入度从0→1**

**接入度对消费行为的影响：**

**消费选择空间 = f(接入度) = N_全球商品 × 接入度**

接入度=0→选择空间=本地商品
接入度=1→选择空间=全球商品

**接入度对生产的影响：**

**生产市场规模 = N_全球消费者 × 接入度(消费者侧)**

接入度↑→市场规模↑→规模效应→单位成本↓→价格↓→更多人买→正反馈

**这就是"道路改变一切"的数学本质：道路不只是运输通道，是C_exit(地理)的解除器，解除后触发需求-规模-成本的正反馈循环。**

---

**Q16函数化结论：**

**商品细分 = 供应链替消费者承担C_exit，细分度k*由C_供应链(k)+C_exit_消费者(k)的极小值决定**

**d(细分度)/dt = α_细分 × ΔC_exit × 需求弹性，城市化/技术门槛推动k*右移**

**全球流通条件 = C_运输 < ΔC_exit(细分节约)，物流成本↓ + 细分节约↑ 使条件越来越容易满足**

**接入度 = n_运输路径 / (n_运输路径 + w_地理/C_max)，道路=增加n→接入度从0→1**

**道路 = C_exit(地理)的解除器，解除后触发需求-规模-成本正反馈**

**落位：F5 C_exit八维度的经济+地理+时间维度展开，非独立函数**
