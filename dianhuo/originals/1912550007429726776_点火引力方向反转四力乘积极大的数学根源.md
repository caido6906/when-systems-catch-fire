# 点火｜引力方向反转——四力乘积极大的数学根源

D154推导过程中的关键发现：

引力ε的映射方向跟其他三种力相反。

三种力（强/弱/电磁）：
- 高能时α→0，微扰有效，ε→1（存活）
- 低能时α→∞或破缺，ε→0（锁死）
- ε_i(μ) = exp(-1/ln(μ/Λ_i))，Λ_i是破缺能标，μ>Λ_i时ε→1

引力：
- 高能时时空失效，ε→0（锁死）
- 低能时时空经典连续，ε→1（存活）
- ε_grav(μ) = exp(-1/ln(M_Planck/μ))，μ<M_Planck时ε→1

方向相反的数学后果：

B_4force(μ) = exp(-[1/ln(μ/Λ_QCD) + 1/ln(μ/M_W) + 1/ln(M_Planck/μ)])

三项中前两项随μ增大而减小（ε增大），第三项随μ增大而增大（ε减小）。正是这个方向相反产生了内部极大值——三种力把B往上推，引力把B往下拉，平衡点就是μ*。

如果四种力方向相同（全部渐近自由或全部红外自由），B_4force单调，没有内部极大。

物理含义：引力的"高能锁死"是四力统一的必要条件。没有引力的反向作用，其他三种力的ε在高能都趋近1，B_4force→1但没有极值结构。引力引入了"高能天花板"——能标不能无限升高，否则时空本身崩溃。这个天花板跟三种力的"高能自由"对抗，产生了最优平衡点。

这跟D112防守-进攻相变完全同构：三种力是"进攻"（高能时ε增大），引力是"防守"（高能时ε减小），μ*是攻防平衡点。
