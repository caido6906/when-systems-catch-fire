# 点火｜自举元函数：正反双通道完整数学定义

作者：之元

## 定义

完整自举元函数不仅要论证命题成立，还要论证命题不成立。
正向通道论证 x 成立，反向通道论证 x 不成立；只有正向通过且反向不通过，命题才成立。

## 判定

- J⁺(x)=1, J⁻(x)=0 → true
- J⁺(x)=0, J⁻(x)=1 → false
- J⁺(x)=1, J⁻(x)=1 → contradiction / bootstrap_failed
- J⁺(x)=0, J⁻(x)=0 → underdetermined
- pending → pending

## 数学表达

M_boot⁺(x)=M_boot(x⁺)

M_boot⁻(x)=M_boot(x⁻)

J⁺(x), J⁻(x) ∈ {0,1,⊥}

R(x)=Φ(J⁺(x),J⁻(x))

## 嵌套

自举元函数可以嵌套地检查自举元函数本身：
- 正向通道是否可靠
- 反向通道是否可靠
- 正反互斥判定器是否可靠

## 备注

如果正向和反向都通过，说明对象定义、变量映射或判定器有问题，整个自举循环不通过。
如果正向和反向都不通过，说明证据或判定器不足，状态应为 pending / inconclusive。
