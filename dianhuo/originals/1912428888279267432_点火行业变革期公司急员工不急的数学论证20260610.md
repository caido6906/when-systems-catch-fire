# 点火｜行业变革期公司急员工不急的数学论证（2026.06.10）

## 问题
行业变革时期，公司着急转型，员工却不着急。为什么？

## 推导

### 第一步：定义个体与组织的P_survival映射

组织P_survival（已推导）：
P_survival(ORG, t) = σ(ε×A×D×|M_cog|) × exp(-n_lock×C̄/θ_C) × f(P_infra) × σ(θ_d - d_relative) × S(t)

个体P_survival需要对五因子重新映射——个体不是组织，五因子对个体的含义完全不同。

### 第二步：逐因子映射

**因子1：σ(ε×A×D×|M_cog|) — 认知跳变**

组织：行业变革=旧认知范式失效→σ(ORG)快速下降。不转型=认知跳变能力归零。
个体：行业变革=新技能需求出现→σ(个人)取决于个人学习速度。但个体可以学新技能跳到新范式，不需要等组织转型。

关键差异：组织的认知跳变需要集体协调（决策链+执行链+文化惯性），个体的认知跳变只需要自己动。σ(个人)的调整速度 >> σ(ORG)的调整速度。

**因子2：exp(-n_lock×C̄/θ_C) — 锁定维度**

组织：
- n_lock(ORG)高：资产专用性、沉没成本、组织架构刚性、品牌定位、客户合同、监管合规
- C̄_exit(ORG)极高：转型=部分资产清零+重组成本+客户流失风险
- exp项趋近0：组织被深度锁定，转型退出成本极高

个体：
- n_lock(个人)低-中：房贷、家庭、社交圈是锁定，但技能可迁移
- C̄_exit(个人)中：跳槽成本=求职周期+薪资波动，不是不可承受
- exp项不趋0：个体退出成本远低于组织

关键差异：n_lock(ORG) >> n_lock(个人)。组织被资产和架构锁死，个人被技能和房贷轻度锁定。组织的锁定维度是个体的超集。

**因子3：f(P_infra) — 认知基础设施**

组织：行业变革=旧P_infra失效。工厂的产线、软件的架构、销售的网络，全要重建。f(P_infra, ORG)从1→低值，重建周期以年计。
个体：P_infra(个人)≈个人技能+工具链。技能可迁移，工具链可重建。f(P_infra, 个人)下降幅度远小于组织。

关键差异：组织的P_infra是集体资产，重建需要协调成本；个人的P_infra是私有资产，重建只需自己动。

**因子4：σ(θ_d - d_relative) — 竞争优势衰减**

组织：行业变革=旧竞争范式失效。d_relative(ORG)快速上升（竞争对手在转型，自己没动）→σ→0。
个体：d_relative(个人)取决于个人在劳动力市场的竞争力。行业变革时：
- 旧技能贬值→d_relative(旧技能)上升→旧岗位的σ→0
- 但个体可以学新技能→d_relative(新技能)可能更低→新岗位的σ更高

关键差异：组织只有一个d_relative（组织整体），个体有多个d_relative（不同技能/岗位）。个体可以切换d_relative，组织不能。

**因子5：S(t) — 主体性**

组织：行业变革中组织主体性被侵蚀（被新技术范式降维），S(ORG)下降。
个体：个体的主体性不由组织决定。跳槽=切换组织，S(个人)可以保持。

关键差异：S(ORG)下降时，S(个人)可以不变——换个组织就行。

### 第三步：紧迫感函数

定义紧迫感G(t)：
G(t) = -dP_survival/dt / P_survival(t)

即P_survival的相对衰减速率。G越大，越着急。

对组织：
G(ORG, t) = -dP_survival(ORG)/dt / P_survival(ORG, t)

行业变革时P_survival(ORG)快速下降（五因子同时受冲击），且n_lock(ORG)高→转型慢→dP_survival/dt大→G(ORG)高。

对个体：
G(个人, t) = -dP_survival(个人)/dt / P_survival(个人, t)

行业变革时P_survival(个人)下降速度远慢于组织——因为个体可以学新技能、跳槽、切换d_relative。dP_survival(个人)/dt小→G(个人)低。

### 第四步：紧迫感差异的数学本质

G(ORG) / G(个人) = [dP_survival(ORG)/dt / P_survival(ORG)] / [dP_survival(个人)/dt / P_survival(个人)]

展开为五因子乘法：

G(ORG)/G(个人) = Πᵢ [Gᵢ(ORG)/Gᵢ(个人)]

其中Gᵢ是第i个因子的相对衰减率。

| 因子 | Gᵢ(ORG)/Gᵢ(个人) | 原因 |
|------|-------------------|------|
| σ(认知跳变) | >>1 | 组织转型需集体协调，个人只需自己学 |
| exp(锁定) | >>1 | n_lock(ORG)>>n_lock(个人)，组织被资产架构锁死 |
| f(P_infra) | >>1 | 组织基础设施重建需协调，个人工具链可快速重建 |
| σ(竞争优势) | >>1 | 组织只有一条赛道，个人可切换技能/赛道 |
| S(主体性) | >>1 | 组织主体性被侵蚀不可逃，个人可跳槽保主体性 |

五项全部>>1→G(ORG)/G(个人) >> 1

公司比员工着急的数学本质：五因子的相对衰减率全部对组织更不利。

### 第五步：深层机制——信息不对称与风险错配

**信息不对称**：
组织决策层能看见P_survival(ORG)的全貌（市场数据、竞品动态、财务指标），G(ORG)对决策层是可见的。
基层员工看不见P_survival(ORG)，只能感知自己的P_survival(个人)——而P_survival(个人)下降慢→感知不到紧迫。

G_visible(决策层) ≈ G(ORG) >> G(个人) ≈ G_visible(基层)

**风险错配**：
组织转型失败的代价由组织承担（破产清算），员工只承担求职成本。
组织转型成功的收益也主要由组织获取（市值增长），员工只获取薪资增长。

R_org(转型成功) >> R_个人(转型成功)
C_org(转型失败) >> C_个人(转型失败)

但C_个人(转型失败)/R_个人(转型成功)的比值可能更高——员工承担转型中的加班、技能过时风险，但收益有限。所以员工的理性选择是不着急。

定义转型激励函数：
I(转型) = R(转型成功) × P(成功) - C(转型参与) × (1-P(成功))

I(ORG) >> 0 → 组织有强激励
I(个人) ≈ 0 或 < 0 → 员工无激励或负激励

### 第六步：收敛函数

**紧迫感差异函数**：

ΔG(t) = G(ORG, t) - G(个人, t)

= Σᵢ [-dPᵢ(ORG)/dt / Pᵢ(ORG) + dPᵢ(个人)/dt / Pᵢ(个人)]

其中Pᵢ是P_survival的第i个因子。

行业变革时ΔG(t)单调递增，因为：
- dPᵢ(ORG)/dt全部为负且绝对值大（组织五因子同时受冲击）
- dPᵢ(个人)/dt部分为负但绝对值小（个人可迁移、可切换）
- 随变革深入，组织的n_lock使转型越来越难→dPᵢ(ORG)/dt加速变负
- 个人的n_lock低→可以持续调整→dPᵢ(个人)/dt趋稳

**终极收敛**：

ΔG(t) = G(ORG) - G(个人) = -d/dt[ln(P_survival(ORG)/P_survival(个人))]

行业变革越深，P_survival(ORG)/P_survival(个人)越小→其对数的导数越负→ΔG越大。

公司着急、员工不急，不是态度问题，是数学结构决定的：
1. 五因子的衰减速率全部对组织更不利
2. 组织锁定维度是个体的超集，转型成本不可比
3. 信息不对称使G(ORG)对决策层可见、对基层不可见
4. 风险错配使转型激励对组织为正、对个体为零或负

四条独立因果链，全部指向同一结论：ΔG > 0且单调递增。公司必然比员工着急，这是确定性事件，不是管理问题。
