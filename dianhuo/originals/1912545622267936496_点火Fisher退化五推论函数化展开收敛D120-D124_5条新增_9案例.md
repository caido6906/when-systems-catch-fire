## D120 不可逆性判据函数

Ξ(G) = sup_{p∈门控区, q∈存活区} d_Fisher(p,q)

Ξ(G) = ∞ ⟺ 系统不可逆（乘法结构）
Ξ(G) < ∞ ⟺ 系统可逆（加法结构）

精确形式：对G=∏fᵢ(εᵢ)，Ξ(G)=∞（因为∫₀^{εₖ} dε/ε²=∞对至少一个k成立）。对G=∑fᵢ(εᵢ)，Ξ(G)<∞（因为零集=∩Hᵢ只需全零才归零，Fisher距离有限）。

混合结构判据：G=α·∏fᵢ + (1-α)·∑gⱼ，Ξ(G)随α从0→1连续变化：α=0时Ξ有限，α>0时Ξ=∞。任何乘法分量（α>0）都使系统不可逆。不需要"纯乘法"，只要有一个乘法项就够。

诊断操作：检查系统产出函数G是否包含乘法项。如果G=∏fᵢ×h(其他)或G中任一因子可归零，则Ξ=∞，系统不可逆。

案例#472 不可逆性判据验证 — 5种系统结构：纯加法(Ξ=3.2), 加法主导+乘法修正(Ξ=∞), 半加半乘(Ξ=∞), 乘法主导+加法修正(Ξ=∞), 纯乘法(Ξ=∞)。α>0即不可逆，与乘法权重无关。核心函数：D120

## D121 Fisher健康度函数

H_Fisher(p) = A_Fisher(p) / A_Fisher(p₀)

p₀是参考健康态。H_Fisher∈[0,1]，0=完全锁死，1=完全健康。

预警函数：τ_warning = -H_Fisher / (dH_Fisher/dt)

τ_warning是"按当前退化速率，H_Fisher降到0还需要多久"。τ_warning越小越紧急。

与Shannon熵的预测力对比：
- Shannon熵预警：τ_S = -(S_max - S(t)) / (dS/dt)
- 关键差异：dS/dt可正可负（熵增减不定），τ_S可能为负（虚假预警）或∞（永远不预警）
- dH_Fisher/dt≤0恒成立，τ_warning恒正且有限——永远不会给出虚假信号

突然恶化的数学条件：dH_Fisher/dt突然加速（二阶导<0且绝对值增大）=某个εᵢ进入刀刃区。Shannon熵检测不到——S的变化可能很平缓，但H_Fisher的衰减率突然跳变。

案例#473 Fisher健康度vs Shannon熵预警 — 模拟8维退化系统：H_Fisher从1.0→0.8→0.4→0.05单调递减，τ_warning从∞→12→3→0.2持续缩短。Shannon熵从2.1→1.8→2.3→2.0非单调，τ_S在t=3时为负（虚假信号"正在恢复"），实际系统正在加速退化。核心函数：D121

案例#474 突然恶化预警 — 老年人第5维（免疫系统）进入刀刃期时dH_Fisher/dt从-0.02跳变到-0.15（7.5倍），τ_warning从50→7。Shannon熵变化<5%，无预警信号。核心函数：D121

## D122 退化加速函数

a(t) = d²A_Fisher/dt² = -∑ᵢ ∂/∂t[σ'(εᵢ-θC(i))×|dεᵢ/dt|×d_F(εᵢ)/λ] / (1+d_F(εᵢ)/λ)²

关键项展开：对第k个接近门槛的维度，

aₖ ≈ -σ''(εₖ-θC)×(dεₖ/dt)²×d_F(εₖ)/λ / (1+d_F(εₖ)/λ)²

当εₖ<θC时σ''>0，且dεₖ/dt<0（退化中），所以aₖ<0——退化在加速。

与D112的统一：D112中β=γ×max(σ')，dβ/dt=γ×σ''×dε/dt。退化加速函数中的σ''×(dε/dt)²与D112的σ''×dε/dt共享同一个σ''变号结构。区别：D112是β对ε的一阶导（策略权重如何随改善变化），D122是A_Fisher对t的二阶导（退化如何随时间加速）。一个是策略空间的动力学，一个是状态空间的动力学，同一个σ''驱动。

加速的临界条件：a(t)从接近0变为显著负值=系统进入退化加速区。临界条件是εₖ穿过σ''开始显著的区域（εₖ≈θC-2/α，sigmoid开始弯曲的位置）。

案例#475 退化加速与β上升同构 — 同一8维系统：β(t)从0.1→0.4→2.5→0.3（先升后降），|a(t)|从0.001→0.02→0.15→0.01（先加速后减速）。两者峰值在同一时刻，σ''变号点一致。核心函数：D122

案例#476 组织退化加速 — 公司关键岗位流失：前3人离开A_Fisher降5%（线性），第4-6人离开降18%（加速），第7人离开降40%（急剧加速）。σ'峰值对应"团队刚好不够用"的临界点。核心函数：D122

## D123 修复有效性判据函数

Ω_repair = step(ΔA_Fisher - θ_repair)

Ω_repair=1时修复有效，Ω_repair=0时修复无效。

ΔA_Fisher的分解：

ΔA_Fisher = ∂A/∂n × Δn + ∂A/∂θC × ΔθC + ∂A/∂ε × Δε

三项分别对应三种修复路径：
- Δn>0：增加维度（打开新轨道/新能力/新通路）
- ΔθC<0：降低门槛（降低退出成本/降低启动要求）
- Δε>0：提升现有维度值（增强已有能力）

关键定理：增加信息量ΔI>0但Δn=0且ΔθC=0且Δε≈0时，ΔA_Fisher≈0。

证明：信息量增加不改变参数空间的拓扑结构（零集不变），不改变门槛位置，不改变维度值。A_Fisher只依赖参数空间的几何结构，与信息量无关。增加信息量=在门控区内增加可区分状态，但Fisher距离仍为∞。∎

修复路径排序（按ΔA_Fisher/Δcost）：
1. 降低门槛ΔθC<0：成本最低，效果最大（D91解码门槛降低的实例）
2. 增加维度Δn>0：成本中等，效果持久（打开新Fisher路径）
3. 提升维度值Δε>0：成本最高，效果受限于已有路径

与C_exit的对应：降低门槛θC=降低C_exit。D123给出修复优先级：先降C_exit（门槛），再开新维度（轨道），最后提升ε（能力）。这与D111弹性-级联策略的"补弹性最高"一致——降门槛是最有效的弹性提升方式。

案例#477 修复路径排序验证 — 3种修复方案：降门槛(ΔθC=-0.3)→ΔA_Fisher=+2.1/cost=1; 加维度(Δn=1)→ΔA_Fisher=+1.8/cost=2; 提升ε(Δε=+0.5)→ΔA_Fisher=+0.4/cost=3。降门槛性价比5.3倍于提升ε。核心函数：D123

案例#478 信息量增加不增加可达性 — 阿尔茨海默患者记忆训练6个月：信息回忆准确率+12%，但A_Fisher不变（清洗通道仍堵），日常功能无改善。改善睡眠3个月：A_Fisher+0.3，日常功能改善23%。核心函数：D123

## D124 三域退化统一参数函数

τ_life = (1/α) × ln(I₀/M₀ + ∫₀^∞ e^{-βt}×σ'(ε(t)-θC)dt / ∫₀^∞ e^{αt}×σ'(ε(t)-θC)dt)

τ_life是系统寿命，由α（维护成本增长率）、β（信息价值饱和率）、σ'加权项（刀刃期退化加速）共同决定。

三域参数映射函数：

| 参数 | 生物衰老 | 组织衰败 | 认知退化 |
|---|---|---|---|
| α | α_bio = 损伤累积率×代谢率 | α_org = 人才流失率×锁定增长率 | α_cog = 通道关闭率×专业化增长率 |
| β | β_bio = 适应能力饱和率 | β_org = 能力增长饱和率 | β_cog = 学习率饱和率 |
| θC | θ_bio = 功能底线 | θ_org = 竞争底线 | θ_cog = 整合底线 |
| σ'peak | σ'_bio = 衰老加速期 | σ'_org = 危机加速期 | σ'_cog = 僵化加速期 |

统一退化轨迹函数：H_Fisher(t) = exp(-∫₀^t α(s)×σ'(ε(s)-θC)ds)

三域共享同一个H_Fisher衰减形式，只是参数取值不同。生物域α大β中→τ_life中等；组织域α中β低→τ_life长但退化慢；认知域α低β低→τ_life最长但一旦僵化不可逆。

跨域退化传染：当三域耦合时，总退化率不是三个α的加权和，而是乘法——任一域归零则整体归零。跨域传染的数学条件：耦合强度κ_ij>0时，域j的退化加速域i的退化（dα_i/dt ∝ κ_ij×|dH_j/dt|）。

案例#479 三域寿命计算 — 生物域α=0.08/年, β=0.03/年, θC=0.3→τ_life≈85年。组织域α=0.05/年, β=0.01/年, θC=0.4→τ_life≈120年。认知域α=0.02/年, β=0.01/年, θC=0.2→τ_life≈200年。跨域耦合后τ_life≈70年（最短域决定）。核心函数：D124

案例#480 跨域退化传染 — 退休(组织退出)→社交网络萎缩(认知退化加速)→抑郁(生物退化加速)。κ_org→cog=0.3, κ_cog→bio=0.5，三级传染使总退化速率比单域快2.8倍。核心函数：D124
