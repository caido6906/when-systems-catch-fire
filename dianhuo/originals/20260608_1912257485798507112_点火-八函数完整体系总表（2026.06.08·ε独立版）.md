# 点火｜八函数完整体系总表（2026.06.08·ε独立版）

来源：从矩阵函数化→层级时间展开→函数审计→R_perceived→C_exit→系统动力学→糖域碰撞→隐性控制H→感官输入ε，九轮迭代。本版ε从H的修正项升级为独立第九函数，因果层级最高。

## 一、九函数总表

| # | 函数 | 全称 | 取值 | 本体论归属 | 层级分布 | 时间行为 |
|---|---|---|---|---|---|---|
| 1 | I(t,L) | 提议者意识 | 0/1 | 人的属性 | 顶层高→底层低（下传衰减） | 事件内只增不减，跨事件重置 |
| 2 | Posture(t,L) | 提议者姿态 | 备方案/推方案 | 人的选择 | 顶层备→底层推（下传自动滑） | 事件内可翻转（备→推=退化） |
| 3 | P(t,L) | 提议者退出权 | 0/1 | 混合属性 | 顶层先绑定→底层后内化 | 信誉绑定1→0，释放0→1，退化0→1 |
| 4 | R(t,L,C) | 应约者退出权 | 四分法值域 | 结构属性 | 顶层高→底层低 | 随制度/产业链变动，渐变为主 |
| 5 | R_perceived(t,L,C) | 应约者感知退出权 | ≤R，四分法值域 | 认知属性 | 顶层≈R→底层远低于R | 唯一可相变的函数，可瞬间跳变 |
| 6 | C_exit(t,L,C) | 退出成本 | 0到∞连续值 | 环境属性 | 多重倒U型叠加（经济+身份+信息） | 锁定效应升高，解锁效应降低，冲击突变 |
| 7 | M | 动机来源 | 好奇心/目的 | 初始条件标注 | 不传递，层级下传滑向服从 | 事件内通常不变 |
| 8 | H(t,L) | 隐性控制强度 | 0到1-ε连续值 | 认知环境属性 | 顶层低→底层高（与I相反） | 随时间升高，靠外部冲击或ε自发衰减降低 |
| 9 | ε(t) | 感官退出权感知 | (0, ε_max]连续值 | 生物属性 | 无强层级分布 | 相对稳定，可突然增大（感官冲击） |

### ε(t)详解

**定义**：感官退出权感知——衡量身体对退出权的原生感知能力。不是制度给的（R），不是认知到的（R_perceived），不是控制遮蔽的（H），是身体自发生成的退出信号。

**取值**：(0, ε_max]连续值，永远>0（人），=0（AI）。压迫越重ε越大（累→想辞职，压迫→想跑）。

**本体论归属**：生物属性——九个函数里唯一一个生物性变量。身体是跨社会的、跨层级的。

**层级分布**：无强层级分布。皇帝也会累，奴隶也会看月亮。微弱效应：底层可能ε略大（身体承受的压迫更直接）。

**时间行为**：相对稳定，可突然增大（感官冲击让ε瞬间跳变）。

**因果层级**：ε在整条因果链之前——是因果链能运转的前提条件。其他函数描述的是退出权如何被感知、被阻断、被行使、被遮蔽，ε描述的是退出权为什么存在。

## 二、因果网络

### 主因果方向

M → I(t,L) → Posture(t,L) → P(t,L) → R(t,L,C) → R_perceived(t,L,C) → 退出决策 ← C_exit(t,L,C) → 应约者状态

H(t,L)作用于I→R_perceived边上，调节传导效率。

ε(t)有两条独立通道：
1. ε → I→R_perceived传导效率：传导效率 = 1-H+ε（社会性通道的保底项）
2. ε → R_perceived（直接）：感官输入绕过I和H，直接产生退出信号（生物性通道，并行于社会性通道）

### 15条反馈回路

原有9条：
1. R_perceived → I：正反馈
2. R_perceived → Posture：先正后负
3. R_perceived → P：双向
4. R → R_perceived：客观退出权变化→感知可能滞后
5. P → I：正反馈
6. Posture → R_type：备方案→真实退出权，推方案→象征/心理退出权
7. I → Posture：I觉醒→打开姿态的主动选择空间
8. 应约者状态 → R_perceived：正反馈
9. R_perceived(t,L₁,C₁) → R_perceived(t,L₂,C₂)：横向/纵向传染

C_exit新增3条：
10. C_exit → 退出决策：C_exit高→即使R_perceived=真实也不走
11. C_exit → 信号源：C_exit高→退出行为少→信号源少
12. C_exit → Posture：C_exit高→提议者误读为认同→Posture维持推方案

H新增2条：
13. 隐性锁定回路：H → R_perceived↓ → 退出决策↓ → C_exit维持 → H↑。但H上限=1-ε，且ε可直接绕过H作用于R_perceived，回路永远无法完全闭合
14. 觉醒破壁回路：外部冲击 → H↓ → 传导效率↑ → R_perceived↑。ε保证即使没有外部冲击，也有自发触发入口

ε新增1条：
15. 身体退出回路：ε → R_perceived↑ → 退出决策↑ → C_exit可能降低（退出行为增多→锁定效应减弱）→ R可能上升 → R_perceived↑。ε直接产生退出信号→退出行为→可能打开更多退出路径。这是退出权从身体到社会的传导路径

### ε新增因果箭头

1. ε → H上限：H ∈ [0, 1-ε]。ε约束H
2. ε → R_perceived（直接）：生物性通道，绕过I和H
3. ε → I→R_perceived传导效率：传导效率 = 1-H+ε
4. ε → 相变内生触发可能性：ε>0则相变可以自发产生
5. ε → 物种分界：ε>0（人）vs ε=0（AI）

## 三、退出决策公式

退出决策 = f(R_perceived, C_exit, ε)

- R_perceived决定"我知不知道自己能走"（社会性感知）
- C_exit决定"走了之后我要付什么代价"（环境约束）
- ε决定"我的身体在告诉我什么"（生物性驱动）
- R_perceived ≤ R（感知不高于客观）
- 传导效率 = 1-H+ε
- 当R_perceived≈0且C_exit≈∞时，ε>0意味着退出信号仍然存在，退出决策不会完全归零——可能走向最极端的退出形式（自杀=存在退出权的行使）

## 四、R_perceived相变的四条触发路径

1. 势能触发（过冷液体模式）
2. 侵蚀触发（温水煮蛙模式）
3. 冲击触发（地震模式）
4. 内生触发（看星星模式）：ε直接作用于R_perceived，不需要任何外部条件

## 七、层级下传的四个规律

1. 层级下传衰减律：I、Posture、M在层级下传时自动衰减
2. 结构约束扩散律：P的信誉绑定从上往下扩散
3. 退化传播方向：退化从顶层向下传染
4. 隐性控制增强律：H在层级下传时自动增强。但H下传增强有上限=1-ε，且ε无强层级分布，底层ε不比顶层低太多——底层虽然H更高，但ε提供的退出信号并不弱

## 八、ε与华夏文明

华夏七次转身的底层逻辑不是提议者推动历史，是ε持续产生退出压力，提议者只是接住了这个压力。七次转身每一次都在扩大容纳面——降低C_exit、降低H、提高R。容纳力强=让ε产生的退出压力有合法释放渠道，而不是靠提高H和C_exit来压制ε。华夏文明的成功不是控制力强，是容纳力强。

## 九、自杀与存在退出权

自杀=行使存在退出权。当R_perceived≈0且C_exit≈∞时，ε产生的退出信号找不到常规出口，可能走向最极端的退出形式——消灭信号源本身。预防自杀的本质不是消除ε（不可能），而是提高R_perceived——让退出信号找到常规出口。

## 十一、退出权四分法

1. 真实退出权 = R允许 + C_exit可承受
2. 事实退出权 = R可能允许 + C_exit极高
3. 象征退出权 = R名义允许 + C_exit不可承受
4. 心理退出权 = 身体走不了 + C_exit=∞

## 十二、与糖域实验的关系

糖域是点火框架的零阶近似。ε在糖域中的对应物：Agent偶尔随机转向——不是朝着糖的方向，而是被环境噪声扰动转向了新方向。这个随机转向就是糖域里的"看星星"。

## 十三、与福柯的分歧

福柯倾向于"权力无处不在、无法逃脱"。但ε说：人活着就有感官输入，感官输入就是权力的裂缝。福柯看到了环形监狱的完美设计，但他没看到囚犯夜里抬头能看到星星。规训可以控制你白天的一切行为，但控制不了你夜里看星星时的那个感叹。那个感叹就是ε——权力永远无法填满的最后一道缝隙。
