# 《点火》章节结构（2026.06.03·故事驱动版·九次转身·三层结构）

定位：讲故事的人，不是学者。文明为主线，政策和公司穿插。框架藏在故事排列里，读者不需要知道的东西不写。

书名《点火：系统重写的五种趋势》

## 核心定义

系统 = 协作系统 + 认同系统
- 协作系统回答"我们怎么一起做事"——提议者搭房
- 认同系统回答"我们是谁、为什么愿意留在这"——提议者点火

提议者与应约者：一方提方案，一方接方案。退出权内置在"应不应"里。
- 被动应约：有退出权但没意识到，是认知盲区
- 被强迫应约：无退出权，不构成应约，是服从

## 三层结构

**第一层：条件（24格矩阵）**
提议者意识（有/无）× 提议者退出权（有/无）× 应约者退出权（有/无）× 应约者状态（积极/消极/无人）= 24格

**第二层：方向（6种）**
建在应约者身上 / 建在部分应约者身上 / 建在外部 / 建在个人权威上 / 建在旧系统的否定上 / 没建

**第三层：状态（5种）**
点着了·活着 / 点着了·灭了 / 没点着·火种还在 / 没点着·火种也没了 / 从未点火

## 章节结构

**引言：空降CEO为什么推不动**
主案例：互联网公司空降CEO改革失败
点题：只搭房不点火，系统必然无法存续
闪回：周公制礼、孝文帝汉化——先点一下，后面会展开

**第一章：秦——只搭房不点火，十五年就塌了**
主故事：秦始皇统一六国，郡县制+书同文+车同轨，制度完备但六国遗民从未认同
穿插：空降CEO只改制度不改文化
同组案例：查士丁尼复兴罗马、洋务运动

**第二章：周公——点火点对了，八百年基业**
主故事：周公制礼，异姓诸侯从战俘变贵族，礼乐秩序让所有人愿意留下
穿插：创业公司认同锚定在融资估值上的反面案例
同组案例：穆罕默德创伊斯兰、美国建国、新加坡

**第三章：唐——点歪了，但不是死刑**
主故事：唐朝认同锚定在胡汉二元上，安史之乱暴露裂缝，但通过内部整合续命150年
穿插：企业将认同锚定在行业红利上
同组案例：苏联、清满汉一体、阿拔斯革命

**第四章：戈尔巴乔夫——先拆旧房子再建新房子**
主故事：苏联解体，先拆旧系统再建新的，负时间差直接崩盘
穿插：裸辞、分手——旧支撑没了新的还没建起来
同组案例：法国大革命初期、戊戌变法、奥斯曼晚期

**第五章：元朝——从来没点着过**
主故事：元朝四等人制靠暴力维持，从未建立值得群体应约的共同方案
穿插：从创立之初就只追求收割而非共建的企业
同组案例：蒙古帝国、卡拉卡拉普发公民权、西罗马灭亡

**第六章：华夏为什么扛得住——文化协议的秘密**
主故事：华夏文明与罗马帝国的存续差异，叠加式升级 vs 替换式升级
穿插：文化驱动型企业与KPI驱动型企业的抗风险差异
同组案例：罗马从城邦到帝国、奥斯曼米利特制度、孔子作春秋、印度被殖民

**第七章：九次转身——华夏的完整故事**
主故事：华夏文明九次关键转型，串联前文所有华夏案例
穿插：凯末尔改革作为对照案例

华夏九次点火（按时间线排列·层级校准版）：
1. 周公制礼（约前1040年代）——文明级，格7→格1，点对了
2. 孔子作春秋（约前5世纪）——文明级，格9→格1，点对了（300年后汉武帝补火）
3. 秦统一（前221年）——文明级，格6→格1，只搭房没点火（汉补火）
4. 孝文帝汉化（471-499年）——文明级，格10，点对了
5. 唐朝开放（618年起）——文明级，格10→格11，点歪了但转向成功
6. 清朝满汉一体（1644-1912年）——文明级，格10，点歪了但转向成功
7. 新中国成立（1949年）——国家级，格10，点对了
8. 改革开放（1978年启动）——国家级（可能升级为文明级），格22→格19→格7，点火中
9. 元朝（1271-1368年）——折戟，格12，异族拒绝全盘接入

全球对照穿插：日本大化改新、明治维新、阿育王皈依

**第八章：你的系统**
引导读者自测：所在系统的提议者/应约者，自身认同锚点，提议者退化信号
个人案例：裸辞=负时间差，KPI认同=点歪了，企业文化认同=点对了

**尾声：让火被看见**
前八次转身，当时的人不知道自己在参与一次系统重写，事后才被命名。第九次不一样——这是第一次转身还在进行中就被识别出来。发现者不是点火的人，是让点火被看见的人。应约者一旦看见自己在应约，应约的方式会变。让火被看见，本身就在改变点火的走向。

历史先例：路德、西耶斯、福泽谕吉——但都是提议者看见自己的提议在生效，不是应约者看见机制本身。

读者读到这里，意识到自己正在应约——这个认知本身就是点火的一部分。

**附录：这个框架能看到什么，看不到什么**
适用边界：可精确覆盖文明、政策、公司；婚姻、教育等角色模糊场景边界衰减
点火判定核心：认同建在哪？
框架验证：四个案例库外案例（张居正改革、福耀美国工厂、美国芯片法案、美国国家认同危机）全部自然落格，框架不是萝卜坑
