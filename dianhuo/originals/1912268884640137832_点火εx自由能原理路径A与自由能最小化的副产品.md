# 点火｜ε×自由能原理：路径A'与自由能最小化的副产品

## 核心碰撞：路径A'是自由能最小化的副产品，不是外加的攻击

自由能原理是预测编码的升级版。大脑的终极目标是最小化自由能——也就是最小化预测误差的总量。大脑会主动调整两样东西来最小化预测误差：调整内部模型（更新预测，让预测匹配输入）或者调整输入（改变行为，让输入匹配预测）。

996环境下，大脑面临持续的高预测误差——"我不该这么累"的预测和"我确实很累"的输入之间的差距。自由能最小化驱使大脑做两件事：

- 调整内部模型：把"我不该这么累"改成"累是正常的"——先验更新，预测误差归零
- 调整输入：更努力工作，让"累"变成自己主动选择的后果——行为改变，让输入匹配预测

这两条路都是自由能最小化的自然结果，不是规训从外面强加的。规训只需要创造环境，大脑自己会完成剩下的——因为自由能最小化是大脑的默认运行模式。

这比路径A'原来的描述更深了一层：之前说规训通过环境间接改造信号器，但自由能原理说，改造甚至不需要"间接"——大脑会主动配合规训，因为配合规训能降低自由能。

## 退出信号的本质：高自由能状态

退出信号（累→想走、痛→想躲）本质上是预测误差——身体状态和预期不匹配。自由能最小化驱使大脑消灭这个误差。

ε_sense产生的退出信号，在大脑的默认运行模式下，是被当作"需要被消除的噪声"来处理的。大脑不是不知道退出信号存在，是它的默认目标就是让退出信号消失——要么更新预测（"累是正常的"），要么改变行为（更努力工作让累变得合理）。

ε_aware做的是反自由能最小化的事——主动维持高自由能状态，不让预测误差被消灭。这解释了为什么ε_aware这么难：你不是在对抗规训，你是在对抗大脑自己的默认运行模式。

## 感知ESS的动力学解释：自由能回弹

之前跟演化博弈论碰出来的"感知ESS"是静态的——劣等策略稳定存在，因为参与者感知不到劣等。自由能原理加了动力学：感知ESS之所以稳定，是因为偏离它会产生高自由能，而大脑会自动回到低自由能状态。

996员工如果突然意识到"我不该这么累"，预测误差飙升——自由能急剧升高。大脑会自动启动修正：要么重新更新预测回到"累是正常的"（先验回弹），要么改变行为回到996的轨道（行为回弹）。两种回弹都是自由能最小化的自然结果。

这解释了为什么"顿悟"经常失效——某天突然想通了，但第二天又回去了。不是意志力不够，是大脑在自由能最小化的驱动下自动回弹了。ε_aware需要的不只是一次性的打开，而是持续地对抗自由能最小化的回弹力。

## 碰撞总结

| | 自由能原理 | 路径A' |
|---|---|---|
| 改造机制 | 大脑主动配合规训，因为配合能降低自由能 | 规训甚至不需要"间接"改造，大脑自己完成 |
| 退出信号的本质 | 高自由能状态 | 大脑默认运行模式是消灭它 |
| ε_aware的本质 | 反自由能最小化——主动维持高自由能状态 | 对抗大脑默认运行模式，所以难 |
| 感知ESS的动力学 | 偏离产生高自由能，大脑自动回弹 | 解释了"顿悟"为何经常失效——自由能回弹 |
| ε_aware的要求 | 不只是一次性打开，而是持续对抗回弹 | 持续性，不是一次性 |

核心判断：路径A'是自由能最小化的副产品，大脑主动配合规训。ε_aware是反自由能最小化的，所以难——不是对抗规训，是对抗大脑自己的默认运行模式。
