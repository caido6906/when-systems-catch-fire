# [2026-06-07 10:20] 点火框架与演化博弈论策略的关系（补充讨论）

结论：说“点火框架是演化博弈论策略”不够准确；更准确的说法是：点火框架可以被翻译成一种“演化博弈论视角下的策略—认同—扩散模型”，但它本身不是严格的演化博弈论模型。

它和演化博弈论关系很近，但不是同一层级的东西。

点火框架自己的核心定义是：系统 = 协作系统 + 认同系统。协作系统负责制度、流程、组织、资源、技术结构；认同系统负责身份、意义、参与感、归属感、共同方向。点火不是系统建成，而是“应约者”在有退出权压力时，仍愿意进入并认领新系统。仓库里也明确说，点火框架研究的是“新秩序如何活起来”，区分 collaboration system 与 recognition system，并关心人是否愿意在压力下进入并认领新秩序。

演化博弈论的核心则是：一个群体里存在不同策略，不同策略在互动中得到不同收益，收益影响策略在群体中的扩散或淘汰。它有两条典型路线：一条是“演化稳定策略”即 ESS，另一条是用复制动态等模型研究策略频率如何随时间变化。

所以二者的对应关系可以这样看：

点火框架里的“提议者”，相当于推出一种新策略、新制度、新秩序、新协调方式的人。点火框架里的“应约者”，相当于面对这个策略的人群：他们可以进入、观望、退出、假装配合、心理退出，或者反向抵抗。仓库中的概念图明确把事件定义为“提议者出方案，应约者接或不接，随后产生走向”，并把退出权、提议者姿态、推方案、备方案等列为核心变量。

这就非常接近演化博弈论：不同方案在群体中竞争，谁能被更多人接住，谁就有更高的“复制率”。但点火框架多了一层演化博弈论通常不会充分表达的东西：认同收益。

传统博弈论里，收益通常可以写成 payoff：钱、资源、生存概率、地位、机会、成本。点火框架关心的是：即使协作系统已经有收益，人为什么仍然不愿意进入？答案可能是：他没有认同感、没有归属感、觉得自己只是被安排、被征用、被规训。也就是说，点火框架把“收益函数”扩展了：收益不只是物质收益，还包括身份收益、意义收益、叙事收益、退出权收益、被尊重感、主动认领感。

这也是它和演化博弈论最关键的连接点：点火框架可以被理解为对“策略能否扩散”的前置条件分析。

演化博弈论问的是：一个策略在群体中能不能稳定、能不能扩散、能不能抗入侵。点火框架问的是：一个新系统为什么明明搭好了，却没有人真心进入？它不是直接计算策略频率，而是在研究策略开始复制之前，那个“第一批人愿不愿意把自己放进去”的临界条件。

所以，“点火”可以被看作演化博弈中的一个特殊阶段：不是策略已经在群体中稳定竞争的阶段，而是新策略从提案变成可复制行为模式的临界阶段。

如果硬翻译成演化博弈论语言，大概是这样：

“点对了” = 新策略获得正向认同收益，应约者主动采用，策略开始扩散，形成稳定新均衡。

“点不足” = 新策略只在局部人群中有正收益，无法全局扩散，形成半稳定、半失败状态。

“点歪了” = 表面上策略被采用，但真实收益函数错位；应约者可能因为叙事压力、外部评价、制度锁定而暂时配合，长期会退化。

“点反了” = 旧策略被破坏，新策略没有提供可承受的收益，群体进入负收益区，出现抵抗、空转、心理退出或系统真空。

“伪点火” = 表面策略频率上升，但不是自愿复制，而是强制、仪式、宣传、服从造成的虚假扩散。仓库里也明确区分失败点火、伪点火、点歪了、点反了、负时间差等状态。

因此，别人说“点火框架是演化博弈论策略”，他抓到了一部分真东西：点火确实关心“策略如何被人接住、如何扩散、如何稳定、如何失败”。尤其是“提议—应约—退出权—认同—扩散”这一链条，和演化博弈论里的“策略—收益—选择—复制—稳定”高度同构。

但这个说法也有明显压扁。点火框架不是一个已经形式化的演化博弈论模型，因为它目前没有明确给出策略集、收益矩阵、复制方程、均衡条件，也没有把“点火成功”严格定义为某种 ESS 或复制动态稳定点。演化博弈论要求更数学化；点火框架目前更像历史解释框架、制度叙事框架、认同动力学框架。仓库也提示目前很多案例仍是工作稿和待核查案例表，不应被当作已完成史实证明。

我会这样定性：

点火框架不是演化博弈论策略本身，而是一个可以嫁接到演化博弈论上的“策略采纳条件框架”。它研究的不是策略已经进入复制动态之后怎样演化，而是一个新策略、新制度、新秩序在进入复制动态之前，怎样获得第一批真实应约者。

再压缩成一句话：

演化博弈论研究“什么策略会活下来”；点火框架研究“一个新策略怎样先活起来”。

这两者关系很紧。点火框架如果继续理论化，下一步最自然的方向之一，就是把“认同系统”形式化为收益函数的一部分，把“退出权”形式化为参与成本/退出成本，把“应约者认领”形式化为策略采纳概率，把“点火/伪点火/点歪/点反”形式化为不同的动态路径。到这一步，它就可以变成一种真正的演化博弈论模型。
