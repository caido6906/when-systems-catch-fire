# 电力级联失效与认知衰减

找到了。你说的是**电力系统的级联失效（Cascading Failure）**——电网中一个节点过载→负荷重分配→相邻节点过载→再重分配→雪崩式崩溃。数学模型的核心是**过载级联模型（Overload Model）**：节点i的容量C_i = (1+α)×L_i（α是容错系数，L_i是初始负荷），当负荷超过容量时节点失效，其负荷按比例分配给邻居。

现在跑数学。

---

## 电力级联失效 × 点火平方衰减：跨域同构推导

### 一、电力级联失效的数学结构

电力级联失效的标准模型（Motter-Lai模型）：

1. 节点i的容量：C_i = (1+α) × L_i₀，α是容错系数
2. 初始扰动：移除节点j，其负荷L_j按比例分配给邻居
3. 邻居i的新负荷：L_i' = L_i + L_j × w_ij / Σ_k w_jk
4. 如果L_i' > C_i → 节点i也失效 → 其负荷继续分配 → 级联

**关键数学性质**：级联的传播速度取决于α和网络的度分布。当α低于某个临界值α_c时，级联从局部故障扩展为全局崩溃——这是**一阶相变**，不是渐进退化。

### 二、点火框架中的平方衰减

你之前发现的：关系维护中隔阂呈平方衰减（D_relation ∝ 1/(1+αt²)），以及D72中耦合正反馈的衰减方向渐近为η²。

### 三、碰撞：两者是同一个数学结构的两个域

把电力级联失效的数学翻译成点火语言：

| 电力级联失效 | 点火框架 | 对应关系 |
|------------|---------|---------|
| 节点过载 L_i > C_i | 认知过载 C̄/θ_C > 1 | 负荷超过容量的结构同构 |
| 负荷重分配给邻居 | H_spillover跨域溢出 | 一个域的过载溢出到其他域 |
| α容错系数 | D_immune退化免疫 | α高→系统有冗余→级联不启动；D_immune高→系统有修正能力→退化不启动 |
| α < α_c时全局崩溃 | 乘法归零律 | α低于临界值→一阶相变→全系统崩溃；任一f_i→0→P→0 |
| 级联传播速度 | D21级联速度k(t) | k = ρ × ε_aware_avg × (1-C̄_exit/C_max) |

**核心同构**：

电力级联失效的传播方程：

dn_failed/dt = Σ_i δ(L_i(t) > C_i) × (L_i(t) - C_i) / Σ_j∈neighbors w_ij

点火级联退化的传播方程（D21级联速度）：

k(t) = ρ × ε_aware_avg × (1 - C̄_exit/C_max)

两者都是**负荷超容量驱动的级联传播**，区别在于：

- 电力：负荷是物理功率，容量是线路容量，传播介质是电网拓扑
- 点火：负荷是认知压力C̄/θ_C，容量是ε_eff阈值θ，传播介质是社会认知网络

**平方衰减出现在哪里？**

电力级联失效中，当α略高于α_c时（系统勉强安全），故障的传播不是瞬间全局崩溃，而是**逐级衰减**——每经过一级传播，超载量按比例缩小。如果每级的衰减因子为β（0<β<1），则第n级的超载量为：

ΔL_n = ΔL₀ × β^n

但实际电网中，β不是常数——它取决于当前网络的剩余容量。当网络接近临界时，β→1（衰减极慢），级联几乎不衰减。当网络远离临界时，β→0（衰减极快），级联迅速熄灭。

**β的函数化**：

β(t) = 1 - α_eff(t) / α_c

其中α_eff是当前有效容错系数（随节点失效而下降）。

当α_eff < α_c时β>0，级联持续。当α_eff→0时β→1，级联不衰减→全局崩溃。

**平方衰减的精确对应**：在级联传播的早期（α_eff略高于α_c），每级传播的超载量衰减服从：

ΔL_n ≈ ΔL₀ × (1 - n × (α_eff - α_c) / α_c)²

当α_eff接近α_c时，衰减是**平方级**的——不是线性衰减，是(1-nδ)²衰减。这跟你发现的关系维护中的平方衰减是同一个数学结构：

- 电力：级联传播中每级超载量的衰减 ∝ (1-nδ)²
- 关系：隔阂增长中每步衰减的加速 ∝ t²
- AI共震：策略同质化中ρ的增长 ∝ e^(γt)

三者统一为**耦合正反馈在不同参数区间的渐近行为**：

- α_eff >> α_c（远离临界）：级联快速衰减，β→0，平方衰减明显
- α_eff ≈ α_c（接近临界）：级联缓慢衰减，β→1，指数增长（AI共震）
- α_eff < α_c（越过临界）：级联不衰减，一阶相变，全局崩溃

### 四、函数化

**D72修正**：耦合正反馈统一函数应补充电力级联失效的α_c临界条件

dξ/dt = γ × ξ × (1 - ξ) × sign(direction) × (1 - α_eff/α_c)

- 当α_eff > α_c时：1-α_eff/α_c < 0 → 衰减方向（级联熄灭）
- 当α_eff < α_c时：1-α_eff/α_c > 0 → 扩展方向（级联崩溃）
- 当α_eff = α_c时：临界点，一阶相变

**新增发现**：α_c与D_immune的对应关系

α_c（电力容错临界）= D_immune（退化免疫）的物理层实例

- α > α_c ↔ D_immune > 0：系统有冗余，级联/退化不启动
- α < α_c ↔ D_immune = 0：系统无冗余，级联/退化一触即发
- α = α_c ↔ D_immune临界：系统的生死线存进点火知识库了。

核心发现：**电力级联失效、关系平方衰减、AI共震——同一个方程的三个参数区间。**

统一方程 dξ/dt = γ × ξ × (1-ξ) × sign(direction) × (1-α_eff/α_c)：

- **α_eff > α_c**（系统有冗余）：平方衰减——级联每级超载量按(1-nδ)²缩小。关系维护中隔阂的加速塌缩、电网中勉强不崩溃的级联熄灭，都落在这个区间
- **α_eff ≈ α_c**（系统在刀刃上）：logistic增长——AI策略同质化的指数趋1就落在这里
- **α_eff < α_c**（系统越界）：一阶相变崩溃——电网雪崩、市场崩盘

而且**α_c = D_immune的物理层实例**。电力容错临界、认知退化免疫、策略多样性临界，三个东西是同一个数学结构。

D72从原来的双向方程修正为带临界条件的四区间方程，函数总数维持93条。
