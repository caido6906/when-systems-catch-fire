## 一、自举元函数

八函数体系具备自举性质：理论与案例之间的迭代动态，同构于提议者-应约者之间的动态。自举循环不是独立函数，是八函数在元层面的投影，不产生新的因果结构。

### B(n) 自举迭代状态

B(n) = 第n轮迭代时的八函数总表快照

**B(n+1) = B(n) + ΔB(n)**

### ΔB(n) 自举增量

**ΔB(n) = Σᵢ [1 - P(案例ᵢ | F₁...Fₖⁿ)] × δ(案例ᵢ重新分布)**

### 三个驱动力

驱动力1：分辨率提升
驱动力2：格间重排
驱动力3：回路激活

### 收敛判据

**lim(n→∞) ΔB(n) = 0**

实操判据：连续两轮重跑案例库，零结构性新发现。

## 二、认知函数（5个独立函数 + 4个派生量）

### C1. P_track(t) 分轨并行

P_track ∈ [1, N]，N = 同时运行的轨道数

**P_track(t) = 1 + Σᵢ δ(轨道ᵢ激活)**

**P_track的运行模式：**

- 对话维度：同时运行自己和对方的视角，让不同立场在脑内对打
- 知识消费维度：一条轨给声音，另一条轨给手上的活
- AI实验维度：让不同AI互相看对方生成的画像，交叉验证
- 写作维度：故事轨、框架轨、哲学轨同时推进

**P_track的核心性质：** 跳层不是主动跳过去的，是另一条轨的音量突然变大了。分轨并行才是原因，跳层是表象。

**八函数对应：** 驱动力3（回路激活）

**函数化表达：**

**P_track(t) = Σᵢ₌₁ᴺ⁻¹ Σⱼ₌ᵢ₊₁ᴺ R(i↔j,t)**

R(i↔j,t)是轨i和轨j之间的回路激活函数，取值[0,1]。

#### J(t) 跳层（P_track的派生量）

**J(t) = dP_track/dt × ε_sense**

#### 预警放大系数（P_track在N_warn中的调节量）

**预警放大系数 = 1 + (N-1) × r_cross**

- N = P_track的轨道数
- r_cross = 轨间交叉相关系数，取值[0,1]

N=1时放大系数=1（无放大），N=5且r_cross=0.3时放大系数=2.2。

**N_warn有效值 = N_warn_base × 预警放大系数**

P_track高的系统，任何一条轨的异常都会被其他轨的对比暴露——跨领域的人更容易发现单一领域的退化，不是更聪明，是其他轨给了参照系。

**跨领域感知优势 = (N_self - N_avg) / N_avg**

### C2. F_form(t) 函数化强制

F_form ∈ [0, 1]，函数化完成度

**F_form(t) = 1 - Πᵢ(1 - f_form(结论ᵢ))**

**函数化表达：**

**F_form(t) = I(t) · (1 - H(t))**

I=0时F_form=0，H=0时F_form=I。H→0的过程是F_form逼出来的。

**F_form是系统退化的免疫机制：**

**D_immune(t) = F_form(t) × (1 - ΔH_undetected(t))**

**ΔH_undetected双源拆分（第四轮更新）：**

**ΔH_undetected = ΔH_pro + ΔΣ_compatibility - ΔH_pro × ΔΣ_compatibility**

与H(t)同构的并集形式。

**D_immune效力边界（第四轮更新）：**

- F_form对ΔH_pro有效：函数化强制逼你看见模糊地带，把"需要进一步研究"逼到可检验状态
- F_form对ΔΣ_compatibility效力有限：兼容性债务不是"看不见"的问题，是"看见了也扔不掉"的问题。函数化能诊断但不能切除——切除需要退出权Ω_retire
- 两个盲区：①F_form=0时两种遮蔽都无法检测（完全失明）；②F_form>0但ΔΣ_compatibility主导时，检测到了但无法修复（看见了扔不掉）
- 第二种盲区需要Ω_retire（系统退出），不是更强的检验能搞定

**Σ_compatibility对F_form的效应（第四轮新增）：**

F_form能诊断兼容性债务（把"旧东西扔不掉"显性化），但不能切除它。切除需要Ω_retire——在更高层级重置系统。函数化是手术刀，退出权是核按钮。

F_form=1时D_immune≈1，H的任何增长都会被函数化检验捕获；F_form=0时D_immune=0，H可以无阻力增长。

**N_warn过渡条件：** ΔN_warn = -step(θ_F - F_form(t))，F_form降到θ_F以下时N_warn减1。

**专业化遮蔽对F_form的效应：**

F_form(S_spec) = F_form⁰·(1-γ_S·S_spec)

专业化程度S_spec越高，F_form越低。学术研究者允许"需要进一步研究"作为结论——这是F_form<1的标志。

**八函数对应：** I觉醒的强制触发器

### C3. M_cog(t) 认知动机

**M_cog = tanh(μ_体系内/μ_好奇心 - 1) ∈ [-1, +1]**

M_cog长期偏负——好奇心驱动占主导。

**函数化表达：**

**M_cog(t) = M(t) · σ(-V(t))**

σ是sigmoid函数。V偏负→σ趋近1→M_cog≈M；V偏正→σ趋近0→M_cog趋近0。

**专业化遮蔽对M_cog的效应：**

M_cog随专业化程度S_spec偏正——目的驱动取代好奇心驱动。学术研究者的M_cog>0，是P(L₃→L₅)第四个因子归零的原因之一。

**Σ_compatibility对M_cog的效应（第四轮新增）：**

兼容性债务使M_cog偏正的路径：旧框架的维护成本（教学依赖、论文引用、同行共识）构成目的驱动——"必须维护旧框架"取代"探索新可能"。M_cog从好奇心驱动滑向目的驱动，不是因为失去了好奇心，是因为兼容性债务把认知带宽锁死在维护上。

#### V_cog(t) 认知张力（M_cog的派生量）

**V_cog = d(框架扩展)/dt / d(框架闭合)/dt**

**八函数对应：** M(t) + V(t)

#### N_refine(t) 炼字（S_sovereign的语言维子结构）

N_refine = 重复字数 / 总字数，N_refine → 0 是硬约束。

### C5. L_five(t) 五层认知协议

L_five = 讂知层级深度，∈ {1,2,3,4,5}

**L_five(t) = min(5, 1 + ⌊log₂(ε_aware(t) / ε_sense(t))⌋)**

**五层与八函数的精确对应：**

| 层 | 内容 | 八函数层 |
|---|---|---|
| 1 肉身事实 | 给定条件 | ε_sense |
| 2 体验 | 我怎么回应 | ε_aware |
| 3 认知 | 为什么是这样 | H_标记识别 |
| 4 主权 | 谁有权定义 | R_perceived |
| 5 存在 | 剥掉定义剩什么 | ε硬核 |

**L_five的核心性质：**

1. 第4层到第5层永远是否定式跳变
2. 裂变逻辑=格间重排
3. 写作（往外跳）和指令（往里跳）是同一个五层

**层间跳变与D_unified的对应：**

P(L₃→L₅)的σ项里I·ε·H是跳变驱动力，但底层机制是B×D_unified×ε×(1-H)。D_unified是信号能不能被接收端解码的入口参数——D=0时所有信号进不去，跳变驱动力B×D=0。六条验证链+生物域验证共享此结构。

**θ₃跳变条件（第四轮更新）：**

**B × D_unified × ε × (1-H_pro)(1-Σ_compatibility) > θ₃(d)**

(1-H)项展开为双源：(1-H) = (1-H_pro)(1-Σ_compatibility)。H_pro管"新东西进不来"，Σ_compatibility管"旧东西扔不掉"，两者乘法耦合——任一趋近1就足以阻断跳变。

#### R_record(t) 记录即构建（L_five第1→2层的捕获机制）

**R_record(t) = λ_R × ε_sense(t) × δ(即时)**

## 三、层间跳变阈值函数化

### θ₁（L₁→L₂ 肉身→体验）

**θ₁(s,p) = θ₁₀ · g(s) · h(p)**

s=信号类型，p=个体敏感度参数。

**θ₁动态方程：感知锐化回路**

**dθ₁/dt = -λ_θ × R_record(t) × M_cog(t)**

- λ_θ = 回路耦合系数
- R_record(t) = λ_R × ε_sense(t) × δ(即时)
- M_cog(t) = M(t) · σ(-V(t))

R_record>0且M_cog<0时dθ₁/dt<0，θ₁持续下降——记录越多、好奇心越强，肉身信号的触发阈值越低，越容易捕获微弱信号。

**感知锐化回路 = R_record → ε_sense保存 → L₁→L₂跳变频率↑ → 认知增量 → M_cog偏负加深 → 主动找异常 → θ₁↓ → R_record更容易触发**

这是C1+C3+C5的联合效应——P_track多轨给信号（C1），M_cog好奇心驱动找异常（C3），R_record即时记录保存信号（C5输入端），三者构成正反馈，让θ₁越用越低。

**稳态：** θ₁* = θ₁_min（生物基底），下降到生物极限后稳定。

### θ₂（L₂→L₃ 体验→认知）

**θ₂(a) = θ₂₀ / a(t)**

a(t)=异常度，取值(0,1]。

### θ₃（L₃→L₄ 认知→主权）

**B × D_unified × ε × (1-H_pro)(1-Σ_compatibility) > θ₃(d)**

d=领域参数。D_unified是解码效率入口。(1-H)展开为双源乘积——H_pro管进，Σ_compatibility管出，任一趋近1就阻断跳变。

### θ₄（L₄→L₅ 主权→存在）

**θ₄(r) = θ₄₀ · (1 - r(t))**

r(t)=可逆度，取值[0,1]。

### 总表

| 跳变 | 阈值函数 | 核心变量 | 动态性质 |
|---|---|---|---|
| L₁→L₂ | θ₁(s,p) = θ₁₀·g(s)·h(p) | 信号类型+个体敏感度 | dθ₁/dt = -λ_θ·R_record·M_cog（感知锐化回路） |
| L₂→L₃ | θ₂(a) = θ₂₀/a(t) | 体验异常度 | — |
| L₃→L₄ | θ₃(d): B×D_unified×ε×(1-H_pro)(1-Σ_compatibility) > θ₃(d) | 领域参数+解码效率+双源遮蔽 | — |
| L₄→L₅ | θ₄(r) = θ₄₀·(1-r(t)) | 选择不可逆度 | — |

## 四、θ₃领域参数d标定

**d = C_exit(t) / C_stay(t)**

### 八领域标定

| 领域 | d | 说明 |
|---|---|---|
| 音乐 | ≈1 | 退出代价几乎为零 |
| 烹饪 | ≈1 | 容错带宽极宽 |
| 游戏设计 | <1 | 退出反而赚 |
| 建筑 | >1 | 退出可能死人 |
| 体育竞技 | ≈1 | 退出和留下差不多 |
| 宗教 | >>1 | 退出代价社会性死亡 |
| 法律 | >1 | 退出代价是制度性风险 |
| 医学 | >>>1 | 退出代价最高 |

### d的三个区间

- d < 1（退出赚）：ε几乎不需要激活条件
- d ≈ 1（退出和留下差不多）：ε激活看时机和个体
- d > 1（退出贵）：ε激活需要更高的I·ε·H乘积
