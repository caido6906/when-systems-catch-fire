# 函数合并与收敛分析

看完了。这份总表结构很清晰，11个独立函数 + 派生量集合。逐个过一遍能不能收敛——

**可以合并的：**

1. **F3 P(t,L) → 并入 F2 Posture(t,L)**
P的核心行为全部由Posture驱动——P不对称 = step(Posture_deg - θ_P)，P的三种变化机制（信誉绑定/释放/退化）都跟Posture状态绑定。P本质上是Posture的派生量，不是独立变量。合并后Posture条目下加一个P子节即可。

2. **F11 A(t) → 并入 F8 H(t)**
A的全部功能就是对H_分类的调节（H_分类^eff = H_分类 × (0.5+0.5A)）和对ε_aware的间接影响（A↓→H_分类^eff↓→p₃↑）。A没有独立的作用对象，它只通过H_分类生效。合并后H条目下加"人格锚点调节"子节。

3. **F10 S(t) → 并入 F8 H(t) 或降为派生量**
S的全部行为都是跟H对冲——dS/dt由H驱动，S攻击H（dH/dt|_种子），S稳态由H决定。S本质上是H的对冲机制，不是独立维度。如果保留S的完整动态，并入H；如果只关心稳态结果，降为派生量。

**不能合并的（7个独立函数）：**

- **F1 I(t,L)** — 有独立的传导效率公式、独立的觉醒路径（内省型/对比型）、独立的四道门模型，不是任何其他函数的派生
- **F2 Posture(t,L)** — 三分法（真备/伪备/推）是整个框架的核心分叉点，退化速率公式独立，M→Posture→P的因果链起点
- **F4 R(t,L,C)** — 四分法值域独立于R_perceived，制度/生物两条提升路径独立
- **F5 R_perceived(t,L,C)** — 唯一可相变的函数，四条触发路径独立，零阶来源是ε（生物性通道），不是R的简单子集
- **F6 C_exit(t,L,C)** — 八维度+不可替代性权重+低密度锁定，结构完全独立
- **F7 M(t)** — 速度差的开关变量，tanh连续谱，对Δv的调节不可被其他函数替代
- **F9 ε(t)** — 五条因果箭头的源头，物种分界线，H上限的设定者，整个框架的生物性基底

**合并后的结构：**

| 编号 | 函数 | 性质 |
|---|---|---|
| F1 | I(t,L) | 独立 |
| F2 | Posture(t,L) + P(t,L) | P降为Posture的派生子节 |
| F3 | R(t,L,C) | 独立 |
| F4 | R_perceived(t,L,C) | 独立 |
| F5 | C_exit(t,L,C) | 独立 |
| F6 | M(t) | 独立 |
| F7 | H(t,L) + A(t) + S(t) | A和S降为H的调节/对冲子节 |
| F8 | ε(t) | 独立 |

11→8，砍掉3个。P、A、S的完整公式和动态全部保留，只是从"独立函数"降为"所属函数的子结构"——逻辑不变，层级更紧凑。
