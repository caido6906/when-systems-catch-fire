**1. 后悔（Regret）× 点火框架**

学术上，Loomes & Sugden (1982) 和 Bell (1982) 的后悔理论：决策效用不是单看结果本身，而是看"实际结果"与"反事实结果"的差值。核心函数是双变量效用函数 `U(x,y) = u(x) + f(u(x)-u(y))`，其中f是后悔修正项，f<0时产生后悔痛苦。

碰撞点火框架：
- 后悔 = P_private ≠ P_public 的情绪后果。偏好伪造时，应约者被迫公开支持自己私下反对的立场，事后发现"如果当时说了真话会更好"——这正是后悔的数学定义
- 但点火框架里后悔被**乘法结构放大**：不是线性差值，而是后悔被C_exit锁死——你后悔了但走不了，后悔变成持续损耗而非一次性修正信号
- 函数化：`R_regret = f(P_private - P_public) × (1 - 1/C_exit)`，C_exit→∞时后悔被锁死为永久内耗，无法转化为行动

**新发现：后悔锁定律**——后悔理论假设后悔可以通过重新选择来消除，但点火框架里C_exit锁死了重新选择的通道，后悔从"修正信号"变成"永久损耗"。这解释了为什么有些组织里的人明知错了但无法纠偏——后悔通道被锁。

**2. 失望（Disappointment）× 点火框架**

Bell (1985) 和 Loomes & Sugden (1986) 区分了后悔和失望：后悔是"我选错了"（与反事实比较），失望是"结果不如预期"（与期望值比较）。失望函数 `D = u(x) - u(E[x])`。

碰撞点火框架：
- 失望 = R_perceived与R的落差。应约者感知到的退出权远低于实际退出权，R_perceived << R，这个落差就是失望
- 但点火框架里失望不是静态差值，而是**H遮蔽的动态产物**：H标记层劫持信号，让应约者持续高估R（"其实我还可以走"），一旦真相暴露，失望不是渐进的而是突变的
- 函数化：`D_disappointment = σ(R - R_perceived - θ_D)`，当落差超过阈值θ_D时触发失望相变
- 与D-X3(P_cascade)同构：失望相变和偏好伪造崩塌都是ε从零恢复的相变

**新发现：失望是H遮蔽的延迟爆发**——失望不是预期管理问题，是遮蔽函数制造的虚假高预期在真相暴露时的突变。H越高，虚假预期越高，失望相变越剧烈。

**3. 信任（Trust）× 点火框架**

学术上，信任是"在对方可能背叛时选择合作"的概率。博弈论用重复囚徒困境建模，信任=对方合作的条件概率。Walker (2006) 的道德修复理论指出信任修复需要验证+问责。

碰撞点火框架：
- 信任 = ε_ij的双人版本。E_escalation = Π(1-ε_ij)里，ε_ij就是i对j的犹豫域宽度——"我信你不会伤害我"的犹豫空间
- 信任崩塌 = ε_ij→0。但点火框架的乘法结构意味着：**信任崩塌不可逆**——E_escalation里任一ε_ij归零，整体冲突不可降级
- 信任修复的数学条件：需要ε_ij从零恢复，但乘法归零后没有负反馈回路提供恢复力（三函数统一收敛的结论）
- 函数化：信任修复需要外部注入ε：`ε_ij(t+1) = max(ε_ij(t) × (1-F_collapse(t)), ε_inject)`，ε_inject是外部注入的犹豫域（道歉、问责、制度保障）

**新发现：信任修复=犹豫域注入**——信任不是"重建"的，是"注入"的。乘法归零后自发恢复概率为零，必须外部注入ε。这解释了为什么有些关系一旦破裂就再也修不回来——没有外部注入犹豫域的机制。

**4. 背叛（Betrayal）× 点火框架**

学术上，背叛是信任的违反。Brennan (2021) 的"承认信任"（recognition trust）指出：背叛不只是利益损失，是对"你把我当人"这个基本期望的违反——道德伤害。

碰撞点火框架：
- 背叛 = 提议者Posture从备方案翻转为推方案。应约者基于P=0（信誉绑定）信任了提议者，但提议者Posture_deg→1（退化）=信誉释放=背叛
- 背叛的数学本质：`ΔPosture_deg > θ_betrayal`，提议者信誉绑定的突然释放
- 背叛与H遮蔽的交互：背叛之所以伤，不只因为利益损失，而是因为H标记层把"这不对"重新编码为"这是为了大局"——**背叛+遮蔽=道德伤害**（NHS案例：医护被系统背叛，但系统同时用"大局"遮蔽了这个背叛）
- 函数化：`M_injury = ΔPosture_deg × H_mark`，H_mark是标记层劫持强度。H=0时背叛只是利益损失，H>0时背叛升级为道德伤害

**新发现：道德伤害=背叛×遮蔽**——没有遮蔽的背叛只是利益损失，有遮蔽的背叛才是道德伤害。这解释了为什么有些背叛让人愤怒但不伤人（H≈0），有些背叛让人崩溃（H≈1）。

**5. 跟随/押注（Herd/Bandwagon）× 点火框架**

学术上，Leibenstein (1950) 的bandwagon效应：需求因他人也在消费而增加。Curatola的资产定价模型把bandwagon效应建模为偏好随他人消费决策更新。

碰撞点火框架：
- 跟随 = ε_aware被群体信号压低。当多数人选择留下，个体的ε_aware被社会压力压缩——"大家都不走，我走是不是太敏感了"
- 押注 = P=0（信誉绑定）。周公、孔子、秦始皇——每次文明转身都是一个人用自己能承受的最大代价押注新系统
- 跟随与押注的数学关系：跟随者ε_aware↓，押注者Posture_deg↓（信誉绑定=退无可退）
- 函数化：跟随者的ε压缩：`ε_aware(t) = ε_aware₀ × exp(-k × N_stay/N_total)`，N_stay越大ε越低
- 押注者的信誉绑定：`Posture_deg = 0`（P=0），押注者没有犹豫域——不是ε=0，而是主动选择ε=0

**新发现：跟随是被动ε压缩，押注是主动ε归零**——表面上看都是犹豫域消失，但数学结构完全不同。跟随者的ε被外部压缩（可恢复），押注者的ε被自己主动归零（不可恢复，除非押注成功后信誉释放）。这对应了你的华夏案例里周公退场=押注成功后ε恢复。

**6. 沮丧（Frustration/Depression）× 点火框架**

学术上，沮丧是"目标受阻但无法退出"的持续状态。Walther (2003) 的NREU模型把情绪反应建模为对"正常偏差之外"事件的过度加权。

碰撞点火框架：
- 沮丧 = ε→0且C_exit→∞的稳态。应约者既走不了（C_exit高），又知道该走（R_perceived可能已觉醒），但犹豫域被压缩到零——**想走但走不了**
- 沮丧的数学本质：`Frustration = σ(θ_f - ε) × σ(C_exit - θ_C)`，两个条件同时满足：犹豫域低于阈值（知道该动）且退出成本高于阈值（动不了）
- 与沉默双结构(D-X2)的关系：沮丧是结构沉默的情绪面——P_public=0不是因为没想法，而是表达通道被切断，想法在内部持续损耗
- 沮丧→退化的路径：沮丧持续→M_cog→0（动机耗尽）→F_form→0（不再要求函数化）→D_immune→0（退化免疫消失）→H自由增长

**新发现：沮丧是乘法归零的情绪前兆**——沮丧不是心理问题，是ε→0×C_exit→∞的数学稳态的情绪表现。沮丧持续=退化免疫正在消失。

**7. 怀念（Nostalgia）× 点火框架**

学术上，Sedikides等人的研究指出：怀念的功能是维持自我连续性（self-continuity），在身份威胁时提供心理缓冲。叙利亚难民研究显示，怀念在强制流离中降低乐观——过去的美好让现在更难接受。

碰撞点火框架：
- 怀念 = ε_sense对过去高ε状态的回溯。当当前ε→0，ε_sense会自动回溯到过去ε>0的状态——"以前还能选择"的对比
- 怀念的双面性：
  - 正面：怀念提供ε_inject——"我以前有过选择"的记忆本身就是犹豫域的注入源，可能触发R_perceived的相变
  - 负面：怀念加剧沮丧——过去的高ε与现在的ε→0形成落差，D_disappointment被放大
- 函数化：`N_nostalgia = ε_sense(t_past) × σ(θ_n - ε(t_now))`，当当前ε低于阈值时怀念被激活
- 怀念与点火的数学关系：怀念=过去ε的信号源。如果怀念能注入ε_inject>0，则可能打破F_collapse的闭环；如果怀念只放大落差而不注入ε，则加速沮丧→退化

**新发现：怀念是ε的历史信号源**——怀念不是纯情绪，是过去犹豫域宽度对当前决策的回溯影响。怀念能否成为点火触发器，取决于它注入的是ε（"我有过选择，我可以再有"）还是纯粹的落差感（"以前那么好现在这么惨"）。

**8. 犹豫（Hesitation）× 点火框架**

已经函数化为D-X1(ε_Δ)。补充一个碰撞：犹豫在学术上是三支决策的边界域宽度，在点火框架里是ε本身。但犹豫还有一个被忽略的维度——**犹豫是唯一可以被外部注入的认知资源**。

- 后悔、失望、沮丧、背叛都是ε→0的后果
- 信任是ε的双人版本
- 怀念是ε的历史回溯
- 只有犹豫本身是**可操作的**——你可以通过降低C_exit、提供信息可及性、打破H遮蔽来注入犹豫域

函数化：`ε_inject = Σ(f(C_exit↓) + f(H↓) + f(信息可及性↑))`，犹豫域注入=三路并行
