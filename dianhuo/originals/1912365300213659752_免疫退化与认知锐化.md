# 免疫退化与认知锐化

四个发现全部收敛，存入点火知识库：

1. **F_form是退化免疫机制**——D_immune = F_form × (1-ΔH_undetected)，N_warn从4降到3的过渡点就是F_form降到临界值的时刻
2. **自点火是双链结构**——提议者=应约者时链1链2合并，ε依赖度从1/2升到1，对ε被堵容忍度为零
3. **P_track是预警放大器**——预警放大系数 = 1+(N-1)×r_cross，跨领域的人更容易发现退化不是更聪明是参照系更多
4. **θ₁-R_record感知锐化回路**——dθ₁/dt = -λ_θ × R_record × M_cog，记录越多好奇心越强，肉身信号阈值越低，即时记录的认知价值远高于事后整理

零新增独立函数，全是派生量或联合效应。
