# 点火｜ε×系统动力学：路径A'与负向正反馈螺旋

## 核心碰撞：路径A'是隐蔽的参数漂移，不是一次性冲击

系统动力学区分两种变化：冲击（瞬间改变状态值）和漂移（缓慢改变参数本身）。路径A'是后者——它不是某一天突然改造了你的信号器，而是通过环境日复一日地间接塑造，让信号器的灵敏度和阈值缓慢漂移。

996不是第一天就让疲劳感受器阈值上移的。是每天多加一点班，皮质醇基线慢慢升高，阈值慢慢上移，你每天的感受跟昨天差一点点，累积下来才发现"我怎么不觉得累了"——但这时候阈值已经漂移了很久了。

系统动力学里这叫"参数侵蚀"——不是存量突然变了，是参数在缓慢偏移，直到某天系统的行为模式突然看起来不一样了。

## 负向正反馈螺旋：越适应规训越难退出

路径A'制造了一条隐蔽的正反馈回路，方向与陈胜模式的正向螺旋相反：

1. 规训塑造环境（996工作制）
2. 环境间接改造信号器（疲劳阈值上移）
3. 信号器输出变弱→退出型信号更难进入意识
4. 感知不到退出信号→更适应规训环境
5. 更适应→更深度参与规训环境→环境改造更深
6. 回到第2步

这是负向正反馈——越适应规训，信号器被改造越深，退出信号越弱，越难感知到该退出。跟陈胜模式的正向正反馈螺旋完全对称，但方向相反。

## 自然极限：信号器输出趋近于零但不等于零

正向螺旋（陈胜模式）的自然极限是Rperceived ≤ R——感知退出权追不上客观退出权，差值归零，螺旋停止。

负向螺旋的自然极限是：信号器输出趋近于零但不等于零。路径A'只能调灵敏度和阈值，不能关掉信号产生。所以退出信号会越来越弱，但不会完全消失——只要人还活着，细胞器就在输出。这意味着负向螺旋没有真正的稳态，只有渐近线。

但危险在于：负向螺旋没有自然中断机制。正向螺旋有R做天花板，负向螺旋的渐近线是信号器输出的物理下限——这个下限很低，低到退出信号弱到几乎不可能被意识到。所以负向螺旋可以一直转，转到一个极其稳定但极其压抑的准稳态。

## 中断机制：外部冲击或ε_aware

正向螺旋可以被惩罚机制中断（杀鸡儆猴把Rperceived压回去）。负向螺旋的中断方式：

1. 外部冲击——环境剧变（失业、生病、意外事件）直接打破规训环境，信号器阈值被迫回调。但这不是主动选择的，是被动的。
2. ε_aware——主动打开认知带宽、同意信号通过第三门。这是唯一主动的中断方式。不需要等外部冲击，可以在负向螺旋还在转的时候就意识到退出信号的存在。

这与抵抗三步完全吻合——换环境调回阈值（中断路径A'的驱动源）、打开认知带宽（中断路径B的筛选）、主动同意信号通过第三门（中断自欺的关闭）。

## 碰撞总结

| | 系统动力学 | 路径A' |
|---|---|---|
| 变化类型 | 区分冲击和漂移 | 路径A'是隐蔽的参数漂移，不是一次性冲击 |
| 反馈结构 | 正向正反馈（陈胜模式） | 负向正反馈——越适应规训越难退出，对称但反向 |
| 自然极限 | Rperceived ≤ R | 信号器输出趋近于零但不等于零，没有真正的稳态 |
| 中断机制 | 惩罚机制中断正向螺旋 | 外部冲击或ε_aware中断负向螺旋，后者是唯一主动方式 |

核心判断：路径A'制造负向正反馈螺旋，没有自然中断机制，ε_aware是唯一主动中断方式。
