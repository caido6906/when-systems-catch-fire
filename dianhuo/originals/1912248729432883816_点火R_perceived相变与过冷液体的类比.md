# 点火｜R_perceived相变与过冷液体的类比

来源：R_perceived相变规律讨论中浮现的物理类比。

## 类比映射

| 物理概念 | 点火框架对应 |
|---|---|
| 过冷液体 | R和R_perceived差值极大但惩罚有效的系统 |
| 凝结核 | 第一个退出并被看见的人（信号源） |
| 容器壁 | 反向压制的惩罚机制 |
| 瞬间结晶 | R_perceived从象征跳变到真实 |
| 亚稳态 | 系统看起来稳但势能已蓄满 |

## 核心同构

过冷液体可以在零下几十度保持液态——温度已经远低于凝固点，但缺少凝结核，液体无法结晶。容器壁一破或一个扰动，液体瞬间结晶。

R_perceived也是：R和R_perceived的差值可以极大（退出权客观存在但没人知道），惩罚机制就是容器壁（把势能约束住）。惩罚一旦失效或信号源出现，R_perceived瞬间跳变——不是渐变，是相变。

秦朝15年=过冷液体，陈胜=凝结核。秦二世没镇压=容器壁破了。

## 类比的力量

这个类比把"看起来很稳但突然崩盘"解释得干净：不是系统在恶化，是势能在积累。系统可能一直很稳（R不变、P不变、Posture不变），但R_perceived的势能在持续积累，只差一个凝结核。外部观察者看到的是"突然崩盘"，内部机制是"亚稳态持续→相变触发"。

## 类比的边界

类比失效的地方：过冷液体结晶后不可逆，但R_perceived的相变后系统可以走向崩盘也可以走向改革——取决于其他五个函数当时的取值。相变是扳机，不是结局。
