# 点火框架·全案例校准总表（71案例·退出权四分法版·2026.06.03）

框架核心：系统 = 协作系统（搭房）+ 认同系统（点火）
角色：提议者 / 应约者
退出权四分法：真实 / 事实 / 心理 / 象征（选主导型，不标混合）
五步流程：①旧系统失灵 → ②提议者出现 → ③应约者应约 → ④点火跑通 → ⑤稳定期足够

## 六、退出权四分法分布统计

|退出权类型|案例数|典型特征|
|---|---|---|
|真实|34|框架精确成立，认同可验证，退化信号最清晰（人才流失/用脚投票）|
|事实|14|温室效应，法理有退出权但事实无，认同靠利益绑定未经验证，退出权出现时冲击最大|
|心理|1|系统空转，人还在但认同已撤回，最隐蔽的退化信号|
|象征|9|强制力的优雅版本，退出权被仪式化，框架衰减最大|
|边际（不适用）|6|框架不覆盖|

## 七、核心发现

1. **退出权类型决定框架精度**：真实退出权→框架精确成立；事实退出权→框架成立但验证是假的（温室）；心理退出权→框架衰减（空转）；象征退出权→框架衰减最大（与无退出权同构）
2. **事实退出权是最危险的盲区**：14个案例中，应约者以为自己有选择但实际走不了。认同从未被验证但应约者不知道——一旦退出权暴露，冲击最大（新中国、清朝、凯末尔改革、海尔·塞拉西）
3. **象征退出权是暴力系统的合法化外衣**：9个案例中，系统保留了退出的形式但把代价设计到不可承受。秦统一、元朝、计划生育、安然——都是"你可以走但走了等于死"
4. **心理退出权是文明级崩盘的前兆**：苏联后期、恒大——人还在但心已走，系统表面正常实际空转，没有崩盘的声响只有缓慢的空转
5. **公司级几乎全部是真实退出权**：13个案例中10个真实退出权，因为劳动力市场提供了退出通道。公司级点火更快见效但更脆——应约者随时可以走
