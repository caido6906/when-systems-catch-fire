# 点火案例总表（事件框架版·2026.06.06）

框架核心：事件 = 提议者出方案 + 应约者接或不接。点火框架拆的是事件走向。
三个变量：提议者意识（有/无）× 提议者退出权（有/无）× 应约者退出权（有/无）= 8格
五步走向：旧事件失灵 → 提议者出现 → 应约者应约 → 点火跑通 → 稳定期
五种走向：点对了 / 点不足 / 点歪了 / 点反了 / 负时间差
退出权四层：真实 / 事实 / 心理 / 象征
提议者姿态：推方案（僭越）vs 备方案（管家）
