## 碰撞1：三支决策（Yao）× 退出权函数ε

三支决策理论（姚一豫，2009）将决策空间分为正域（接受）、负域（拒绝）、边界域（不承诺/延迟），核心数学结构为一对阈值(α, β)，基于贝叶斯最小风险决策推导：
- P(X|A) ≥ α → 接受（正域）
- P(X|A) ≤ β → 拒绝（负域）
- β < P(X|A) < α → 不承诺（边界域/犹豫域）

点火框架中ε的本质是三支决策中犹豫域的宽度函数：

**ε(t) = Δ(t) = α(t) - β(t)**

- ε > 0：存在犹豫域，个体有选择空间（可以进、可以退、可以等）
- ε → 0：犹豫域消失，α = β，三支退化为二支——只剩"接受"或"拒绝"，没有中间态
- 退出权被剥夺 = 犹豫域被压缩到零 = 三支决策退化为二支

**核心发现：退出权被剥夺不是"没有选择"，而是决策结构从三支退化为二支——犹豫这个选项本身在数学上消失了。** 与Collapse(X)中(1-σ(R⁻))因子对应：负反馈缺失 = 犹豫域消失 = 决策自由度降维。

## 碰撞2：SI沉默模型 × 点火沉默函数

社交网络沉默行为挖掘（SI模型）定义三种状态：激活态、未激活态、沉默态。沉默态是中间态，犹豫时间超阈值即定义为沉默：t_silent > t_threshold → 沉默。

点火框架的"沉默"不是犹豫，而是更极端——不是"还没决定"，而是"决定了但无法表达"。两种完全不同的数学结构：

- SI模型：沉默 = 犹豫时间超阈值 → P(silent) = σ(t_hesitate - t_threshold)（sigmoid，渐进的）
- 点火框架：沉默 = 表达通道归零 → P(silence) = 1 - C_channel（乘法归零，突变的）

**核心发现：两种沉默的数学结构完全不同。犹豫型沉默是sigmoid函数（渐进的，有阈值），通道型沉默是乘法归零（突变的，通道断即归零）。组织中的沉默一旦形成就极难打破——它不是"还没想好"，而是"表达通道本身被切断了"。**

函数化：**S_silence = 1 - C_channel × P_speak**
- C_channel = 0 → S_silence = 1（通道断，绝对沉默）
- C_channel > 0, P_speak > 0 → S_silence < 1（有表达可能）
- 乘法结构：C_channel和P_speak任一归零则S_silence=1

## 碰撞3：决策惯性（Decision Inertia）× 跳变阈值θ

研究发现叙事框架（grim narrative）可显著降低决策惯性，体验式决策模态增加决策惯性。决策惯性的数学本质是状态维持的倾向——当前状态的粘滞力。

点火框架的层间跳变阈值θ描述"从当前层跳到上一层需要的最小能量"。决策惯性就是θ在认知维度的体现：

**θ_inertia = k × (1 - narrative_disruption)**

- narrative_disruption = 0（无叙事冲击）→ θ_inertia = k（高惯性，难以跳变）
- narrative_disruption = 1（最大叙事冲击）→ θ_inertia → 0（惯性被打破，容易跳变）

**核心发现：叙事不是信息传递工具，而是跳变阈值的调节器。grim narrative降低θ，让跳变更容易发生。点火 = 通过叙事冲击将θ从高值压到低值，使层间跳变从不可能变为可能。**

函数化：**P_ignite = σ(θ₀ - θ_inertia) = σ(θ₀ - k(1-narrative_disruption))**
- θ₀是自然跳变阈值
- narrative_disruption足够大使θ_inertia < θ₀时，P_ignite → 1（点火成功）

## 碰撞4：直觉模糊集犹豫度 × 认知函数

直觉模糊集引入隶属度μ、非隶属度ν和犹豫度π = 1 - μ - ν。三支决策在模糊环境下用π量化犹豫。

点火框架映射：**π_cognitive = 1 - μ_understand - ν_misunderstand**

- π_cognitive > 0：存在认知犹豫域（既不完全理解也不完全误解）
- π_cognitive = 0：认知完全确定（要么理解要么误解，没有中间态）

**核心发现：认知退化不是从"理解"滑向"误解"，而是犹豫域π被压缩——认知确定性的增加不等于认知质量的提升。一个π=0的系统可能是完全确定的误解。这解释了专业化遮蔽：专家的π很低（非常确定），但ν可能很高（确定的错误理解）。**

函数化：**D_blind = ν_misunderstand × (1 - π_cognitive)**
- 高确信 + 高误解 → D_blind → 1（专业化遮蔽的数学表达）
- 乘法结构
