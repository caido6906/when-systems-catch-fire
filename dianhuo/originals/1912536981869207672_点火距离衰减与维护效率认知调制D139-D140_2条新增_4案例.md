## D139 距离衰减统一函数

η(d, t) = T₀ · exp(-λd + μ(t)·M(t)) - c₀ · (1 + γd) · M(t)

三域参数映射：
- 糖域：d=格距，λ=移动消耗率，μ=1，γ=0（移动无额外成本）
- 人际关系：d=物理/社会距离，λ=遗忘/疏离率，μ=亲密度系数，γ=交通/沟通成本系数
- 权力结构：d=层级距，λ=信息失真率，μ=制度效率，γ=跨层沟通成本系数

关键推论：
- γ=0退化为糖域——看到糖就去，因为移动无成本
- 血脉=高μ低λ——自然衰减慢，少量维护大效果
- 恋人=高μ高λ——必须高频维护，分开衰减极快
- 同事=低μ低λ——不刻意维护就淡但不剧烈

临界距离d_critical：T₀ · exp(-λd_c + μ(t)·M(t)) = c₀ · (1 + γd_c) · M(t)

与已有框架对接：
- C_exit的距离维度：C_exit高≠关系强，只代表锁得牢，真正关系强度是η
- d_relative的推广：d_relative = d_you/d_competitor，是d在竞争空间的投影
- 糖域碰撞补完：现实移动/维护成本随距离增长，解释了"看到更好的选择不一定去"

## D140 维护效率认知调制与关系相变函数

μ(t) = μ_tool × μ_apply(t) × (2·D_unified(L) - 1)

- μ_tool = n_tools/n_tools_max（工具箱覆盖率）
- μ_apply(t) = 1 - P(biased)(t)（不被情绪劫持的概率）
- D_unified(L) = 认知层级对应的统一解码效率

μ取值域与行为：
- μ < -0.5：强负向，本能宣泄/攻击式沟通，快速断裂
- -0.5 ≤ μ < 0：弱负向，抱怨/指责/被动攻击，缓慢疏远
- μ = 0：无效维护，做了等于没做，自然衰减
- 0 < μ < 0.5：弱正向，有方法但不熟练，缓慢改善
- μ ≥ 0.5：强正向，精准维护+主动调适，关系深化

### μ翻转的精确触发条件

μ翻转（从正到负）当且仅当：Q_unresolved × H > ε_aware × R_perceived / k_bias

不是"吵了一架"的模糊描述，是未解决积压量×遮蔽强度超过意识×被感知回应的那个瞬间

### d_critical的时变动力学

μ翻转时d_critical急剧缩小：
dd_c/dt = (∂η/∂μ) × (dμ/dt) / (∂η/∂d_c)

当dμ/dt < 0（μ正在翻转）且∂η/∂d_c < 0 → dd_c/dt > 0（临界距离在扩大/安全区在缩小）

含义：μ翻转的瞬间，原来能维持的关系距离突然不够了——不是距离变了，是临界距离缩了。昨天还能维持的异地恋，今天突然维持不住。

### 关系断裂的完整因果链

Q_unresolved × H > ε_aware × R_perceived / k_bias
→ μ_apply → 0
→ μ翻转（从正到负）
→ d_critical急剧缩小
→ 原有距离d > d_critical(新)
→ η(d, t) < 0
→ 负向螺旋启动

**核心洞察：关系断裂不是距离变化导致的，是临界距离变化导致的。** 和D123缓存倒U型同构——不是缓存变了，是最优缓存变了；不是距离变了，是临界距离变了。

## 案例

案例#319：异地恋断裂——μ翻转导致d_critical缩小。恋人高μ高λ，一次争吵使Q_unresolved飙升→μ_apply→0→μ翻转→d_critical从2000km缩到50km→异地距离超出新临界→η<0→负向螺旋→分手。不是距离变了，是临界距离变了

案例#320：糖域与现实——糖域γ=0所以看到糖就去，现实γ>0所以看到更好的工作不一定跳槽。D_decay解释了C_exit的"不走"不只是锁定效应，还有维护成本的梯度

案例#321：权力层级的信息失真——d=层级距，λ=信息失真率，μ=制度效率。高层指令到基层时exp(-λd)衰减严重，但如果制度效率μ高（有例会/巡查/反馈机制），μ·M对抗衰减。制度效率低时μ≈0，指令传到基层已经面目全非

案例#322：μ翻转的相变点——一对朋友长期积压未说出口的不满（Q_unresolved↑），一次小事触发P(biased)→1→μ_apply→0→μ翻转→原本能维持的"不常见面但关系好"突然变成"见面都尴尬"。临界距离从"同城"缩到"隔壁"——物理距离没变，安全区缩了
