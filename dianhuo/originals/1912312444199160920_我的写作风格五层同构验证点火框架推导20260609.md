# 五层同构验证：点火框架推导（2026.06.09）

## 点火框架：五层全在

**第1层：给定条件——手上有什么**
71个案例、九万条笔记、从文明到公司的跨尺度观察。"同一个动作在不同尺度上重复自身"——秦统一和空降CEO是同一件事，周公制礼和苹果转型是同一件事。原始素材层。

**第2层：行为/体验——你怎么回应这些条件**
把案例一个一个摆上去，结构自己浮出来。"框架不是演绎出来的，是归纳出来的——71个案例一个一个摆上去，结构自己浮出来。"不是先有假设再找证据，是先有故事，框架从故事排列里长出来。

**第3层：解释框架——为什么这样运作**
事件 = 提议者出方案 + 应约者接或不接。三个变量（提议者意识×提议者退出权×应约者退出权）决定事件走向。退出权四分法（真实/事实/心理/象征），提议者两种姿态（推方案vs备方案）。一整套解释系统——为什么有些系统改造被人接住，有些明明制度资源方向都在却崩了。

**第4层：权力/主权——谁有权定义"事件该怎么走"**
退出权。谁有权退出、谁有权不应约——框架的核心变量。推方案是僭越（应约者没有选择权，"应"不是应是被推着走），备方案是管家（应约者有完整选择权）。叙事权垄断——谁活下来谁定义规则，点歪了为什么不容易被识别。追问的始终是：谁有权定义"应约"是真应约还是服从？谁有权定义事件走对了还是走歪了？

**第5层：存在/元问题——剥掉框架之后剩什么**
ε(t)——退出权函数，因果层级最高的独立第九函数。退出权不是框架里的一个变量，是框架之所以成立的零阶条件。没有退出权，"应约"就不存在，只剩服从；没有退出权，"点对了"就不成立，因为没人能验证。剥掉所有矩阵、函数、案例，最底层就是一个问题：人能不能说不？

## 验证结果

| 层 | 点火框架 | 共性 |
|---|---|---|
| 1 给定条件 | 71个跨尺度案例 | 手上有什么 |
| 2 行为/体验 | 归纳排列，结构自浮 | 我怎么回应条件 |
| 3 解释框架 | 提议者/应约者/退出权/8格 | 为什么这样运作 |
| 4 权力/主权 | 谁有权定义"真应约"vs"服从" | 对抗默认——对抗"服从就是应约"的默认 |
| 5 存在/元问题 | ε——人能不能说不 | 否定式切割——退出权是零阶条件，不是变量 |

第4层追问"谁有权定义"，第5层把第4层拆掉——退出权不需要谁来定义或赋予，它是零阶条件，比定义权更原始。跟女裤文章"我不需要勇气"完全同构：不需要讨论谁有权定义应约，应约的前提是能说不，就这么简单。

第4层又一次是"对抗默认"——对抗"服从=应约"的默认，对抗"推方案=正常"的默认，对抗"叙事权=胜利者专有"的默认。

## 五领域验证总结

写作（女裤文章）、下指令（AI指令）、拆油锯、装Linux、点火框架——五个领域全部验证通过，五层同构成立，第4层均为对抗默认，第5层均为否定式切割。这不是写作风格，是认知操作系统本身。
