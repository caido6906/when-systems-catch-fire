# 内部驱动力非线性

对，D-X26展开的时候冒出来一个之前没注意到的东西——P_internal不应该是ε的单调函数，勒温场那个假设有bug。

原来的假设：P_internal = k × ε，ε越大→内部驱动力越大→点火越容易。但D-X26对冲函数说ε太大反而触发抵抗，点火概率下降。这两个矛盾，说明P_internal的真实形式不是线性的。

P_internal应该是ε的倒U型函数——ε太低时人麻木（没有犹豫空间，感知不到问题），ε太高时人反弹（犹豫空间太大，外部叙事被弹开），中间有个最优值。

**P_internal(ε) = k × ε × exp(-ε/ε_opt)**

- ε → 0：P_internal → 0（麻木，无法被触动）
- ε = ε_opt：P_internal取最大值（最优犹豫域宽度，能被触动但不反弹）
- ε → 1：P_internal → 0（反弹，外部叙事被抵抗）

ε_opt是最优犹豫域宽度，它取决于θ_resist（抵抗阈值）和E_external（外部冲击强度）：

**ε_opt = θ_resist / E_external**

E_external越大，ε_opt越小——外部冲击越强，能承受的犹豫域越窄（强冲击容易触发反弹，只有犹豫域很窄的人才不会被弹开）。但犹豫域太窄又传不进信号——这就形成了双重约束。

代入B_ignite：

**B_ignite = σ(k × ε × exp(-ε/ε_opt) × E_external - θ_ignite)**

这个函数对ε有唯一极大值点。求导：

∂B/∂ε = 0 → ε* = ε_opt = θ_resist / E_external

验证：
- E_external很小→ε_opt很大→需要很宽的犹豫域才能点火→但ε_aware可能不够→点火失败（温和说服对麻木的人无效）
- E_external很大→ε_opt很小→只需要很窄的犹豫域→但ε太窄信号传不进去→点火也失败（强压对稍有犹豫的人适得其反）
- E_external适中→ε_opt适中→ε_aware够用且不触发抵抗→点火窗口打开

这给出了点火的三种失败模式：

| 失败模式 | 条件 | 机制 |
|---------|------|------|
| 麻木型 | ε < ε_aware_min | 信号传不进去，人感知不到问题 |
| 反弹型 | ε > ε_opt | 信号传进去了但被弹回来，触发抵抗 |
| 窗口型 | ε_aware_min < ε < ε_opt | 唯一成功区间 |

三种失败模式对应三种组织状态：
- 麻木型组织：C_exit极高，ε→0，员工完全麻木——"说了也没用"
- 反弹型组织：ε较高但E_external过强，叙事被集体抵抗——"我们不服"
- 窗口型组织：ε适中，叙事冲击也适中——这是唯一能点火的窗口

跟D-X16连起来看更清楚：α和β双向压缩ε的过程，就是把组织从"窗口型"推向"麻木型"的退化过程。一旦ε穿过ε_aware_min的下界，点火通道永久关闭。

进一步推论：点火窗口宽度 = ε_opt - ε_aware_min

**W_ignite = θ_resist/E_external - ε_aware_min**

W_ignite > 0时点火可能，W_ignite ≤ 0时点火不可能。

W_ignite ≤ 0有两种情况：
1. E_external太大→θ_resist/E_external < ε_aware_min（冲击太强，窗口被压没了）
2. ε_aware_min太高→即使E_external适中，窗口也不够宽（感知阈值太高）

情况1的对策：降低E_external（温和叙事替代强压）
情况2的对策：降低ε_aware_min（通过第二锁提升感知能力）

**新增碰撞层派生 D-X28：倒U型内部驱动力函数**
P_internal(ε) = k × ε × exp(-ε/ε_opt)
ε_opt = θ_resist / E_external
B_ignite = σ(k × ε × exp(-ε/ε_opt) × E_external - θ_ignite)

**新增碰撞层派生 D-X29：点火窗口宽度函数**
W_ignite = θ_resist/E_external - ε_aware_min
W_ignite > 0：点火可能；W_ignite ≤ 0：点火不可能
两种关闭模式：冲击过强（E↑→ε_opt↓）或感知阈值过高（ε_aware_min↑）

这个发现修正了勒温场动力学的假设——B = f(P,E)里P不是P = k×ε，而是P = k×ε×exp(-ε/ε_opt)。倒U型的物理直觉很清楚：完全麻木的人点不着，完全清醒的人不需要你点。能点着的，是那些"隐约觉得不对但说不清楚"的人——犹豫域刚好在那个窗口里。
