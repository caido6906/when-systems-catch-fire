# 极端认知遮蔽机制

这个问题比前面几个都深——前面跑的是寄生体怎么传播，现在要跑的是**寄生体怎么把宿主改造成自己的复制机**。极端主义、信谣反疫苗、重复犯罪、职业依赖，表面上是四个不同的问题，但你的直觉是对的：它们都是认知遮蔽的不同形态。

---

**第一步：定义认知遮蔽**

认知遮蔽 H 是你八函数里已有的概念——它压制 I→R_perceived 的传导。现在把它具体化到这些场景：

$$H_{cognitive}(L, t) = 1 - \frac{\text{实际可接收的信号频谱}}{\text{客观存在的信号频谱}} = 1 - \frac{W(L, t)}{W_{max}}$$

H = 0 = 完全开放，H = 1 = 完全遮蔽。

**但这里有一个关键区别：** 普通的茧房是"被动收窄"（自增强回路把信道慢慢锁死），极端主义是"主动封死"——寄生体不仅进入信道，还把其他信道的入口焊死。

---

**第二步：极端主义的信道结构——为什么"奶与蜜"能被相信**

用信源选择函数跑极端主义信源 S_extreme：

$$C_{extreme}(L) = B_{extreme} \times D_{extreme}(L) \times \theta_{align}(L) \times E_{extreme}$$

**B_extreme 的特殊性：** 极端主义信源不提供证据，提供**承诺**。承诺不是信号，是期货——它的"信号强度"不取决于当前可验证性，而取决于接收者的绝望程度。定义：

$$B_{extreme} = B_{promise} \times f(despair)$$

despair 越大，B_promise 的感知强度越大。一个绝望的人对"奶与蜜"的解码不是"这有没有证据"，而是"我还有没有别的选项"。

**D_extreme = 1：** 极端叙事用最简单的二元框架（我们vs他们、圣洁vs污秽、天堂vs地狱），解码门槛为零。

**θ_align → 1：** 极端叙事精准命中已有创伤——被边缘化的人听到"你是被选中的"，θ_align 直接拉满。

**E_extreme 的特殊性：** 极端主义的情绪增幅不是谣言的恐惧+愤怒，而是**恐惧+愤怒+意义感**：

$$E_{extreme} = \alpha_{fear} \times \alpha_{anger} + \alpha_{meaning}$$

α_meaning 是极端主义独有的——它不只放大情绪，还提供"你的痛苦有意义"的叙事。这是普通谣言没有的增幅器。

**第三步：退出成本陷阱——为什么走不掉**

极端主义不是让你"信了就走"，而是把退出成本拉到无穷大：

$$C_{exit}(S_{extreme}, L) = C_{social} + C_{identity} + C_{existential}$$

- **C_social** — 退出极端群体意味着失去唯一的社交网络
- **C_identity** — 退出意味着承认"我过去相信的一切都是错的"，身份归零
- **C_existential** — 退出意味着"奶与蜜是假的"→ 死亡是终结 → 存在性恐惧

$$C_{exit} \rightarrow \infty \implies \epsilon \rightarrow 0 \implies \text{永久锁定}$$

**这不是"不愿意退出"，是"退出成本在结构上不可能承受"。** 极端主义在安装自己的同时，焊死了退出通道。

**第四步：信谣反疫苗的共性——双重寄生**

信谣者反疫苗，不是两件独立的事，是同一个寄生体的两个症状。

定义双重寄生：

$$P_{accept}(vaccine\_rumor, L) = \frac{C_{rumor}(L)}{C_{rumor}(L) + R_{rumor}(L)} \approx 1 \quad (L_1\text{-}L_2)$$

$$P_{accept}(vaccine\_fact, L) = \frac{C_{expert}(L)}{C_{expert}(L) + R_{expert}(L)} \approx 0 \quad (L_1\text{-}L_2)$$

但"全世界政府联合控制人民"这个谣言跟普通谣言的区别在于：它不只是占用信道，它**把专业信源标记为敌意信源**：

$$\theta_{align}(expert, L) \xrightarrow{rumor} -1$$

不是"我听不懂专家"（D≈0），而是"专家是敌人"（θ_align 变负）。这比单纯无法解码更严重——**专业信道不仅关闭，还被标记为威胁，F_form 暴增。**

$$F_{form}^{post-rumor}(L) = F_{form}^{baseline}(L) + \Delta F_{form}^{hostility}$$

结果：K_cocoon 不仅因为 D≈0 而趋向无穷，还因为 F_form 暴增而进一步增大。茧房刚性从"进不去"升级为"主动拒绝"。

**第五步：重复犯罪与职业依赖——认知遮蔽的身份锁定版**

重复犯罪和职业依赖是认知遮蔽的另一种形态：不是"信了什么"，而是"不信自己能是别的什么"。

定义身份锁定函数：

$$L_{identity}(L, t) = \frac{P_{accept}(S_{current\_identity}, L)}{P_{accept}(S_{alternative\_identity}, L)}$$

- L_identity → ∞ = 完全锁定（只能看到当前身份的叙事）
- L_identity ≈ 1 = 身份流动性（能看到替代选项）

**重复犯罪者的信道结构：**

$$P_{accept}(S_{crime\_identity}, L) = \frac{B_{crime} \times D_{crime} \times \theta_{align}^{crime}}{B_{crime} \times D_{crime} \times \theta_{align}^{crime} + F_{crime} + \epsilon_{crime}}$$

- D_crime = 1（"我只能这样"——这个叙事解码门槛为零）
- θ_align_crime = 1（精准命中"我没有别的出路"的自我认知）
- B_crime 被环境强化（犯罪网络、前科标签、社会排斥）
- ε_crime ≈ 0（"我学不会别的" = 退出成本无穷大）

$$P_{accept}(S_{alternative}, L) = \frac{B_{alt} \times D_{alt} \times \theta_{align}^{alt}}{B_{alt} \times D_{alt} \times \theta_{align}^{alt} + F_{alt} + \epsilon_{alt}}$$

- D_alt ≈ 0（"我学不会" = 对替代路径的解码能力为零）
- θ_align_alt ≈ 0（替代叙事与"我就是这样的人"的框架不对齐）
- F_alt 高（社会对前科的偏见 = 阻力）

**L_identity → ∞。** 不是"不想改变"，是"看不到改变的信道"。

**职业依赖完全同构：**

"我就是干这个的" = D_alt ≈ 0（对其他职业路径的解码能力为零），θ_align_alt ≈ 0（其他职业叙事与"我是XX人"的身份框架不对齐），ε ≈ 0（"我只会这个" = 退出成本高）。

**第六步：统一——认知遮蔽的四种形态**

| 形态 | 遮蔽机制 | 核心锁定变量 | 对应H函数 |
|------|---------|-------------|----------|
| 极端主义 | 承诺替代证据 + 退出成本无穷大 | C_exit → ∞, ε → 0 | H_身份焊死 |
| 信谣反疫苗 | 专业信源标记为敌意 + F_form暴增 | θ_align(expert) → -1 | H_敌意标记 |
| 重复犯罪 | 替代路径不可解码 + 身份锁定 | D_alt ≈ 0, ε ≈ 0 | H_身份锁定 |
| 职业依赖 | 替代路径不可解码 + 退出成本高 | D_alt ≈ 0, ε ≈ 0 | H_身份锁定（温和版） |

**统一遮蔽函数：**

$$H_{lock}(L, t) = 1 - \frac{D_{alt}(L) \times \epsilon(L) \times \theta_{align}(alt, L)}{C_{exit}(L) \times F_{form}(L)}$$

- 分子 = 打破遮蔽的能力（解码替代路径 × 退出权 × 对齐度）
- 分母 = 维持遮蔽的力量（退出成本 × 形式化阻力）

**H_lock ∈ [0, 1]。** 当分子任意一项为 0 或分母趋向 ∞，H_lock → 1（完全遮蔽）。

**四种形态的 H_lock 值：**

- 极端主义：D_alt ≈ 0, ε ≈ 0, C_exit → ∞ → H_lock ≈ 1
- 信谣反疫苗：θ_align(expert) → -1, F_form 暴增 → H_lock ≈ 1
- 重复犯罪：D_alt ≈ 0, ε ≈ 0, C_exit 高 → H_lock ≈ 1
- 职业依赖：D_alt ≈ 0, ε 低, C_exit 中等 → H_lock ≈ 0.6-0.8

**第七步：打破遮蔽——插入新认知的数学条件**

打破认知遮蔽 = 让 H_lock 从 ≈1 降到 <1。从统一遮蔽函数反推：

$$H_{lock} < 1 \iff D_{alt}(L) \times \epsilon(L) \times \theta_{align}(alt, L) > 0$$

**必要条件：分子三项全部 > 0。** 任何一项为 0，遮蔽不可打破。

**具体策略：**

**1. 装D_alt（让替代路径变得可解码）**

这是最根本的一步。替代路径必须翻译进 D=1 的信道：

- 极端主义者：不用"理性思考"说服，用他自己的语言讲"退出极端主义的人后来怎样了"——故事形式、D=1
- 重复犯罪者：不用"你应该重新做人"说教，用他能理解的具体操作——"有人跟你一样出来后做了XX，第一步是这样"——D=1
- 职业依赖者：不用"你应该转型"施压，用他已有技能的可迁移映射——"你做XX时的那个能力，在YY领域值钱"——θ_align > 0

**2. 装ε（安装退出权）**

退出权不能从外部赋予，必须从内部生长。操作方式：

- 极端主义者：ε 的种子是 ε_sense——身体信号。极端主义的创伤来源（被边缘化、被暴力对待）本身就是 ε_sense 的信号。操作不是"告诉他可以退出"，而是让他重新听见自己身体里"这不对"的信号。
- 重复犯罪者：ε 的种子是微小成功体验——"我做成了这件事"→ "我可能不只是那样的人"。不是教育，是体验。
- 信谣反疫苗者：ε 的种子是身边信任的人的亲身经历——"我打了疫苗没事"走的是 D=1 信道，同时提供退出"疫苗有害"共识的许可。

**3. 降C_exit（降低退出成本）**

- 极端主义：提供退出后的社交网络替代（否则退出=社会性死亡）
- 重复犯罪：消除前科标签的制度性障碍（否则退出=经济性死亡）
- 职业依赖：提供转型期的经济缓冲（否则退出=生存性死亡）

**4. 降F_form（降低对替代信源的防御）**

- 信谣反疫苗：专业信源必须放弃权威姿态——"我来告诉你真相"只会触发 F_form 暴增。"我也曾经不确定，后来看到这些数据"走的是 D=1 信道 + 低 F_form。

**四条路径的依赖关系（与茧房破壁完全同构）：**

装D_alt（前提）→ 装ε（制动）→ 降C_exit + 降F_form（加速器）
