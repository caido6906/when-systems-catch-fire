### **🔍 背景：美国对PCB产业链的安全焦虑**

**核心事件**  
6月3日，美国消费者新闻与商业频道（CNBC）刊文渲染**中国制造的印刷电路板（PCB）** 对美国构成“国家安全担忧”，称AI领域（如英伟达等企业）使用的电路板几乎全部由中国制造，可能存在“恶意元件植入”风险。美国政府以此为由推动**本土PCB制造业复兴**。

**PCB的战略地位**  
- **定义**：物理基板（绝缘材料+薄铜层），通过蚀刻、钻孔、电镀等工艺形成导电走线和焊盘。  
- **功能**：为电子元器件（芯片、电阻等）提供**物理支撑、电气连接路径和散热通道**，是“芯片的载体”（迅达科技高管凯茜·格德利语）。

### **📊 全球PCB产业格局与美国困境**

#### **(一) 产能对比与供应链依赖**

| 指标 | 美国 | 中国大陆 | 全球 |
| :--- | :--- | :--- | :--- |
| **历史产能占比** | 30% | - | - |
| **当前产能占比** | **4%** | **60%** | - |
| **2026年市场规模** | - | - | 960亿美元（同比增长12.5%） |
| **2030年预测规模** | - | - | 1230亿美元 |

#### **(二) 美国产业痛点**
1. **产能不足**：无法满足AI和国防需求，本土仅2家PCB上市企业（迅达科技、Sanmina）。  
2. **供应链脆弱**：关键原材料依赖单一供应商（如美国本土仅1家铜箔供应商）。  
3. **价格波动**：3-4月全球PCB价格上涨40%，迅达科技5月宣布提价5%-25%（不同层数PCB价格差异大，从个位数到10万美元不等）。

### **🛡️ 美国政策应对与产业动作**

#### **(一) 政治层面：国家安全叙事与立法**
- **核心论点**：  
  - 美国助理国防部长迈克·卡德纳齐：被入侵的PCB可能导致“导弹飞行故障”“弹药落点错误”。  
  - 前国防部副部长帮办阿尔·谢弗：PCB是“最容易破坏电子供应链的环节”，可在基板和层中隐藏恶意元件。  
- **立法进展**：  
  - 5月两党参议员提出“**保护电路板和基板法案**”，提供25%税收抵免。  
  - 众议院配套法案呼吁30亿美元拨款支持本土制造商。

#### **(二) 企业层面：产能扩张与市场表现**
- **迅达科技（TTM Technologies）**：  
  - 运营副总裁Raffi Araradian展示本土生产的PCB。  
  - 扩产计划：美国工厂数量将达18家（亚洲7家，最大工厂仍在中国）。  
  - 股价表现：过去一年上涨近**500%**。  
- **Sanmina**：股价上涨超**200%**。

### **🌍 外部因素与中方回应**

#### **(一) 地缘冲突的影响**
- **原材料供应**：中东、俄乌冲突导致铜和树脂价格上涨（中企胜宏科技4月警告）。  
- **军事需求激增**：美企需同时满足AI和国防订单。

#### **(二) 中方立场**
- 外交部发言人毛宁：  
  - 美国才是“搞不公平竞争、施压胁迫”的一方。  
  - “小院高墙”挡不住中国创新，不利于全球产业健康发展。  
  - 呼吁美方遵守市场经济原则，支持良性竞争。

### **📝 补充细节**
- **技术细节**：PCB层数是价格关键因素，高端AI和国防用PCB价格可达10万美元。  
- **产业关联**：PCB是芯片封装的必要环节，无PCB则芯片技术无法应用于设备。  
- **历史对比**：美国PCB产能占比从30%降至4%，反映全球产业链转移趋势。
