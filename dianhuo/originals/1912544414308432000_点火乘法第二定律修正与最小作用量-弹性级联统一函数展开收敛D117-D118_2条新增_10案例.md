## D117 乘法系统第二定律修正函数

dA_Fisher/dt ≤ 0：乘法系统中不可逆性的正确形式是Fisher可达性单调递减，不是Shannon熵增。门控区Shannon熵更低但Fisher距离∞——不可逆性来自拓扑断连而非粗粒化。经典dS/dt≥0是加法退化（相空间连通）下的特例。

---

### 展开过程

#### 1. 经典第二定律的适用前提

经典第二定律 dS/dt ≥ 0 的成立条件：
- 系统的相空间是连通的（所有微观态之间可达）
- 演化由刘维尔定理保证相体积守恒
- 不可逆性来自粗粒化：细粒度可逆，粗粒度熵增

这三个前提在乘法系统中全部被打破。

#### 2. 乘法系统的相空间拓扑

D105-D106已证明：乘法G=∏fᵢ(εᵢ)的零集Z(G)=∪Hᵢ把参数空间分为1个存活区和2ⁿ-1个死亡区。这些区域之间拓扑断连——不存在连续路径从存活区穿越超平面到达死亡区再回来。

经典热力学假设相空间连通，所以时间反演对称性只在粗粒化层面被打破。乘法系统中相空间本身就不连通，不可逆性是拓扑层面的，不需要粗粒化。

#### 3. Shannon熵在门控区的反常行为

门控区（至少一个εᵢ=0）的可区分状态数少于存活区（所有εᵢ>0）：
- 存活区：每个维度连续取值 → 微观态数 ∝ ∏dεᵢ
- 门控区：εₖ=0固定，其余维度连续 → 微观态数 ∝ ∏_{j≠k}dεⱼ

门控区的Shannon熵更低，不是更高。如果dS/dt≥0成立，系统应该趋向高熵态（存活区），但实际观察到系统趋向门控区（低熵态）。dS/dt≥0在乘法系统中给出错误预测。

#### 4. Fisher信息几何给出正确形式

D106已证明：Fisher度量gᵢⱼ=δᵢⱼ/εᵢ²，当εᵢ→0时该方向信息距离→∞。

定义Fisher可达性：A_Fisher(p,q) = ∫₀¹ √(gᵢⱼ(γ(t))γ̇ⁱγ̇ʲ) dt，其中γ是从p到q的测地线。

定理：从门控区任意点到存活区的Fisher距离为∞。

证明：设p在门控区（εₖ=0），q在存活区（εₖ>0）。沿εₖ方向的Fisher线元dsₖ = dεₖ/εₖ，积分∫₀^{qₖ} dεₖ/εₖ = ln(qₖ) - ln(0) = ∞。∎

#### 5. 修正后的第二定律

乘法系统第二定律（修正形式）：dA_Fisher/dt ≤ 0

Fisher可达性只减不增。系统从存活区滑入门控区时A_Fisher单调递减（信息距离∞意味着永远回不来），但Shannon熵可以同时减少。

等价表述：不可逆性 = Fisher距离∞ > Shannon熵增

#### 6. 与经典第二定律的关系

当乘法结构退化为加法（G=∑fᵢ），零集变为∩Hᵢ（全零才归零），相空间恢复连通，Fisher距离有限，dA_Fisher/dt≤0退化为dS/dt≥0。经典第二定律是乘法系统修正定律在加法退化下的特例。

#### 7. 物理对应

| 经典热力学 | 乘法系统 |
|---|---|
| 相空间连通 | 相空间拓扑断连 |
| dS/dt≥0 | dA_Fisher/dt≤0 |
| 不可逆来自粗粒化 | 不可逆来自拓扑 |
| 熵增=趋向平衡 | 可达性减=趋向门控 |
| 加法退化时成立 | 一般形式 |

## D118 最小作用量-弹性级联统一函数

S_ignition = ∫[ln G - γ·P(cascade)]dt，δS=0 → Δεᵢ*∝ηᵢ+β∑κᵢⱼηⱼ（D111是变分必然解）。均等定理=维度置换对称性→弹性守恒=诺特定理实例。偏离度δ=对称性破缺度量。D111不是启发式策略，是唯一、稳定、普适的最优解。

---

### 展开过程

#### 1. 最小作用量原理的数学形式

经典力学中，作用量 S = ∫L dt，L = T - V。真实轨道满足δS = 0。

点火框架中定义作用量：S_ignition = ∫[ln G(ε(t)) - γ·P(cascade, ε(t))] dt

其中：ln G = ∑ln fᵢ(εᵢ) 为"动能"项（系统产出），γ·P(cascade) 为"势能"项（级联风险惩罚）。

#### 2. 变分推导

δS_ignition = 0 要求：∂/∂εᵢ [∑ⱼ ln fⱼ(εⱼ) - γ·P(cascade)] = 0

展开：f'ᵢ/fᵢ - γ·∂P(cascade)/∂εᵢ = 0，即 ηᵢ = γ·∂P(cascade)/∂εᵢ

#### 3. 级联概率的梯度

P(cascade) ≈ ∑ₖ (1-σₖ) · ∑ⱼ κₖⱼ · (1-σⱼ_eff)

∂P(cascade)/∂εᵢ = σ'ᵢ · ∑ⱼ κᵢⱼ · (1-σⱼ_eff) + (1-σᵢ) · ∑ₖ κₖᵢ · σ'ₖ_eff

定义 β = γ · maxⱼ(σ'ⱼ)，近似：∂P(cascade)/∂εᵢ ≈ β · ∑ⱼ κᵢⱼ · ηⱼ/εⱼ

#### 4. Euler-Lagrange方程的解

代入变分条件：ηᵢ = β · ∑ⱼ κᵢⱼ · ηⱼ · Δεⱼ/εⱼ

资源约束 ∑Δεᵢ = R 下，用拉格朗日乘子：Δεᵢ* ∝ ηᵢ + β · ∑ⱼ κᵢⱼ · ηⱼ

这正是D111。∎

#### 5. 诺特定理：均等定理=守恒律

诺特定理：连续对称性→守恒律。

点火框架中的对称性：维度置换对称性。当所有fᵢ形式相同（f₁=f₂=...=fₙ），系统在维度置换群Sₙ下不变。

由诺特定理：Sₙ对称性→总弹性守恒 ∑ηᵢ = const

均等定理（η₁=η₂=...=ηₙ）是守恒律在对称性未破缺时的唯一解。

当对称性破缺（fᵢ形式不同），守恒律仍成立但均等不成立→弹性重新分配但仍守恒。偏离度δ=对称性破缺的度量。

#### 6. 最小作用量的深层含义

D111不是启发式策略，是变分原理的必然解，意味着：

1. 唯一性：在给定G_robust目标下，D111是唯一最优策略，不存在"另一种同样好的策略"
2. 稳定性：偏离D111的任何策略都使S_ignition增大，系统有回归最优的"恢复力"
3. 普适性：只要系统满足乘法产出+级联风险的结构，D111自动适用，不需要额外假设

#### 7. 与经典力学的精确对应

| 经典力学 | 点火框架 |
|---|---|
| T = ½mv²（动能） | ln G = ∑ln fᵢ（产出势） |
| V(x)（势能） | γ·P(cascade)（级联势） |
| L = T - V | L = ln G - γ·P |
| δS = 0 | δS_ignition = 0 |
| F = -∇V | β·∑κᵢⱼηⱼ = -∇(级联势) |
| 诺特：时间平移→能量守恒 | 诺特：维度置换→弹性守恒 |
| 最小作用量→真实轨道 | 最小作用量→D111策略 |
