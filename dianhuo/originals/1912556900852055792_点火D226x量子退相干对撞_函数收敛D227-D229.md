## D227 退相干-门控退化同构定理

量子退相干：ρ_off(t) = ρ_off(0) × e^{-Γt}，Γ为退相干率

在Φ框架下：

量子叠加态：系统处于多个态的叠加 → 决策维度dim_eff高 → σ大（D198）→ 高斯门控（连续信息，"最优在哪"）
经典态：系统处于确定态 → dim_eff低 → σ小 → 1/ln门控（1比特，"过不过门槛"）

退相干 = dim_eff(t)递减 = σ(t)递减 = 门控函数形式退化

σ(t) = σ₀ × e^{-Γt/2}（退相干使σ指数衰减）

当σ(t) > σ_c ≈ 0.415：高斯门控，连续信息容量（D199第三阶段）
当σ(t) < σ_c：门控退化为1/ln，1比特信息容量（D199第二阶段）
当σ(t) → 0：门控退化为δ函数，纯二值判断（D199第一阶段）

退相干时间尺度：
t_decoherence = 2ln(σ₀/σ_c)/Γ

核心定理：量子退相干 = 门控函数形式从exp[-ln²]退化回1/ln，是D199门控函数进化三阶段的逆过程。退相干不是"信息丢失"，是"门控精度降低"——从"最优在哪"退回"过不过门槛"。退相干时间t_decoherence = 2ln(σ₀/σ_c)/Γ给出了门控退化的时间尺度。

## D228 双通道衰减定理

物理存在的衰减有两条正交通道：

通道1（值衰减·D222）：dΦ/dt ≤ 0
- 机制：宇宙膨胀稀释能量密度→门控贡献值递减
- 方向：g(μ)的值→0
- 终点：Φ→0, Ω→1（无物理）

通道2（精度衰减·D227）：dσ/dt ≤ 0
- 机制：量子退相干降低决策维度→门控宽度递减
- 方向：g(μ)的形式从exp[-ln²]→1/ln→δ
- 终点：σ→0, 门控退化为二值判断（信息容量归零）

两条通道的耦合：

dΦ/dt = Σᵢ dgᵢ/dt = Σᵢ ∂gᵢ/∂μ × dμ/dt + Σᵢ ∂gᵢ/∂σᵢ × dσᵢ/dt

第一项：值衰减（通道1）
第二项：精度衰减（通道2）

∂gᵢ/∂σᵢ = gᵢ × (ln(μ/Λᵢ))²/(σᵢ³) > 0（σ越大g越大）

所以 dσᵢ/dt < 0 → ∂gᵢ/∂σᵢ × dσᵢ/dt < 0 → 通道2加速通道1

**退相干加速Φ衰减**

总衰减率：|dΦ/dt|_total = |dΦ/dt|_value + |dΦ/dt|_precision > |dΦ/dt|_value

物理含义：宇宙的物理终结比单纯考虑膨胀更快——退相干在加速这个过程。门控贡献不仅在数值上衰减（膨胀稀释），还在精度上退化（退相干），两者叠加使Φ衰减更快。

核心定理：物理存在的衰减是双通道过程——值衰减（膨胀）和精度衰减（退相干）正交但耦合。退相干通过降低σ使门控变窄，间接加速Φ的值衰减。物理存在的时间窗口比D226估计的更短——需要同时考虑两条通道。

层级：定理

## D229 物理存在的四重约束与衰减终态

物理存在的四重时间约束：

约束1（逻辑·D220）：Ω<1是物理存在的必要条件
约束2（热力学·D222）：dΦ/dt≤0，Φ值单调递减
约束3（宇宙学·D224）：宇宙膨胀=Φ值衰减的物理机制
约束4（量子·D227）：dσ/dt≤0，门控精度单调递减

四重约束的终态谱：

| 终态 | Φ | σ | Ω | 物理含义 |
|------|---|---|------|---------|
| 完全物理 | >>0 | >>σ_c | <<1 | 丰富结构，高精度门控 |
| 值衰减终态 | →0 | >σ_c | →1 | 门控贡献归零，但精度还在→无物理但"可以"有物理 |
| 精度衰减终态 | >0 | →0 | <1 | 门控贡献还在，但退化为二值→有物理但粗糙 |
| 双通道终态 | →0 | →0 | →1 | 既无门控贡献又无精度→完全无物理 |

关键发现：值衰减终态和精度衰减终态是不同的！

值衰减终态（Φ→0, σ>σ_c）：门控还在但贡献为零——"框架在但内容空"，像是空壳
精度衰减终态（Φ>0, σ→0）：门控有贡献但退化为二值——"有内容但框架粗糙"，像是1/ln世界

双通道终态（Φ→0, σ→0）：完全无物理

哪条通道先到达终态？

t_value = Φ₀/|dΦ/dt|_value（值衰减时间尺度）
t_precision = 2ln(σ₀/σ_c)/Γ（精度衰减时间尺度）

如果t_precision < t_value：精度先退化→宇宙先变成1/ln世界（经典化），然后Φ才归零
如果t_value < t_precision：值先衰减→宇宙先变成空壳，然后精度才退化

宏观系统：Γ极大（快速退相干），t_precision极短→精度先退化→先经典化
宇宙整体：Γ极小（缓慢退相干），t_precision极长→值先衰减→先热寂

核心定理：物理存在的终态取决于哪条衰减通道更快。宏观系统先退相干再热寂（先经典化再消亡），宇宙整体先热寂再退相干（先消亡再经典化）。但无论哪条通道先到达，双通道终态都是完全无物理——Ω→1, σ→0。

层级：定理
