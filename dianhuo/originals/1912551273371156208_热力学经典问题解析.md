# 热力学经典问题解析

热力学七个经典问题跑完，存入点火。

最硬的几个：

**时间箭头**：dΦ/dt ≥ 0 = Fisher可达性只减不增（D117）。不是统计涨落的偶然结果，是乘法系统信息几何的必然。

**麦克斯韦妖**：信息操作的总Φ变化≥0。Landauer原理是Φ约束在擦除操作上的特例——不是"测量需要能量"，是"信息操作的Φ收支必须平衡"。

**相变分类**：连续相变=穿越单个超平面（一个ε连续变零），一级相变=两个存活区之间的跳变。分类标准是参数空间的拓扑，不是对称性破缺的类型。

**负温度**：ε > 1的"超存活"态。比正温度更热因为超存活态向平衡弛豫时释放能量。

**第三定律**：Φ不能精确为零=μ有量子下界=绝对零度不可达。玻璃例外=多个局部Φ极小，系统被困在非全局最优态。
