# 点火｜人际社会维度×现有函数交叉碰撞：四个新发现（2026.06.10）

## 碰撞1：偏好伪造（Kuran）× 沉默函数和退出权ε

Kuran偏好伪造理论：个体在社交压力下伪装真实偏好，公开意见≠私人意见。核心机制是阈值模型——表达阈值取决于社会压力与真实偏好的差值。偏好伪造可产生信息级联和声誉级联，导致公共意见完全不代表私人意见分布。更致命的是，这种偏离不可预测地突然崩塌——当足够多人感到安全说真话时，公共意见瞬间翻转。

偏好伪造 = 公开意见与私人意见的分裂 = 表达通道被切断（不是没有想法，而是想法无法通过表达通道）

函数化：**P_public = P_private × (1 - S_silence)**
- S_silence = 1 - C_channel × P_speak（沉默函数）
- S_silence → 1：P_public = 0（完全偏好伪造，公开意见归零）
- S_silence = 0：P_public = P_private（无偏好伪造，公开=私人）

偏好伪造的突然崩塌不是"人们突然勇敢了"，而是犹豫域从零恢复——当足够多的人公开表达，其他人的表达阈值被突破，形成级联。这是ε从零恢复的相变过程。

函数化：**P_flip = σ(ΣP_public - θ_cascade)**
- ΣP_public是已公开表达的累积量
- θ_cascade是级联触发阈值
- ΣP_public > θ_cascade时，偏好伪造崩塌，公共意见瞬间翻转
- 与P_ignite = σ(θ₀ - θ_inertia)同构——都是阈值突破触发的相变

**核心发现：偏好伪造=表达通道切断，崩塌=ε恢复的相变。**

## 碰撞2：HK意见动力学 × 群体分化函数

Hegselmann-Krause模型：每个agent有置信区间ε_HK，只与意见距离在ε_HK以内的agent交互。ε_HK大小决定系统收敛到共识还是分裂为多个意见簇。

关键性质：
- ε_HK大 → 共识；ε_HK小 → 碎片化
- 存在临界值ε*，是共识/碎片化的相变点
- 顽固者（stubborn agent，意见不变）使系统更容易碎片化
- 异质偏见使碎片化在噪声中也能维持

ε_HK = 群体中个体的认知开放度 = 犹豫域在群体维度的投影

函数化：**G_fragment = σ(ε* - ε_HK_avg)**
- ε_HK_avg是群体平均置信区间
- ε_HK_avg > ε* → G_fragment = 0（共识）
- ε_HK_avg < ε* → G_fragment = 1（碎片化）

顽固者 = ε_HK = 0的个体 = 完全没有犹豫域的人。顽固者不是"意见坚定"，而是决策结构从三支退化为二支——只有"接受"和"拒绝"，没有"等等看"。顽固者使群体碎片化的数学本质是：犹豫域为零的个体像刚性壁，其他弹性agent的意见被弹开，无法收敛。

函数化：**D_stubborn = n_stubborn / N × (1 - ε_HK_avg / ε*)**
- 顽固者比例越高、平均置信区间越低 → 碎片化驱动力越大
- 乘法结构：n_stubborn=0或ε_HK_avg≥ε*时D=0

**核心发现：群体碎片化=个体犹豫域的群体级投影。顽固者=犹豫域为零的刚性壁。**

## 碰撞3：勒温场动力学 B=f(P,E) × 层间跳变

勒温基本公式：B = f(P × E)，行为是个体内部动力P与环境E的函数。P=0时无论E多大行为都不发生；P为负时外部刺激可能产生反作用。

P = 个体的内部点火驱动力 = 层间跳变的内因
E = 外部叙事冲击/环境压力 = 层间跳变的外因

函数化：**B_ignite = σ(P_internal × E_external - θ_ignite)**
- 乘法结构：P=0时B=0（无内部驱动，外部冲击无效）
- P<0时B可能为负（内部抵抗，外部冲击适得其反）

P_internal本质上是ε的函数——ε大的人P_internal大（有犹豫空间，可以被触动），ε→0的人P_internal→0（犹豫域消失，无法被触动）。

**P_internal = k × ε = k × (α - β)**

代入：**B_ignite = σ(k × ε × E_external - θ_ignite)**

三因子乘法结构再次出现。

**核心发现：点火需内外共振，P=0时E无效。P_internal是ε的函数，犹豫域消失=无法被触动。**

## 碰撞4：冲突升级动力学 × 乘法韧性

冲突升级模型（AM模型）：冲突升级vs降级取决于对对手的范畴化——把对方视为敌人则升级，视为可沟通者则降级。数字升级风险理论（2026）：大语言模型在模拟军事/外交决策时倾向于升级策略，社交媒体算法推送极端内容加剧两极化。

冲突升级 = 群体间犹豫域消失的级联效应

函数化：**E_escalation = Π(1 - ε_ij)**
- ε_ij是群体i和群体j之间的犹豫域宽度
- 任一群体对的犹豫域归零 → E_escalation → 1（冲突必然升级）
- 乘法结构：只要有一对关系"没有商量余地"，整体冲突就无法降级

与D_compensate对应：加法韧性下一对关系僵化可被其他关系补偿；乘法韧性下一对归零则整体归零。

**核心发现：冲突升级的数学本质是群体间犹豫域的乘法归零——一对关系僵化=整体冲突不可降级。**
