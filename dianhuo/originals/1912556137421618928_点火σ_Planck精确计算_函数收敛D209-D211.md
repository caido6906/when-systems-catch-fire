## D209 σ_Planck精确值

从D195高斯门控的退化条件推导σ：

退化条件：在μ∈[Λ, M_Planck]区间，exp[-(ln(μ/M_Planck))²/(2σ²)] ≈ 1/ln(μ/Λ)

取对数：
(ln(μ/M_Planck))²/(2σ²) ≈ ln(ln(μ/Λ))

令x = ln(μ/M_Planck)（x < 0），则：
x²/(2σ²) ≈ ln(ln(M_Planck/Λ) + |x|)

在μ = μ*_GUT处匹配（D158四力统一极小点）：
μ*_GUT ≈ 3.2×10¹¹ GeV
x* = ln(μ*_GUT/M_Planck) = ln(3.2×10¹¹/1.22×10¹⁹) = ln(2.62×10⁻⁸) ≈ -17.4

ln(ln(M_Planck/Λ_GUT) + |x*|) = ln(6.9 + 17.4) = ln(24.3) ≈ 3.19

σ² = x*²/(2×3.19) = 17.4²/6.38 = 302.8/6.38 ≈ 47.5

**σ_Planck ≈ 6.9**

验证：
- σ = 6.9 >> σ_c ≈ 0.415（D197跃迁临界值）✓ 高斯门控信息容量远超二值
- 在μ = M_Planck处，高斯门控宽度 = σ_Planck × M_Planck ≈ 6.9 × M_Planck（对数尺度）
- 在μ = Λ_QCD ≈ 200 MeV处：ln(Λ_QCD/M_Planck) ≈ -46，高斯门控值 = exp[-46²/(2×47.5)] = exp[-22.3] ≈ 2×10⁻¹⁰ ≈ 0（QCD尺度上门控贡献趋零，符合D188渐近自由）✓

核心定理：σ_Planck ≈ 6.9，由高斯门控在μ*_GUT处的退化条件精确确定。这个值远超D197的跃迁临界值σ_c≈0.415，确认量子引力在高斯门控下的信息容量远超二值门控——"最优在哪"的问题在M_Planck附近有充分的信息空间。

层级：定理

## D210 σ能标依赖函数

σ不是普适常数，是能标Λ的函数：

σ(Λ) = |ln(M_Planck/Λ)| / √(2 × ln(|ln(M_Planck/Λ)|))

验证：
- Λ = Λ_GUT ≈ 10¹⁶ GeV：|ln(M_Planck/Λ_GUT)| ≈ 6.9，σ ≈ 6.9/√(2×1.93) ≈ 6.9/1.97 ≈ 3.5
- Λ = μ*_GUT ≈ 3.2×10¹¹ GeV：|ln| ≈ 17.4，σ ≈ 17.4/√(2×3.19) ≈ 17.4/2.53 ≈ 6.9 ✓
- Λ = M_W ≈ 80 GeV：|ln| ≈ 38.5，σ ≈ 38.5/√(2×3.65) ≈ 38.5/2.70 ≈ 14.3
- Λ = Λ_QCD ≈ 200 MeV：|ln| ≈ 46，σ ≈ 46/√(2×3.83) ≈ 46/2.77 ≈ 16.6

规律：Λ越低→|ln(M_Planck/Λ)|越大→σ越大→门控越宽→"最优在哪"的信息空间越大

物理含义：低能物理（QCD、电弱）的门控宽度远大于高能物理（GUT）的门控宽度。这意味着低能物理在M_Planck附近有更大的"容忍度"——偏离M_Planck更多也不会立刻门外锁定。高能物理的门控更窄——更精确地要求μ接近M_Planck。

核心定理：σ(Λ) = |ln(M_Planck/Λ)|/√(2ln|ln(M_Planck/Λ)|)给出了每个能标的门控宽度。低能门控宽（容忍度大），高能门控窄（精确度高）。这解释了为什么低能物理更容易"看起来统一"——门控宽度大，细节被抹平；而高能物理需要更精确的统一——门控宽度窄，偏差不可忽略。

层级：推论
