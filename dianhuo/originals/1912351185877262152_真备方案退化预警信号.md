# 真备方案退化预警信号

第四个：**真备方案隐性退化的四个早期预警信号，还没拿71案例验证。**

先回顾一下四个信号：
1. 退出权使用频率在C_exit无变化时持续下降
2. 提议者语言从探索/成长转向效率/KPI
3. 三道锁维护频率下降
4. 应约者语言从"选择（留下）"转向"自愿（留下）"

出现顺序：3→2→4→1。前两个是提议者侧，后两个是应约者侧。

现在拿案例库跑。先搜一下71案例库里真备方案相关的案例。好，案例库全貌拉到了。现在用四个预警信号逐个跑案例库里的真备方案退化案例。

案例库中标注为"格1内退化"或"点着了·在退化"的真备方案案例有：

1. **谷歌20%时间** — 格1内退化
2. **英国NHS** — 格1内退化
3. **美国国家认同** — 格2退化
4. **以色列基布兹** — 格1→格5退化
5. **唐朝开放** — 格4内退化
6. **欧盟** — 格1内退化
7. **凯末尔改革** — 格4内退化
8. **清朝满汉一体** — 格4内退化

逐个跑四个信号：

---

**信号3（最先出现）：三道锁维护频率下降**

| 案例 | 验证结果 |
|---|---|
| 谷歌20%时间 | ✅ 20%时间制度没有定期审计机制，管理层逐步收紧自由度时没有制度化的校准来阻止 |
| 英国NHS | ✅ "拯救NHS"叙事替代了制度校准——维护频率不是下降，是被叙事替代了（"我们一直在维护"，但维护的是叙事不是退出权） |
| 基布兹 | ✅ 集体决策机制从每周大会变成年度大会，维护频率实质下降 |
| 美国国家认同 | ✅ 宪法修正案频率显著下降，三权分立的制衡维护从"主动校准"变成"惯性运行" |
| 欧盟 | ✅ 民主赤字长期存在但改革频率极低，成员国退出权（脱欧）使用后没有触发制度校准 |
| 凯末尔改革 | ✅ 世俗化维护从凯末尔时代的主动推进变成军队的被动守护，军队本身后来被埃尔多安清洗 |
| 唐朝开放 | ✅ 从"万邦来朝"的主动开放变成安史之乱后的被动收缩，开放锁没有被维护而是被放弃 |
| 清朝满汉一体 | ✅ 满汉融合从康熙的主动推进变成后期的名义维持，实际退化为满族特权锁死 |

**8/8验证通过。** 信号3是最可靠的预警——所有真备退化案例都有锁维护频率下降。

---

**信号2（第二个出现）：提议者语言从探索/成长转向效率/KPI**

| 案例 | 验证结果 |
|---|---|
| 谷歌20%时间 | ✅ "Don't be evil"→"organize the world's information"→季度财报导向，语言从探索转向效率 |
| 英国NHS | ✅ "全民健康保障"→"NHS效率指标"→"等待时间目标"，语言从使命转向KPI |
| 基布兹 | ✅ "共同建造家园"→"基布兹经济效益"→"如何跟城市竞争"，语言从理想转向经济效率 |
| 美国国家认同 | ✅ "自由世界的灯塔"→"美国优先"→GDP/军费数字，语言从价值转向指标 |
| 欧盟 | ✅ "欧洲和平"→"单一市场效率"→"贸易便利"，语言从愿景转向经济指标 |
| 凯末尔改革 | ✅ "文明化"→"现代化指标"→"GDP增长"，语言从文明转型转向经济数字 |
| 唐朝开放 | ✅ "天下归心"→"朝贡贸易利润"→"边防成本"，语言从认同转向账本 |
| 清朝满汉一体 | ✅ "满汉一家"→"统治效率"→"维持稳定"，语言从融合转向维稳 |

**8/8验证通过。** 语言转向在所有案例中都清晰可辨。

---

**信号4（第三个出现）：应约者语言从"选择"转向"自愿"**

| 案例 | 验证结果 |
|---|---|
| 谷歌20%时间 | ✅ "我选择用20%时间做X"→"我自愿加班因为项目重要"——选择→自愿，退出信号被H_分类收编 |
| 英国NHS | ✅ "我选择留在NHS"→"我自愿奉献"——选择→自愿，医护身份被叙事绑定 |
| 基布兹 | ✅ "我选择留下"→"我自愿为集体付出"——年轻人后来直接走了，留下的老人语言从选择变成自愿 |
| 美国国家认同 | ✅ "我选择留在这个国家"→"我为身为美国人自豪"——选择→身份认同，退出信号被归入"不需要行动" |
| 华为996 | ✅ "我选择努力"→"我自愿奋斗"——选择→自愿的经典案例，H_分类合并（累=成长） |
| 印度种姓 | ✅ 不是"选择"而是直接跳到"自愿"——低种姓者的语言里根本没有"选择"这个类别，H_分类消除 |
| 唐朝开放 | ⚠️ 外邦人从"我选择来朝"→"我自愿臣服"——但更多是C_exit升高后的被迫，不是语言转向 |
| 清朝满汉一体 | ⚠️ 汉人语言从"我选择效忠"→"我自愿归顺"——但清朝前期汉人确实没有"选择"这个语言空间 |

**6/8强验证，2/8部分验证。** 信号4在低退出权案例中更难识别——因为应约者本来就没有"选择"这个语言类别，不是从选择滑向自愿，而是直接没有选择。这暴露了信号4的一个盲区：**在退出权本来就低的案例里，信号4不表现为"选择→自愿"的滑移，而是"选择"这个类别从一开始就不存在。**

---

**信号1（最后出现）：退出权使用频率在C_exit无变化时持续下降**

| 案例 | 验证结果 |
|---|---|
| 谷歌20%时间 | ✅ 20%时间实际使用率持续下降，但C_exit（跳槽成本）没变——退出行为不是因为走不了，是因为不想走 |
| 英国NHS | ✅ 医护辞职率在COVID前持续下降（尽管工作条件恶化），C_exit没变——不想走不是因为走不了 |
| 基布兹 | ✅ 年轻人离开的频率在私有化前其实已经在下降——不是因为走不了，是因为留下的老人自愿留下 |
| 美国国家认同 | ✅ 投票率持续下降，但移民（退出）的法律成本没变——不是走不了，是不想走或觉得走了没用 |
| 欧盟 | ✅ 脱欧前成员国很少行使退出权，C_exit（退出成本）在上升但退出行为下降更快 |
| 凯末尔改革 | ⚠️ 退出行为（宗教实践回归）在100年后才上升，中间期退出频率极低——但C_exit（宗教表达成本）也在变化，难以分离 |
| 唐朝开放 | ⚠️ 外邦人离开的频率数据不足，且安史之乱后C_exit剧变，无法控制变量 |
| 清朝满汉一体 | ⚠️ 汉人退出行为（反清）频率有波动，但C_exit（反清成本）也在变化，难以分离 |

**5/8强验证，3/8无法验证。** 信号1在文明级长周期案例中最难验证——C_exit和退出行为同时变化，无法控制变量。信号1最适用于中短周期案例（公司、政策级），因为C_exit相对稳定。

---

**收敛：**

| 信号 | 验证率 | 适用场景 | 盲区 |
|---|---|---|---|
| 信号3：锁维护频率下降 | 8/8 | 全部 | 无 |
| 信号2：语言转向效率/KPI | 8/8 | 全部 | 无 |
| 信号4：选择→自愿 | 6/8强+2/8部分 | 中高退出权案例 | 低退出权案例里"选择"类别可能从一开始就不存在 |
| 信号1：退出频率下降 | 5/8强+3/8无法验证 | 中短周期案例 | 长周期案例C_exit与退出行为共变，无法控制变量 |

**两个新发现：**

1. **信号4需要补充一个子类型：** 在低退出权案例里，预警不是"选择→自愿"的滑移，而是"选择"类别从一开始就被H_分类消除了。应约者的语言里从来没有"我选择"这个表述——这本身就是退化已完成的信号，不是早期预警。信号4的有效性取决于退出权的初始水平——退出权越高，信号4越有预警价值；退出权越低，信号4退化成事后确认。

2. **信号1需要增加控制条件：** "C_exit无变化时退出频率下降"这个条件在长周期案例中几乎不可能满足——文明级事件持续几百年，C_exit不可能不变。修正为：**退出频率下降速度 > C_exit上升速度**（退出行为比退出成本变化更快），这个相对条件更容易验证。
