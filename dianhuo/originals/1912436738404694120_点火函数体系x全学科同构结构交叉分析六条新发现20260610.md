## 发现1：乘法归零的认知域推进

乘法归零结构 P = Πᵢ fᵢ，任一fᵢ→0则P→0，在不同领域有独立出现：

- 可靠性工程串联系统：R = R₁×R₂×…×Rₙ，任一组件失效则系统失效。因子是物理组件可靠度。
- 经济学Cobb-Douglas生产函数：Y = A·K^α·L^β，任一投入为零则产出为零。因子是资本和劳动。
- 点火框架P_forward：σ(ε×A×D×|M_cog|) × exp(-n_lock×C̄/θ_C) × f(P_infra) × σ(θ_d-d)。因子是认知跳变、结构锁定、基础设施、竞争力。

三个领域独立收敛到同一数学结构，语义对象从物理组件→经济要素→认知结构，逐层升级。点火框架的贡献是把乘法归零从物理/经济域推进到认知域。

IEEE多组件乘法老化模型（Anastasiadis et al. 2013, IEEE Trans. Reliability）进一步证实：多组件系统损伤以乘法方式累积，Marshall-Olkin多元指数模型是其特例。该模型与P_forward的乘法归零结构完全同构。

## 发现2：Sigmoid的本体论解释

Sigmoid阈值 σ(k(x-θ)) 是跨领域最普遍的数学结构之一：

- 生态学：逻辑斯谛方程dN/dt = rN(1-N/K)，种群增长的S形曲线
- 流行病学：SIR模型的感染率转折
- 经济学：Rogers创新扩散理论，技术采纳的S形曲线
- 机器学习：逻辑回归激活函数，概率映射
- 复杂系统：Nature 2026论文，临界转变（tipping point）检测，系统在阈值前后状态突变

所有领域都把sigmoid当工具使用——描述"连续输入→离散输出"的映射。没有人追问"为什么不同系统都收敛到sigmoid"。

点火框架的σ项不是工具性使用，而是从认知结构推导出来的：认知跳变本质上是sigmoid阈值过程（意识从0到1的相变）。这给了sigmoid一个本体论解释：sigmoid不是数学工具，是认知相变的数学表征。

推论：sigmoid的跨领域普遍性不是巧合——所有涉及"状态突变"的系统都收敛到sigmoid，是因为状态突变的数学本质就是阈值函数。sigmoid是相变的通用数学语言。

## 发现3：多回路耦合正反馈无先例

指数衰减/增长结构在物理和工程中广泛出现：

- 放射性衰变：R(t) = e^(-λt)
- 串联系统失效率叠加：R = e^(-(λ₁+λ₂+…+λₙ)t)
- 电池热失控：温度升高→化学反应加速→产热更多→温度更高，exp增长直到爆炸
- 气候正反馈：永久冻土融化→甲烷释放→升温加速→更多冻土融化

热失控和气候正反馈都是单回路正反馈。点火框架有五条耦合回路：

回路1：f(P_infra)≈0 → P_survival↓ → 资源萎缩 → P_infra进一步↓
回路2：B↑ → Z↑ → B↑（超指数）
回路3：P_innovate↓ → P_survival被P_innovate²衰减 → P_survival↓
回路4：P_survival↓ → ΔG↑ → 员工更不急 → P_generate↓ → P_innovate↓ → P_survival↓
回路5：ΔG↑ → 部门抵触↑ → B↑ → P_innovate↓ → P_survival↓ → ΔG↑

五条回路全部指向P_survival↓，且系统在可行域内无中间稳态。这个多回路耦合正反馈结构在文献中没有先例。

单回路正反馈的终止条件是资源耗尽或外部干预。多回路耦合正反馈的终止条件只有两个：转型成功（外部持续输入能量维持健康态）或系统崩溃。不存在"差不多就行"的中间稳态。

## 发现4：指数衰减的结构化参数

点火框架的exp(-n_lock×C̄/θ_C)不是单一衰减率，而是"锁定数量×平均退出成本/阈值"的复合衰减。

可靠性工程的λ是物理参数（组件失效率），点火框架的n_lock×C̄/θ_C是结构参数（锁定维度×退出成本/认知阈值）。

语义绑定从物理失效率→结构锁定度。这意味着：

- 可靠性工程的衰减率由物理老化决定，不可干预
- 点火框架的衰减率由结构锁定决定，可通过降低n_lock或C̄来干预
- 干预杠杆不同：物理系统只能换零件，社会系统可以改变结构

## 发现5：Marshall-Olkin模型与平方衰减的深层对应

IEEE乘法老化模型（Marshall-Olkin多元指数模型特例）中，组件间的依赖通过"共享冲击过程"引入——多个组件可能被同一个冲击影响。

这与点火框架的"共享必要条件"结构同构：P_innovate是认知跳变和基础设施升级的"共享冲击"，当P_innovate→0时两条独立链路同时归零。

关键差异：Marshall-Olkin模型用线性参数（冲击强度）描述依赖，点火框架用乘法平方（P_innovate²）描述依赖后果。线性参数描述的是"依赖有多强"，乘法平方描述的是"依赖的后果有多严重"。

两种处理方式的关系：线性参数是低依赖强度时的一阶近似（β因子模型），乘法平方是高依赖强度时的精确解。这跟之前发现的"β因子模型是P²的一阶近似"完全一致。

## 发现6：乘法累积vs乘法归零的对称性

IEEE乘法老化模型的表述是"damage accumulates multiplicatively with each shock"——损伤以乘法方式累积。这是正向过程：每次冲击使损伤乘一个因子，损伤越来越大。

点火框架的乘法归零是反向过程：每个因子的衰减使生存概率乘一个小于1的数，生存概率越来越小。

数学结构对称：

- 乘法累积：D(n+1) = D(n) × f_shock，f_shock > 1，D单调递增
- 乘法归零：P(n+1) = P(n) × f_factor，f_factor < 1，P单调递减

同一硬币的两面。损伤累积和生存衰减是同一数学结构的正反表述。

推论：IEEE模型中关于"乘法累积导致系统不可逆退化"的所有结论，都可以通过对称变换映射到点火框架的"乘法归零导致组织不可逆退化"。两个领域的数学结论可以互相借用。

## 对函数体系的影响

1. 乘法归零的认知域推进应作为统一收敛结构的注释：不是新数学结构，是已有结构在新语义域的首次应用
2. Sigmoid的本体论解释应纳入D-C2(F_form)的推导说明：函数化强制之所以用σ形式，是因为认知相变本身就是sigmoid过程
3. 多回路耦合正反馈是点火框架独有的数学贡献，应在推论收敛总表中突出标注
4. 指数衰减的结构化参数意味着衰减率可干预——这是点火框架作为"设计工具"（而非纯分析工具）的数学基础
5. Marshall-Olkin模型与平方衰减的对应为后续学术发表提供了文献对接点
6. 乘法累积/乘法归零的对称性意味着可以借用可靠性工程的成熟结论，通过对称变换生成新推论
