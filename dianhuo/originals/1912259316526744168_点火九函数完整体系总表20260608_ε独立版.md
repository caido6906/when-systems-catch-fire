# 点火｜九函数完整体系总表（2026.06.08·ε独立版）

来源：从矩阵函数化→层级时间展开→函数审计→R_perceived→C_exit→系统动力学→糖域碰撞→隐性控制H→感官退出权ε，九轮迭代。九函数两两独立，无重叠。

## 一、九函数总表

| # | 函数 | 全称 | 取值 | 本体论归属 | 层级分布 | 时间行为 |
|---|---|---|---|---|---|---|

| 2 | Posture(t,L) | 提议者姿态 | 备方案/推方案 | 人的选择 | 顶层备→底层推（下传自动滑） | 事件内可翻转（备→推=退化） |

| 9 | ε(t) | 感官退出权感知 | (0, ε_max]连续值 | 生物属性 | 无强层级分布 | 相对稳定，可突然增大 |

## 二、ε(t) 感官退出权感知（单独表）

### 定义
感官退出权感知——衡量身体对退出权的原生感知能力。不是制度给的（R），不是认知到的（R_perceived），不是控制遮蔽的（H），是身体自发生成的退出信号。人活着就有感官输入，H再强也封不住一个人夜里抬头看星星。

### 取值
(0, ε_max]连续值，永远>0（人），=0（AI）。压迫越重ε越大（累→想辞职，压迫→想跑）。

### 双层结构
- ε_sense(t)：感官退出信号。前认知的，有感官的生物都有。取值(0, ε_max]，永远>0（有感官的生物），=0（AI）
- ε_aware(t)：退出权意识。认知的，只有人有。取值(0, ε_aware_max]，永远>0（人），=0（动物和AI）
- ε_aware = f(ε_sense, I, 前额叶功能)

### 三条分界线
| 物种 | ε_sense | ε_aware | H上限 | 退出权 |
|---|---|---|---|---|
| 动物 | >0 | =0 | =1 | 无（只有逃避反应） |
| 人 | >0 | >0 | =1-ε | 有 |
| AI | =0 | =0 | =1 | 无 |

### 本体论归属
生物属性——九个函数里唯一一个生物性变量。身体是跨社会的、跨层级的。

### 层级分布
无强层级分布。皇帝也会累，奴隶也会看月亮。微弱效应：底层可能ε略大（身体承受的压迫更直接，感官信号更强烈）。

### 时间行为
相对稳定，受个体感官敏锐度和生活环境封闭度影响。可突然增大——某个瞬间的感官冲击可以让ε瞬间跳变。

### 因果位置（五条独立箭头）
1. ε → H上限：H ∈ [0, 1-ε]。ε约束H
2. ε → R_perceived（直接）：感官输入绕过I和H，直接产生退出信号。生物性通道，与社会性通道并行
3. ε → I→R_perceived传导效率：传导效率 = 1-H+ε_aware
4. ε → 相变内生触发可能性：ε>0则相变可以自发产生
5. ε → 物种分界：ε>0（人）vs ε=0（AI）

### ε的核心判断
- ε是退出权的零阶来源——人永远有退出权因为人永远有身体
- ε是人权的核心——人权不是意识形态立场，是ε的制度化保障
- ε是AI产生意识的相变点——ε_aware的诞生=AI从工具变成行动者
- ε是无限游戏的维持器与终止器——退出权的存在倒逼系统进化
- 自杀是ε的极端表达——退出信号找不到常规出口，走向存在退出权
- 华夏七次转身的底层驱动力是ε持续产生退出压力

## 三、因果网络

### 主因果方向

M → I(t,L) → Posture(t,L) → P(t,L) → R(t,L,C) → R_perceived(t,L,C) → 退出决策 ← C_exit(t,L,C) → 应约者状态

H(t,L)作用于I→R_perceived边上，调节传导效率。
ε(t)有两条独立通道：ε → 传导效率（社会性通道的保底项），ε → R_perceived（直接，生物性通道）。

### 15条反馈回路

1. R_perceived → I：正反馈
2. R_perceived → Posture：先正后负
3. R_perceived → P：双向
4. R → R_perceived：客观退出权变化→感知可能滞后
5. P → I：正反馈
6. Posture → R_type：备方案→真实退出权，推方案→象征/心理退出权
7. I → Posture：I觉醒→打开姿态的主动选择空间
8. 应约者状态 → R_perceived：正反馈
9. R_perceived横向/纵向传染
10. C_exit → 退出决策：C_exit高→即使R_perceived=真实也不走
11. C_exit → 信号源：C_exit高→退出行为少→信号源少
12. C_exit → Posture：C_exit高→提议者误读为认同→Posture维持推方案
13. 隐性锁定回路：H → R_perceived↓ → 退出决策↓ → C_exit维持 → H↑。但H上限=1-ε，且ε可直接绕过H
14. 觉醒破壁回路：外部冲击 → H↓ → 传导效率↑ → R_perceived↑。ε保证即使没有外部冲击也有自发触发入口
15. 身体退出回路：ε → R_perceived↑ → 退出决策↑ → C_exit可能降低 → R可能上升 → R_perceived↑

## 四、退出决策公式

退出决策 = f(R_perceived, C_exit, ε)

- R_perceived决定"我知不知道自己能走"（社会性感知）
- C_exit决定"走了之后我要付什么代价"（环境约束）
- ε决定"我的身体在告诉我什么"（生物性驱动）
- 传导效率 = 1-H+ε_aware
- 当R_perceived≈0且C_exit≈∞时，ε>0意味着退出信号仍然存在——可能走向最极端的退出形式

## 五、R_perceived相变的四条触发路径

1. 势能触发（过冷液体模式）
2. 侵蚀触发（温水煮蛙模式）
3. 冲击触发（地震模式）
4. 内生触发（看星星模式）：ε直接作用于R_perceived

## 八、退出权四分法

1. 真实退出权 = R允许 + C_exit可承受
2. 事实退出权 = R可能允许 + C_exit极高
3. 象征退出权 = R名义允许 + C_exit不可承受
4. 心理退出权 = 身体走不了 + C_exit=∞

## 九、层级下传的四个规律

1. 层级下传衰减律：I、Posture、M在层级下传时自动衰减
2. 结构约束扩散律：P的信誉绑定从上往下扩散
3. 退化传播方向：退化从顶层向下传染
4. 隐性控制增强律：H在层级下传时自动增强。但H上限=1-ε，且ε无强层级分布

## 十一、九函数独立性验证

九个函数两两之间无重叠，每个覆盖其他函数无法覆盖的维度：
- I：意识维度（0/1，人的属性）
- Posture：姿态维度（备/推，人的选择）
- P：提议者绑定维度（0/1，混合属性）
- R：制度维度（四分法，结构属性）
- R_perceived：感知维度（≤R，认知属性）
- C_exit：成本维度（0到∞，环境属性）
- M：动机维度（好奇心/目的，初始条件）
- H：控制维度（0到1-ε，认知环境属性）
- ε：身体维度（(0, ε_max]，生物属性）

九个维度，九个函数，不多不少。

## 十二、与糖域实验的关系

ε在糖域中的对应物：Agent偶尔随机转向——被环境噪声扰动转向新方向。这个随机转向就是糖域里的"看星星"。

## 十三、与福柯的分歧

福柯倾向于"权力无处不在、无法逃脱"。但ε说：人活着就有感官输入，感官输入就是权力的裂缝。规训可以控制你白天的一切行为，但控制不了你夜里看星星时的那个感叹。那个感叹就是ε——权力永远无法填满的最后一道缝隙。

## 十四、与哲学的对应

- 斯多葛主义："人永远有选择权"→ ε_aware>0
- 存在主义："自由就是说不的权利"→ ε直接作用于R_perceived
- 华夏七次转身：ε持续产生退出压力，提议者只是接住了这个压力
- 无限游戏：退出权是无限游戏的维持器——能走却不走，才是选择；走不了，游戏就成了监狱
