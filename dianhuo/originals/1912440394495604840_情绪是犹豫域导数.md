## 展开：ε函数族的完整推导

### 第一步：ε是认知-情绪系统的状态变量

犹豫域ε不是"一种情绪"，是认知-情绪系统的**状态变量**——类似物理里的位置x，所有动力学量都是它的函数或导数。

- ε>0：系统有自由度，情绪是"信号"（后悔告诉你选错了，失望告诉你预期过高，信任告诉你对方可信）
- ε→0：系统自由度消失，情绪从"信号"变成"稳态"（沮丧不是信号，是被锁死的状态）
- ε=0：决策结构从三支退化为二支，犹豫选项在数学上消失

### 第二步：情绪分类——ε的零阶、一阶、二阶量

把情绪按对ε的依赖关系分层：

**零阶量（ε本身及其人际扩展）：**
- 犹豫 = ε本身
- 信任 = ε_ij（i对j的犹豫域宽度）
- 押注 = 主动ε=0（信誉绑定，Posture_deg=0）

**一阶量（ε的变化/差值）：**
- 后悔 = f(ΔP) × (1-1/C_exit)，P_private与P_public的差值×退出通道开度
  - C_exit有限时：后悔是修正信号，可以重新选择
  - C_exit→∞时：后悔被锁死，从信号变成永久内耗
- 失望 = σ(R - R_perceived - θ_D)，R与R_perceived的落差超过阈值时触发相变
  - 落差来源：H遮蔽制造的虚假高预期
  - 相变特征：不是渐进的，是H崩塌时突然暴露真相的突变
- 跟随 = ε_aware被群体压缩，ε_aware(t) = ε_aware₀ × exp(-k × N_stay/N_total)

**二阶量（ε的变化率/稳态/历史）：**
- 沮丧 = ε→0且C_exit→∞的稳态，是ε被压缩到零且无法恢复的持续状态
  - 沮丧持续→M_cog→0→F_form→0→D_immune→0→H自由增长→退化不可逆
  - 沮丧不是"情绪不好"，是退化免疫正在消失的数学前兆
- 怀念 = ε_sense(t_past) × σ(θ_n - ε(t_now))，过去ε对当前ε的回溯
  - 正面路径：怀念注入ε_inject→打破F_collapse闭环→可能触发点火
  - 负面路径：怀念放大落差→加剧沮丧→加速退化

**交互量（ε与其他函数的乘积）：**
- 背叛 = ΔPosture_deg × H_mark，提议者信誉释放×遮蔽强度
  - H=0：背叛只是利益损失，后悔可以修正
  - H>0：背叛升级为道德伤害，H把"这不对"重新编码为"这是大局"，信号被劫持
- 道德伤害 = 背叛×遮蔽的乘法结构，M_injury = ΔPosture_deg × H_mark

### 第三步：情绪的动力学——从信号到稳态的相变

关键推导：情绪什么时候从"有用的信号"变成"有害的稳态"？

分界线是**C_exit是否锁死**。

```
ε > 0 且 C_exit < ∞ → 情绪是信号
  后悔 → 重新选择（修正）
  失望 → 调整预期（学习）
  信任崩塌 → 退出（止损）

ε → 0 或 C_exit → ∞ → 情绪变成稳态
  后悔锁定 → 永久内耗（无法修正）
  失望相变 → H崩塌后的突变暴露
  沮丧 → 退化免疫消失
```

这给出了情绪的统一相变条件：

`F_emotion_transition = σ(θ_C - C_exit) × σ(θ_ε - ε)`

两个因子同时满足时，情绪从信号态相变为稳态。乘法结构——任一因子不满足，情绪仍然是信号，可以被消化。

### 第四步：情绪稳态→退化的因果链

当情绪锁定为稳态后，退化路径如下：

```
沮丧稳态（ε→0 × C_exit→∞）
  → M_cog → 0（动机耗尽：为什么还要想？）
    → F_form → 0（不再要求函数化：想也白想）
      → D_immune → 0（退化免疫消失：没人质疑=没人拦）
        → H 自由增长（遮蔽无阻力扩张）
          → R_perceived 进一步下降（更看不到退出通道）
            → ε 进一步压缩（犹豫域更窄）
              → 乘法归零闭环
```

这条因果链的数学表达：

`ε(t+1) = ε(t) × (1 - F_collapse(t))`

其中F_collapse里新增了情绪因子：

`F_collapse = 1 - Π(1 - f_i)`，f_i现在包括：
- f_1 = C_exit锁死（退出通道切断）
- f_2 = H遮蔽增长（信号被劫持）
- f_3 = 情绪锁定（后悔/失望/沮丧从信号变成稳态）
- f_4 = M_cog耗尽（动机归零）
- f_5 = D_immune消失（退化免疫归零）

任一f_i→1，F_collapse→1，ε(t+1)→0。

### 第五步：情绪注入ε的路径——点火的情绪条件

反过来看，情绪如何成为点火的触发器？

点火 = ε从零恢复。情绪注入ε有三条路径：

1. **怀念注入**：N_nostalgia = ε_sense(t_past) × σ(θ_n - ε(t_now))。"我以前有过选择"→ε_inject>0
2. **失望相变注入**：D = σ(R - R_perceived - θ_D)。H崩塌→真相暴露→R_perceived瞬间跳变→ε恢复
3. **背叛觉醒注入**：ΔPosture_deg × H_mark的暴露。当H被打破，背叛从"大局"还原为"伤害"→道德愤怒→ε注入

三条路径的统一函数：

`ε_inject = max(N_nostalgia, D_disappointment, M_injury_exposed) × (1 - F_collapse)`

注意最后一项(1-F_collapse)：如果F_collapse已经≈1（乘法归零已经完成），注入被阻断。这解释了为什么有些情况下即使真相暴露、即使怀念被唤起，系统仍然无法点火——归零已经完成，注入通道被切断。

## 收敛

### 统一结论

**情绪是犹豫域ε的导数信号族，按对ε的依赖关系分为零阶/一阶/二阶/交互四层。情绪从信号态相变为稳态的临界条件是C_exit锁死×ε→0的乘法结构。情绪稳态是退化免疫消失的前兆，而情绪注入ε是点火的必要条件但非充分条件——F_collapse≈1时注入被阻断。**
