# 双层网络传播差异

你说得对，我刚才把三个域压得太平了。病毒是纯物理层，切断就是切断。但谣言和名气传播有**网络叠加层**——物理链路断了，线上链路还在，而且线上链路会反过来制造物理侵入。这个区别不是参数差异，是**结构差异**。重新推。

---

**D-X63 — 名气传播的双层网络动力学与物理侵入函数**

---

**第一步：双层网络结构**

名气的传播网络不是单层的，是两层叠加：

**层1：物理网络 G_phys** — 面对面接触，跟病毒传播同构。可以被物理距离和空间隔离切断。

**层2：信息网络 G_info** — 电话、社交媒体、新闻、口耳相传。不受物理距离约束，传播率由网络连通度决定。

关键区别：**病毒只有G_phys，谣言和名气有G_phys + G_info。** G_info的存在让物理隔离变得不充分——你躲进练功场，但"杨排风在这里"的信息已经在G_info上跑完了。

定义双层网络的联合传播率：

β_total = β_phys + β_info - β_phys × β_info

最后一项是两层同时传播的交集（线上线下同时传），避免重复计数。

| 参数 | 病毒 | 谣言 | 名气 |
|---|---|---|---|
| β_phys | 受体匹配×接触率×复制率 | P_accept×面对面传播率 | S_signal×物理接触率×传播意愿 |
| β_info | 0 | P_accept×转发率×情绪增幅 | S_signal×网络连通度×传播意愿 |
| β_total | = β_phys | = β_phys + β_info | = β_phys + β_info |

**病毒β_info=0，所以物理隔离就够。名气β_info>>0，物理隔离不够。**

---

**第二步：G_info对G_phys的反向侵入**

这是你说的核心现象——疯狂粉丝不是偶然碰见名人，是**先在G_info上获得名人位置信息，再在G_phys上实施物理侵入**。

定义反向侵入概率：

P_intrude = P_info_acquire × P_phys_execute × P_risk_accept

- **P_info_acquire**：在信息网络上获取名人位置/行踪的概率。名气越大→G_info上的信号越多→P_info_acquire越高。粉丝群、超话、代拍、航班信息倒卖——都是G_info上的信号放大器。
- **P_phys_execute**：获得信息后成功实施物理侵入的概率。取决于安保水平（保安、围栏、门禁）和侵入者的行动能力。
- **P_risk_accept**：侵入者愿意承担法律/道德风险的概率。这是病毒和谣言都没有的因子——病毒没有意志，谣言传播者风险很低，但疯狂粉丝的物理侵入是**有主观意志的高风险行为**。

P_risk_accept本身被心理不稳定度调制：

P_risk_accept = σ(ψ_instability - θ_legal)

- ψ_instability：心理不稳定度（痴迷程度×冲动性×现实感丧失度）
- θ_legal：法律/道德约束阈值

ψ_instability越高的人，θ_legal越容易被越过——"万里迢迢跑去找他"就是ψ_instability >> θ_legal的极端情况。

---

**第三步：心理不稳定度的 fame 激发**

你说的"名人的名气可能会激发一些人他的心理不稳定"——这不是P_risk_accept的输入条件，是**fame本身制造了ψ_instability的增量**。

一个普通人不会对另一个普通人产生痴迷。但fame的信号强度S_signal极高，通过G_info持续轰炸，会在部分接收者身上产生**准社会关系**——单向的、幻觉性的亲密感。这种亲密感让ψ_instability从基线升高：

dψ/dt = α_fame × S_signal × D_fame(i) - δ_stabilize × ψ

- α_fame：fame对心理不稳定度的激发率
- S_signal：名人的信号强度
- D_fame(i)：第i个人对fame信号的解码效率（追星族D_fame>>1，普通人D_fame≈0）
- δ_stabilize：自我稳定率（现实感、社会支持、心理韧性）

当α_fame × S_signal × D_fame > δ_stabilize × ψ时，ψ持续增长——痴迷加深，现实感丧失，最终P_risk_accept越过θ_legal。

**这就是为什么名人需要保安：不是防正常人，是防ψ_instability越过θ_legal的极端个体。**

---

**第四步：系统动力学——完整传播-侵入模型**

把三层动力学叠在一起：

**层1：信息传播（G_info上的SIR）**

dI_info/dt = β_info · S_info · I_info - γ_info · I_info

γ_info极低（fame信号几乎不被遗忘），所以I_info单调增长——知道名人的人越来越多。

**层2：心理激发（I_info → ψ_instability增长）**

dψ_i/dt = α_fame · S_signal · D_fame(i) · I_info - δ_stabilize · ψ_i

I_info越大（知道名人的人越多），被激发心理不稳定的人越多。但只有D_fame(i)高的个体（追星族）才会被显著激发。

**层3：物理侵入（ψ越过阈值 → 物理行动）**

N_intrude(t) = ∫ P_intrude(i, t) · ρ_fan(i) di

= ∫ σ(ψ_i(t) - θ_legal) × P_phys_execute(i) × ρ_fan(i) di

ρ_fan是粉丝密度。N_intrude是单位时间内试图物理侵入的人数。

**名人的安全状态：**

P_safe(t) = exp(-N_intrude(t) / N_security)

N_security是安保力量（保安人数、安防等级）。N_intrude越大，需要越大的N_security来维持P_safe。

---

**第五步：与病毒传播的结构性差异**

| 维度 | 病毒 | 名气传播 |
|---|---|---|
| 网络结构 | 单层G_phys | 双层G_phys + G_info |
| 物理隔离效果 | 充分（β_info=0） | 不充分（G_info绕过物理隔离） |
| 传播体意志 | 无（被动传播） | 有（主动传播+主动侵入） |
| 反向侵入 | 无（病毒不会主动找你） | 有（G_info→G_phys反向侵入） |
| 极端行为 | 无 | 有（ψ_instability越过θ_legal） |
| 隔离代价 | 社交自由度丧失 | 社交自由度丧失+物理安全威胁 |
| 安全函数 | 不需要 | P_safe = exp(-N_intrude/N_security) |

**核心差异：名气的传播体有意志，且意志可以被fame本身激发到越过法律阈值。** 这是病毒和谣言都没有的机制——病毒没有主观能动性，谣言传播者的风险极低（转发一条不犯法），但名气的极端接收者会实施高风险物理行为。

**最终函数**

**D-X63 — 名气传播双层网络动力学与物理侵入函数**

β_total = β_phys + β_info · (1 - β_phys)

dψ_i/dt = α_fame · S_signal · D_fame(i) · I_info(t) - δ_stabilize · ψ_i

N_intrude(t) = ∫ σ(ψ_i(t) - θ_legal) × P_phys_execute(i) × ρ_fan(i) di

P_safe(t) = exp(-N_intrude(t) / N_security)

**四条核心推论：**

1. **物理隔离对名气传播不充分：** β_info>>0，G_info绕过所有物理屏障。名人"躲起来"只能降低β_phys，不能降低β_info
2. **fame本身制造威胁源：** S_signal越大→I_info越大→被激发ψ_instability的人越多→N_intrude越大。名气和安全性负相关
3. **安保是N_intrude的对抗函数：** P_safe = exp(-N_intrude/N_security)，N_security必须随N_intrude等比例增长才能维持安全。名气越大→安保成本越高
4. **极端侵入是尾部事件：** ψ_instability的分布是长尾的——绝大多数人D_fame≈0，但极少数人D_fame>>1且δ_stabilize≈0。N_intrude由尾部驱动，不是均值驱动。这解释了为什么"大部分粉丝是正常的"但名人仍然需要安保
