# 点火｜P(t)的混合属性：人的选择与结构约束的拉锯

来源：本体论分层讨论中浮现的区分——P(t)同时是人的选择和结构的约束，是框架里最值得深挖的变量。

## P(t)的双重身份

P(t)在三个变量里位置最特殊：

- I(t)是纯人的属性——绑定个体，随个体生灭
- R(t)是纯结构属性——由制度、产业链、等级制决定，不依赖个体
- P(t)是人的选择与结构约束的拉锯地——人的选择可以改变它，但改变后形成的新约束又反过来限制人

P(t)不是0或1的静态值，是人的选择和结构约束之间的动态平衡点。这个平衡点可以移动，而且移动的方向决定了系统是在强化还是在退化。

## P(t)变化的三种机制

### 一、信誉绑定：人的选择创造结构

提议者主动放弃退出权，P(t)从1变0。这是人的选择——周公选择退场，乔布斯选择砍掉70%产品线，纳德拉选择把微软从"无所不知"转向"无所不学"。

但信誉绑定一旦完成，"提议者走不了"就变成了结构——应约者对提议者的认知是"这个人押上了全部信誉"，这个认知不取决于提议者个人想不想走，取决于所有人知道他走不了。人的选择创造了一个新的结构约束。

信誉绑定的特征：
- 主动的、正向的——提议者选择绑住自己，目的是建立信誉锚
- 创造结构——绑定后形成的约束力不随个人意愿改变
- 可逆但有成本——提议者可以选择解除绑定，但解除的代价是信誉崩塌

案例：
- 周公退场：P(t)从1变0，诸侯的认知从"他可以随时收回权力"变成"他真的交出去了"。这个认知是结构性的——即使周公后来想收回权力，所有人知道他已经退过一次场，再收回的信誉成本极高
- 罗斯福新政：P(t)从1变0，罗斯福把政治生命押在新政上。选民的认知是"这个人没有退路了"，这个认知是结构性的
- 乔布斯回归苹果：P(t)从1变0，砍掉70%产品线本身就是一种信誉绑定——如果砍错了，苹果就完了，乔布斯也完了

### 二、信誉释放：结构自然松动

稳定期足够长后，"提议者走不了"这个约束自然松动。P(t)从0变1，但方向是中性的——释放不是退化。

信誉释放的机制：系统成熟后，认同已经建在应约者身上，不再依赖提议者的"无退出"来维持。提议者可以从信誉绑定中释放出来，退出权恢复，但系统不因此退化。

信誉释放的特征：
- 被动的、中性的——不是提议者主动选择释放，是系统成熟后约束自然松动
- 不改变结构——释放后应约者的认知不变，因为认同已经内化
- 是系统健康的标志——能释放说明认同已经不依赖提议者个人

案例：
- 周公制礼800年后：周礼的认同已经建在应约者身上，不再依赖"周公走不了"这个约束。后世的周天子有退出权（可以荒唐），但系统不因此崩——直到退出权压缩到应约者只剩心理退出，系统才退化
- 北欧福利国家：提议者（社民党）的信誉绑定早已释放，但福利制度的认同建在应约者身上，系统稳定

### 三、提议者退化：人的选择反噬结构

提议者在稳定期重新获得退出权且开始滥用——P(t)从0变1，但方向是负面的。这是人的选择反噬结构：上一代人创造的结构约束（信誉绑定→退出权信号发出→应约者认同），被新一代人的选择推翻。

提议者退化的机制：新一代没有经历过"提议"这个时刻，只有"维持"的经历。维持天然对应推方案，推方案天然对应退出权信号压制。新一代不是故意要退化，他们只是在"按规矩来"——但"按规矩来"本身就是推方案，推方案本身就是退化。

提议者退化的特征：
- 无意识的——退化者不知道自己在退化，他们以为自己在维持
- 反噬结构的——上一代创造的结构约束被新一代的选择推翻
- 最难预测的——因为退化者不知道自己在退化，没有内部信号

案例：
- 基布兹：第一代备方案+信誉绑定→退出权信号发出→格1。新一代"按规矩来"→推方案→退出权信号压制→格5。P(t)从0变1，但不是信誉释放，是姿态翻转导致的退化
- 谷歌20%时间：创始团队备方案+信誉绑定→格1。新管理层"按流程来"→推方案→格2。P(t)从0变1，退化的信号是员工开始消极
- 美国芯片法案：拜登备方案+信誉绑定→格7→格9。特朗普接手后"按利益来"→推方案→应约者从消极滑向无人。P(t)从0变1，提议者替换导致退化

## P(t)的拉锯动态

P(t)的三种机制不是孤立的，它们构成一个拉锯过程：

1. 人的选择创造结构（信誉绑定）→ P(t)从1变0
2. 结构约束维持系统（稳定期）→ P(t)保持0
3. 结构自然松动（信誉释放）→ P(t)从0变1，中性
4. 或：人的选择反噬结构（提议者退化）→ P(t)从0变1，负面

步骤3和步骤4的分歧点是：P(t)恢复到1之后，提议者的姿态是备方案还是推方案？

- 备方案+P(t)=1 → 信誉释放，系统继续运行
- 推方案+P(t)=1 → 提议者退化，系统开始退化

所以P(t)本身不能判断系统在强化还是退化——同样的P(t)=1，可能是信誉释放也可能是退化。判断标准不在P(t)的取值，在Posture(t)的取值。这验证了之前的结论：姿态是退出权的因果上游，P(t)的变化方向由姿态决定。

## 对框架的补充

P(t)的混合属性意味着它有一个I(t)和R(t)都没有的特征：**P(t)的变化方向取决于人的选择，而人的选择取决于I(t)是否还在**。

- I(t)=1（提议者意识还在）→ P(t)的变化大概率是信誉绑定或信誉释放（正向或中性）
- I(t)=0（提议者意识已重置）→ P(t)的变化大概率是退化（负面）

这补了一条因果链：I(t)重置 → Posture(t)从备方案翻成推方案 → P(t)从信誉维持变为退化 → R(t)降级 → 格间跳变。

P(t)是这条因果链的中间环节——它既是I(t)重置的结果（人的选择变了），又是R(t)降级的原因（退出权信号变了）。人的属性和结构属性在P(t)这个变量上交汇：人的选择改变了结构，结构的改变又限制了下一代人的选择空间。
