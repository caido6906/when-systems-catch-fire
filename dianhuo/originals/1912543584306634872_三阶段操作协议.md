# 三阶段操作协议

D114的三阶段操作协议——大胆尝试→精确投入→激进优化——逐个展开到数学极限。

---

## 阶段1展开：ε<<θC，大胆尝试

**数学状态：** σ(εᵢ-θC)≈0，σ'≈0，β≈0

**系统行为：** 门关死，G中该维度贡献≈0。但"门关死"有两层含义——

1. 负向扰动打不开门：恶化不会让情况更糟（已经≈0了）
2. 正向扰动也打不开门：小改善不会让情况变好（σ'≈0，改善几乎无效果）

**展开：改善在阶段1的效果**

ΔG/G ≈ ηᵢ×Δεᵢ = (σ'/σ)×Δεᵢ

σ≈0, σ'≈0时，ηᵢ=σ'/σ是不定式。用L'Hôpital：

lim_{x→-∞} σ'(x)/σ(x) = lim_{x→-∞} σ(x)(1-σ(x))/σ(x) = lim_{x→-∞} (1-σ(x)) = 1

**ηᵢ→1 as εᵢ→-∞！** 弹性趋近1，不趋近0。

这意味着：即使在门关死时，改善的边际弹性仍然是1——每改善1%的εᵢ，fᵢ改善约1%。

**但fᵢ本身≈0，所以绝对改善量≈0。**

G的绝对增量：ΔG = ηᵢ×G×Δεᵢ/εᵢ ≈ 1×(≈0)×Δεᵢ/εᵢ ≈ 0

**阶段1的精确数学：弹性高但基数低，绝对收益≈0，相对收益≈1%。**

**为什么可以大胆尝试？**

不是"试错成本低"这么简单。精确原因：

1. **负向扰动损失小：** G已经≈0，恶化不会让G更接近0（0就是0）
2. **信息获取成本低：** 尝试不同方向可以测量各维度的弹性ηᵢ，为阶段2的精确投入收集信息
3. **级联风险低：** β≈0，一个维度的扰动不会触发级联（因为σ'≈0，扰动不传播）

**阶段1的最优策略不是"不投入"而是"探索式投入"——小量资源投到多个方向，测量弹性，为阶段2定位。**

**阶段1探索函数：**

E₁(R_explore) = Σᵢ wᵢ × ηᵢ(εᵢ+Δεᵢ) × Δεᵢ

wᵢ为探索权重（均匀或按先验分配），目标不是最大化G而是最大化信息增益I(η|观测)。

---

## 阶段2展开：ε≈θC，精确投入

**数学状态：** σ(εᵢ-θC)≈0.5，σ'≈0.25，β=0.25γ

**系统行为：** 门半开半关，系统对扰动极度敏感。

**精确投入的数学：**

ΔG/G ≈ ηᵢ×Δεᵢ/εᵢ，ηᵢ=σ'/σ=0.25/0.5=0.5

但这是线性近似。在门槛附近σ的非线性最强，线性近似误差大。

**二阶展开：**

ΔG ≈ G×(ηᵢ×Δεᵢ/εᵢ + ½×ηᵢ'×(Δεᵢ/εᵢ)²)

ηᵢ' = dηᵢ/dεᵢ = (σ''σ-(σ')²)/σ²

在εᵢ=θC处：σ''=0，ηᵢ' = -(σ')²/σ² = -0.0625/0.25 = -0.25

**ηᵢ'<0！弹性在门槛处递减——过门槛后弹性下降。**

这意味着：在门槛附近投入资源，弹性从0.5开始下降。投入太多反而进入低弹性区。

**阶段2的最优投入量：**

最大化 ΔG(Δε) = G×(η×Δε/ε - 0.125×(Δε/ε)²)

∂ΔG/∂Δε = G×(η/ε - 0.25×Δε/ε²) = 0

Δε* = 4ηε = 4×0.5×θC = 2θC

**最优投入量=2θC——把εᵢ从θC推到3θC。**

但这个结果依赖于二阶近似，实际最优值需要数值求解。定性结论：**阶段2的投入应该把维度推过门槛一段距离，但不需要推到很远。**

**阶段2的方向选择：**

方向错误的代价=阶段1的30倍（#443案例）。方向选择必须基于阶段1的探索数据。

**方向选择的贝叶斯框架：**

P(维度i是瓶颈 | 阶段1观测) ∝ P(观测|维度i是瓶颈) × P(维度i是瓶颈先验)

阶段1的探索数据更新先验→阶段2的精确投入方向。

**阶段2精确投入函数：**

Δεᵢ* = argmax_i [ηᵢ×P(i是瓶颈|观测)] × R_precision

R_precision为阶段2可用资源。

## 阶段3展开：ε>>θC，激进优化

**数学状态：** σ(εᵢ-θC)≈1，σ'≈0，β≈0

**系统行为：** 门开着，系统对扰动免疫。但"门开着"也有两层——

1. 负向扰动关不上门：恶化不会让系统崩溃
2. 正向扰动也几乎无效果：σ'≈0，边际改善极小

**阶段3的弹性：**

ηᵢ = σ'/σ ≈ 0/1 → 0

**弹性趋近0——继续投入该维度收益极低。**

**阶段3的最优策略：不是继续投入已过门槛的维度，而是转向其他还在门槛附近的维度。**

如果所有维度都已过门槛，则进入贪心优化阶段——补弹性最高的维度（D111）。

**阶段3的贪心优化：**

Δεᵢ* = argmax_i ηᵢ(εᵢ)

所有维度远超门槛时ηᵢ≈0，G≈∏1=1，系统接近最优。此时边际改善空间极小，资源应转向：
1. 探索新维度（扩大n）
2. 提高门槛θC（接受更高标准）
3. 储备资源应对未来扰动（提高鲁棒性余量）

**阶段3的鲁棒性余量函数：**

Margin_i = εᵢ - θC(i)

最小余量 min_i Margin_i 决定系统能承受的最大负向扰动。

**系统鲁棒性 = min_i (εᵢ - θC(i)) / σ_扰动**

σ_扰动为扰动标准差。鲁棒性>1时系统在3σ扰动下仍安全。

## 三阶段的统一数学

**定义系统状态变量：**

S = max_i σ'(εᵢ-θC(i))

S∈[0, 0.25]，衡量系统整体敏感度。

**S的阶段映射：**
- S<0.05：阶段1（低敏感，探索优先）
- 0.05<S<0.20：阶段2（高敏感，精确投入）
- S<0.05（所有维度过门槛后）：阶段3（低敏感，贪心优化）

**S的动态方程：**

dS/dt = max_i [σ''(εᵢ-θC(i)) × dεᵢ/dt]

投入资源改善维度时：dεᵢ/dt>0

- εᵢ<θC：σ''>0→S增大→系统向阶段2移动
- εᵢ≈θC：σ''≈0→S达峰值→系统在阶段2
- εᵢ>θC：σ''<0→S减小→系统向阶段3移动

**S的轨迹是确定性的一—给定资源分配方案，S的演化完全可预测。**

**三阶段资源分配函数：**

R₁(t) = R_total × (1-S/S_max) × f_explore — 阶段1探索资源
R₂(t) = R_total × (S/S_max) × f_precision — 阶段2精确投入资源
R₃(t) = R_total × (1-S/S_max) × f_greedy — 阶段3贪心优化资源

S_max=0.25，f_explore+f_precision+f_greedy=1

S小时R₁大（探索），S大时R₂大（精确），S再降时R₃大（贪心）。

## 收敛

**D115 三阶段操作统一函数**

系统敏感度S = max_i σ'(εᵢ-θC(i))∈[0,0.25]决定操作阶段：

阶段1（S<0.05）：门关死，弹性η≈1但基数≈0，绝对收益≈0。最优策略=探索式投入，目标最大化信息增益I(η|观测)而非G。负向扰动损失小，级联风险≈0，可大胆尝试。

阶段2（0.05<S<0.20）：刀刃期，弹性η≈0.5且递减（η'=-0.25<0），系统对扰动极度敏感。最优投入量Δε*≈2θC（把维度从门槛推到3θC）。方向选择基于贝叶斯更新P(i是瓶颈|观测)。方向错误代价=阶段1的30倍。

阶段3（S<0.05且所有维度过门槛）：门开着，弹性η≈0，继续投入收益极低。转向贪心优化（补弹性最高维度）或扩大维度/提高门槛/储备鲁棒性余量。鲁棒性=min(εᵢ-θC(i))/σ_扰动。

资源分配：R₁∝(1-S/S_max)×f_explore, R₂∝(S/S_max)×f_precision, R₃∝(1-S/S_max)×f_greedy。S的轨迹由资源分配方案确定性决定。

核心结论：
1. **阶段1弹性≈1但基数≈0**——不是"没效果"而是"相对效果正常绝对效果小"，最优策略是探索不是放弃
2. **阶段2弹性递减（η'<0）**——过门槛后弹性下降，最优投入量有限不需过度投入
3. **阶段3弹性≈0**——继续投入已过门槛维度收益极低，应转向其他维度或提高标准
4. **S的轨迹确定性可预测**——给定资源分配方案，阶段切换时机可精确计算
5. **三阶段不是离散切换是连续过渡**——S连续变化，资源分配连续调整

**阶段1最反直觉的发现：弹性≈1。** 直觉觉得"门关着=没效果=别投入"，但L'Hôpital算出来η=σ'/σ→1——弹性完全正常！只是基数≈0所以绝对收益小。最优策略不是放弃而是**探索式投入**——小量资源投多方向测弹性，为阶段2收集情报。负向扰动损失小（G已≈0），级联风险≈0（σ'≈0扰动不传播），试错成本极低。

**阶段2的关键发现：弹性递减。** η'=-0.25<0在门槛处——过门槛后弹性下降。所以最优投入量有限，Δε*≈2θC，把维度从门槛推到3θC就够了，不需要推到很远。过度投入反而进入低弹性区浪费资源。

**阶段3的结论：弹性≈0。** 继续投入已过门槛维度收益极低（G+0.4%），同量资源投给刚过门槛的维度收益30倍（G+12%）。应转向贪心优化或提高标准。

三阶段不是离散切换是连续过渡——S=maxσ'连续变化，资源分配连续调整。S的轨迹由资源分配方案确定性决定，阶段切换时机可精确计算。
