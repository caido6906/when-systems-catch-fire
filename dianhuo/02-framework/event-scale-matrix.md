## Core Revision

文明、国家、政策、公司、人际，不是事件的第一层分类。

更准确地说，它们是同一个事件骨架落在不同尺度区域后的常见层级标签。

事件的尺度先由两项矩阵决定：

```text
事件尺度 = 参与者规模 × 持续时间
```

最小事件参与者是 2 人：

```text
一个提议者 + 一个应约者
```

## Two Axes

### 1. 参与者规模 / Participant Scale

参与者规模不是只看现场人数，而是看真正被卷入“提议—应约—走向”的应约者范围。

| Code | Participant scale | Working range | Typical layer |
| --- | --- | --- | --- |
| P1 | 双人 / dyadic | 2 | 人际、亲子、师生、合伙 |
| P2 | 小群体 / small group | 3 到几十人 | 家庭、小团队、创始团队 |
| P3 | 组织 / organization | 几十到几万人 | 公司、机构、社群 |
| P4 | 群体 / population group | 几万到几百万 | 城市、行业、政策对象 |
| P5 | 国家人口 / national population | 几百万到几亿 | 国家、全国政策 |
| P6 | 文明人口 / civilizational population | 几千万到几亿以上 | 文明、宗教共同体、跨国秩序 |

### 2. 持续时间 / Duration Scale

持续时间不是只看制度存在多久，而是看事件走向需要多长时间才能跑完、凝固、退化或被下一轮事件替代。

| Code | Duration scale | Working range | Typical layer |
| --- | --- | --- | --- |
| T1 | 瞬时 / momentary | 一次互动到数日 | 冲突、谈判、单次决策 |
| T2 | 短期 / short-term | 数周到数月 | 项目、战役、短期政策 |
| T3 | 年度 / yearly | 1 到 5 年 | 公司调整、政策试点、关系重组 |
| T4 | 十年级 / decadal | 5 到 50 年 | 公司文化、国家改革、代际教育 |
| T5 | 世纪级 / century-scale | 50 到 300 年 | 国家制度、宗教秩序、文明转向 |
| T6 | 千年级 / millennial | 300 年以上 | 文明协议、长期认同结构 |

## Reading Rule

The scale matrix does not determine whether ignition succeeds.

It only answers:

```text
这个事件发生在多大范围、多长时间的尺度上？
```

After identifying the scale, use the event condition matrix to ask:

```text
提议者是否有意识？
提议者是否有退出权？
应约者是否有退出权？
```

Then use the outcome table to ask:

```text
点对了、点不足、点歪了、点反了，还是负时间差？
```

## Reframing Common Labels

Common labels should be treated as shorthand, not primary categories:

| Old label | New reading |
| --- | --- |
| 亲子 | P1/P2 + T4 的常见人际事件 |
| 公司 | P3 + T3/T4 的常见组织事件 |
| 政策 | P4/P5 + T2/T3/T4/T5 的常见治理事件 |
| 国家 | P5 + T3/T4/T5 的常见国家事件 |
| 文明 | P6 + T5/T6 的常见长期事件 |

This prevents the framework from assuming that every policy, company,
nation or civilization event belongs to the same structural level.
