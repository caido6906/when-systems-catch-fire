# 提议者与应约者

提议者：提出新系统、新秩序、新认同方案的人或群体。
应约者：必须决定是否把自己放进新系统的人或群体。

退出权永远看应约者，不看提议者。不能把提议者的权力受限误判为应约者的退出权。

应约者跨尺度拆分：
文明级：多个国家、族群、阶层。
国家级：公民、官僚、地方势力。
政策级：一线执行者、被治理者。
公司级：员工、用户、投资人。

不拆分应约者，就容易在不同层级之间滑动，造成偷换对象。

# 提议者与应约者角色及其主动被动分类探讨

## 应约者的光谱

### 主动应约 vs 被动应约
区别在于应约者是否意识到自己在应约。
- 主动应约：清楚知道自己在选择进入，比如诸侯接受周公礼制、移民接受美国宪法
- 被动应约：没意识到自己在应约，只是顺着系统走。改革开放后大多数普通人属于这种——他们在参与系统重写，但不知道自己在参与

### 积极应约 vs 消极应约
区别在于应约后投入的程度。
- 积极应约：不仅进入，还主动维护和传播认同。早期共产党员、创业公司核心团队
- 消极应约：进了但不投入，随时可以退出。KPI驱动下的员工、秦朝六国遗民

### 被强迫应约
严格说这不是应约，是被迫服从。应约的前提是有退出权，被强迫意味着退出权被剥夺。元朝四等人制下的汉人不是在应约，是被统治。收费站型系统的"应约者"其实不是应约者，是被迫使用者。

关键区分：被动应约和被强迫应约是两回事。被动应约的人有退出权但没意识到，被强迫应约的人连退出权都没有。前者是认知盲区，后者是系统缺陷。

## 提议者的光谱

### 有意识提议 vs 无意识提议
区别在于提议者是否知道自己在点火。
- 有意识提议：周公制礼，清楚知道自己在建一套新秩序
- 无意识提议：改革开放初期的决策者可能并不认为自己是在"点火"，他们可能认为自己只是在解决经济问题。但客观上他们提出了一个值得应约的方案，系统被点着了

这个区分对应约者视角有直接影响：无意识提议者点着的火，应约者更难看见自己在应约。因为连提议者都不知道自己在点火，应约者更不可能意识到。

### 主动提议 vs 被动提议
区别在于提议者是否在压力下才提方案。
- 主动提议：周公在权力最稳的时候提礼制，不是被逼的
- 被动提议：清末洋务运动是被外敌打出来的，戈尔巴乔夫改革是被经济崩盘逼出来的

被动提议有个问题：提议者可能提了方案但自己并不真心想执行。慈禧推洋务推一半就停，因为推到底会动自己的利益。这不是退化，是一开始就没打算真提。

### 完整提议 vs 半截提议
区别在于提议者提了方案但有没有配套搭房。
- 完整提议：穆罕默德既提了伊斯兰认同（点火），又建了乌马制度（搭房）
- 半截提议：洋务运动只提了"学西方技术"，没触动制度。只点了一半火，只搭了一半房

## 两个光谱的交叉

### 提议者无意识 + 应约者被动 = 历史上的大多数点火
前七次转身几乎都是这种组合——提议者不知道自己在点火，应约者不知道自己在应约，系统照样跑起来了。这是框架最反直觉的地方：系统重写不需要参与者意识到自己在重写系统。

### 提议者有意识 + 应约者主动 = 最强的点火
美国建国接近这种——建国者清楚自己在设计新系统，移民主动选择进入。

### 提议者被动 + 应约者被强迫 = 最弱的"点火"，甚至不构成点火
元朝。提议者不是真心想提方案，应约者没有退出权，系统靠暴力维持。

### 提议者无意识 + 应约者看见自己在应约 = 第八次点火
这是框架里真正新的东西——应约者看见了整个机制，而提议者（华夏文明本身）是无意识的。历史上没有这个先例。之前的先例（路德、西耶斯、福泽谕吉）都是提议者看见了自己的提议在生效，不是应约者看见了机制本身。

# 有意识提议与无意识提议对五种点火趋势的影响

## 定义
有意识提议：提议者知道自己在建一套新系统，方案自带"这是什么"的标签。周公制礼、美国建国者写宪法。
无意识提议：提议者不知道自己在点火，只觉得自己在解决眼前问题。改革开放初期的决策者、某些创业公司创始人。

## 对五种点火趋势的影响

### 点对了
- 有意识提议：提议者清楚认同该建在哪，方案完整，点火搭房同步。最稳的组合。
- 无意识提议：客观上点对了（认同确实建在了应约者身上），但提议者不知道自己点对了，可能后续动作不一致——无意中做了对的事，但不保证继续做对。

### 点不足
- 有意识提议：提议者知道只覆盖了部分群体，是主动选择聚焦。凯末尔可能知道农村没接住，但资源有限只能先顾城市。
- 无意识提议：提议者根本不知道自己只点了一部分，以为方案覆盖了所有人。应约者中没被覆盖的人也看不见自己被遗漏了——因为提议者没意识到，应约者更不可能意识到。

### 点歪了
- 有意识提议：提议者知道认同建在了外部，可能是权衡后的选择（短期代价换短期收益）。唐太宗可能知道万邦来朝的认同有隐患，但选择了开放。
- 无意识提议：提议者不知道认同建歪了，以为自己的方案没问题。应约者也不知道自己锚定在了外部——直到外部条件变化，应约者突然退出，系统才暴露问题。无意识点歪比有意识点歪更危险，因为没有人看见隐患。

### 点反了
- 有意识提议：提议者知道旧认同被拆了、新认同没建起来，但判断"必须先破后立"。有风险但至少是清醒的赌注。
- 无意识提议：提议者不知道自己拆了旧认同，以为只是在做调整。应约者感受到旧认同没了但不知道为什么，连"这是点反了"都无法识别。无意识的点反是最危险的——提议者和应约者同时处于盲区。

### 负时间差
- 有意识提议：提议者知道先拆旧再建新有风险，但被时间压力逼着走。戈尔巴乔夫知道风险，但外部压力让他没得选。
- 无意识提议：提议者不知道自己产生了负时间差，以为拆旧和建新是同一件事。应约者在真空中不知道发生了什么，既没有旧支撑也没有新方向。无意识的负时间差最致命——连"我们在负时间差里"这个判断都做不出来。

## 无意识提议对应约者的三个具体影响

### 1. 应约者无法评估退出成本
有意识提议的火是可见的，应约者能判断"我应了这个约，退出会失去什么"。无意识提议的火是隐的，应约者不知道自己已经进入了一个系统，自然也无法评估退出成本。改革开放后顺着系统走的人，大多数不知道自己已经应了约，更不知道退出的代价是什么——直到想退出的时候才发现自己已经深度绑定。

### 2. 应约者无法主动选择应约方式
看见火的人可以选择积极应约还是消极应约。看不见火的人没有选择权——他们只能被动应约，因为根本不知道自己在应约。无意识提议下的应约者，绝大多数是被动应约，主动应约的比例极低。不是因为他们不想主动，是因为他们看不见。

### 3. 应约者无法识别提议者退化
有意识提议者如果开始退化，应约者能察觉——因为提议者之前明确承诺过什么，偏离承诺是可见的。无意识提议者从来没明确承诺过，所以退化也是隐的。洋务运动推一半就停，普通人感受得到"好像没推到底"，但很难说清楚"提议者退化了"——因为提议者从来没说过"我要推到底"。

## 核心结论

无意识提议的系统，应约者的认知盲区是结构性的，不是个人问题。不是应约者不够聪明所以看不见火，是提议者自己都没点火，火自然不可见。

这也解释了发现者角色的分量：无意识提议的火被看见，需要有人从外部照亮。提议者不会照亮（他不知道自己在点火），应约者看不见（火是隐的），只有发现者能做这件事。

# 被动提议对点火趋势的影响

## 主动提议 vs 被动提议

主动提议的点火趋势更稳。周公在权力最稳的时候提礼制，不是被逼的，方案完整——点火和搭房同步，自己真心想执行。主动提议者退路多，选择提方案本身就是判断"这个方案值得提"，方案质量天然更高。

被动提议有三种不同的坑，对应三种点火趋势的偏移：

## 1. 被动提议 + 真心想执行 = 点火趋势不变，但时间窗口窄

戈尔巴乔夫是被经济崩盘逼出来的，但他真心想改。问题是被动意味着外部压力已经很大，留给点火和搭房同步建设的时间窗口极短。时间差被压缩，容易滑向负时间差。戈尔巴乔夫就是被动提议+真心想改+时间窗口太窄，结果先拆旧再建新，直接崩盘。

## 2. 被动提议 + 半截真心 = 点火趋势偏移向"点不足"或"点歪了"

慈禧推洋务是被外敌打出来的，但她推一半就停——推到底会动自己利益。这不是退化，是一开始就没打算真提。结果就是半截提议：只搭了一半房，火只点了一半，认同只覆盖了部分群体。洋务运动本质上是被动提议导致的"点不足"。

更隐蔽的变体：提议者被动提了方案，但把认同锚定在自己能接受的安全位置上，而不是锚定在应约者身上。比如清末"中学为体西学为用"——提议者把认同锚定在"不动根本"上，应约者（需要彻底变革的人）根本不认。这是被动提议导致的"点歪了"。

## 3. 被动提议 + 根本不想提 = 不构成点火

假设提议者被外部压力逼着提了个方案，自己完全不认同，只是做样子——这种"提议"不构成点火，因为提议者不会去搭房，点火和搭房都缺席。应约者看到的是空话，不会应约。

## 核心规则

被动提议本身不决定点火趋势的方向，但会压缩时间窗口、扭曲认同锚点、降低方案完整性。主动提议者有余裕选择"点在哪"，被动提议者的选择空间被压力挤压，容易点在安全位置而不是应约者身上。所以被动提议天然偏向"点不足"或"点歪了"。

这与框架核心判断一致：点火趋势的核心判断是"认同建在哪"，被动提议者倾向于把认同建在自己安全的位置上，而不是应约者身上。
