## Current Rule

The current matrix is the 8-cell Event Condition Matrix.

It is not the total matrix of the Ignition Framework.

Use the framework in this order:

1. Event Scale Matrix: participant scale × duration scale.
2. Event Condition Matrix: proposer awareness × proposer exit right × responder exit right.
3. Event path and outcome: responder state, exit layer, step reached, and event outcome.
4. Case status: working hypothesis, validation hypothesis, prediction, boundary, or fact-checked.
