# 提议者姿态：事件条件矩阵内的次级变量

## 结论

提议者姿态（备方案 vs 强推）是事件条件矩阵内的次级变量，不是独立的新维度。

## 理由

1. 姿态不是与现有六维度并列的第七维度，而是嵌在现有维度内部——它主要决定"退出权真假"和"事件走向"的取值。

2. 姿态是退出权的因果上游：强推姿态的提议者不会给真退出权，给了就不是强推；备方案姿态天然对应真退出权。姿态决定退出权，退出权决定走向，是因果链而非并列关系。

3. 如果把姿态提成独立维度，它与退出权高度共变，几乎不会出现"强推+真退出权"或"备方案+假退出权"的格子。一个几乎为空的行列说明它不是独立维度，而是已有维度的内在分化。

## 案例验证

- 秦统一：强推 → 退出权为假 → 崩盘
- 周公制礼：备方案 → 退出权为真 → 点对
- 基布兹：备方案+真退出权 → 提议者退化 → 格1滑到格5

## 退化分析中的标注方式

初始姿态是备方案不等于永远备方案——提议者退化可以把备方案扭成强推。姿态是时间维度的变量，同一事件内部可以翻转。这个翻转不需要新维度描述，用事件走向的格间滑动即可刻画（如格1→格5）。姿态标注在退出权那一格的取值来源上即可。
