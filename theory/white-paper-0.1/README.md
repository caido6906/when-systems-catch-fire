## 白皮书定位

白皮书是点火框架的「理论成立说明书」。它比正文更硬、更短、更可被学术圈检验。

- **正文**（`book/`）：文学性、历史感、叙述性
- **白皮书**（`theory/white-paper-0.1/`）：理论性、边界性、可检验性
