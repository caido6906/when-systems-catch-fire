# 点火框架理论定位声明

> 版本：2026-06-07 / 定位声明 v1.0

---

## 点火框架是中层理论

按照 Robert K. Merton 对中层理论（Middle-Range Theory）的经典定义：

> 中层理论介于日常研究中零散的工作假设和包打一切的宏大概念体系之间。它足够抽象，可以处理社会行为和社会结构的不同领域；又足够贴近经验材料，可以导出可检验的命题。

点火框架完全符合这一定义：

| 判据 | 点火框架的位置 |
|------|-------------|
| **不是宏大理论** | 不解释所有社会变迁、全部合作形式、一切制度演化 |
| **不是零散假设** | 有 14 个可操作化的核心概念，有结构化的编码矩阵 |
| **有明确边界** | 只解释「旧系统不足→新系统被提出→应约者认领」这一特定链条 |
| **有竞争对照** | 与演化博弈论、创新扩散、制度理论、社会认同、Hirschman E/V/L 有明确的增量解释关系 |
| **可导出可检验命题** | 7 条可检验命题，附检验方式和证伪条件 |
| **有死亡条件** | 三类可推翻框架的反例，附处理流程 |

## 为什么不是「原创性辩论」

点火框架不需要参与「是不是全新」的辩论。理论成立不靠喊「别人没讲过」，而靠它能否稳定回答：

1. 它解释什么？（见 `boundary-statement.md`）
2. 它不解释什么？（同上）
3. 它比旧理论多解释了什么？（见 `competing-theories.md`）
4. 它在什么情况下会失败？（见 `counter-examples.md`）
5. 别人怎么检验它？（见 `testable-propositions.md`）
